# About Restricted Tiles (DCL)

Do not use restricted tiles in your dialog definitions.

Your DCL files should not use the tiles
cluster or
tile. Also, do not use the basic exit button types (cancel\_button,
help\_button,
info\_button, and
ok\_button) unless you redefine the standard exit button subassemblies.
