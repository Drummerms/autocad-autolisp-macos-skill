# To Change the Background Color of the Drawing Area in Model Space

1. In the Application menu, click Preferences.
2. In the Application Preferences dialog box, left column, click Look & Feel.
3. Under Interface Theme, click in the Model box, and then click a color, or click Select Color.

   The default dark gray background color has an RGB value of 33,40,48.
4. If you clicked Select Color, the Color Palette dialog box is displayed. At the top of the dialog box, click either Index Color,
   True Color, or Color Books, and then make your color selection. Click OK to exit the Color Palette dialog box.
5. Click OK.
