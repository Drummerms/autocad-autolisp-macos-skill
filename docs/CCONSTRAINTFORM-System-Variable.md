# CCONSTRAINTFORM (System Variable)

Controls whether annotational or dynamic constraints are applied to objects.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Dynamic |
| 1 | Annotational |

When this system variable is set to 0, dynamic constraints are created, by default, when you use the [DIMCONSTRAINT](DIMCONSTRAINT-Command.md) command.
