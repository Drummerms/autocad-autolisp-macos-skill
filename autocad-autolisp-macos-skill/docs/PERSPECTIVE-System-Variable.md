# PERSPECTIVE (System Variable)

Specifies whether the current viewport displays a perspective view.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | Varies |

| 0 | Perspective view turned off |
| 1 | Perspective view turned on |

Perspective views are available only in model space, and only with 3D visual styles.

NOTE:PERSPECTIVE is set to 0 when the drawing file or DXF file is saved to a file format earlier than AutoCAD 2007.
