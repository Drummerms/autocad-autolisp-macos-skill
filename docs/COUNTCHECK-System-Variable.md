# COUNTCHECK (System Variable)

Controls the types of errors to check in the count.

|  |  |
| --- | --- |
| Type: | Bitcode |
| Saved in: | Registry |
| Initial value: | 2 |

The setting is stored as a bitcode using the sum of the following values:

| Value | Description |
| 1 | Checks for duplicate objects that overlap on top of each other |
| 2 | Checks for duplicate objects that overlap on top of each other, renamed, or exploded blocks |
