# To Rotate 3D Objects Along an Axis

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) 3D Rotate.
   ![](../images/GUID-A09A2043-CFA8-4806-B587-BDDFF9C4AE62-low.png)
2. Select the objects and subobjects you want to rotate using the following methods:
   * Press and hold Ctrl to select subobjects (when no subobject selection filter is enabled).
   * Release Ctrl to select entire objects.
3. When you have selected all objects, press Enter.
4. Hover the cursor over an axis path on the gizmo until it turns yellow and the vector that represents the axis of rotation
   is displayed. Click the path.
5. Click or enter a value to specify the angle of the rotation.
