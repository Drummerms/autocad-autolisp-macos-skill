# ACTIVITYINSIGHTSPATH (System Variable)

Specifies the location Activity Insights activities are recorded.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Registry |
| Initial value: | /Users/{username}/Library/Application Support/Autodesk/ActivityInsights/Common |

Change this to a shared location so that all drawing activity is logged in a single location regardless of who works on it.
