# To Modify a Block Reference

Use this procedure to modify the properties of a selected block reference without affecting the block definition.

1. Select the block to modify.
2. Right-click the block and choose Properties.
3. In the Properties Inspector, select and modify *X* and *Y* position, scale, rotation values, or other properties.
