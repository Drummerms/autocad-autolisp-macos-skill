# To Insert a Code Snippet (AutoLISP/VS Code)

NOTE:You can define your own code snippets in the
*snippets.json* file. For information on adding or editing code snippets, see
[To Add or Edit Code Snippet Definitions](To-Add-or-Edit-Code-Snippet-Definitions-AutoLISP-VS-Code.md).

1. Open an AutoLISP source (LSP) or DCL file, or switch to an LSP or DCL file that is already open.
2. Type the name of the code snippet you want to insert.

   For example, type
   *ifp*.
3. Click the name of the code snippet to insert it from the IntelliSense functions list or press Tab when the item is highlighted.

   ![](../images/GUID-1828984C-CDFF-4D95-B0AD-8D5967D6A0F5-low.png)

   ![](../images/GUID-ECC72C39-97DC-4790-8D0F-148304ED09E2-low.png)
4. If the code snippet contains a placeholder, type an expected value or expression. Press Tab to move to the next placeholder
   in the code snippet if more than one is available.
