# To Share a link (URL) to a drawing

Use Share URL (SHARE command) to share a link to current drawing for viewing or editing in AutoCAD on the web ([web.autocad.com](https://web.autocad.com/)) or AutoCAD on mobile. All related DWG xrefs and images are included. For drawings saved in Autodesk Docs, you can create
a link that allows recipients to make changes to the actual drawing file. For drawings that aren't saved to Autodesk Docs,
share a version of the file that can be saved as a copy.

1. Click
   Collaborate tab ![](../images/ac.menuaro.gif) Share panel ![](../images/ac.menuaro.gif) Share Drawing.
   ![](../images/GUID-832FD002-F679-4ED2-A285-E874490367C4-low.png) Find
2. Select whether to share the drawing as view only or with editing permissions.
3. Click Copy Link.

   Send the link to your collaborators. The link opens the drawing in the AutoCAD on the web or AutoCAD on mobile.
