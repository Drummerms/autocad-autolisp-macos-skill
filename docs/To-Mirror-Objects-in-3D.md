# To Mirror Objects in 3D

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) 3D Mirror.
   ![](../images/GUID-9C86B6FB-83F2-4569-91F9-D08837AE0004-low.png)
2. Select the object to mirror.
3. Specify three points to define a mirroring plane.
4. Press Enter to retain the original objects, or enter
   *y*to delete them.
