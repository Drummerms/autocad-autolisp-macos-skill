# 2D Graphics Display

This release of AutoCAD for Mac includes a new graphic engine, which provides a better display effect on some high-end GPUs.

![](../images/GUID-75810A16-F103-415B-820A-8446C683E97B-low.png)

The new graphics engine provides:

* Clearer and smoother lines and curves
* Hatch pattern display at a small scale
* Improved selection highlight effects
