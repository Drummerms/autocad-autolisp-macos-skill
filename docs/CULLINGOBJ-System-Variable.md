# CULLINGOBJ (System Variable)

Controls whether 3D subobjects that are hidden from view can be highlighted or selected.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | No subobject culling   * Rolling over 3D objects highlights all 3D subobjects, including hidden subobjects. * Selecting 3D objects by dragging selects all 3D subobjects, including hidden subobjects. |
| 1 | Apply subobject culling   * Rolling over 3D objects highlights only the subobjects that are normal in the current view. * Selecting 3D objects by dragging selects only the subobjects that are normal in the current view. |
