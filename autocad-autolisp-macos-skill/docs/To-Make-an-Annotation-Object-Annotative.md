# To Make an Annotation Object Annotative

1. Select an annotation object in the drawing area.
2. Right-click in the drawing area and choose Properties.
3. On the Properties Inspector, select the Annotative property and choose Yes.

NOTE:When creating new annotation objects, you should use an annotation style with the Annotative property enabled.
