# To Watch Variables and Expressions while Debugging an LSP File (AutoLISP/VS Code)

NOTE:Debugging is not available in AutoCAD LT.

1. In Visual Studio Code,
   [start debugging an AutoLISP source (LSP) file](To-Launch-and-Attach-an-Instance-of-AutoCAD-to-VS-Code-for-Debugging-AutoLISP-VS-Code.md).
2. On the Activity Bar, click Debug and Run (or click View menu > Debug).
3. In the editor window containing the LSP file, select the variable or expression to watch.
4. Right-click and choose Debug: Add to Watch to add the selected variable or expression to the Watch section of the Debug and
   Run view.

   ![](../images/GUID-A4DBC406-437F-4A7D-AE13-46E34C924E37-low.png)

   NOTE:Right-click a variable or expression in the Watch section of the Debug and Run view, and choose Remove Expression to stop
   watching that item.

You can watch the value of the most recently evaluated expression by adding the
\*last-value\* variable to the Watch section. For more information on how to add the
\*last-value\* variable, see
[To View the Return Value of the Most Recent Expression Evaluated](To-View-the-Return-Value-of-the-Most-Recent-Expression-Evaluated-AutoLISP-VS-Code.md).
