# CONTENTCLOSE (Command)

Closes the Blocks palette.

## Access

*Menu*:
Window ![](../images/ac.menuaro.gif) Blocks.

*Command entry*: Cmd-2

## Summary

Closes the Blocks palette that is displayed with the
[CONTENT](CONTENT-Command.md) command.
