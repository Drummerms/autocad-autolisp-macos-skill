# HPBOUND (System Variable)

Controls the object type created by HATCH and BOUNDARY.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 1 |

| 0 | Creates a region |
| 1 | Creates a polyline |

A polyline or region object is created only if the HPBOUNDRETAIN system variable is turned on.
