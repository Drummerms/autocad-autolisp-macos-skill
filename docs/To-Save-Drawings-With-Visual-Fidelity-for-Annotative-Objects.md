# To Save Drawings With Visual Fidelity for Annotative Objects

Visual Fidelity allows you to maintain visual fidelity for annotative objects when viewed in earlier releases.

1. In the command line, enter SAVEFIDELITY.
2. Enter 0 to turn off visual fidelity.

   This setting is recommended If you work primarily in model space.
3. Enter 1 to turn on the visual fidelity.

   This setting is recommended If you need to exchange drawings with other users.

NOTE:The SAVEFIDELITY system variable does not affect saving a drawing to the AutoCAD 2010 drawing or DXF file formats.
