# SURFTYPE (System Variable)

Controls the type of surface-fitting to be performed by the Smooth option of the PEDIT command.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 6 |

| 5 | Quadratic B-spline surface |
| 6 | Cubic B-spline surface |
| 8 | Bezier surface |
