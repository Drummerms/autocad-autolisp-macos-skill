# To Assign Colors to Multiple Hatch Types

Assigns color to multiple hatch type

Use this method to change the color assignment for solid and pattern hatch types simultaneously.

1. In the drawing area, select a solid hatch and a pattern hatch. The Hatch Visor opens up.

   NOTE:When multiple hatches or fills of different types are selected in a drawing, some of the controls on the Hatch visor are disabled.
2. In the Hatch visor, select the color you want to use from the drop-down list.

   The solid color and the pattern (but not its background) change to the color you selected. This overrides the current color
   for hatch pattern and solid fills.
3. Click X to close.
