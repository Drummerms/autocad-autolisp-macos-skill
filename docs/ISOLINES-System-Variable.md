# ISOLINES (System Variable)

Specifies the number of contour lines displayed on the curved surfaces of 3D solids.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 4 |

Valid settings range from 0 to 2047.
