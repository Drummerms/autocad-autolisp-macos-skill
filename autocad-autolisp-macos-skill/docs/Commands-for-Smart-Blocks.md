# Commands for Smart Blocks

These commands and system variables are used to work with smart blocks.

#### Commands

* [-BSEARCH](GUID-4DD16D1E-6700-4BB7-A2C9-21FD71767D76.htm " At the Command prompt, converts the selected entities and identical instances into blocks.
  		")
* [BSEARCH](GUID-E5FD2025-3E4A-4A82-AE63-946D0A68C3BF.htm " Displays the Convert dialog box, which provides options to convert selected entities and identical instances into blocks.
  		")

#### System Variables

* [AUTOPLACEMENT](GUID-84265932-8B02-4740-A178-EF1BBD7CB01F.htm " Controls whether placement suggestions are displayed as you insert a block.
  		")
* [BSEARCHINCLUDEEXISTINGBLOCKS](GUID-2B943110-79F8-41B3-98D7-8E9094DFF678.htm " Controls whether existing block instances are selected by BSEARCH command by default. Users can still make adjustments to the selection during the review process.
  		")
* [TEXTTOATTRIBUTE](GUID-7654D6E4-C18B-4003-8EC9-6A49768D96DF.htm " In the BSEARCH command, if text objects are selected, controls whether text variations will be searched for conversion into block attributes.
  		")
