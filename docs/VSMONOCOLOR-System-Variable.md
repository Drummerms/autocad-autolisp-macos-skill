# VSMONOCOLOR (System Variable)

Sets the color for monochrome and tint display of faces in the visual style applied to the current viewport.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | 255,255,255 |

The initial value is white.

NOTE:Existing visual styles are not changed when you enter a new value for this system variable. Any new value entered for this
system variable temporarily creates an unsaved new visual style.
