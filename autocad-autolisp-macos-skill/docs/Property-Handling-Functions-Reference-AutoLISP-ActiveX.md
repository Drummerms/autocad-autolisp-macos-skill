# Property-Handling Functions Reference (AutoLISP/ActiveX)

NOTE:ActiveX support in AutoLISP is limited to Windows only; not available on Mac OS or Web.

The following table provides summary descriptions of the AutoLISP property-handling functions.

| Property-handling functions | | Platforms | | | | |
| Windows | | Mac OS | | Web |
| Function | Description | AutoCAD | AutoCAD LT | AutoCAD | AutoCAD LT | AutoCAD |
| [(vlax-get-property obj property)](vlax-get-property-AutoLISP-ActiveX.md) | Low-level property get function. May be used for custom ActiveX object | ✓ | ✓ | -- | -- | -- |
| [(vlax-property-available-p obj prop [T])](vlax-property-available-p-AutoLISP-ActiveX.md) | Determines whether an object has a specified property | ✓ | ✓ | -- | -- | -- |
| [(vlax-put-property obj property arg)](vlax-put-property-AutoLISP-ActiveX.md) | Low-level property set function | ✓ | ✓ | -- | -- | -- |
