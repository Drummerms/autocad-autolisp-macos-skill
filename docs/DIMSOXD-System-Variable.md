# DIMSOXD (System Variable)

Suppresses arrowheads if not enough space is available inside the extension lines.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Drawing |
| Initial value: | Off |

| Off | Arrowheads are not suppressed |
| On | Arrowheads are suppressed |

If not enough space is available inside the extension lines and  [DIMTIX](DIMTIX-System-Variable.md) is on, setting DIMSOXD to On suppresses the arrowheads. If DIMTIX is off, DIMSOXD has no effect.
