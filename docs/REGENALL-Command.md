# REGENALL (Command)

Regenerates the drawing and refreshes all viewports.

## Access Methods

![](../images/ac.mouse.gif) Menu:   View  ![](../images/ac.menuaro.gif)  Regen All Not available in menus in the current workspace

## Summary

REGENALL regenerates the entire drawing and recomputes the screen coordinates for all objects in all viewports. It also reindexes
the drawing database for optimum display and object selection performance.
