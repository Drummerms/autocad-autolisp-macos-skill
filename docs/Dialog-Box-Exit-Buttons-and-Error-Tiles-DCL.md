# Dialog Box Exit Buttons and Error Tiles (DCL)

Exit buttons and error tiles are used to close a dialog box and provide information when a problem is encountered.

The
*base.dcl* file provides standard button subassemblies for exiting (or “retiring”) a dialog box. Use these standard versions to maintain
a consistent appearance across applications.

You can customize the text in these buttons by using the prototype
retirement\_button.
