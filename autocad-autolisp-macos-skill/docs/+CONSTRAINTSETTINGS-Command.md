# +CONSTRAINTSETTINGS (Command)

Controls the tab that is displayed by default when the Constraint Settings dialog box is invoked.

| Tab Index | Tab Displayed |
| 0 | Geometric tab |
| 1 | Dimensional tab |
| 2 | AutoConstrain tab |
