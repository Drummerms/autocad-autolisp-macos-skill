# OSNAPNODELEGACY (System Variable)

Controls whether the Node object snap can be used to snap to multiline text objects.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

The number and location of nodes on a multiline text object snap depends on the vertical and horizontal justification of the
multiline text object.

| 0 | Node object snap can be used with multiline text objects |
| 1 | Node object snap ignores multiline text objects |
