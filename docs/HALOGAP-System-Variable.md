# HALOGAP (System Variable)

Specifies a gap to be displayed where an object is hidden by another object.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

The value is specified as a percent of one unit and is independent of the zoom level.

HALOGAP is available only in 2D views. In 3D views, the
[VSHALOGAP](VSHALOGAP-System-Variable.md) system variable is used.
