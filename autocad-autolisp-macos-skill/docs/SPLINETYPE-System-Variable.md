# SPLINETYPE (System Variable)

Sets the type of curve generated by the Spline option of the PEDIT command.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 6 |

| 5 | Quadratic B-spline |
| 6 | Cubic B-spline |
