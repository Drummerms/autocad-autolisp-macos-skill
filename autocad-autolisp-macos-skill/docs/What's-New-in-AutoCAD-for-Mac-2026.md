# What's New in AutoCAD for Mac 2026

Learn about new features in this release of AutoCAD for Mac 2026 release.

## 2026 Highlights

* *[Smart Blocks: Search and Convert](Smart-Blocks-Search-and-Convert.md)* - Convert repetitive geometry, including text variations, into blocks for improved efficiency and organization.
* *[Activity Insights](Activity-Insights.md)* - This release includes several enhancements such as simplified search and filter options for ease of use.
* *[Added Layer Option to CENTERLINE, CENTERMARK, and MLEADER](Added-Layer-Option-to-CENTERLINE,-CENTERMARK,-and-MLEADER.md)* - In this release, we integrated the CENTERLAYER system variable into the workflows of creating new centerlines, center marks,
  and multileaders.

## Additional enhancements

*Text:* Below is a list of improvements to the text functionality.

* A new layout engine supports right-to-left text input on macOS. When the input and font language are set to Arabic or Hebrew,
  characters align correctly from right to left. During text entry, the cursor automatically positions itself at the leftmost
  end of the text, ensuring a better typing experience.
* The font searching experience in the Text Style dialog box is improved. It now supports fuzzy searches, allowing you to enter
  only part of the characters to search for a font.
* You can now see recent input results in the Find and Replace dialog box for one or multiple drawings in the same AutoCAD
  session. This improvement makes it easier and more efficient to select recently entered text from the dropdown list.

*Sheet Set Terminology:* The term
*Project Manager* is now updated to
*Sheet Set Manager* in this release. This change, along with other terminology updates related to sheet set, ensures a more consistent interface
and experience for cross-platform users, making it easier to manage and organize your sheet sets across different devices
and operating systems.

*3D Graphics*: In a continued effort to modernize the 3D graphics system, the Conceptual and Hidden visual styles are now supported by
Fast Shaded mode.

*File Open Performance:* Improvements have been made to reduce the amount of time it takes to open 2D drawing files.

*Dimensions:* The DIMCONTINUE and DIMBASELINE commands now inherit all annotation scales from the selected dimension.
