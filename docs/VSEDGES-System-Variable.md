# VSEDGES (System Variable)

Controls the types of edges that are displayed in the viewport.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

![](../images/GUID-052B7410-7519-4DF2-8408-F1B65C172D9A-low.gif)

isolines displayed

![](../images/GUID-67046069-B69F-44AD-BED9-EA4A7B182B3A-low.gif)

facet edges displayed

| 0 | No edges are displayed |
| 1 | Isolines are displayed |
| 2 | Facet edges are displayed |

NOTE:Existing visual styles are not changed when you enter a new value for this system variable. Any new value entered for this
system variable temporarily creates an unsaved new visual style.
