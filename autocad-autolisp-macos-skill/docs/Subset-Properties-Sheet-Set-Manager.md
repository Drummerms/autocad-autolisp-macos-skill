# Subset Properties (Sheet Set Manager)

Displays information that is specific to the selected subset.

SHEETSET (Command)

*Menu:* Window ![](../images/ac.menuaro.gif) Sheet Set Manager: Select a subset in Sheet Set Manager

![](../images/GUID-04E2B090-B697-471C-BD95-193B842C707E-low.png)

## List of Options

The following options are displayed.

### Subset

Displays the name and if the subset is included when publishing.

Subset Name
:   Displays the name of the subset.

Publish Sheets in Subset
:   Controls whether the sheets in the subset are added to the Batch Publish dialog box when publishing the subset or all sheets
    in the sheet set.

### File Locations

Displays information about the folder on disk that the subset represents in the sheet set.

Sheet Storage Location
:   Specifies the folder location for new drawing (DWG) files created as a result of adding a new sheet to the subset.

    Does not impact the folder location of a drawing (DWG) file when adding sheets from an existing drawing.

Reveal in Finder
:   Displays the resolved absolute path to the folder on disk that the subset represents.

### Template Location

Displays settings used when creating a new sheet in the subset.

Sheet Creation Template
:   Displays the layout to use when creating a new sheet.

    The layouts displayed in the drop-down list come from the file specified in the Default Template property.

Sheet Creation Template Location
:   Specifies the drawing template (DWT) or drawing (DWG) file to use when creating new drawing files for the subset.

Prompt for Template
:   Controls whether the
    [New Sheet dialog box](New-Sheet-Dialog-Box.md) is displayed in the expanded or collapsed state each time a new sheet is added.
