# vlr-add (AutoLISP/ActiveX)

Enables a disabled reactor object

*Supported Platforms:* Windows only; not available on Mac OS or Web

## Signature

```
(vlr-add obj)
```

*obj*
:   *Type:* VLR object

    A object representing the reactor to be enabled.

## Return Values

*Type:* VLR object

The
*obj* argument.
