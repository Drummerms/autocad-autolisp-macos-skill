# DYNDIVIS (System Variable)

Controls how many dynamic dimensions are displayed during grip stretch editing.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

DYNDIGRIP controls which dynamic dimensions are displayed during grip stretch editing.

| 0 | Only the first dynamic dimension in the cycle order |
| 1 | Only the first two dynamic dimensions in the cycle order |
| 2 | All dynamic dimensions, as controlled by the  [DYNDIGRIP](DYNDIGRIP-System-Variable.md) system variable |

The  [DYNMODE](DYNMODE-System-Variable.md) system variable turns Dynamic Input features on and off.
