# acet-sys Functions

#### Topics in this section

* [acet-sys-beep (AutoLISP/Express Tools)](acet-sys-beep-AutoLISP-Express-Tools.md)

  Makes a noise.
* [acet-sys-command (AutoLISP/Express Tools)](acet-sys-command-AutoLISP-Express-Tools.md)

  Runs a command in a shell and waits for the shell to complete execution before returning.
* [acet-sys-foreground (AutoLISP/Express Tools)](acet-sys-foreground-AutoLISP-Express-Tools.md)

  Forces AutoCAD's application window to be the topmost and receive focus.
* [acet-sys-keystate (AutoLISP/Express Tools)](acet-sys-keystate-AutoLISP-Express-Tools.md)

  Checks the state of a virtual key.
* [acet-sys-lasterr (AutoLISP/Express Tools)](acet-sys-lasterr-AutoLISP-Express-Tools.md)

  Reports the last error.
* [acet-sys-procid (AutoLISP/Express Tools)](acet-sys-procid-AutoLISP-Express-Tools.md)

  Returns the current process identifier.
* [acet-sys-sleep (AutoLISP/Express Tools)](acet-sys-sleep-AutoLISP-Express-Tools.md)

  Delay to allow other processes to run.
* [acet-sys-spawn (AutoLISP/Express Tools)](acet-sys-spawn-AutoLISP-Express-Tools.md)

  Executes an external program.
* [acet-sys-term (AutoLISP/Express Tools)](acet-sys-term-AutoLISP-Express-Tools.md)

  Terminates the given process.
* [acet-sys-wait (AutoLISP/Express Tools)](acet-sys-wait-AutoLISP-Express-Tools.md)

  Waits for a process to complete.
