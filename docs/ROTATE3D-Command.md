# ROTATE3D (Command)

Moves objects about a 3D axis.

## Access Methods

*Tool Sets*:
Modeling tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) 3D Rotate.
![](../images/GUID-A09A2043-CFA8-4806-B587-BDDFF9C4AE62-low.png)

*Menu*:
Modify ![](../images/ac.menuaro.gif) 3D Operations ![](../images/ac.menuaro.gif) 3D Rotate.

## Summary

It is recommended that you use the gizmos available through the 3DMOVE and 3DROTATE commands to manipulate 3D objects.

## List of Prompts

The following prompts are displayed.

Select objects: *Use an object selection method and press*Enter *when you finish*

Specify first point on axis or define axis by [[Object](ROTATE3D-Command.md)/[Last](ROTATE3D-Command.md)/[View](ROTATE3D-Command.md)/[Xaxis/Yaxis/Zaxis](ROTATE3D-Command.md)/[2points](ROTATE3D-Command.md)]: *Specify a point, enter an option, or press*Enter

![](../images/GUID-98F48295-A788-4950-B881-7A5B454B33AA-low.png)

Object

Aligns the axis of rotation with an existing object.

Line

Aligns the axis of rotation with the line selected.

Rotation Angle
:   Rotates the object about the selected axis the specified amount from the current orientation.

Reference
:   Specifies the reference angle and the new angle.

    The difference between the starting angle and the ending angle is the computed rotation angle.

Circle

Aligns the axis of rotation with the 3D axis of the circle (perpendicular to the plane of the circle and passing through the
center of the circle).

* [Rotation Angle](ROTATE3D-Command.md)
* [Reference](ROTATE3D-Command.md)

![](../images/GUID-D496F30A-9EFF-4FC4-9666-01694756036D-low.png)

Arc

Aligns the axis of rotation with the 3D axis of the arc (perpendicular to the plane of the arc and passing through the center
of the arc).

* [Rotation Angle](ROTATE3D-Command.md)
* [Reference](ROTATE3D-Command.md)

2D Polyline Segment

Aligns the axis of rotation with a segment of the polyline. Treats a straight segment as a line segment. Treats an arc segment
as an arc.

* [Rotation Angle](ROTATE3D-Command.md)
* [Reference](ROTATE3D-Command.md)

Last

Uses the last axis of rotation.

* [Rotation Angle](ROTATE3D-Command.md)
* [Reference](ROTATE3D-Command.md)

View

Aligns the axis of rotation with the viewing direction of the current viewport that passes through the selected point.

* [Rotation Angle](ROTATE3D-Command.md)
* [Reference](ROTATE3D-Command.md)

X Axis, Y Axis, Z Axis

Aligns the axis of rotation with one of the axes (*X*,
*Y*, or
*Z*) that pass through the selected point.

* [Rotation Angle](ROTATE3D-Command.md)
* [Reference](ROTATE3D-Command.md)

2 Points

Uses two points to define the axis of rotation.

![](../images/GUID-238ADDC3-C83B-4584-A62E-A054DD0FBBBD-low.png)

* [Rotation Angle](ROTATE3D-Command.md)
* [Reference](ROTATE3D-Command.md)
