# To Edit a Dimensionally Constrained Object Using Grips

You can use grips to edit a dimensionally constrained object either by selecting the object, or by selecting the constraint.

1. Select a dimensionally constrained *object*.
2. Click one of its grips.
3. Press and release the Shift key to relax the dimensional constraint temporarily.
4. Continue the editing operation with the grip.

or

1. Select a *dimensional constraint*.
2. Click one of its grips:
   * The square grips relocate the text
   * The arrow grips change the length of the object
3. Press Esc to cancel the selection.
