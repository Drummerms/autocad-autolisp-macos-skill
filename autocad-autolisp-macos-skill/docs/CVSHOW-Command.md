# CVSHOW (Command)

Displays the control vertices for specified NURBS surfaces or curves.

## Access Methods

*Tool Set*:
Modeling tab ![](../images/ac.menuaro.gif) Surface panel ![](../images/ac.menuaro.gif) Surface CV drop-down ![](../images/ac.menuaro.gif) Surface CV - Show.
![](../images/GUID-8983AD29-5CBC-4507-A16D-95A497CABA9E-low.png)

*Menu*:
Modify ![](../images/ac.menuaro.gif) Surface Editing ![](../images/ac.menuaro.gif) NURBS Surface Editing
![](../images/ac.menuaro.gif) Show CV.

## Summary

Prompts you to select the NURBS surfaces and curves whose control vertices you want to display. Non-NURBS surfaces do not
have control vertices. You can convert objects to NURBS surfaces with the CONVTONURBS command.

![](../images/GUID-A0227B2E-2D5B-4B00-991C-13F6D76C940B-low.png)
