# To Enter Relative 2D Polar Coordinates

* At a prompt for a point, enter UCS coordinates using the following format:

  *@* *distance* *<* *angle*

* For WCS coordinates, turn off dynamic input and, at a prompt for a point, enter the coordinates using the following format:

  *@ \**  *distance* *<*  *angle*
