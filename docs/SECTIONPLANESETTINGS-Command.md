# SECTIONPLANESETTINGS (Command)

Sets display options for the selected section plane.

## Access Methods

*Tool Sets*:
Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Section Plane drop-down ![](../images/ac.menuaro.gif) Section Plane Settings.
![](../images/GUID-96950BDA-DB2D-46FC-B877-42437F9D31E9-low.png)

*Shortcut Menu*: Select a live section object. Right-click in the drawing area and click Live Section Settings.

## Summary

The
[Section Settings dialog box](Section-Settings-Dialog-Box.md) is displayed.
