# To Access the Call Stack while Debugging an LSP File (AutoLISP/VS Code)

NOTE:Debugging is not available in AutoCAD LT.

1. In Visual Studio Code,
   [start debugging an AutoLISP source (LSP) file](To-Launch-and-Attach-an-Instance-of-AutoCAD-to-VS-Code-for-Debugging-AutoLISP-VS-Code.md).
2. Set one or more breakpoints to interrupt the execution of the program.

   For information on setting breakpoints, see
   [To Add, Remove, or Disable a Breakpoint while Debugging an LSP File](To-Add,-Remove,-or-Disable-a-Breakpoint-while-Debugging-an-LSP-File-AutoLISP-VS-Code.md).
3. On the Activity Bar, click Debug and Run (or click View menu > Debug).

   Calls made are shown in the Call Stack section of the Debug and Run view.

   ![](../images/GUID-4EB2C083-77B6-4177-B64C-8D015AA4624D-low.png)
