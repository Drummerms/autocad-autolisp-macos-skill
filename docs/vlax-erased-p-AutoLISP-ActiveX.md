# vlax-erased-p (AutoLISP/ActiveX)

Determines whether an object was erased

*Supported Platforms:* Windows only; not available on Mac OS or Web

## Signature

```
(vlax-erased-p obj)
```

*obj*
:   *Type:* VLA-object

    An object.

## Return values

*Type:* T or nil

T if the object was erased; otherwise
nil.
