# FLATTEN (Command)

Converts 3D geometry to a projected 2D representation.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Flatten Objects.
![](../images/GUID-5AA5F605-B691-4CC9-824B-E1BB186D9191-low.png)

*Menu*:
Modify ![](../images/ac.menuaro.gif) Flatten Objects.

FLATTEN creates a 2D representation of selected objects and projects them onto the current viewing plane. FLATTEN results
in 2D objects that retain their original layers, linetypes, colors and object types where possible. You can use FLATTEN to
create a 2D drawing from a 3D model, or you can use it to force the thickness and elevations of selected objects to 0.

NOTE:Some degradation of drawing precision might occur. This precision loss is usually insignificant and it occurs mainly in 3D
objects. Perspective view is not supported, therefore the FLATTEN command switches to parallel view before proceeding.
