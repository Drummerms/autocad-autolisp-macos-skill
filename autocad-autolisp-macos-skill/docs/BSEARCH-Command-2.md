# BSEARCH (Command)

Finds matching instances of the selected object and provides options to convert them into blocks.

When you specify a geometry for conversion, AutoCAD identifies and highlights all instances of the same geometry.

The Search Review toolbar displays.

The following prompts are displayed.

Objects to convert to blocks
:   Specifies the geometry to convert into blocks.
