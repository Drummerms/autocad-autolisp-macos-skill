# About Publishing Sheet Sets

Publishing provides a streamlined alternative to plotting multiple drawings by providing compressed representations that are
easy to view and distribute.

Using the Sheet Set Manager, you can publish an entire sheet set, a subset, or a single sheet. It is faster to publish a sheet
set in the Sheet Set Manager compared to using the Batch Publish dialog box to add drawings individually and then publish
them as a single set.

When publishing from the Sheet Set Manager, you have the option to create an electronic sheet set by publishing to a PDF
file or a paper set by publishing to the plotter specified in the page setup associated with each sheet. You can also use
a page setup saved in the page setup overrides drawing template file associated with the sheet set. This page setup overrides
the current page setup settings for the individual publish job.
