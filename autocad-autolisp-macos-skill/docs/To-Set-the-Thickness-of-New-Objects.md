# To Set the Thickness of New Objects

This process changes the default thickness setting for 2D solids, arcs, circles, lines, polylines, single-line text, and points.

1. Make sure no objects are selected.
2. If the Properties Inspector is not displayed, click Windows ![](../images/ac.menuaro.gif) Properties Inspector.
3. In the Properties Inspector, under General, enter the value for Thickness.

   When new objects are created, they have the specified 3D thickness.
