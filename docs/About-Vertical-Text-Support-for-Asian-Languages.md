# About Vertical Text Support for Asian Languages

The Chinese, Japanese, and Korean languages can require text to be created in a vertical orientation.

*Vertical SHX fonts.* Vertical text can be created specifying an AutoCAD SHX Font and an SHX Big Font in the Text Style dialog box (STYLE command).
You choose an SHX font for non-Asian characters, click the Use Big Font option, and choose a Big Font that is appropriate
for the Asian language you want to use. Finally, click the Vertical option and save the text style.

*Vertical TrueType fonts.* TrueType text fonts that start with the @ sign in their names are designed for vertical display in their respective Asian
languages. TrueType fonts are not designed for vertical display of text for non-Asian fonts.
