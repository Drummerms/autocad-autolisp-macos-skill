# To Edit Plot Style Settings

1. Click
   File menu ![](../images/ac.menuaro.gif) Plot Styles.
2. Double-click the plot style table you want to modify.
3. In the Plot Style Table Editor, under Plot Styles, select the plot style you want to modify.
4. Under Properties, click the arrow next to the property you want to change. Select an option from the list.
5. Edit other properties or plot styles as needed.
6. Click Save & Close.
