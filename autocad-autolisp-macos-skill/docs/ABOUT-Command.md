# ABOUT (Command)

Displays information about
AutoCAD 2026.

## Access Methods

*Menu*:
AutoCAD 2026 ![](../images/ac.menuaro.gif) About AutoCAD

*'about* for transparent use

Copyright information and product information is displayed. Product information includes the version number and service pack,
serial number, license type and expiration date, and the text of the license agreement. You can manage your license as needed.
