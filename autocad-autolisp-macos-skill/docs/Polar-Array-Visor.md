# Polar Array Visor

Modifies the properties of an associative polar array, such as base point and item rotation.

![](../images/GUID-084DA4D3-7755-4E5C-9FF1-04CAA06D19E1-low.png)

## Summary

The Polar Array visor is displayed when you select an associative array created with the Polar option. The options on this
visor can also be accessed using the
[ARRAYEDIT](ARRAYEDIT-Command.md) command from the Command prompt.

## List of Options

The following options are displayed.

Base Point

Redefines the base point of the array. Path arrays are repositioned relative to the new base point.

Edit Source

Activates an editing state in which you can update the associative array by editing one of its items.

Replace Item

Replaces the source objects for selected items or for all items referencing the original source objects.

Reset Array

Restores erased items and removes any item overrides.

Rotate Items

Specifies whether items are rotated as they are arrayed.
