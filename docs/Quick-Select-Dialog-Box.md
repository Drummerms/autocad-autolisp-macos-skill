# Quick Select Dialog Box

Specifies the selection filtering criteria and how you want to create the selection set from that criteria.

QSELECT (Command)

*Toolbar*:
![](../images/GUID-A942A435-ACF7-4704-80C7-22D3E46458E3-low.png)

![](../images/GUID-96F8EE91-B6DE-4070-89BF-C7F0445C7951-low.png)

## List of Options

The following options are displayed.

Where to Select
:   Applies the filtering criteria to the entire drawing or to the current selection set (if one exists).

    If Append to Previous Selection Set is selected, the filtering criteria is added to the selection set made before.

Pick New Selection Set (button)
:   Temporarily closes the Quick Select dialog box so that you can select the objects to which you want to apply the filter criteria.

Object Type
:   Specifies the type of objects to include in the filtering criteria. If the filtering criteria are being applied to the entire
    drawing, the Object Type list includes all object types, including custom. Otherwise, the list includes only the object types
    of the selected objects.

New Property Rule button (![](../images/GUID-0FAB92D9-3B8A-4BF5-AE3C-EEC6E12C7484-low.jpg))
:   Specifies the object property for the filter. This list includes all searchable properties for the selected object type. The
    property you select determines the options available in Operator and Value.

Operator
:   Controls the range of the filter. Depending on the selected property, options can include Equals, Not Equal To, Greater Than,
    and Less Than.

Value
:   Specifies the property value for the filter.

Preview
:   Displays a preview of selection set in the filtering criteria.

Select Inverse
:   Specifies whether the selection set created by QSELECT replaces or is appended to the current selection set.

Append to Previous Selection Set
:   Specifies whether the selection set created by QSELECT replaces or is appended to the selection set previously chosen.
