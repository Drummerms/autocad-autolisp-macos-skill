# To Lengthen an Object

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Lengthen.
   ![](../images/GUID-E835CDF6-D2B4-4965-87FD-E6841924D37D-low.png)

   NOTE:Hidden by default. Click
   ![](../images/GUID-78DED619-51B3-49C1-AD71-F928B5F69C0A-low.png) to display the icon on the tool set panel.
2. Enter
   *dy* (Dynamic Dragging mode).
3. Select the object you want to lengthen.
4. Drag the endpoint closest to the point of selection, and specify a new endpoint.
