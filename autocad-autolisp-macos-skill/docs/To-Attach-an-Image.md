# To Attach an Image

1. Click
   Insert menu ![](../images/ac.menuaro.gif) Raster Image Reference.
2. In the dialog box, select a file name from the list. Click Open.
3. In the Attach Image dialog box, use one of the following methods to specify insertion point, rotation, or scale:
   * Select Specify On-Screen to use the pointing device to insert the image at the location, angle, or scale you want.
   * Clear Specify On-Screen and enter values under Insertion Point, Rotation, or Scale.

   To view the unit of measurement for the image, expand the image size in the Details section.
4. Click OK.

## Attach by Dragging

1. From Finder, drag the image file into the drawing canvas.
2. Move the mouse to the location you want to place the image.

   When you release the button, the image is attached as an external reference.
3. Use the Image visor to change brightness, contrast, and fade as needed.
