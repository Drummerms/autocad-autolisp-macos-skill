# Count Palette

Displays a palette from which you can view and manage the counted blocks.

COUNTLIST (Command)

*Menu*:
Window ![](../images/ac.menuaro.gif) Count.

![](../images/GUID-40BA8FD5-7C95-4A7F-9A41-9866B065F4C3-low.png)

The Count palette displays the block names and the corresponding count of each block. When you are in an active count, the
Count palette displays count details including the error report, if any. The report lists the count errors such as overlapping,
exploded, or renamed objects.

![](../images/GUID-BFB3B1D3-FBFB-43A2-A91C-B1817AAC7A72-low.png)

## List of Options

Generate Table ![](../images/GUID-DF348FE5-DE3B-475F-AB7B-4A4165EE5AC6-low.png)
:   Inserts a table containing the block names and corresponding count of each block in the drawing.

Search
:   Filters the available blocks as soon as you enter the first few letters of the name or its keywords.

List of Blocks
:   Displays the block names and the corresponding count of each block found in the drawing. When multiple copies of a block are
    inserted in the drawing, and these copies have different properties, you can filter the count based on the general block properties
    such as the layer, scale, and mirror state. Additional filters are displayed for user-defined parameters and attributes of
    the counted blocks. Right-click to see filters.
