# Work with Smart Blocks

#### Topics in this section

* [About Smart Blocks](About-Smart-Blocks.md)

  Smart Blocks is a powerful family of features designed to enhance your block workflow in your drawings.
* [About Block Insertion With Placement Suggestions](About-Block-Insertion-With-Placement-Suggestions.md)

  The automatic block placement allows you to easily insert blocks using the Blocks palette, providing suggestions based on
  similar geometry and learned patterns from previous block instances in your drawing.
* [About Converting Repetitive Geometry](About-Converting-Repetitive-Geometry.md)

  The BSEARCH command allows you to easily find and convert multiple instances of the specified geometry into blocks.
* [Commands for Smart Blocks](Commands-for-Smart-Blocks.md)

  These commands and system variables are used to work with smart blocks.
