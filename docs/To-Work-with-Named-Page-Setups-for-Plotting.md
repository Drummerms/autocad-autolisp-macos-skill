# To Work with Named Page Setups for Plotting

You can use named page setups to store plot settings to publish a drawing.

## Create a New Named Page Setup

1. Click Page Setup Manager.
   ![](../images/GUID-C6B05F20-1FAB-48DF-8CA1-621D77A17680-low.png)
2. In the Page Setup Manager, Page Setups area, click
   ![](../images/GUID-9174311E-7DC8-49C8-B39F-991EC3DAD0E2-low.png).
3. In the New Page Setup dialog box, enter a name for the new page setup.
4. Under Start From, select a page setup in the list.
5. Click Continue. The settings specified in the selected page setup will be displayed in the Page Setup dialog box.
6. In the Page Setup dialog box, change any required settings. Click OK.

   The new page setup is displayed in the Page Setups list in the Page Setup Manager.
7. To apply the new page setup to the current layout, in the Page Setup Manager, right-click the page setup and select Set Current.
8. In the Page Setup Manager, click Close.

## Modify the Settings of a Layout's Page Setup

1. Click the layout tab for which you want to modify the page setup settings.
2. Click Page Setup Manager.
   ![](../images/GUID-C6B05F20-1FAB-48DF-8CA1-621D77A17680-low.png)
3. In the Page Setup Manager, Page Setups area, select the page setup that you want to modify.
4. Click
   ![](../images/GUID-78DED619-51B3-49C1-AD71-F928B5F69C0A-low.png) ![](../images/ac.menuaro.gif) Edit.
5. In the Page Setup dialog box, modify the required settings. Click OK.
6. In the Page Setup Manager, click Close.

## Apply a Layout's Named Page Setup to Another Layout

Use one of the following methods:

1. Right-click on the layout tab to which you want to apply another layout's page setup settings.
2. Expand Choose Named Page Setup and select the page setup.

1. Click the layout tab to which you want to apply another layout's page setup settings.
2. Click Page Setup Manager.
   ![](../images/GUID-C6B05F20-1FAB-48DF-8CA1-621D77A17680-low.png)
3. In the Page Setup Manager, Page Setups area, select a named page setup that you want to apply to the layout selected in the
   drawing area.
4. Right-click and select Set Current.
5. Click Close.

## Apply a Layout's Named Page Setup to Multiple Layouts

1. Click Show Drawings & Layouts located next to the file tabs.
   ![](../images/GUID-C2BA3961-A036-4F07-A055-D7A43119520E-low.png)
2. Use the Shift or Command key to select the layouts to which you want to apply a named page setup.
3. Right-click, expand Choose Named Page Setup, and select the page setup.

## Import a Named Page Setup from Another Drawing

1. Click Page Setup Manager.
   ![](../images/GUID-C6B05F20-1FAB-48DF-8CA1-621D77A17680-low.png)
2. In the Page Setup Manager, click
   ![](../images/GUID-78DED619-51B3-49C1-AD71-F928B5F69C0A-low.png) ![](../images/ac.menuaro.gif) Import.
3. Browse to and select a drawing file from which you want to import one or more named page setups. Click Open.
4. In the Import Page Setups dialog box, select one or more page setups to import. Click OK.

   If a page setup with the same name already exists in the drawing, you can redefine the settings of the existing one with the
   settings of the imported page setup, or you can cancel the operation.

   The imported page setups are displayed in the Page Setup Manager in the list of page setups.

   NOTE:You can import both model space and layout page setups at the same time. However, an imported model space page setup is shown
   in the Page Setup Manager only if the model tab was current when you opened the Page Setup Manager. Similarly, an imported
   layout page setup is shown in the Page Setup Manager only if a layout tab was current when you opened the Page Setup Manager.
5. In the Page Setup Manager, click Close.

## Override Settings

Named page settings can be overridden on the fly, as you're printing. The new settings can be saved as a new named page setup.

1. Click Plot.
   ![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png)
2. In the Plot dialog box, change the required settings.
3. Under Page Setup, click Save As.
   ![](../images/GUID-1AD93785-6805-43C2-AD24-AC5B5C169D85-low.png)
4. Enter a name for the page setup. Click OK.
