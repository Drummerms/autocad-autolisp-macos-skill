# To Set a Plot Style Table Type for New Drawings

1. Click Windows ![](../images/ac.menuaro.gif) Properties Inspector .
2. In the Properties Inspector, under Plot Style tab, select Color-dependent Plot Style or Use Named Plot Style from the Plot
   Table Type list.

   NOTE: Setting plot style table types for new drawings does not affect existing drawings.
