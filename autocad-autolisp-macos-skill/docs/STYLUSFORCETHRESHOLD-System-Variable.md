# STYLUSFORCETHRESHOLD (System Variable)

For those using a stylus for input, controls the force threshold which means the intensity needed to press the stylus for
it to be considered a click or selection.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 2 |

| Value | Description |
| 0 | Force press is disabled |
| 1 | Low intensity |
| 2 | Medium intensity |
| 3 | High intensity |
