# PDF Underlay Visor

Displays options for modifying PDF underlays.

![](../images/GUID-E2C4AB27-7057-4DC4-8568-A54963C419C8-low.png)

## Summary

The PDF Underlay visor is displayed when you select a PDF underlay.

## List of Options

The following options are displayed.

Scale
:   Enlarges or reduces the size of the PDF underlay. ([SCALE](SCALE-Command.md) command)

Fade
:   Controls the appearance of the linework. The greater the value, the lighter the linework in the PDF underlay is displayed.
    ([PDFADJUST](PDFADJUST-Command.md) command)

Visibility
:   Controls the visibility of the graphics and text within the PDF underlay.

Monochrome
:   Controls the color saturation of all linework while maintaining the luminance. ([PDFADJUST](PDFADJUST-Command.md) command)

Enable Object Snapping
:   Controls whether object snapping is active for geometry in all attached PDF underlays that contain vector graphics. ([PDFOSNAP](PDFOSNAP-System-Variable.md) system variable)

Create the Clipping Boundary
:   Crops the display of the selected PDF underlay to a specified boundary. ([PDFCLIP](PDFCLIP-Command.md) command)

Delete the Clipping Boundary
:   Removes the clipping boundary from the selected PDF underlay. ([PDFCLIP](PDFCLIP-Command.md) command)

Underlay Layers
:   Controls the display of layers in a PDF underlay. ([ULAYERS](ULAYERS-Command.md) command)

Import PDF
:   Imports the geometry, fills, raster images, and TrueType text objects from the selected PDF underlay.

External Reference Manager
:   Opens or closes the Reference Manager palette. ([EXTERNALREFERENCES](EXTERNALREFERENCES-Command.md) command)
