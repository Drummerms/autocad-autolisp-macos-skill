# To Display or Hide Dynamic Dimensional Constraints

1. Click
   Tools menu ![](../images/ac.menuaro.gif) Parametric ![](../images/ac.menuaro.gif) Dynamic Dimensions ![](../images/ac.menuaro.gif) Show All/Hide.
2. Select the constrained objects and press Enter.
3. Enter one of the options.
