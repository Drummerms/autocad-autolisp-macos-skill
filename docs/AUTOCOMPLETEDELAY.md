# AUTOCOMPLETEDELAY

Controls the amount of time that elapses before automated keyboard features display at the Command prompt.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0.3 |

The time delay setting in the [AUTOCOMPLETEMODE](GUID-73E36D7D-DE9A-4D60-BCEE-45E052D3E62C.htm#WS1A9193826455F5FF-4231BC7412BAC55746E-2555) system variable must be turned on for AUTOCOMPLETEDELAY to have an effect.

Valid values are real numbers from 0 to 10, which represent seconds.
