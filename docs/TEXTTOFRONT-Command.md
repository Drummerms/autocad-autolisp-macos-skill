# TEXTTOFRONT (Command)

Brings text, dimensions, and multileaders in front of all other objects in the drawing.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Draw Order drop-down.

*Menu*:
Tools ![](../images/ac.menuaro.gif) Draw Order ![](../images/ac.menuaro.gif) Bring Annotations to Front.

The following prompts are displayed.

NOTE: Text and dimensions that are contained within blocks and xrefs cannot be brought to the front separately from the containing
object.

Text ![](../images/GUID-D0F23D60-EDF8-46E1-A002-D0BD87DF97C1-low.png)
:   Brings all text in front of all other objects in the drawing.

Dimensions ![](../images/GUID-30BB5C29-A448-492C-BA7E-4D21131BAEF5-low.png)
:   Brings all dimensions in front of all other objects in the drawing.

Leaders ![](../images/GUID-C86BE356-3FEE-4A90-8BFC-BC1B1D5893AF-low.png)
:   Brings all leaders in front of all other objects in the drawing.

All ![](../images/GUID-896C4F88-9C94-457A-B631-4E4B58EB360C-low.png)
:   Brings all text, leader, and dimension objects in front of all other objects in the drawing.
