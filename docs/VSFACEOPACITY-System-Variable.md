# VSFACEOPACITY (System Variable)

Turns on and off a preset level of transparency for 3D objects.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | -60 |

The level of transparency for 3D solids and surfaces can be set between 100 percent opaque and 0 percent opaque. Negative
numbers store the value but turn off the effect.

![](../images/GUID-D0F5DB5C-4D49-42B7-851E-14D0D863AD1F-low.gif)

NOTE:Existing visual styles are not changed when you enter a new value for this system variable. Any new value entered for this
system variable temporarily creates an unsaved new visual style.
