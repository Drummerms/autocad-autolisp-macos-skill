# HPPATHWIDTH (System Variable)

Specifies the width of new drawn paths to be hatched or filled.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Registry |
| Initial value: | 0.2500 |

Width is used to define the final shape and size of the closed hatch boundary based on the path drawn and the current alignment
specified.
