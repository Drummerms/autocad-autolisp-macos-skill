# RASTERDPI (System Variable)

Controls paper size and plot scaling when changing from dimensional to dimensionless output devices, or vice versa.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 300 |

Converts millimeters or inches to pixels, or vice versa. Accepts an integer between 100 and 32,767 as a valid value.
