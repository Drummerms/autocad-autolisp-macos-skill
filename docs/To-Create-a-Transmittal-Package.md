# To Create a Transmittal Package

1. Do one of the following:
   * Click Package on toolbar.
     ![](../images/GUID-D5ECD674-84D1-4B3A-AB6A-F2264AA36BC2-low.png)
   * Click
     File menu ![](../images/ac.menuaro.gif) Package Drawing.
   * At the Command prompt, enter ETRANSMIT.

   The current drawing is added to the list of files to package.
2. To add files to the package:
   * Click the plus sign (+).
   * Browse to and select the files you want to include.
   * Click Open.
3. To remove files from the package:
   * Select the files you want to remove.
   * Click the minus sign (-).
4. Click Package Settings to specify more transmittal options that you want to use.
   * To specify the package type, select an option from the Package Type drop-down to create either a zip file or a folder.
   * To specify the DWG file format, select an option from the File Compatibility drop-down list.
   * Check any additional options as needed.
   * Click Done.
5. In the Package Notes text box, specify any additional comments to include in the report file.
6. Click Package Files.
7. Select the folder or enter the zip file name and location when prompted.
