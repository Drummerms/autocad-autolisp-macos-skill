# To Access Model Space

Switches from a named (paper space) layout to the Model layout

* Click the Model tab below the drawing window or enter
  *Model* at the command prompt.
* If you want to access model space from within a layout viewport, double-click inside the layout viewport. When you are done,
  you can click within any other layout viewport to make it the current one, or you can double-click anywhere else in the layout
  to return to paper space.
