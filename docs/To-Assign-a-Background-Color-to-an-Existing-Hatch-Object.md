# To Assign a Background Color to an Existing Hatch Object

1. Enter Hatch in the command prompt.
2. Select a Hatch Colour from the listed options in the command prompt.
3. To set the hatch color back to the current color for objects, enter “.” or Use Current.
4. To turn off the background color, enter “.” or none.
5. When Hatch Type is set to Pattern, you can also specify a new background color. The background color value is stored in the
   HPBACKGROUNDCOLOR system variable.
