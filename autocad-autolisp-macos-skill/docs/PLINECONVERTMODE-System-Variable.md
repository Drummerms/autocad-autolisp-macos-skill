# PLINECONVERTMODE (System Variable)

Specifies the fit method used in converting splines to polylines.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Polylines are created with linear segments |
| 1 | Polylines are created with arc segments |
