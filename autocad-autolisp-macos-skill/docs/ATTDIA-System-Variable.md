# ATTDIA (System Variable)

Controls whether the INSERT command uses a dialog box for attribute value entry.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Issues Command prompts |
| 1 | Uses a dialog box |
