# PLOTSTYLE (Command)

Create and edit plot style files.

## Access Methods

*Menu*:
Format ![](../images/ac.menuaro.gif) Plot Style.

## Summary

Displays the
[Edit Plot Style Table dialog box](Edit-Plot-Style-Table-Dialog-Box.md).

If you enter
*-plotstyle* at the Command prompt,
[options are displayed](PLOTSTYLE-Command-2.md).
