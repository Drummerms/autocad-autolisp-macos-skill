# SCALELISTEDIT (Command)

Controls the list of scales available for layout viewports, page layouts, and plotting.

## Access Methods

*Menu*:
Format
![](../images/ac.menuaro.gif) Scale List.

![](../images/ac.keyboard.gif) Command entry:  *'scalelistedit* for transparent use

## Summary

Displays a list of predefined scales used throughout the program. This list can be easily modified to display only the scales
you need.

The
[Edit Drawing Scales dialog box](Edit-Drawing-Scales-Dialog-Box.md) is displayed.

If you enter
*-scalelistedit* at the Command prompt,
[options are displayed](SCALELISTEDIT-Command-2.md).
