# Create STL File Dialog Box

Saves a drawing as an STL file.

STLOUT (Command)

## Summary

The Create STL File dialog box is a standard file selection dialog box.

The STL file format is preferred by 3D printing service providers. If you have prepared your drawing for 3D printing, you
save it with the STLOUT command.

The
[FACETRES](FACETRES-System-Variable.md) system variable determines how the solid is triangulated. A higher value creates a finer mesh that more accurately represents
the model. This also results in a much larger file.
