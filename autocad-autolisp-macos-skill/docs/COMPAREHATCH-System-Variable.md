# COMPAREHATCH (System Variable)

Controls whether hatch objects are included in the drawing comparison.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| Value | Description |
| 0 | Hatch objects are excluded from the drawing comparison |
| 1 | Hatch objects are included in the drawing comparison |
