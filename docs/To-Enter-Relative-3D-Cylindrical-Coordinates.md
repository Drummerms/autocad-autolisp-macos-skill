# To Enter Relative 3D Cylindrical Coordinates

* At a prompt for a point, enter the coordinates using the following format:

  *@* *X* *<* *angle* *,* *Z*

For example, @4<60,2 represents a location that is 4 units along the *X* axis from the last point measured at 60 degrees from the positive *X* axis and at 2 units in the positive *Z* direction.
