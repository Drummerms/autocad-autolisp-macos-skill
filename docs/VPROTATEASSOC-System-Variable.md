# VPROTATEASSOC (System Variable)

Controls whether the view within a viewport is rotated with the viewport when the viewport is rotated.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | When a viewport is rotated, the view inside is not rotated. |
| 1 | When a viewport is rotated, the view inside is rotated to match the rotation of the viewport. |

NOTE:After rotating a viewport, VPROTATEASSOC is set to 0 for a new viewport created in the same layout.
