# To Attach a PDF Underlay

1. Click
   Insert menu ![](../images/ac.menuaro.gif) PDF Underlay.
2. In the dialog box, select a PDF file name from the list. Click Open.
3. In the Attach PDF Underlay dialog box, select one page, or use the Shift or Command key to select multiple pages.
4. Use one of the following methods to specify insertion point, rotation, or scale:
   * Select Specify On-Screen to use the pointing device to attach the underlay at the location, angle, or scale you want.
   * Clear Specify On-Screen and enter values under Insertion Point, Rotation, or Scale.
5. Click OK.

## Attach by Dragging

1. From Finder, drag the PDF file into the drawing canvas.
2. Move the mouse to the location you want to place the PDF file.

   When you release the button, the first or only page of the PDF file is attached as a PDF underlay.
3. Use the PDF Underlay visor to change scale, fade, and object snapping as needed.
