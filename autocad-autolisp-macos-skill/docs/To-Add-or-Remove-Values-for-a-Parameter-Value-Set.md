# To Add or Remove Values for a Parameter Value Set

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block Editor.
   ![](../images/GUID-E3DDBE51-C92A-4D07-A51D-C7C685E4F578-low.png)
2. In the Edit Block Definition dialog box, select the block you want to modify and click Edit Block.
3. In the Block Editor visor, click the parameter you need to use.
4. In the Properties palette, under Value Set, select List in the drop-down list next to one of the following items:
   * Ang Type
   * Dist Type
   * Hor Type
   * Ver Type
5. In the Properties palette, under Value Set, click the [...] button next to Value List.
6. In the Add Distance Value/Angle dialog box, do one of the following:
   * To add a value, enter the value in the entry field and click Add.

     You can add one value or enter two or more values separated by commas.
   * To delete a value, select the value and click Delete.
7. Click OK.
8. Click Save in the Block Editor visor
9. (Optional) If you are finished using the Block Editor, click Close Block Editor.
