# ACISIN (Command)

Imports an ACIS (SAT) file and creates 3D solid, body, or region objects.

## Access Methods

*Menu*:
Insert
 ![](../images/ac.menuaro.gif)  ACIS File

A standard file selection box is displayed. Select the file to import. The SAT (ASCII) ACIS file is imported into the drawing.

NOTE:ACISIN imports SAT files up to ACIS version 7.0.
