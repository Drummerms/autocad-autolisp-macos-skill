# SMOOTHMESHMAXFACE (System Variable)

Sets the maximum number of faces permitted for mesh objects.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 1000000 |

The initial value of this system variable is adjusted for best performance based on your system configuration. Permissible
values are from 108 to 16,000,000. Use this limit to prevent creating extremely dense meshes that might affect program performance.
