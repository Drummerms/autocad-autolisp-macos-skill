# Polyline Extend

A new grip option, Extend Vertex, adds a new vertex to your polyline extending from the selected end grip. You no longer
have to reverse the polyline direction.

![](../images/GUID-ABDB962D-01EA-40F8-BEE5-AF1AF95F17E7-low.png)

The existing Add Vertex option could add the vertex between the end and the previous vertex. It depended on the direction
the polyline was drawn.

![](../images/GUID-BB10E627-B822-451D-95CB-CEE202ADB75C-low.png)

With the new Extend Vertex grip option, the vertex is always extended from the selected end grip, no matter which direction
the polyline was drawn.

![](../images/GUID-6FF85E6D-C047-4193-BFD0-4B6200F20BFA-low.png)
