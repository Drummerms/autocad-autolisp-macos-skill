# Named Page Setups Reference

You can also apply different named page setups to the same layout to achieve specific results when plotting. For example,
you might create the named page setups in the following table to control scaling and paper size.

| Page setup name | Description |
| NoScaling | Plot at scale 1:1, E-size sheet |
| Scale 1 to 2 | Plot at scale 1:2, C-size sheet |
| Draft | Plot to the draft-quality plotter |
| Final | Plot to the high-quality plotter |
| Fit-to-Paper | Fit to Paper, A-size sheet |
