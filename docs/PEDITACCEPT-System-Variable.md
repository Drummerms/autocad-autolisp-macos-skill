# PEDITACCEPT (System Variable)

Suppresses display of the Object Selected Is Not a Polyline prompt in PEDIT.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

The prompt is followed by “Do you want it to turn into one?” Entering y converts the selected object to a polyline. When the
prompt is suppressed, the selected object is automatically converted to a polyline.

| 0 | The prompt is displayed |
| 1 | The prompt is suppressed |
