# BSAVE (Command)

Saves the current block definition.

## Access Methods

![](../images/ac.mouse.gif) Toolbar: Block Editor visor ![](../images/ac.menuaro.gif) Save.
![](../images/GUID-5E815E0C-AED9-4DCE-AC3D-3AA0C6FF0DA6-low.png)

## Summary

Saves changes to the current block definition.

You can only use the BSAVE command in the
[Block Editor](Block-Editor.md).
