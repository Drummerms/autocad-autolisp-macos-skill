# To Move a List Item in Multiline Text Down a Level

1. Double-click a multiline text object to open the In-Place Text Editor.
2. Place the cursor at the beginning of a list item and press TAB.

   The item moves down one level and begins a nested list.
3. Press ENTER to start the next item at the same level, or press SHIFT+TAB to move the item up a level.
4. To save your changes and exit the editor, click in the drawing outside the editor. Or, Press CMD + ENTER.
