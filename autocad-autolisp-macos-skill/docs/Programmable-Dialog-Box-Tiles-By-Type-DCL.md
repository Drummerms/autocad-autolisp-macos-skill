# Programmable Dialog Box Tiles By Type (DCL)

Listing of tiles that are used to implement dialog boxes.

The programmable dialog box tiles are used to layout and define which controls are to be used to collect input from the user.
This section lists each tile, grouping them by type.

#### Topics in this section

* [Cluster Tiles Reference (DCL)](Cluster-Tiles-Reference-DCL.md)

  You can group tiles into composite rows or columns (known collectively as clusters).
* [Decorative Tiles (DCL)](Decorative-Tiles-DCL.md)

  Image and Spacer tiles are used to provide messages to the user when a dialog definition is displayed.
* [About Predefined Active Tiles (DCL)](About-Predefined-Active-Tiles-DCL.md)

  Predefined controls make it possible to implement commonly used Windows controls.
* [Text Cluster Tiles (DCL)](Text-Cluster-Tiles-DCL.md)

  A text tile is surrounded by margin space (like any other kind of tile), which presents a problem when you want to combine
  pieces of text.
* [Dialog Box Exit Buttons and Error Tiles (DCL)](Dialog-Box-Exit-Buttons-and-Error-Tiles-DCL.md)

  Exit buttons and error tiles are used to close a dialog box and provide information when a problem is encountered.
* [About Restricted Tiles (DCL)](About-Restricted-Tiles-DCL.md)

  Do not use restricted tiles in your dialog definitions.
