# POLARANG (System Variable)

Sets the polar angle increment.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Registry |
| Initial value: | 90 |

Values are 90, 45, 30, 22.5, 18, 15,10, and 5.
