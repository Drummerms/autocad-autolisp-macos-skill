# To Draw a Polyline with Straight Segments

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Polyline.
   ![](../images/GUID-30386B3B-BD89-447D-A95A-EFCE56889866-low.png)
2. Specify the first point of the polyline.
3. Specify the endpoint of the first polyline segment.
4. Continue specifying segment endpoints as needed.
5. Press Enter to end, or enter
   *c* to close the polyline.

   To start a new polyline at the endpoint of the last polyline drawn, start the PLINE command again and press Enter at the Specify
   Start Point prompt.
