# Commands for Importing PDF Files

#### Commands

* [PDFIMPORT](GUID-69BFD177-438C-446C-AAFB-0754B71FBB72.htm " Imports the geometry, fills, raster images, and TrueType text objects from a specified PDF file.
  		")

#### System Variables

* [PDFIMPORTFILTER](GUID-79A946C3-7E9C-4E7F-B175-23CC55978028.htm "Controls what types of data are imported from a PDF file and converted to AutoCAD objects.
  		")
* [PDFIMPORTIMAGEPATH](GUID-B55151E1-E885-4F98-875C-06D402A910F5.htm "Specifies the folder where referenced image files are extracted and saved when importing PDF files.
  		")
* [PDFIMPORTLAYERS](GUID-F5A62383-B2F2-4003-8BE2-5E3E5419593D.htm "Controls what layers are assigned to objects imported from PDF files.
  		")
* [PDFIMPORTMODE](GUID-669F39FF-972F-4C24-8841-E42185A44843.htm "Controls the default processing when importing objects from a PDF file.
  		")
