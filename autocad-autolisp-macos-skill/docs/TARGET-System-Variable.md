# TARGET (System Variable)

Stores the location (as a UCS coordinate) of the target point for the current viewport.

(Read-only)

|  |  |
| --- | --- |
| Type: | 3D-point |
| Saved in: | Drawing |
| Initial value: | 0.0000,0.0000,0.0000 |
