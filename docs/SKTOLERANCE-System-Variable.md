# SKTOLERANCE (System Variable)

Determines how closely the spline fits to the freehand sketch.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 0.5 |

Valid values are between 0 and 1.
