# About Rollover Tooltips

Rollover tooltips display current values of certain properties when the cursor hovers over an object in the drawing area.

The following properties are included:

* Color
* Layer
* Linetype
* Block Name

![](../images/GUID-6DD28AD0-76E4-4A19-BEBF-E48671355011-low.png)

During a drawing compare, the rollover tooltip includes the following additional information:

* Drawing name
* Drawing path
* Color for objects only in the current drawing
* Color for objects in compared drawing

![](../images/GUID-1613757D-AC15-4235-847D-783316389C50-low.png)

NOTE:The tooltip disappears when you move the cursor.

Control the display of rollover tooltips with
AutoCAD menu ![](../images/ac.menuaro.gif) Preferences ![](../images/ac.menuaro.gif) General ![](../images/ac.menuaro.gif) Window Elements: Show Rollover Tooltips. You can also set the display with the system variable, ROLLOVERTIPS.

NOTE:The properties shown on the rollover tooltips can't be customized.
