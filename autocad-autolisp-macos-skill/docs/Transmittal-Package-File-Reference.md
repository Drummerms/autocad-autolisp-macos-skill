# Transmittal Package File Reference

Use eTransmit to create a package of files for sharing. Drawing files in the transmittal package automatically include all
related dependent files such as external references and font files.

| File Types Added Automatically | Description |
| *\*.dwg* | Root drawing file and any attached external references |
| *various image file types* | Raster image files attached externally to the root drawing or external reference drawings, or as part of a material |
| *\*.pdf* | PDF files that are attached externally to the root drawing or external reference drawing |
| *\*.shx* | Compiled shape files used for text styles or complex linetypes |
| *\*.fmp* | Font Mapping File used for the in-place text editor when working with Multi-Line Text objects |
| *\*.ctb* | Color-dependent plot style files used to control plotting of objects in transmittal set drawings |
| *\*.stb* | Named plot style files used to control the plotting of objects in transmittal set drawings |
| *\*.ttf* | True Type font files used for text styles |
| *\*.pmp* | Plotter model parameter files used to store plotter configuration information |
| *\*.lin* | Linetype definition files |
| *\*.pat* | Hatch pattern files |
| *\*.pfa, \*.pfb* | PostScript Type 1 font files |
