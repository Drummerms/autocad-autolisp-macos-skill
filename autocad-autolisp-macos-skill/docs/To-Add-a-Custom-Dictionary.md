# To Add a Custom Dictionary

You can change dictionaries by specifying the dictionary name in the DCTMAIN or DCTCUST system variable.

1. At the Command prompt, enter DCTCUST.
2. Enter the file path of the new custom dictionary when prompted for the new value of DCTCUST.
3. The newly selected dictionary is highlighted as the current custom dictionary.
