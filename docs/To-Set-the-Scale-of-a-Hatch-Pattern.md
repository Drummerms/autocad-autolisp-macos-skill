# To Set the Scale of a Hatch Pattern

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Hatch panel ![](../images/ac.menuaro.gif) Hatch.
   ![](../images/GUID-6968D6F8-F92C-458E-B670-6599C70B316E-low.png)
2. In the Hatch visor, click Pattern from the Hatch Type drop-down list.
3. On the Pattern panel, click a hatch pattern.
4. On the Properties panel, enter a hatch pattern scale.
5. Specify a point inside each area that you want to hatch.
6. Press Enter, or click Close on the visor to apply the hatch and exit the command.
