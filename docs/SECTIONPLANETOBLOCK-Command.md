# SECTIONPLANETOBLOCK (Command)

Saves selected section planes as 2D or 3D blocks.

## Access Methods

*Tool Sets*:
Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Section Plane drop-down ![](../images/ac.menuaro.gif) Section Plane to Block.
![](../images/GUID-CBCC93AA-BCC1-4116-BF90-5E50597DB4F3-low.png)

*Shortcut Menu*: Select a live section object. Right-click in the drawing area and click Generate 2D/3D Section.

## Summary

The
[Generate Section /Elevation dialog box](Generate-Section-Elevation-Dialog-Box.md) is displayed.
