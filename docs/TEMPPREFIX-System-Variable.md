# TEMPPREFIX (System Variable)

Contains the directory name (if any) configured for placement of temporary files, with a path separator appended.

(Read-only)

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Not-saved |
| Initial value: | Varies |

You can modify the location from the Application Preferences dialog box, Application tab, Files section, under Temporary Drawing
File Location.
