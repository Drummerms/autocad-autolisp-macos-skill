# To Save a Drawing With Layer and Spatial Indexes

1. Click File menu ![](../images/ac.menuaro.gif)Save As.
2. In the Save Drawing As dialog box, click Tools ![](../images/ac.menuaro.gif) Options.

   ![](../images/GUID-D47126C9-9F39-48CD-96A4-7E5C38301661-low.png)
3. In the Saveas Options dialog box, under Index Type, select Layer, Spatial, or Layer & Spatial and click OK.

NOTE: If you are saving a partially open drawing that does not already contain layer and spatial indexes, this option is disabled.

4. Click Save.
