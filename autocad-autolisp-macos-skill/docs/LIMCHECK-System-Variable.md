# LIMCHECK (System Variable)

Controls whether you can create objects outside the grid limits.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| Value | Description |
| 0 | Objects can be created outside the grid limits |
| 1 | Objects cannot be created outside the grid limits |
