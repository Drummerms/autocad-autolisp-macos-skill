# To Change the Appearance of the Drafting Tooltips for Dynamic Input

1. At the Command prompt, enter
   *dsettings*.
2. In the Drafting Settings dialog box, Dynamic Input tab, click Drafting Tooltip Appearance.
3. In the Tooltip Appearance dialog box, click Colors to display the Drawing Window Colors dialog box. Under Context, make a
   choice and then specify a color.
4. Under Size, drag the slider to adjust the size of the drafting tooltips.
5. Under Transparency, drag the slider to adjust the transparency of the drafting tooltips.

   The lower the setting, the more transparent the tooltip. A value of 100 sets the tooltip to opaque.
6. Under Apply To, choose an option:
   * *Override OS Settings for All Drafting Tooltips*. Applies the settings to all tooltips, overriding the settings in the operating system.
   * *Use Settings Only for Dynamic Input Tooltips*. Applies the settings only to the drafting tooltips used in dynamic input.
