# STYLE (Command)

Creates, modifies, or specifies text styles.

## Access Methods

*Tool Sets*:
Drafting tab ![](../images/ac.menuaro.gif) Text panel ![](../images/ac.menuaro.gif) Style.
![](../images/GUID-A1A489F9-EDAC-453E-B777-A21DD0ADC60D-low.png)

*Menu*:
Format ![](../images/ac.menuaro.gif) Text Style.

*Command entry*:
*'style* for transparent use

## Summary

The
[Text Style dialog box](Text-Style-Dialog-Box.md) is displayed.

If you enter *-style* at the Command prompt,
[options are displayed](STYLE-Command.md).

You can specify the current text style to determine the appearance of all new text. A text style includes the font, size,
obliquing angle, orientation, and other text characteristics.
