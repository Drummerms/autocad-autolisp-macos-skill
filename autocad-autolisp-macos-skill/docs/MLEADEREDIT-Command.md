# MLEADEREDIT (Command)

Adds leader lines to, or removes leader lines from, a multileader object.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Leader panel ![](../images/ac.menuaro.gif) Add Leader.
![](../images/GUID-CEFE5B90-CCDB-4419-835A-9AF11F85694E-low.png)

*Menu*:
Modify
![](../images/ac.menuaro.gif) Object
![](../images/ac.menuaro.gif) Multileader
![](../images/ac.menuaro.gif) Add Leader.

NOTE:Hover over Landing grip and choose Add Leader; hover over leader endpoint grip and choose Remove Leader.

## List of Prompts

The following prompts are displayed.

[Specify leader arrowhead location](MLEADEREDIT-Command.md) or [[Remove Leaders](MLEADEREDIT-Command.md)]

Add Leader
:   Adds a leader line to a selected multileader object. The new leader line is added to the left or right of the selected multileader,
    depending on the location of the cursor.

    ![](../images/GUID-88CAB3E2-6DC7-4825-A545-2DC89DE3D1A6-low.gif)

Remove Leaders
:   Removes a leader line from a selected multileader object.

    For example:

    ![](../images/GUID-365C69E6-54A3-4031-B109-BB04EBF5DCCE-low.gif)
