# To Specify a Font Mapping File

1. Click
   AutoCAD 2026 menu ![](../images/ac.menuaro.gif) Preferences.
2. On the Application tab, under the Files list, double-click Text Editor, Dictionary, and Font File Names.
3. Double-click Font Mapping File.
4. To change the font mapping file, double-click the arrow line to open the Open dialog box. Select a file and click Open. Click
   OK.
5. At the Command prompt, enter
   *regen* to convert existing text using the new font mappings.
