# BURST (Express Tool)

Explodes blocks, converting attribute values to text objects.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Attributes to Text.
![](../images/GUID-2A9B641C-0BA0-41FB-A29F-57FCBED887D1-low.png)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Express Tools ![](../images/ac.menuaro.gif) Explode Attributes to Text.

Explodes selected blocks. Any attribute objects are converted into to text objects. The exploded attributes assume the original
attribute definition's layer and style.
