# LAYOUTCREATEVIEWPORT (System Variable)

Controls whether a viewport is created automatically on each new layout added to a drawing.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Newly created layouts do not include any viewports. |
| 1 | A single layout viewport is created with each new layout. |
