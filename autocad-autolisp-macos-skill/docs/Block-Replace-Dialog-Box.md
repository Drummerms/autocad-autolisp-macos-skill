# Block Replace Dialog Box

Replaces all instances of a specified block with a different block.

BLOCKREPLACE (Command)

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Replace Block with another Block.
![](../images/GUID-120A4B05-D747-4D2E-89E0-39A812CB8D86-low.png)

![](../images/GUID-17CAA1AE-C148-4A92-9F67-68787AF6AB33-low.png)

## List of Options

The following options are displayed:

Select original
:   Select a block to replace.

Replace With
:   Select the block to be replaced with.

Purge Unreferenced Items when finished
:   Removes unreferenced block definition from the drawing.
