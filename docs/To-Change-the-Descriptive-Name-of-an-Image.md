# To Change the Descriptive Name of an Image

1. Click Insert ![](../images/ac.menuaro.gif) Reference Manager![](../images/GUID-6EC866D6-17B4-4BF1-81D1-9AB5FB49A6CB-low.png).
2. In the Reference Manager palette, in the Detail and Preview Panel, select the image name, and then modify the name.
