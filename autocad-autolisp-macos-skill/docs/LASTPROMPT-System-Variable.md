# LASTPROMPT (System Variable)

Stores the last string echoed to the Command prompt.

(Read-only)

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Not-saved |
| Initial value: | "" |

This string is identical to the last line seen at the Command prompt and includes any user input.
