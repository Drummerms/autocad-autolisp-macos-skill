# LOGFILENAME (System Variable)

Specifies the path and name of the command history log file for the current drawing.

(Read-only)

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | Varies |

The initial value varies depending on the name of the current drawing and the installation folder.
