# About Curved Objects

Curved objects include arcs, circles, polyline arcs, donuts, ellipses, and splines.

![](../images/GUID-6E1BE1CC-A3C0-4FD0-A44D-8A242CAF780B-low.png) Arcs--You can create arcs by specifying any three points.

![](../images/GUID-7520F49C-B7B7-4285-8460-6FFC4EA34DF9-low.png) Circles--You can create circles by specifying the center and the radius. In addition to this default method, you can specify
two points to define the diameter, three points that define the circumference, or two tangent points and the radius.

![](../images/GUID-30386B3B-BD89-447D-A95A-EFCE56889866-low.png) Polyline Arcs--Multi-segmented polyline arcs provide editing capabilities unavailable for single arcs, as you can adjust
their width and curvature. When you draw arc segments in a polyline, the first point of the arc is the endpoint of the previous
segment.

![](../images/GUID-CFEBBE80-DA08-4094-8650-EA70BA259027-low.png) Donuts--Donuts appear as filled rings or sold-filled circles; they are actually closed polylines with width.

![](../images/GUID-36ACD897-212E-469F-96D1-0FF1356A0DA3-low.png) Ellipses--The shape of an ellipse is determined by two axes that define its length and width. The longer axis is called the
*major* axis; the shorter axis is called the
*minor* axis.

![](../images/GUID-BE4EBA83-10D1-4F23-ABDB-0641CDE98831-low.png) Splines--A spline is a smooth curve that passes through or near a set of points that influence the shape of the curve. You
can create or edit splines using either
*control vertices* or
*fit points*.

![](../images/GUID-0242F2CF-6CD3-4C35-9C8B-C2D12D3F2998-low.png) Helixes--A helix is an open 2D or 3D spiral. You can use a helix as a path along which to sweep an object to produce an image.
For example, you might sweep a circle along a helix path to create a sold model of a spring.
