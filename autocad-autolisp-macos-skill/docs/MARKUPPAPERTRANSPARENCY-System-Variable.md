# MARKUPPAPERTRANSPARENCY (System Variable)

Controls the level of transparency when a digital markup is active.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 90 |

The higher the number, the more transparent the overlay.

Valid range is 0 to 90.
