# ok\_cancel Tile (DCL)

The
ok\_cancel tile is a combination of the OK and Cancel buttons, and is the standard combination for dialog boxes that can originate changes
to data.

*Supported Platforms:* Windows, Mac OS, and Web

```
ok_cancel;
```

![](../images/GUID-72DE206B-E95C-492E-9C4D-A156BB1ABB15-low.png)

The key of the Cancel button is
"cancel".

The
ok\_cancel tile is defined in the
*base.dcl* file.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
