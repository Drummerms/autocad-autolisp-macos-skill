# Count Visor

Displays the number of objects and issues, along with other controls to manage the counted objects.

## Visor Commands and Options

When you're in an active count, the Count visor displays at the top of the drawing area.

![](../images/GUID-3951D146-B66F-4393-B135-8CEDEB6C08F1-low.png)

The following controls are available.

## Count

Displays the count of the selected object.

## Count Details

Displays the count information of the selected object. The count details icon changes depending on whether the current count
contains errors. Click
![](../images/GUID-FBAC9A26-4A4D-4E99-BF12-68A302626888-low.png) or
![](../images/GUID-4ADA7BAA-4104-4C0C-B3AB-8EAE788DCAAC-low.png) to open the Count palette and view more details.

| ![](../images/GUID-BF2DBC8C-3390-4661-BFD8-AF1CE0A36153-low.png) | Displays the count criteria including the general properties of the counted objects and any user-defined block attributes and parameters. |
| ![](../images/GUID-525D7C63-B386-4CE0-8067-475C63D0D592-low.png) | Displays the count criteria including the general properties, user-defined block attributes and parameters, and the error report of the counted objects. Count errors can include overlapping, exploded, or renamed objects. |

## Previous or Next

Zooms to the previous or next object in the count.

## Insert Count Field

Inserts a field that is set to the current count value.

## End Count

Closes the Count visor and ends the count.
