# REBUILD2DOPTION (System Variable)

Controls whether to delete the original curve when rebuilding a spline.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Retains original curve |
| 1 | Deletes original curve |
