# To Create a Network Surface

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Surface panel ![](../images/ac.menuaro.gif) Surface drop-down ![](../images/ac.menuaro.gif) Surface Network.
   ![](../images/GUID-F700457A-FF55-43F7-A84D-DCA8F93D52CA-low.png)
2. In the drawing area, select cross section curves in the first direction (U or V) and press Enter.
3. Select cross sections in the second direction and press Enter.
