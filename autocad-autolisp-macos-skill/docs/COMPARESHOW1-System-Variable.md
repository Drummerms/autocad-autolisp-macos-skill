# COMPARESHOW1 (System Variable)

Displays the objects that exist only in the first drawing.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| Value | Description |
| 0 | Does not display the objects in the first drawing |
| 1 | Displays the objects in the first drawing |
