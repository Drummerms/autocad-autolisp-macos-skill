# Model Space and Layouts

There are two working environments, or spaces, in which you can work, model space and paper space layouts.

* Model space is used to create 2D drawings and 3D models
* Paper space is used create layouts for plotting

While you can plot from model space, layouts are more convenient for scaling views, changing the location of views, and controlling
the area and settings used in plotting.

To switch between model space and a layout, click the drop-down near the left side of the status bar.
