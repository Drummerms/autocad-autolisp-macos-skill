# About Analyzing Draft Angle

Evaluates whether a part has adequate taper to allow it to be removed from its mold.

If you create shapes or parts that need to be molded, the draft analysis tool evaluates whether the draft angles on the part
make it possible to remove the part from the mold (based on the pull direction). You can set the maximum and minimum angle
requirements on the Draft Angle tab of the Analysis Options dialog box and then use a color analysis to see how the model
meets the requirements.

NOTE: Analysis tools only work in the 3D visual styles; they will not work in 2D.
