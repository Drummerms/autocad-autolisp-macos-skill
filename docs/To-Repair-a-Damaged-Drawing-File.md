# To Repair a Damaged Drawing File

1. At the Command Prompt, enter Recover.
2. In the Select File dialog box, select a file. Click Open.

   After the audit, all objects with errors are placed in the Previous selection set for easy access. The output from the audit
   is written to an audit log (ADT) file if the AUDITCTL system variable is set to 1 (On).
