# To Specify Style Settings for Lateral Tolerances

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Dimension panel ![](../images/ac.menuaro.gif) Dimension Style.
   ![](../images/GUID-89D5BC05-30EE-4627-B80B-8680F08A01A9-low.png)
2. In the Dimension Style Manager, select the style you want to change. Click Options
   ![](../images/GUID-0888F513-2A7D-45B1-AD99-683EBA803A86-low.png) ![](../images/ac.menuaro.gif) Modify.
3. In the Modify Dimension Style dialog box, Tolerances tab, under Tolerance Format, choose a tolerance style from the Method
   list, and then do one of the following:
   * If you choose Limits, enter upper and lower tolerance deviation in the Upper Value and Lower Value boxes.
   * If you choose Symmetrical tolerances, Lower Value is not available, because you need only one tolerance value.
   * If you choose Basic, enter a value in Offset from Dim Line (on the Text tab) to represent the gap between the text and its
     enclosing box.
4. Click OK.
5. Click Close to exit the Dimension Style Manager.
