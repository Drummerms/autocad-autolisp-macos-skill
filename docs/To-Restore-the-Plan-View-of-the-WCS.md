# To Restore the Plan View of the WCS

* Click
  View menu ![](../images/ac.menuaro.gif)3D Views![](../images/ac.menuaro.gif)Plan View![](../images/ac.menuaro.gif)World UCS.

The menu bar is not displayed by default, so an alternative method is to enter PLAN at the Command prompt and use the World
option.
