# To Export Selected Objects to an ACIS File

1. Click File ![](../images/ac.menuaro.gif) Export .
2. In the Export Data dialog box, enter a file name.
3. In the File Format pane, select ACIS (\*.sat). Click Save.
4. Select the objects you want to save to an ACIS file.

   The file extension *.sat* is automatically appended.
