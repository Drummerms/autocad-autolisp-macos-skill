# To Re-create the Boundary Object for a Hatch Object

1. Select the hatch object.
2. At the prompt, specify the type of object to create as the new boundary, and whether to associate the boundary with the hatch.
3. Use Recreate Boundary option to generate a closed polyline or a region object around a selected hatch or fill.
4. Press Enter to apply the hatch and exit the command.

![](../images/GUID-4A76702A-DAEB-4030-838A-AA6FA7294104-low.png)

NOTE:

* You can also right-click an unbounded hatch object and choose Generate Boundary; however, this option does not allow you to
  create an associative boundary.
* In some cases, it is easier to delete the hatch object, and create a new hatch object from the desired boundary objects.
