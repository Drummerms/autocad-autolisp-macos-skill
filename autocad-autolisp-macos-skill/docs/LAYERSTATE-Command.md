# LAYERSTATE (Command)

Saves, restores, and manages sets of layer settings that are called layer states.

![](../images/GUID-507AE55C-B1EB-46BC-99F8-7080DAE7F47C-low.png) Find

## Summary

The Layer States Manager is displayed.

Saves the current layer settings in a drawing as a layer state. Layer states can then be restored, edited, imported, and exported
for use in other drawings.
