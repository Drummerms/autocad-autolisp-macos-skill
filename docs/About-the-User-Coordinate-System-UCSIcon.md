# About the User Coordinate System (UCS) Icon

The drawing area displays an icon representing the XY axis of a rectangular coordinate system called the user coordinate system,
or UCS.

![](../images/GUID-10777399-75DE-460A-87E2-675EA4A56F18-low.png)

You can select, move, and rotate the UCS icon to change the current UCS. The UCS is useful in 2D, and essential in 3D.

You can control whether the UCS icon is visible, and change its appearance, with the UCSICON command, Properties option.
