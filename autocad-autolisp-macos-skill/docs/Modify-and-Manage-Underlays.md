# Modify and Manage Underlays

#### Topics in this section

* [About Modifying the Position, Scale, or Rotation of an Underlay](About-Modifying-the-Position,-Scale,-or-Rotation-of-an-Underlay.md)

  You specify an underlay’s position, scale, or rotation when you attach an underlay file. You can also modify these settings
  later during the drafting process.
* [About Controlling the Display of Layers in an Underlay](About-Controlling-the-Display-of-Layers-in-an-Underlay.md)

  You can turn layers on and off in an underlay.
* [Use Object Snaps With PDF Underlays](Use-Object-Snaps-With-PDF-Underlays.md)

  Use object snaps to draw or edit objects relative to a precise location.
* [About Adjusting the Appearance of Underlays](About-Adjusting-the-Appearance-of-Underlays.md)

  You can modify the contrast, fade, and monochrome settings of a PDF underlay. Also, you can adjust the underlay colors based
  on the current background color of the drawing window to make the underlay visible.
* [Clip Underlays](Clip-Underlays.md)

  You can use a clipping boundary to clip a PDF underlay.
* [About Reference Manager and Underlays](About-Reference-Manager-and-Underlays.md)

  With the Reference Manager, you can view PDF underlays that are attached to a DWG file and edit the paths of the attachments.
* [About Changing File Paths of Underlays](About-Changing-File-Paths-of-Underlays.md)

  You can change the file path to a referenced PDF file or search for an underlay when it is reported as not found.
* [About Publishing, Saving, and ETransmitting Drawings Containing Underlays](About-Publishing,-Saving,-and-ETransmitting-Drawings-Containing-Underlays.md)

  When you eTransmit a file, DWF, DWFx, PDF, and DGN underlays are tracked and managed.
