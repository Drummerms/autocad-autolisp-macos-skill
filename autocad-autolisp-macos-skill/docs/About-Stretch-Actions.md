# About Stretch Actions

Stretch actions move and stretch objects a specified distance in a specified direction relative to the base point specified
by the action.

## Apply a Stretch Action to a Point Parameter

A stretch action applied to a point parameter moves and stretches selected objects a specified distance in any direction.

![](../images/GUID-264E7741-F591-4DDE-AF6B-6D5BE50CE7C2-low.png)

## Apply a Stretch Action to a Linear or Polar Parameter

A stretch action applied to a linear or polar parameter moves and stretches selected objects a specified distance in the direction
of the parameter.

![](../images/GUID-FBA4E1F6-50BE-42AD-8F49-D508D040B0FC-low.png)

## Apply a Stretch Action to an XY Parameter

A stretch action applied to an XY parameter stretches the selected objects only in the X and Y direction specified by the
parameter.

![](../images/GUID-078C9ECC-433D-4158-83E0-014CF7B72764-low.png)
