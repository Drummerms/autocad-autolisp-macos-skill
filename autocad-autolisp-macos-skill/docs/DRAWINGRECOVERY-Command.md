# DRAWINGRECOVERY (Command)

Displays a list of drawing files that can be recovered after a program or system failure.

## Access Methods

*Menu*:
Help ![](../images/ac.menuaro.gif) Recover Files.

## Summary

Opens the
[Files Recovered dialog box](Files-Recovered-Dialog-Box.md).
