# XREFCTL (System Variable)

Controls whether external reference log (XLG) files are created.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| Value | Description |
| 0 | Does not write log files |
| 1 | Writes log files  NOTE:The XLG file records only the most recent XREF function that was performed. |
