# To Save a Drawing Automatically

Using Preferences, you can enable, disable, or change the frequency of, Automatic Save.

1. Right-click the drawing area and choose Preferences.
2. On the General tab, in the Application Preferences Dialog Box, select the Automatic Save check box.
3. Enter a number in Minutes Between Saves.

NOTE:Automatic save is disabled when the Block Editor is open
