# PROXYNOTICE (System Variable)

Displays a notice when a proxy is created.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

A proxy is created when you open a drawing containing custom objects created by an application that is not present. A proxy
is also created when you issue a command that unloads a custom object's parent application.

| 0 | No proxy warning is displayed |
| 1 | Proxy warning is displayed |
