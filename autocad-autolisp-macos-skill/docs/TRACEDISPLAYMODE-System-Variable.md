# TRACEDISPLAYMODE (System Variable)

Indicates whether the tracing paper effect is displayed (front) or not (back) while a trace is active.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 2 |

| Value | Description |
| 0 | Trace mode is not active |
| 1 | Back - tracing paper not displayed, trace geometry dimmed, host geometry fully saturated |
| 2 | Front - tracing paper displayed, trace geometry fully saturated, host geometry dimmed |

NOTE:When trace mode is not active, TRACEDISPLAYMODE is 0.
