# ATTDEF (Command)

Creates an attribute definition for storing data in a block.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Define Attribute.
![](../images/GUID-59191139-7C55-482E-8248-032FA31D2056-low.png)

*Menu*:
Draw ![](../images/ac.menuaro.gif) Block ![](../images/ac.menuaro.gif) Define Attributes.

## Summary

The Attribute Definition dialog box is displayed.

If you enter
*-attdef* at the Command prompt, options are displayed.

An attribute is an object that is created and included with a block definition. Attributes can store data such as part numbers,
product names, and so on.
