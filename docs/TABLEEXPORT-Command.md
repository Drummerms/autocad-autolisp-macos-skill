# TABLEEXPORT (Command)

Exports data from a table object in CSV file format.

## Summary

A [standard file selection dialog box](Standard-File-Selection-Dialog-Boxes.md) is displayed. Table data is exported in the comma-separated (CSV) file format. All formatting of the table and its text is
lost.
