# Table Styles

Create and modify table styles using the new Table Style manager. Table styles control the appearance of a table and the
table cells.

![](../images/GUID-DFEDDB88-74B1-4D2E-A2A9-01D2AF5D5F8D-low.png)

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/GUID-EA6A1B89-9B6F-4552-AD8E-694113F12EA9-low.png), or enter TABLESTYLE at the command prompt.

   Existing table styles for the current drawing are listed in the Table Style Manager dialog box.
2. Do one of the following:
   * To create a table style, click
     ![](../images/GUID-956AE331-FA56-435B-B092-CDB1D4FA6458-low.png).
   * Select the table style you want to modify, right-click and select Modify.

     The New or Modify Table Style dialog box displays.
3. Select the cell style from the drop-down list.

   There are 3 default cell styles: Title, Header, and Data.

   ![](../images/GUID-8E38C430-2D54-40B1-BBAD-645CFB77F2CA-low.png)
4. Update the properties for that cell style. Click the tabs General, Text, and Borders, to see all available properties for
   the selected cell style.
5. Click OK to save all changes to the table style.

The following actions are also supported:

* *Rename*: Select the style, right-click, and select Rename.
* *Delete*: Select a table style, right-click, and select Delete.
* *Set current*: Select a table style, right-click, and select Set Current.

NOTE:You can't rename or delete the Standard table style.

## *New Commands*

[TABLESTYLE](TABLESTYLE-Command.md) - Creates, modifies, or specifies table styles.
