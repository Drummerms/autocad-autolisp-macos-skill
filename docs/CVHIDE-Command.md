# CVHIDE (Command)

Turns off the display of control vertices for all NURBS surfaces and curves.

## Access Methods

*Tool Set*:
Modeling tab ![](../images/ac.menuaro.gif) Surface panel ![](../images/ac.menuaro.gif) Surface CV drop-down ![](../images/ac.menuaro.gif) Surface CV - Hide.
![](../images/GUID-3F95E767-C435-42F8-9E70-B8103E357E14-low.png)

*Menu*:
Modify ![](../images/ac.menuaro.gif) Surface Editing ![](../images/ac.menuaro.gif) NURBS Surface Editing
![](../images/ac.menuaro.gif) Hide CV.

## Summary

Turns off the CV display for all NURBS objects.

![](../images/GUID-D64DAAE1-4349-4FF1-A3FF-CF10E1153768-low.png)
