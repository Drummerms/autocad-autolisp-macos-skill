# NEW Command Prompt

## Summary

Enter a tilde (*~*) at the prompt to display the Select Template dialog box (a [standard file selection dialog box](Standard-File-Selection-Dialog-Boxes.md)).

## List of Prompts

The following prompts are displayed.

Enter template file name or [. (for none)] <*current*>: *Enter a name, enter a period (.), or press* Enter
