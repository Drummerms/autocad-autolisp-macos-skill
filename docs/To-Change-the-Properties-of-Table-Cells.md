# To Change the Properties of Table Cells

1. Select the cells you want to change.
   * To select a single cell, click once inside the cell.
   * To select multiple cells, click inside one cell, hold the Shift key down, and click inside another cell. All cells between
     the two cells are selected.
   * To select a row, click the row label.
   * To select a column, click the column label.
2. If the Properties Inspector isn't open, click
   Window menu ![](../images/ac.menuaro.gif) Properties Inspector.
3. In the Properties Inspector, click the property you want to change and enter a new value.

## Copy the Properties of a Cell to Other Cells

1. Click inside the cell whose properties you want to copy.

   To view the current properties of the selected cell, open the Properties Inspector.
2. Right-click and select Match Cell or click Match Cell from the Table visor.
   ![](../images/GUID-D6937B88-9F92-4985-80AA-2DBD72942E65-low.png)

   The cursor changes to a paintbrush.
3. To copy the properties to another cell, click inside the cell.

   All the properties of the cell are copied except the cell type: text or block.
4. Continue clicking in each cell you want to copy the properties to.
5. Press Esc to end copying properties.

## Remove Property Overrides

1. Click a gridline on the table to select the table.
2. Select Remove All Property Overrides from the Table visor.

   All table properties update to match the table style assigned to the table.
