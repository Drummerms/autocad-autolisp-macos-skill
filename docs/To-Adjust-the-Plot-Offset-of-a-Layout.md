# To Adjust the Plot Offset of a Layout

1. Click the layout tab for which you want to adjust the plot offset.
2. Click
   File ![](../images/ac.menuaro.gif) Page Setup Manager.
3. In the Page Setup Manager, Page Setups area, select the page setup that you want to modify.
4. Click Modify (\*). Click Edit
5. In the Page Setup dialog box, under Offset for Printable Area, enter value for
   *X* or
   *Y* or both. Click OK.
6. In the Page Setup Manager, click Close.
