# Table Visor

Size columns or rows equally, export the table data, change table style, or remove all table style property overrides.

![](../images/GUID-A42B5932-823A-4EFE-8DB9-475844EA1AC2-low.png)

## Summary

The Table visor is displayed when you select an existing table in a drawing.

## List of Options

The following options are displayed.

Size Equally: Column ![](../images/GUID-71FF38CB-B5F7-4D23-A5F6-0342392E5E82-low.png)
:   Size all columns equally based on the table's width.

Size Equally: Row ![](../images/GUID-28C74F04-92DB-47CC-B045-537E23320E89-low.png)
:   Size all rows equally based on the table's height.

Export Data ![](../images/GUID-823417E6-E665-4EF7-8542-E43B728DA01E-low.png)
:   Export the table data to a CSV file.

Table Style
:   Select a table style for the table from the available table styles.

Modify Table Style ![](../images/GUID-201F270F-50B9-4C8B-B153-33D489B4A9FB-low.png)
:   Opens the Modify Table Style dialog box, where you can change the table style currently assigned to the table.

Table Style Manager ![](../images/GUID-C2E86FFC-AEE9-47DB-8E20-18BEF26E9D03-low.png)
:   Opens the Table Style Manager dialog box, where you can create and modify table styles.

Remove All Property Overrides ![](../images/GUID-40CE80FF-E880-47C5-A676-AEE096E2D4C7-low.png)
:   Revert the table properties to match the properties of the table style.
