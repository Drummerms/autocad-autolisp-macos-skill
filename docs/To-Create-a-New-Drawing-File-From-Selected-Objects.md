# To Create a New Drawing File From Selected Objects

1. Open an existing drawing or create a new drawing.
2. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Write Block.
   ![](../images/GUID-E84A38E9-381C-46EC-BE0C-F4E1728D05DA-low.png).
3. In the Write Block dialog box, click Selected Objects.
4. Click Select Objects.

   NOTE:If Delete Objects is selected, the original objects are erased from the drawing when the block is created. If necessary, you
   can use OOPS to restore them.
5. Select the objects to be included in the new drawing. Press Enter to complete object selection.
6. In the Write Block dialog box under Base Point, specify the drawing origin point (0,0,0) for the new drawing using one of
   these methods:
   * Click Pick Point to specify a point using the pointing device.
   * Enter the
     *X,Y,Z* coordinate values of the point.
7. Click Write Block.
8. Enter a file name and path for the new drawing.
9. Click Save.

   A new drawing is created with the selected objects.
