# TSPACETYPE (System Variable)

Controls the type of line spacing used in multiline text.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

At Least adjusts line spacing based on the tallest characters in a line. Exactly uses the specified line spacing, regardless
of individual character sizes.

| 1 | At Least |
| 2 | Exactly |
