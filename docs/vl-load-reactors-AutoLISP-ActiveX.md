# vl-load-reactors (AutoLISP/ActiveX)

Loads reactor support functions

*Supported Platforms:* Windows only; not available on Mac OS or Web

## Signature

```
(vl-load-reactors)
```

No arguments.

## Return Values

*Type:* nil

Always returns
nil.

## Remarks

This function is identical to
vl-load-com and is maintained for backward compatibility.

## Examples

```
(vl-load-reactors)
```
