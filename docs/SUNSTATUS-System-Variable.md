# SUNSTATUS (System Variable)

Turns on and off the lighting effects of the sun in the current viewport.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

Only the lights from sources other than the sun affect the view in the current viewport.

| 0 | Off |
| 1 | On |
