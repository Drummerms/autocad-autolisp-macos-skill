# PREVIEWEFFECT (System Variable)

Specifies the visual effect used for previewing selection of objects.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 2 |

| 0 | Dashed lines (the default display for selected objects) |
| 1 | Thickened lines |
| 2 | Dashed and thickened lines |
