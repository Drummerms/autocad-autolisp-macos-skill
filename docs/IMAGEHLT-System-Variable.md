# IMAGEHLT (System Variable)

Controls whether the entire raster image or only the raster image frame is highlighted.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Highlights only the raster image frame |
| 1 | Highlights the entire raster image |
