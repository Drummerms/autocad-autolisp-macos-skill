# CUI (Command)

Manages the commands, menus, command aliases, and keyboard shortcuts in the product.

## Access Methods

*Menu*:
Tools ![](../images/ac.menuaro.gif) Customize ![](../images/ac.menuaro.gif) Interface (CUI).

## Summary

The Customize dialog box is displayed.

You use the Customize dialog box to add commands, customize the menu bar, manage command aliases and keyboard shortcuts. A
command alias is an abbreviation of a command name, which you can enter at the Command prompt instead of entering the entire
command name, such as L for LINE. A keyboard shortcut is a combination of keystrokes, such as CTRL+S for SAVE.
