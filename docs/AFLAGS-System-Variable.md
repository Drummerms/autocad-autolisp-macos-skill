# AFLAGS (System Variable)

Sets options for attributes.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 16 |

The value is the sum of the following:

| 0 | No attribute mode selected |
| 1 | Invisible |
| 2 | Constant |
| 4 | Verify |
| 8 | Preset |
| 16 | Lock position in block |
| 32 | Multiple lines |
