# ACDBNAVISWORKSMODELDEF (DXF)

The following group codes apply to ACDBNAVISWORKSMODELDEF symbol table entries.

Stored in the ACAD\_BIM\_DEFINITIONS dictionary.

| Coordination Model group codes | |
| Group code | Description |
| 0 | Object name (AcDbNavisworksModelDef) |
| 5 | Handle |
| 102 | Start of persistent reactors group; always “{ACAD\_REACTORS” |
| 330 | Soft-pointer ID/handle to owner dictionary |
| 102 | End of persistent reactors group, always “}” |
| 100 | Subclass marker (AcDbNavisworksModelDef) |
| 1 | Navisworks model path and file name |
| 290 | Persistent load state. Determines whether the Navisworks model will automatically be loaded upon opening DXF file. |
| 10, 20, 30 | Minimum extent coordinates. A 3D point in WCS. |
| 11, 21, 31 | Maximum extent coordinates. A 3D point in WCS. |
| 290 | Host drawing visibility flag. Determines whether the host drawing geometry will be drawn. |
