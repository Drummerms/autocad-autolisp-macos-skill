# To Rotate the Grid and Snap Angle

1. Click Tools ![](../images/ac.menuaro.gif) New UCS ![](../images/ac.menuaro.gif) Z.At the Command prompt, enter ucs.
2. Enter the Z option and specify the rotation angle.

The grid and snap angle rotates with the UCS.
