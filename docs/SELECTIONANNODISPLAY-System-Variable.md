# SELECTIONANNODISPLAY (System Variable)

Controls whether alternate scale representations are temporarily displayed in a dimmed state when an annotative object is
selected.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Off |
| 1 | On |

The dimming intensity is controlled by the [XFADECTL](XFADECTL-System-Variable.md) system variable.
