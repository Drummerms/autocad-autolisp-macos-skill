# Commands for Center Marks and Centerlines

#### Commands

* [CENTERDISASSOCIATE](GUID-96FD0416-DE6C-4220-990B-7F88A5703166.htm " Removes associativity of center marks or centerlines from objects they define.
  		")
* [CENTERLINE](GUID-095D4F06-0DEF-40F3-AA5F-D86834A1B1F6.htm " Creates centerline geometry associated with selected lines and polylines.
  		")
* [CENTERMARK](GUID-068DEC30-B279-4529-95DC-2427A2EA0B42.htm " Creates an associative, cross-shaped mark at the center of a selected circle or arc.
  		")
* [CENTERREASSOCIATE](GUID-E2CC39DA-55B6-4108-A394-86707DCFDE01.htm " Associates or reassociates a center mark or centerline object to selected objects.
  		")
* [CENTERRESET](GUID-F6C8DB68-5A6B-44BB-AAEE-6CFE03909A58.htm " Resets the centerlines to the current value specified in the CENTEREXE system variable.
  		")

#### System Variables

* [CENTERCROSSGAP](GUID-AA5B90C5-AD48-4032-BD7A-39044E26B395.htm " Determines the gap between the center mark and its centerlines.
  		")
* [CENTERCROSSSIZE](GUID-AB005660-A8D7-41AC-85FE-CC5C39B2E61E.htm " Determines the size of the associative center mark.
  		")
* [CENTEREXE](GUID-4BCE09E0-6733-401D-8719-7CE603F878B1.htm " Controls the length of the centerline extensions.
  		")
* [CENTERLAYER](GUID-BAD8055E-71F4-4377-9E31-9E0DDBBAE9ED.htm " Specifies a default layer for new center marks or centerlines.
  		")
* [CENTERLTSCALE](GUID-5C5FD35B-9B78-4C17-865E-B25687BBE121.htm " Sets the linetype scale used by center marks and centerlines.
  		")
* [CENTERLTYPE](GUID-D1BE48F5-2689-4C14-B7F8-917A3AA71246.htm " Specifies the linetype used by center marks and centerlines.
  		")
* [CENTERLTYPEFILE](GUID-2E0C4EC9-141B-4091-A7A2-05B9A01D1E98.htm " Specifies the loaded linetype library file used to create center marks and centerlines.
  		")
* [CENTERMARKEXE](GUID-B5FD3049-94D2-4527-97A4-46F9BABE7EB2.htm " Determines whether centerlines extend automatically from new center marks.
  		")
