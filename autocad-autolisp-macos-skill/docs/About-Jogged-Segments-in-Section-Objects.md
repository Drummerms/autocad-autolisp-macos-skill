# About Jogged Segments in Section Objects

You can create a section plane that has multiple segments (jogs). You can also add a jog to an existing section plane.

![](../images/GUID-D6EECFF1-6FA8-4A06-A0E7-E0727C4EFF90-low.png)

A jog that is added to an existing section object creates a segment that is perpendicular to the selected segment. Its viewpoint
is oriented in the direction set by the Direction grip. The Nearest object snap is temporarily turned on to help you place
the jogs on a section.

You cannot add jogs to the side or back lines of the section object.

After adding jogs, you can reposition and resize the jogged sections by dragging the section object grips.
