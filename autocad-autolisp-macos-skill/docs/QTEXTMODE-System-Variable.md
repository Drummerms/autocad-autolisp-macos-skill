# QTEXTMODE (System Variable)

Controls how text is displayed.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | Turns off Quick Text mode; displays characters |
| 1 | Turns on Quick Text mode; displays a box in place of text |
