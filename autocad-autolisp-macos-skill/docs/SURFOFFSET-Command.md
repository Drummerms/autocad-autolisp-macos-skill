# SURFOFFSET (Command)

Creates a parallel surface a specified distance from the original surface.

## Access Methods

*Tool Sets*:
Modeling tab ![](../images/ac.menuaro.gif) Surface panel ![](../images/ac.menuaro.gif) Surface Creation drop-down ![](../images/ac.menuaro.gif) Surface Offset.
![](../images/GUID-8EF587F6-00BA-4CDA-88FE-B51624F160AF-low.png)

*Menu*:
Draw ![](../images/ac.menuaro.gif) 3D Modeling ![](../images/ac.menuaro.gif) Surfaces ![](../images/ac.menuaro.gif) Offset.

## Summary

Reverse the direction of the offset with the Flip Direction option.

![](../images/GUID-23D7A56B-7860-4E82-8352-C253021E4993-low.gif)

## List of Prompts

The following prompts display.

Specify Offset Distance
:   Specifies the distance between the offset surface and the original surface.

Flip Direction
:   Reverses the offset direction shown by the arrows.

    ![](../images/GUID-E8130230-ED8B-4465-91E8-4BC60DDC5328-low.png)

Both Sides
:   Offsets the surface in both directions (creates two new surfaces in instead of one).

    ![](../images/GUID-CB6D02C7-5EFE-4E2A-8EFD-3999A3F9A4E2-low.png)

Solid
:   Creates a solid from the offset. This is similar to the THICKEN command.

    ![](../images/GUID-970E1EDC-2486-4081-A7BF-80E3CCD26C88-low.png)

Connect
:   Connects multiple offset surfaces, if the original surfaces are connected.

    ![](../images/GUID-CFB9B666-2B42-4DBC-A583-122255A817F9-low.png)

Expression
:   Enter a formula or equation to specify the distance of the surface offset.
