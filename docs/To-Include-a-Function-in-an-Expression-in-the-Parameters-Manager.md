# To Include a Function in an Expression in the Parameters Manager

1. Click Parametric tab![](../images/ac.menuaro.gif)Manage panel![](../images/ac.menuaro.gif)Parameters Manager.![](../images/GUID-6C8237C9-7325-4FDE-9739-185DA7AE46E7-low.png)
2. In the Expression column, double-click the box of the variable to which you want to add the function.
3. Right-click the same box, and click Expressions.
4. Choose the function to insert.
