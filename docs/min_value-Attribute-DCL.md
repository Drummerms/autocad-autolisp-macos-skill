# min\_value Attribute (DCL)

Specifies the lower range of values that a slider returns.

*Supported Platforms:* Windows, Mac OS, and Web

```
min_value = integer;
```

Default minimum value is
0. This value must be a signed, 16-bit integer no less than -32768. The
min\_value can be greater than the
max\_value.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
