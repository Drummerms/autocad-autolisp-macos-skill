# DIMANNO (System Variable)

Indicates whether or not the current dimension style is annotative.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | Nonannotative |
| 1 | Annotative |
