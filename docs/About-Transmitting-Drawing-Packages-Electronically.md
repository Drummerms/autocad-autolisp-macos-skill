# About Transmitting Drawing Packages Electronically

Package and deliver groups of drawings and related and supporting files in electronic transmittal packages.

Packages can be created as:

* a file folder
* a ZIP file

Transmittal packages may be received by team members using earlier versions of AutoCAD. You can use eTransmit to convert
files to previous DWG file formats.
