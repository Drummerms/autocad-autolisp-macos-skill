# To Space Parallel Linear and Angular Dimensions Automatically

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Dimension panel ![](../images/ac.menuaro.gif) Dimension, Dimspace.
   ![](../images/GUID-1FA9FC3B-1BB8-4F4D-9014-8B4B66AEFC4D-low.png)
2. Select the dimension that you want to use as the base dimension when equally spacing dimensions.
3. Select the next dimension to equally space.
4. Continue to select dimensions and then press Enter.
5. Enter
   *a* (Auto) and press Enter.
