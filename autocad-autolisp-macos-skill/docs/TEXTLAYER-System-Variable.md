# TEXTLAYER (System Variable)

Specifies a default layer for new text and multiline text objects in the current drawing.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | use current |

Values other than the "." (use current) value override the current layer (CLAYER system variable).

NOTE: If you specify a currently nonexistent layer as the default layer for new text and multiline text objects, it will be created
automatically when needed.
