# To Specify the Display Style for Drawing Units

1. Click Format ![](../images/ac.menuaro.gif) Units  .![](../images/GUID-EE300CC0-76B4-494E-B511-C48189B99F65-low.png)At the command prompt, enter UNITS.
2. As you change unit settings, you can see the Sample Output change:
   * Under Length, select a unit type and level of precision for linear drawing units.
   * Under Insertion Scale, select the unit that you want used to scale blocks, images, or other content inserted into the drawing.
   * Under Angle, select an angle type, precision, and direction for positive rotation for angular drawing units. Click Direction
     to set the reference direction for angles.
