# To Create a New Subset

1. In the Sheet Set Manager, right-click the sheet set node (at the top of the list), or an existing subset node.
2. Choose New subset.
3. Select the new subset, if necessary.
4. In the Details panel, expand Subset, enter the name and other properties of the new subset.

NOTE:To remove a subset, first remove or relocate any sheets it contains, then right-click the subset and choose Remove Subset.
