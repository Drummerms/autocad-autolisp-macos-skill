# SHEETSET (Command)

Opens the Sheet Set Manager.

## Access Methods

*Menu:* Window ![](../images/ac.menuaro.gif) Sheet Set Manager

*Keyboard:* Cmd-8

## Summary

The
[Sheet Set Manager](Sheet-Set-Manager-Palette.md) organizes, displays, and manages
*sheet sets*, a named collection of drawing sheets. Each
*sheet* in a sheet set is a layout in a drawing (DWG) file.
