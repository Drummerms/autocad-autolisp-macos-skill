# About the Tool Sets

The Tool Sets provide efficient access to AutoCAD commands.

## Tool Sets

Select the Tool Sets tab to display a set of commands based on your current tasks. For example, clicking the
*Modeling* tab replaces the commands in the Tool Set with commands associated with 3D modeling.

Cmd-1 turns the Tool Sets on and off.

![](../images/GUID-12F8A6CF-C95D-441F-AB9B-4C519700341A-low.png)

## Tool Set Panels

The tools on the Tool Sets are organized into
*tool panels*.

* Click
  ![](../images/GUID-78DED619-51B3-49C1-AD71-F928B5F69C0A-low.png) to display the list of commands available for a standard tool panel. Check the commands you want to display on the tool panel.

  NOTE:For a custom panel, the interface for adding and removing commands opens.
* Click
  ![](../images/GUID-9174311E-7DC8-49C8-B39F-991EC3DAD0E2-low.png) (located at the bottom) to create a custom tool panel.
* Click
  ![](../images/GUID-6721239D-925C-4C53-84C8-846E2D234300-low.png) (located at the bottom) to change the panel order.

## Tool Drop-downs

Some of the tools on the Tool Sets have a drop-down indicator.

![](../images/GUID-5C196AD7-7936-4F2A-81AC-B1F5F3BCB9C1-low.png)

Click and hold the drop-down to display the commands.
