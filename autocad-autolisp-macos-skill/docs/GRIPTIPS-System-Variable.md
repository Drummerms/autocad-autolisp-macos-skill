# GRIPTIPS (System Variable)

Controls the display of grip tips when the cursor hovers over grips on dynamic blocks and custom objects that support grip
tips.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Turns off the display of grip tips. |
| 1 | Turns on the display of grip tips. |
