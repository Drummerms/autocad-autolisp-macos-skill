# To Publish Multiple Layouts in a Drawing

1. Click Publish.
   ![](../images/GUID-4C887873-7EE8-497C-B87D-0D8DFF393268-low.png)
2. In the Batch Publish dialog box, Click Add (+) to add drawings.
3. In the Select Drawings dialog box, select layouts from the Add Content drop-down list.

   The selected sheet is added to the Batch Publish dialog box for publishing.

   NOTE:Selecting a subset adds all the sheets in that subset to the Batch Publish dialog box. Selecting the sheet set name adds all
   sheets in the sheet set to the dialog box.
4. In the Batch Publish dialog box, right-click the Sheet Name list to specify the order in which the sheets should be printed.
5. In the Publish To list, select Printer From Page Setup to use the page setups defined in each drawing file.

   Alternatively, select PDF to publish to a PDF file.
6. Select the required output options.
7. Click Publish.
