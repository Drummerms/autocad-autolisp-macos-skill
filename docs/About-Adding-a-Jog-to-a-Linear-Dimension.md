# About Adding a Jog to a Linear Dimension

Jog lines are used to represent a dimension value that does not display the actual measurement in a linear dimension. Typically,
the actual measurement value of the dimension is smaller than the displayed value.

The jog is made up of two parallel lines and a cross line that forms two 40-degree angles. The height of the jog is determined
by the text height defined for the dimension style (DIMTXT) multiplied by the value of the *jog height factor*, which can be set in the Dimension Style Manager, Lines and Arrows tab.

![](../images/GUID-C6270FD1-DD07-418A-8CCA-08289C366E5D-low.png)

Once you add a jog to a linear dimension, you can position it by using grips. To reposition the jog, select the dimension
and then select the grip. Move the grip to another point along the dimension line.
