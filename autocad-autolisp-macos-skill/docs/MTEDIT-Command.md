# MTEDIT (Command)

Edits multiline text.

## Access Methods

Pointing device: Double-click a multiline text object.

## Summary

Displays either the Text Editor visor and the In-Place Text Editor to modify the formatting or content of the selected multiline
text object.

The [In-Place Text Editor](In-Place-Text-Editor.md) is displayed.
