# USERR1-5 (System Variable)

Provides storage and retrieval of real numbers.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 0.0000 |

There are five system variables: USERR1, USERR2, USERR3, USERR4, and USERR5.
