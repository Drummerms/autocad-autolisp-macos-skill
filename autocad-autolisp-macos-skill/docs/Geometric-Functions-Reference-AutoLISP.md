# Geometric Functions Reference (AutoLISP)

The following table provides summary descriptions of the AutoLISP geometric functions.

| Geometric functions | | Platforms | | | | |
| Windows | | Mac OS | | Web |
| Function | Description | AutoCAD | AutoCAD LT | AutoCAD | AutoCAD LT | AutoCAD |
| [(angle pt1 pt2)](angle-AutoLISP.md) | Returns an angle in radians of a line defined by two endpoints | ✓ | ✓ | ✓ | -- | ✓ |
| [(distance pt1 pt2)](distance-AutoLISP.md) | Returns the 3D distance between two points | ✓ | ✓ | ✓ | -- | ✓ |
| [(inters pt1 pt2 pt3 pt4 [onseg])](inters-AutoLISP.md) | Finds the intersection of two lines | ✓ | ✓ | ✓ | -- | ✓ |
| [(osnap pt mode)](osnap-AutoLISP.md) | Returns a 3D point that is the result of applying an Object Snap mode to a specified point | ✓ | ✓ | ✓ | -- | ✓ |
| [(polar pt ang dist)](polar-AutoLISP.md) | Returns the UCS 3D point at a specified angle and distance from a point | ✓ | ✓ | ✓ | -- | ✓ |
| [(textbox elist)](textbox-AutoLISP.md) | Measures a specified text object, and returns the diagonal coordinates of a box that encloses the text | ✓ | ✓ | ✓ | -- | ✓ |
