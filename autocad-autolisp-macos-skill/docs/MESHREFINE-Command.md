# MESHREFINE (Command)

Multiplies the number of faces in selected mesh objects or faces.

## Access Methods

*Tool Set*:
Modeling tab ![](../images/ac.menuaro.gif) Mesh panel ![](../images/ac.menuaro.gif) Mesh Refine.
![](../images/GUID-47DB51A9-5187-42BF-BEBF-1057FFD6E46B-low.png)

*Menu*:
Modify
![](../images/ac.menuaro.gif) Mesh Editing
![](../images/ac.menuaro.gif) Refine Mesh.

## Summary

Refining a mesh object increases the number of editable faces, providing additional control over fine modeling details. To
preserve program memory, you can refine specific faces instead of the entire object.

![](../images/GUID-A992966D-1A6A-41E9-AB1D-467194B0ECEA-low.gif)

Refining an object resets the smoothing level that was assigned to the object to 0 (zero). This level of smoothness becomes
the new baseline for the object. That is, the smoothness level can no longer be decreased beyond that level. Refining a subobject
does not reset the smoothing level.

## List of Prompts

The following prompts are displayed.

Mesh object or face subobjects to refine
:   Specifies 3D mesh objects or mesh faces to be refined. Press Ctrl+click to isolate a specific face.
