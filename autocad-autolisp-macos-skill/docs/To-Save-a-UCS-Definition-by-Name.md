# To Save a UCS Definition by Name

1. Right-click the UCS icon, and click Named UCS![](../images/ac.menuaro.gif)Save.
2. Enter a name.

   You can use up to 255 characters, including letters, digits, and the special characters dollar sign ($), hyphen (-), and underscore
   (\_).
