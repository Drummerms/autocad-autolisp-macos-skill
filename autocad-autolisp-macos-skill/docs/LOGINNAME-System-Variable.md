# LOGINNAME (System Variable)

Displays the current user’s login name and is saved with the file properties statistics of DWG and related files.

(Read-only)

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Not-saved |
| Initial value: | Varies |
