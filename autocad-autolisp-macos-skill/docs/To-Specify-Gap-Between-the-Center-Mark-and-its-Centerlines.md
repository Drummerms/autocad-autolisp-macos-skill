# To Specify Gap Between the Center Mark and its Centerlines

Control the gap between the center mark and its centerlines.

## Specify the Default Gap Between a New Center Mark and its Centerlines

1. At the Command prompt, enter CENTERCROSSGAP.
2. Specify a new value. You can enter any of the following:
   * Relative value - any positive real number followed by
     *x*.
   * Absolute value - any positive real number
   * *ByLineType*

## Specify the Gap Between an Existing Center Mark and its Centerlines

1. If the Properties Inspector is not displayed, select Modify ![](../images/ac.menuaro.gif) Properties.
2. Select one or more center marks.
3. In the Properties Inspector ![](../images/ac.menuaro.gif) All tab, under Geometry, select Cross Gap.
4. Specify a new value.
