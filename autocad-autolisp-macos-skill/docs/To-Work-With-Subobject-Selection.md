# To Work With Subobject Selection

## Limit Selection to Vertex Subobjects

1. Right-click the drawing area ![](../images/ac.menuaro.gif) Subobject Selection Filter ![](../images/ac.menuaro.gif) Vertex.
2. When you select a 3D solid or mesh subobject, only the vertex can be selected.

## Select Multiple Faces, Edges, or Vertices

* Press Ctrl+click as you select faces, edges, or vertices.

  NOTE: If a subobject selection filter is active, you do not need to press Ctrl before clicking a face, edge, or vertex.

## Remove a Subobject From a Selection Set

* Press and hold Ctrl+Shift. Click a selected face, edge, or vertex.

  The grip, if displayed, changes from red to blue.

  NOTE: If a subobject selection filter is active, you do not need to press Ctrl before clicking a face, edge, or vertex.
