# QTEXT (Command)

Controls the display and plotting of text and attribute objects.

## Summary

When QTEXT (Quick Text) is on, each text and attribute object is displayed as a bounding box around the text object. Turning
QTEXT mode on reduces the time it takes the program to redraw and regenerate drawings that contain many text objects.
