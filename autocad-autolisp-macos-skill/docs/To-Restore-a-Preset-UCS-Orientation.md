# To Restore a Preset UCS Orientation

1. Click
   Tools menu ![](../images/ac.menuaro.gif) Named UCS.
2. In the UCS dialog box, Orthographic tab, select a UCS orientation from the list.
3. Click Options > Set Current.
4. Click OK.

   The UCS changes to the selected option.
