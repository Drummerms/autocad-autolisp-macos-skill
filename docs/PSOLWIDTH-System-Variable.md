# PSOLWIDTH (System Variable)

Controls the default width for a swept solid object created with the POLYSOLID command.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 0.2500 (imperial) or 5.0000 (metric) |

The value reflects the last entered height value when using the [POLYSOLID](POLYSOLID-Command.md) command. You cannot enter 0 as the value.
