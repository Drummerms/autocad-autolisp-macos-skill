# LIGHTSINBLOCKS (System Variable)

Controls whether lights contained in blocks are used when rendering.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

By default, this system variable is turned on. When this system variable is off, lights in blocks do not affect 3D objects
in the current viewport when rendering. Some previous versions of AutoCAD did not support rendering lights in blocks.

| 0 (Off) | Lights in blocks are disabled |
| 1 (On) | Lights in blocks are enabled |
