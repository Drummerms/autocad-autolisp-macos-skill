# To Work with Custom Paper Sizes

Add, copy, edit, or delete a custom paper size.

1. Click Plot.
   ![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png)
2. Click Paper Size drop-down and select Manage Custom Sizes.

## Add a Custom Paper Size

1. Select one of the following:
   * ![](../images/GUID-40517BFA-4A82-4F1B-9580-59197EDD94CD-low.png) ![](../images/ac.menuaro.gif) New Paper Size
   * ![](../images/GUID-40517BFA-4A82-4F1B-9580-59197EDD94CD-low.png) ![](../images/ac.menuaro.gif) Select From List, then click a paper size from the list of available paper sizes.
2. Edit the name of the new paper size.
3. Enter the paper height and width.
4. Enter the margins or select predefined margins from the list of printers in the Non-Printable Area drop-down.
5. Click OK.

## Copy a Custom Paper Size

1. Select the paper size you want to copy.
2. Click Duplicate.
3. Edit values as needed.
4. Click OK.

## Edit a Custom Paper Size

1. Select the paper size you want to edit.
2. Edit values as needed.
3. Click OK.

## Delete a Custom Paper Size

1. Select the paper size you want to delete.
2. Click
   ![](../images/GUID-36652FED-6611-41B0-95B6-14B33F2D8DB1-low.png).
3. Click OK.
