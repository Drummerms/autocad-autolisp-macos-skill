# DIVMESHWEDGELENGTH (System Variable)

Sets the number of subdivisions for the length of a mesh wedge along the X axis.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 4 |

![](../images/GUID-517A7322-D49D-4C1A-ADA9-F8C20DFA1E5A-low.png)

This setting affects new mesh wedges that are created with the [MESH](MESH-Command.md) command.
