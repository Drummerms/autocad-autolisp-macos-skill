# OFFSET (Command)

Creates concentric circles, parallel lines, and parallel curves.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Offset.
![](../images/GUID-9F7D4463-198A-410F-8D3A-591E48B9288E-low.png)

*Menu*:
Modify ![](../images/ac.menuaro.gif) Offset.

You can offset an object at a specified distance or through a point. After you offset objects, you can trim and extend them
as an efficient method to create drawings containing many parallel lines and curves.

![](../images/GUID-4FF71583-D4CE-4346-A0AC-74C6EC5E90E1-low.gif)

![](../images/GUID-78D8C435-851E-491D-9821-727EAF755268-low.png)

The OFFSET command repeats for convenience. To exit the command, press Enter.

The following prompts are displayed.

## Offset Distance

Creates an object at a specified distance from an existing object.

![](../images/GUID-9451F109-81A9-4694-8D32-2EE90673973D-low.png)

Exit
:   Exits the OFFSET command.

Multiple
:   Enters the Multiple offset mode, which repeats the offset operation using the current offset distance.

Undo
:   Reverses the previous offset.

## Through

Creates an object passing through a specified point.

NOTE: For best results when you offset a polyline with corners, specify the through point near the midpoint of a line segment,
not near a corner.

![](../images/GUID-F96C9E0E-6FA1-4554-95DC-E7EB38B8DB30-low.png)

* Exit
* Multiple
* Undo

## Erase

Erases the source object after it is offset.

## Layer

Determines whether offset objects are created on the current layer or on the layer of the source object.
