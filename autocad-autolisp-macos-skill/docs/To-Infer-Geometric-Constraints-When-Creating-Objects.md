# To Infer Geometric Constraints When Creating Objects

1. Click
   Tools menu ![](../images/ac.menuaro.gif) Parametric ![](../images/ac.menuaro.gif) Constraint Settings.
2. In the Constraint Settings dialog box, Geometric tab, select Geometric Constraints to infer.

NOTE:Alternatively, you can click the Infer Constraints button on the status bar.
