# To Assign a Fill Style

You can assign a variety of fill style options when plotting wide polylines, donuts, objects hatched with a solid fill, and
solids.

1. Click
   File menu ![](../images/ac.menuaro.gif) Plot Styles.
2. Double-click a plot style (STB or CTB) file in Finder to start the Plot Style Table Editor
3. In the Plot Style Table Editor, in the Properties tab, click the Fill Style field for the plot style you want to change.
4. Select a fill style from the drop-down list.

NOTE:If you assign a fill style, the fill style overrides the object's fill style at plot time.
