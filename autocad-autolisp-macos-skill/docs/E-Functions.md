# E Functions

#### Topics in this section

* [entdel (AutoLISP)](entdel-AutoLISP.md)

  Deletes objects (entities) or restores previously deleted objects
* [entget (AutoLISP)](entget-AutoLISP.md)

  Retrieves an object's (entity's) definition data
* [entlast (AutoLISP)](entlast-AutoLISP.md)

  Returns the name of the last nondeleted main object (entity) in the drawing
* [entmake (AutoLISP)](entmake-AutoLISP.md)

  Creates a new entity in the drawing
* [entmakex (AutoLISP)](entmakex-AutoLISP.md)

  Makes a new object or entity, gives it a handle and entity name (but does not assign an owner), and then returns the new
  entity name
* [entmod (AutoLISP)](entmod-AutoLISP.md)

  Modifies the definition data of an object (entity)
* [entnext (AutoLISP)](entnext-AutoLISP.md)

  Returns the name of the next object (entity) in the drawing
* [entsel (AutoLISP)](entsel-AutoLISP.md)

  Prompts the user to select a single object (entity) by specifying a point
* [entupd (AutoLISP)](entupd-AutoLISP.md)

  Updates the screen image of an object (entity)
* [eq (AutoLISP)](eq-AutoLISP.md)

  Determines whether two expressions are identical
* [equal (AutoLISP)](equal-AutoLISP.md)

  Determines whether two expressions are equal
* [eval (AutoLISP)](eval-AutoLISP.md)

  Returns the result of evaluating an AutoLISP expression
* [exit (AutoLISP)](exit-AutoLISP.md)

  Forces the current application to quit
* [exp (AutoLISP)](exp-AutoLISP.md)

  Returns the constant
  e (a real number) raised to a specified power (the natural antilog)
* [expand (AutoLISP)](expand-AutoLISP.md)

  Allocates additional memory for AutoLISP
* [expt (AutoLISP)](expt-AutoLISP.md)

  Returns a number raised to a specified power
