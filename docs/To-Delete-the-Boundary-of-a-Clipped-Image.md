# To Delete the Boundary of a Clipped Image

1. Click Modify ![](../images/ac.menuaro.gif) Clip ![](../images/ac.menuaro.gif) Image menu  .At the Command prompt, enter IMAGECLIP.
2. Select the clipped image with the boundary you want to delete.
3. Enter*d* (Delete).

   The boundary of the clipped image is deleted, and the original boundary of the image is restored.
