# 3DSWIVEL (Command)

Changes the target of the view in the direction that you drag.

## Access Methods

*Menu*:
View
![](../images/ac.menuaro.gif) Camera
![](../images/ac.menuaro.gif) Swivel.

Shortcut menu: Start any 3D navigation command, right-click in the drawing area, and click Other Navigation Modes ![](../images/ac.menuaro.gif) Swivel (5).

## Summary

Simulates panning with a camera in the direction that you drag. The target of the view changes. You can swivel the view along
the
*XY* plane or along the
*Z* axis.
