# tabs Attribute (DCL)

Specifies the placement of tabs in character width units.

*Supported Platforms:* Windows, Mac OS, and Web

```
tabs = "string";
```

Possible value is a quoted string containing integers or floating-point numbers, separated by spaces (no default). These
values are used for vertically aligning columns of text in a
popup\_list or
list\_box.

For example, the following code specifies a tab stop at every 8 characters.

```
tabs = "8 16 24 32";
```

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
