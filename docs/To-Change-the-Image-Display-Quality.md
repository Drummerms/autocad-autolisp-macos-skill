# To Change the Image Display Quality

1. Click Modify ![](../images/ac.menuaro.gif) Object ![](../images/ac.menuaro.gif) Image ![](../images/ac.menuaro.gif) Quality.At the Command prompt, enter IMAGEQUALITY.
2. Enter *d* (Draft) or *h* (High).

   Images are displayed at the specified quality.
