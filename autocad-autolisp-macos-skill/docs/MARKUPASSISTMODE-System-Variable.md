# MARKUPASSISTMODE (System Variable)

Controls whether identified markups are highlighted.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | Off. Identified markups are not highlighted. |
| 1 | On. Identified markups are highlighted. |
