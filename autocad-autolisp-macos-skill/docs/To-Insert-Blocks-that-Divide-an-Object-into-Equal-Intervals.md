# To Insert Blocks that Divide an Object into Equal Intervals

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Point drop-down ![](../images/ac.menuaro.gif) Divide.
   ![](../images/GUID-6EC11511-8D3E-4D01-92C3-883E60D59ED7-low.png)
2. Select a line, arc, circle, ellipse, polyline, or spline.
3. Enter
   *b* (Block).
4. Enter the name of the block that you want to insert.
5. Enter
   *y* (Yes) to align the blocks with the selected object. Enter
   *n* (No) to use a rotation angle of 0 degrees.
6. Enter the number of intervals between the blocks.
