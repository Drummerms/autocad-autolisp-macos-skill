# To Add Annotation Scales to an Annotative Object

1. Select an annotative object.
2. Right-click in the drawing area and choose Annotative Object Scale ![](../images/ac.menuaro.gif) Add/Delete Scales.
3. Click Add.
4. Select the scale add.

   NOTE:Hold Ctrl to select more than one scale.
5. Click OK twice to return to the drawing.

## Automatically Add Annotation Scales

* On the status bar, click Automatically Add Scales to Annotative Objects When the Annotation Scale Changes ( ![](../images/GUID-E5C92205-BFB7-4A05-8BB7-308331E8B554-low.png) Find) .

NOTE:Adding scales automatically to all annotation objects can increase the overall drawing file size. To keep the drawing file
size down, only add the scales to the annotative objects that need multiple representations.
