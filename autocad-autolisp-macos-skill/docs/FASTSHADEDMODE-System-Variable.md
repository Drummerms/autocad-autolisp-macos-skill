# FASTSHADEDMODE (System Variable)

Specifies whether the cross platform 3D graphics system is turned on or off.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

The cross platform 3D graphics system leverages all the power of modern GPUs and multi-core CPUs to offer a smooth navigation
experience for much larger drawings than in prior releases.

NOTE:This feature only affects the Shaded, Shaded with Edges, and Wireframe visual styles.

| Value | Description |
| 0 or OFF | Turns off the cross platform 3D graphics system |
| 1 or ON | Turns on the cross platform 3D graphics system |
