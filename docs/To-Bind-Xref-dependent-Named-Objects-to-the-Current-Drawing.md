# To Bind Xref-dependent Named Objects to the Current Drawing

1. Click Modify menu![](../images/ac.menuaro.gif)Object![](../images/ac.menuaro.gif)External Reference![](../images/ac.menuaro.gif)Bind.At the Command prompt, enter XBIND.
2. In the Xbind dialog box, click the plus (+) sign next to an external reference.

   The five types of named object definitions (Block, Dimstyle, Layer, Linetype, and Textstyle) are listed.
3. Click the plus (+) sign of one of the definition types.

   The names of the definition table entries are listed.
4. Select a named object definition and click Add.

   The named object definition is listed under Definitions to Bind.
5. If necessary, repeat steps 3 and 4.
