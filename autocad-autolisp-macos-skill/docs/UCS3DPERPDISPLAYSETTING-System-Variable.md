# UCS3DPERPDISPLAYSETTING (System Variable)

Displays the UCS icon when perspective view is on and a 3D visual style is current.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Off. The UCS icon is not displayed when perspective is turned on and a 3D visual style is current. |
| 1 | On. The UCS icon is displayed when perspective is turned on and a 3D visual style is current. |

NOTE:The [UCSICON](UCSICON-Command.md) command must also be set to ON to display the UCS icon.
