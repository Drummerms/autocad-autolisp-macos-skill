# Added Layer Option to CENTERLINE, CENTERMARK, and MLEADER

In this release, we integrated the CENTERLAYER system variable into the workflows of creating new centerlines, center marks,
and multileaders.

The value of the CENTERLAYER system variable can be changed at the Command prompt, using the new Layer option of the CENTERLINE,
CENTERMARK, and MLEADER commands.
