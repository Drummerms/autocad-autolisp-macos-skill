# COUNTAREA (Command)

Defines the area to count the instances of an object or block.

NOTE:This command is available only during an active count.

Specify a rectangular or polygonal selection area, the entire model space, or select a valid object as a count area. A valid
boundary object must be a closed polyline that consists of line segments and does not intersect with itself.

The count area defined in the previous count session can also be used in the current count.

The following prompts are displayed.

## First Corner

Specifies the first corner of the rectangular count area.

Opposite Corner
:   Specifies the diagonally opposite corner of the rectangular count area.

## Current Area

Specifies the last area defined in the previous count session.

## Entire Model Space

Specifies the entire model space as the count area boundary.

## Object

Specifies an existing closed polyline as the count area boundary.

## Polygonal

Specifies a polygonal count area boundary.

Start Point
:   Specifies the starting point of the polygonal count area.

Next Point
:   Specifies the next point of the polygonal count area. Press Enter to close the polygonal area.
