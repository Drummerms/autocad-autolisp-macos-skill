# To Insert a Field in a Cell

1. Click inside a cell.
2. On the Table Cell visor, click Insert Field.
   ![](../images/GUID-A9ECC908-6398-47EF-8DB8-810BF2C77569-low.png)
3. In the Insert Field dialog box, select a category and a field name.

   ![](../images/GUID-8DCCACA2-AC6D-4830-9E2A-20AF1EA6F301-low.png)
4. Specify the options available for the selected field.
5. Click Choose.
