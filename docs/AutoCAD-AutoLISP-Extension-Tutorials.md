# AutoCAD AutoLISP Extension Tutorials

#### Topics in this section

* [Tutorial: Installing and Configuring the AutoLISP Extension (AutoLISP/VS Code)](Tutorial-Installing-and-Configuring-the-AutoLISP-Extension-AutoLISP-VS-Code.md)

  After you have installed Microsoft Visual Studio (VS) Code, you can install and configure the AutoCAD AutoLISP Extension
  which will allow you to manage and debug custom AutoLISP programs.
* [Tutorial: Getting Started with the AutoLISP Extension (AutoLISP/VS Code)](Tutorial-Getting-Started-with-the-AutoLISP-Extension-AutoLISP-VS-Code.md)

  The AutoCAD AutoLISP Extension for Microsoft Visual Studio (VS) Code allows you to write custom routines with the AutoLISP
  programming language that can be used to automate workflows and extend the functionality of AutoCAD.
* [Tutorial: Debugging LSP Files with the AutoLISP Extension (AutoLISP/VS Code)](Tutorial-Debugging-LSP-Files-with-the-AutoLISP-Extension-AutoLISP-VS-Code.md)

  After you create an AutoLISP (LSP) file and add code statements to the file, you can connect Visual Studio Code to AutoCAD
  and step through your custom functions using many of the debug tools that Visual Studio Code offers.
* [Tutorial: Formatting LSP Files with the AutoLISP Extension (AutoLISP/VS Code)](Tutorial-Formatting-LSP-Files-with-the-AutoLISP-Extension-AutoLISP-VS-Code.md)

  The AutoLISP Extension provides tools to format select AutoLISP code statements, define the indent and alignment of new statements
  as they are typed, and mark a statement as a comment.
