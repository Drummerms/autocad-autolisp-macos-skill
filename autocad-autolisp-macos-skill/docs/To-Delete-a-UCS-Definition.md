# To Delete a UCS Definition

1. Click
   Tools menu ![](../images/ac.menuaro.gif) Named UCS.
2. In the UCS dialog box, Named tab, select the UCS definition you want to delete.
3. Click Delete.

   You cannot delete the current UCS or a UCS with the default name UNNAMED.
4. Click OK.
