# Migration

#### Topics in this section

* [About Migrating Custom Files and Settings](About-Migrating-Custom-Files-and-Settings.md)

  Seamlessly migrate your custom files and settings from a previous release to the current release. The migration utility starts
  automatically when you first open AutoCAD.
* [About Exporting and Importing Custom Settings](About-Exporting-and-Importing-Custom-Settings.md)

  Custom settings can be transferred between computers with the same release of the product installed. You can also export the
  settings to create a backup of custom settings for the product.
