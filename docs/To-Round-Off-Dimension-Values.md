# To Round Off Dimension Values

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Dimension panel ![](../images/ac.menuaro.gif) Dimension Style.
   ![](../images/GUID-89D5BC05-30EE-4627-B80B-8680F08A01A9-low.png)
2. In the Dimension Style Manager, select the style you want to change. Click Options
   ![](../images/GUID-0888F513-2A7D-45B1-AD99-683EBA803A86-low.png) ![](../images/ac.menuaro.gif) Modify.
3. In Modify Dimension Style dialog box, Primary Units tab, under Linear Dimensions, enter the round-off value.
4. Click OK.
5. Click Close to exit the Dimension Style Manager.
