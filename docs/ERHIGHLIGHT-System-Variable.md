# ERHIGHLIGHT (System Variable)

Controls whether reference names or reference objects are highlighted when their counterparts are selected in the Reference
Manager palette or in the drawing window.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Referenced objects are not highlighted. |
| 1 | Referenced objects are highlighted. |
