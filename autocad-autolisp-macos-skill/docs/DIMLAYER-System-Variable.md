# DIMLAYER (System Variable)

Specifies a default layer for new dimensions.

![](../images/GUID-507AE55C-B1EB-46BC-99F8-7080DAE7F47C-low.png) Find

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | use current |

Values other than the "." (use current) value override the current layer (CLAYER).
