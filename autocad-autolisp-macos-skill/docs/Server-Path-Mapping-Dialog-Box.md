# Server Path Mapping Dialog Box

Configures server path mapping between Windows and Mac

XREFPATHMAP (Command)

Using this dialog you can open drawings with xrefs from PC to Mac and vice versa by mapping the server path location in the
dialog

![](../images/GUID-AA5A9625-DE6B-4314-A4BD-8B476F20C2BE-low.png)

*List of Options*

The following options are displayed:

*+*

Opens the New Server Path Mapping dialog.

*-*

Removes an existing server path mapping.

*Edit*

Edits the server path mapping.
