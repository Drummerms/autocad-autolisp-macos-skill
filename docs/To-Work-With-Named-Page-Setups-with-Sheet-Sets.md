# To Work With Named Page Setups with Sheet Sets

You can use named page setups to specify the same output options for all sheets in a sheet set.

## Apply a Named Page Setup to a Sheet

1. Click
   Window ![](../images/ac.menuaro.gif) Sheet Set Manager.
2. In the Sheet Set Manager, select a sheet set.
3. Right-click the sheet set. Click New Sheet.
4. In the New Sheet dialog box, enter the sheet information
5. (Optional) Specify a different template.
6. Click Create.
7. In the Save Drawing As dialog box, click Save.
8. In the Sheet Set Manager, right-click the new sheet. Click Open.
9. Click
   File ![](../images/ac.menuaro.gif) Page Setup Manager.
10. In the Page Setup Manager, under Page Setups, right-click a named page setup. Click Set Current.

    Named page setups are not enclosed in asterisks.
11. Click Close.

## Apply a Named Page Setup as an Override When Publishing Sheets

1. Click
   Window ![](../images/ac.menuaro.gif) Sheet Set Manager.
2. In the file navigation dialog box, select a sheet set. Click Open.
3. In the Sheet Set Manager, select a sheet set.
4. In the Details panel, expand Sheet Set, under Page Setup Overrides Fie, specify a drawing template file containing the page
   setup overrides.
5. Click Open.
