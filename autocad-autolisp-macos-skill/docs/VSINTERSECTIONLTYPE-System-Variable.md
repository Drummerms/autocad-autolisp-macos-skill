# VSINTERSECTIONLTYPE (System Variable)

Sets the linetype for intersection lines in the visual style applied to the current viewport.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

The range is 1 to 11.

| 1 | Solid |
| 2 | Dashed |
| 3 | Dotted |
| 4 | Short Dash |
| 5 | Medium Dash |
| 6 | Long Dash |
| 7 | Double Short Dash |
| 8 | Double Medium Dash |
| 9 | Double Long Dash |
| 10 | Medium Long Dash |
| 11 | Sparse Dot |

NOTE:Existing visual styles are not changed when you enter a new value for this system variable. Any new value entered for this
system variable temporarily creates an unsaved new visual style.
