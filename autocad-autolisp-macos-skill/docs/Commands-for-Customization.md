# Commands for Customization

These commands can be used to customize the user interface, automate tasks, and more.

#### Commands

* [APPLOAD](GUID-47621BB1-F29D-4A69-9C99-A6E1495FBA38.htm "Loads and unloads applications and defines which applications to load at startup.

  		")
* [COMPILE](GUID-54F4D0CE-EC71-42DA-A56E-469252287C12.htm "Compiles shape files and PostScript font files into SHX files.
  		")
* [CUI](GUID-7F8F4B26-EFAF-4033-B7B7-CA39FC4E104A.htm "Manages the customized user interface elements in the product.
  		")
* [DSETTINGS](GUID-22D029BC-354B-4D7D-B800-B5308C377D6C.htm " Sets grid and snap, polar and object snap tracking, object snap modes, Dynamic Input, and Quick Properties.
  		")
* [FILL](GUID-A09941AF-919C-4E72-96F4-3DF3C11922DE.htm "Controls the display of filled objects such as hatches, 2D solids, and wide polylines.

* [LINETYPE](GUID-1D3817A0-C5C7-4D52-99A9-F0922F09D4CD.htm "Loads, sets, and modifies linetypes.

  		")
* [REDEFINE](GUID-E2B53E4D-28AE-4F6E-96E4-6FABE3FF823C.htm "Restores AutoCAD internal commands overridden by UNDEFINE.
  		")
* [SCRIPT](GUID-DB55FE5C-6B51-40AE-AE3D-4C3A28ADC5D9.htm "Executes a sequence of commands from a script file.
  		")
* [SETVAR](GUID-01E38833-6D3A-4AA5-A446-C03B44E7AB56.htm "Lists or changes the values of system variables.
  		")
* [SHAPE](GUID-9750CDBE-028A-4E6B-AD51-ABB107737A63.htm "Inserts a shape from a shape file (SHX file) that has been loaded using LOAD.
  		")
* [TEXTSCR](GUID-5F0A5635-0DCB-4A88-BD8E-60EC665342CF.htm "Opens a text window that displays a history of prompts and command line entries for the current session.

  		")
* [TOOLSETSCLOSE](GUID-EE3F4DF0-77B6-4E9A-86CC-6CE3DFB910DB.htm "Closes the Tool Sets palette.
  		")
