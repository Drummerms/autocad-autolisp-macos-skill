# Commands for Displaying Coordinates

#### Commands

* [ID](GUID-F258AC00-5E9F-4B6B-A670-33F7708E3FB6.htm "Displays the UCS coordinate values of a specified location.
  		")
* [STATUSBAR](GUID-C42578AE-EEE5-4034-A35F-56BB5009F537.htm " Displays the status bar.
  		")
* [STATUSBARCLOSE](GUID-6DEA8FE5-C85B-4653-B772-59E108FB140A.htm " Hides the status bar.
  		")
* [UNITS](GUID-1DAE2080-84E4-413E-BB4E-F5D2A96CB14A.htm "Controls the precision and display formats for coordinates, distances, and angles.
  		")

#### System Variables

* [COORDS](GUID-91D3C93C-9C89-4397-B855-92AFD5228422.htm "Controls whether the cursor position on the status bar is updated continuously or at specific times only. It also controls the format in which the coordinates are displayed.
