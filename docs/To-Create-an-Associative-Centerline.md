# To Create an Associative Centerline

Creates an associative centerline between two line segments.

![](../images/GUID-5A5E171D-BD06-410F-BD32-06430823620A-low.png)

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Centerline drop-down ![](../images/ac.menuaro.gif) Centerline.
   ![](../images/GUID-E4521EDC-0353-4060-9278-EF7387188402-low.png)
2. Select first line (1).
3. Select second line (2).
