# TRIMMODE (System Variable)

Controls whether selected edges for chamfers and fillets are trimmed.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | Leaves selected edges intact |
| 1 | Trims selected edges to the endpoints of chamfer lines and fillet arcs |
