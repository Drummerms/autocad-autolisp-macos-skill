# About Importing and Exporting ACIS SAT Files

You can import geometry objects stored in SAT (ASCII) files using the ASCISIN command. It converts the model to a body object
or to 3D solids and regions if the body is a true solid or a true region.

You can export trimmed NURBS surfaces, regions, and 3D solids to an ACIS file in ASCII (SAT) format. Other objects, such
as lines and arcs, are ignored.
