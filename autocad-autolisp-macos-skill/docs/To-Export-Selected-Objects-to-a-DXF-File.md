# To Export Selected Objects to a DXF File

You can use the Write Block command to save selected objects to a DXF file.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Write Block.
   ![](../images/GUID-E84A38E9-381C-46EC-BE0C-F4E1728D05DA-low.png)
2. Select Selected Objects.
3. Under Source Objects, click Select Objects and select the objects to be included.
4. Define a base point.
5. Click Write Block.
6. Enter a file name and path.
7. Select a DXF format from the Files Format drop-down.
8. Click Save.
