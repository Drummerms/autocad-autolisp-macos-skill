# vlr-pers-release (AutoLISP/ActiveX)

Makes a reactor transient

*Supported Platforms:* Windows only; not available on Mac OS or Web

## Signature

```
(vlr-pers-release reactor)
```

*reactor*
:   *Type:* VLR object

    An object.

## Return Values

*Type:* VLR object or nil

The specified reactor object, if successful; otherwise
nil.
