# Import and Export Drawing Data

#### Topics in this section

* [About Importing and Exporting ACIS SAT Files](About-Importing-and-Exporting-ACIS-SAT-Files.md)
* [About Importing and Exporting DXF Files](About-Importing-and-Exporting-DXF-Files.md)

  DXF and DXB files are two types of drawing interchange files used to transfer data between various applications.
* [About Importing PDF Files](About-Importing-PDF-Files.md)

  You can import the geometry, fills, raster images, and TrueType text from a PDF file into the current drawing. The visual
  fidelity along with some properties such as PDF scale, layers, lineweights, and colors can be preserved.
* [About Exporting Drawing Files to PDF](About-Exporting-Drawing-Files-to-PDF.md)

  The Adobe® Portable Document Format (PDF) is a compressed electronic document format that can be viewed on multiple platforms.
  PDF files are widely used to transmit drawing data over the Internet.
* [About Exporting Raster Files](About-Exporting-Raster-Files.md)

  You can create a device-independent raster image of the objects in your drawing.
* [About Exporting PostScript Files](About-Exporting-PostScript-Files.md)

  You can convert a drawing file to a PostScript file, a format that is used by many desktop publishing applications.
* [About Exporting Stereolithography STL Files](About-Exporting-Stereolithography-STL-Files.md)

  You can export 3D solid objects in the STL file format compatible with stereolithography or 3D printing.
