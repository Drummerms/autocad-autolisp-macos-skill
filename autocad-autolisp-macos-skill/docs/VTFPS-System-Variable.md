# VTFPS (System Variable)

Sets the minimum speed of a smooth view transition, in frames per second.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 7 |

When a smooth view transition cannot maintain this speed, an instant transition is used. The valid range is 1 to 30.
