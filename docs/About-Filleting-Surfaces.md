# About Filleting Surfaces

Create a new transition surface that fillets an area between two existing surfaces or regions.

Create a tangent surface between two surfaces or regions (SURFFILLET). The original surfaces are trimmed to meet the fillet
surface.

By default, the fillet surface uses the radius value set in the FILLETRAD3D system variable. Change the radius while you are
creating the surface with the radius option or by dragging the fillet grip after the fillet is created. Use the Properties
palette to change an existing fillet radius or to derive the radius using a mathematical expression.
