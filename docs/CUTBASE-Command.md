# CUTBASE (Command)

Copies selected objects to the Clipboard, along with a specified base point, and removes them from the drawing.

## Access Methods

*Menu*:
Edit
![](../images/ac.menuaro.gif) Cut with Base Point.

Shortcut menu: End any active commands, right-click in the drawing area, and choose
Clipboard ![](../images/ac.menuaro.gif) Cut with Base Point.

## Summary

The selected objects are copied to the Clipboard and removed from the drawing.

* Use
  [PASTECLIP](PASTECLIP-Command.md) to move the copied objects from the Clipboard to a location in the same document or to another document.
* When you paste an object cut with CUTBASE, it is placed relative to the specified base point.
