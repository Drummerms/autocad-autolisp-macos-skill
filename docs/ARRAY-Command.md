# ARRAY (Command)

Creates copies of objects arranged in a 2D or 3D pattern.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Array drop-down.
![](../images/GUID-F8F67338-1FA7-42C2-BBA2-EC313EC197A9-low.png)

*Menu*:
Modify ![](../images/ac.menuaro.gif) Aray.

You can create copies of objects in a regularly spaced rectangular, polar, or path array.

![](../images/GUID-57042C66-B57A-4D48-9355-F18BB10F49DE-low.gif)

The DELOBJ system variable controls whether the source objects of the array are deleted or retained after the array is created.

The following prompts are displayed.

Select objects
:   Specifies the object to be arrayed.

Rectangular
:   Distributes copies of the selected object into any combination of rows, columns, and levels (same as the ARRAYRECT command).

Path
:   Evenly distributes copies of the selected object along a path or a portion of a path (same as the ARRAYPATH command).

Polar
:   Evenly distributes copies of the object in a circular pattern around a center point or axis of rotation (same as the ARRAYPOLAR
    command).

NOTE:Attribute objects cannot be arrayed and are automatically filtered from the selection set.
