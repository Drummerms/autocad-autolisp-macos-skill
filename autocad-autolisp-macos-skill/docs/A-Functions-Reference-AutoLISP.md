# A Functions Reference (AutoLISP)

These AutoLISP functions all start with 'A'.

## Functions

* [abs](abs-AutoLISP.md)

  Returns the absolute value of a number
* [acad\_colordlg](acad_colordlg-AutoLISP.md)

  Displays the standard AutoCAD color selection dialog box
* [acad\_helpdlg](acad_helpdlg-AutoLISP.md)

  Invokes the help facility (obsolete)
* [acad-pop-dbmod](acad-pop-dbmod-AutoLISP.md)

  Restores the value of the DBMOD system variable to the value that was most recently stored with
  acad-push-dbmod
* [acad-push-dbmod](acad-push-dbmod-AutoLISP.md)

  Stores the current value of the DBMOD system variable
* [acad\_strlsort](acad_strlsort-AutoLISP.md)

  Sorts a list of strings in alphabetical order
* [acad\_truecolorcli](acad_truecolorcli-AutoLISP.md)

  Prompts for colors at the Command prompt
* [acad\_truecolordlg](acad_truecolordlg-AutoLISP.md)

  Displays the AutoCAD color selection dialog box with tabs for index color, true color, and color books
* [acdimenableupdate](acdimenableupdate-AutoLISP.md)

  Controls the automatic updating of associative dimensions
* [acet-ent-geomextents](acet-ent-geomextents-AutoLISP-Express-Tools.md)

  Returns the geometric extents of the given object
* [acet-file-attr](acet-file-attr-AutoLISP-Express-Tools.md)

  Gets or sets the protection attributes of a specified file or directory
* [acet-file-chdir](acet-file-chdir-AutoLISP-Express-Tools.md)

  Changes the current working directory
* [acet-file-copy](acet-file-copy-AutoLISP-Express-Tools.md)

  Copies a file
* [acet-file-cwd](acet-file-cwd-AutoLISP-Express-Tools.md)

  Returns the current working directory
* [acet-file-dir](acet-file-dir-AutoLISP-Express-Tools.md)

  Locate files
* [acet-file-mkdir](acet-file-mkdir-AutoLISP-Express-Tools.md)

  Creates a new directory
* [acet-file-move](acet-file-move-AutoLISP-Express-Tools.md)

  Moves or renames files
* [acet-file-remove](acet-file-remove-AutoLISP-Express-Tools.md)

  Deletes one or more files
* [acet-file-rmdir](acet-file-rmdir-AutoLISP-Express-Tools.md)

  Removes a directory
* [acet-ini-get](acet-ini-get-AutoLISP-Express-Tools.md)

  Retrieves data from an INI file
* [acet-ini-set](acet-ini-set-AutoLISP-Express-Tools.md)

  Writes data to an INI file
* [acet-help](acet-help-AutoLISP-Express-Tools.md)

  Displays a help page
* [acet-help-trap](acet-help-trap-AutoLISP-Express-Tools.md)

  Installs a F1 key monitor
* [acet-layerp-mark](acet-layerp-mark-AutoLISP.md)

  Places beginning and ending marks for Layer Previous recording
* [acet-layerp-mode](acet-layerp-mode-AutoLISP.md)

  Queries and sets the LAYERPMODE setting
* [acet-laytrans](acet-laytrans-AutoLISP.md)

  Translates drawing layers to standards defined in another drawing or standards file
* [acet-load-expresstools](acet-load-expresstools-AutoLISP-Express-Tools.md)

  Initializes the Express Tools Utility functions.
* [acet-ms-to-ps](acet-ms-to-ps-AutoLISP.md)

  Converts a real value from model space units to paper space units
* [acet-ps-to-ms](acet-ps-to-ms-AutoLISP.md)

  Converts a real value from paper space units to model space units
* [acet-reg-del](acet-reg-del-AutoLISP-Express-Tools.md)

  Deletes a key or value from the Registry
* [acet-reg-get](acet-reg-get-AutoLISP-Express-Tools.md)

  Returns a value from the Registry
* [acet-reg-machine-prodkey](acet-reg-machine-prodkey-AutoLISP-Express-Tools.md)

  Returns the AutoCAD Windows registry path in the HKLM (HKEY\_LOCAL\_MACHINE)
* [acet-reg-prodkey](acet-reg-prodkey-AutoLISP-Express-Tools.md)

  (Obsolete) Returns the current product key from the Registry
* [acet-reg-put](acet-reg-put-AutoLISP-Express-Tools.md)

  Writes a Registry key or value
* [acet-reg-user-prodkey](acet-reg-user-prodkey-AutoLISP-Express-Tools.md)

  Returns the AutoCAD Windows registry path
* [acet-ss-drag-move](acet-ss-drag-move-AutoLISP-Express-Tools.md)

  Drags a selection set to change location
* [acet-ss-drag-rotate](acet-ss-drag-rotate-AutoLISP-Express-Tools.md)

  Drags a selection set to change rotation
* [acet-ss-drag-scale](acet-ss-drag-scale-AutoLISP-Express-Tools.md)

  Drags a selection set to change scale
* [acet-str-collate](acet-str-collate-AutoLISP-Express-Tools.md)

  Compares for sort order
* [acet-str-equal](acet-str-equal-AutoLISP-Express-Tools.md)

  Compares strings for equality
* [acet-str-find](acet-str-find-AutoLISP-Express-Tools.md)

  Finds a substring in string
* [acet-str-format](acet-str-format-AutoLISP-Express-Tools.md)

  Formats a message with embedded arguments
* [acet-str-replace](acet-str-replace-AutoLISP-Express-Tools.md)

  Replaces one substring with another
* [acet-str-wcmatch](acet-str-wcmatch-AutoLISP-Express-Tools.md)

  Performs a wild-card match similar to
  wcmatch, but allows control over case sensitivity
* [acet-sys-beep](acet-sys-beep-AutoLISP-Express-Tools.md)

  Makes a noise
* [acet-sys-command](acet-sys-command-AutoLISP-Express-Tools.md)

  Runs a command in a shell and waits for the shell to complete execution before returning
* [acet-sys-foreground](acet-sys-foreground-AutoLISP-Express-Tools.md)

  Forces AutoCAD's application window to be the topmost and receive focus
* [acet-sys-keystate](acet-sys-keystate-AutoLISP-Express-Tools.md)

  Checks the state of a virtual key
* [acet-sys-lasterr](acet-sys-lasterr-AutoLISP-Express-Tools.md)

  Reports the last error
* [acet-sys-procid](acet-sys-procid-AutoLISP-Express-Tools.md)

  Returns the current process identifier
* [acet-sys-sleep](acet-sys-sleep-AutoLISP-Express-Tools.md)

  Delay to allow other processes to run
* [acet-sys-spawn](acet-sys-spawn-AutoLISP-Express-Tools.md)

  Executes an external program
* [acet-sys-term](acet-sys-term-AutoLISP-Express-Tools.md)

  Terminates the given process
* [acet-sys-wait](acet-sys-wait-AutoLISP-Express-Tools.md)

  Waits for a process to complete
* [acet-ui-message](acet-ui-message-AutoLISP-Express-Tools.md)

  Displays a message box
* [acet-ui-pickdir](acet-ui-pickdir-AutoLISP-Express-Tools.md)

  Displays a directory selection dialog
* [acet-ui-progress](acet-ui-progress-AutoLISP-Express-Tools.md)

  Displays a progress meter
* [acet-ui-status](acet-ui-status-AutoLISP-Express-Tools.md)

  Displays a modeless status dialog
* [acet-ui-txted](acet-ui-txted-AutoLISP-Express-Tools.md)

  Displays a multiline text editing dialog
* [action\_tile](action_tile-AutoLISP-DCL.md)

  Assigns an action to evaluate when the user selects the specified tile in a dialog box
* [add\_list](add_list-AutoLISP-DCL.md)

  Adds or modifies a string in the currently active dialog box list
* [alert](alert-AutoLISP.md)

  Displays a dialog box containing an error or warning message
* [alloc](alloc-AutoLISP.md)

  Sets the size of the segment to be used by the expand function
* [and](and-AutoLISP.md)

  Returns the logical AND of the supplied arguments
* [angle](angle-AutoLISP.md)

  Returns an angle in radians of a line defined by two endpoints
* [angtof](angtof-AutoLISP.md)

  Converts a string representing an angle into a real (floating-point) value in radians
* [angtos](angtos-AutoLISP.md)

  Converts an angular value in radians into a string
* [append](append-AutoLISP.md)

  Takes any number of lists and appends them together as one list
* [apply](apply-AutoLISP.md)

  Passes a list of arguments to, and executes, a specified function
* [arx](arx-AutoLISP.md)

  Returns a list of the currently loaded ObjectARX applications
* [arxload](arxload-AutoLISP.md)

  Loads an ObjectARX application
* [arxunload](arxunload-AutoLISP.md)

  Unloads an ObjectARX application
* [ascii](ascii-AutoLISP.md)

  Returns the conversion of the first character of a string into its character code (an integer)
* [assoc](assoc-AutoLISP.md)

  Searches an association list for an element and returns that association list entry
* [atan](atan-AutoLISP.md)

  Returns the arctangent of a number in radians
* [atof](atof-AutoLISP.md)

  Converts a string into a real number
* [atoi](atoi-AutoLISP.md)

  Converts a string into an integer
* [atom](atom-AutoLISP.md)

  Verifies that an item is an atom
* [atoms-family](atoms-family-AutoLISP.md)

  Returns a list of the currently defined symbols
* [autoarxload](autoarxload-AutoLISP.md)

  Predefines command names to load an associated ObjectARX file
* [autoload](autoload-AutoLISP.md)

  Predefines command names to load an associated AutoLISP file
