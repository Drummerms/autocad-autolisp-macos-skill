# To Rename Named Objects

1. Click Format menu![](../images/ac.menuaro.gif)Rename.At the Command prompt, enter RENAME.
2. In the list of Named Objects, select an object type and choose the Item to rename.
3. Enter the new name in the Rename To box.
4. Click Rename To to apply the name change.
