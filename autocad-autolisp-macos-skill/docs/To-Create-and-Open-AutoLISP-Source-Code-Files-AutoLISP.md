# To Create and Open AutoLISP Source Code Files (AutoLISP)

AutoLISP source code files can be created and edited using a plain text editor.

## Create a new source code file

Windows
:   1. Click Start ![](../images/ac.menuaro.gif) Windows Accessories ![](../images/ac.menuaro.gif) Notepad.
    2. In Notepad, click File ![](../images/ac.menuaro.gif) Save As.
    3. In the Save As dialog box, click the Save As Type drop-down list and select All Files (\*.\*).
    4. Specify a location for the file and enter a name in the File Name box. Add
       *.lsp* after the file name and click Save.

Mac OS
:   1. In Finder, click Go ![](../images/ac.menuaro.gif) Applications.
    2. In the Applications window, double-click TextEdit.
    3. In TextEdit, if you see the ruler and formatting bar along the top of the window, do the following:
       * Click TextEdit ![](../images/ac.menuaro.gif) Preferences.
       * In Preferences, Format, click Plain Text. Click Close.
       * Click File ![](../images/ac.menuaro.gif) New.
    4. Click File ![](../images/ac.menuaro.gif) Save As.
    5. In the Save As dialog box, specify a location for the file and enter a name in the Save As box. Add
       *.lsp* after the file name and click Save.
    6. In the Extension dialog box, click Use .LSP.

## Open an existing source code file

Windows
:   1. Click Start ![](../images/ac.menuaro.gif) Windows Accessories ![](../images/ac.menuaro.gif) Notepad.
    2. In Notepad, click File ![](../images/ac.menuaro.gif) Open.
    3. In the Open dialog box, click the Save As Type drop-down list and select All Files (\*.\*).
    4. Browse to and select the LSP file to open. Click Open.

Mac OS
:   1. In Finder, click Go ![](../images/ac.menuaro.gif) Applications.
    2. In the Applications window, double-click TextEdit.
    3. In TextEdit, click File ![](../images/ac.menuaro.gif) Open.
    4. In the Open dialog box, browse to and select the LSP file to open. Click Open.
