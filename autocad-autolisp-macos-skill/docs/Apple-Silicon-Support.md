# Apple Silicon Support

AutoCAD for Mac 2024 runs natively on both Intel and Apple Silicon architectures. The performance is improved significantly when running natively
on Apple Silicon machines.

NOTE:You may be prompted to install
[Rosetta 2](https://support.apple.com/en-ca/HT211861) when you install AutoCAD on an Apple Silicon machine. Because the installer is built to run Intel architectures, Rosetta
2 is required to run the AutoCAD installer on Apple Silicon. Once you install Rosetta 2 you will not be prompted to install
Rosetta 2 again.
