# VPORTS (Command)

Creates multiple viewports in model space or paper space.

## Access Methods

*Menu*:
View
![](../images/ac.menuaro.gif) Viewports
![](../images/ac.menuaro.gif) New Viewports.

Shortcut menu: Right-click the drawing area when a named layout is current and click Viewports ![](../images/ac.menuaro.gif) New Viewports.

## Summary

The
[Viewports dialog box](Viewports-Dialog-Box.md) is displayed.

If you enter
*-vports* at the Command prompt,
[options are displayed](VPORTS-Command.md).
