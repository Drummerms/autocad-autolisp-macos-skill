# Choose a Layer Dialog Box

Merges selected layers to a specified layer.

LAYERS (Command)

*Menu*:
Format
![](../images/ac.menuaro.gif) Layers:
Select Layers ![](../images/ac.menuaro.gif) Right-Click ![](../images/ac.menuaro.gif) Combine with Layer

![](../images/GUID-FD7B2476-A85D-4BA9-8B44-D38E998399A4-low.png)

## List of Options

The following options are displayed.

Choose an existing layer to move content to

Lists the available layers in the current drawing that you can merge the selected layers with from the Layers palette. All
objects on the selected layers from the Layers palette are moved to the specified layer and then the layers are removed from
the drawing.
