# Commands for 2D Isometric Drawing

#### Commands

* [DSETTINGS](GUID-22D029BC-354B-4D7D-B800-B5308C377D6C.htm " Sets grid and snap, polar and object snap tracking, object snap modes, Dynamic Input, and Quick Properties. ")
* [GRID](GUID-7EC38AD6-FA34-4115-9E1C-6F13E1BA033D.htm "Displays a grid pattern in the current viewport. ")
* [ISOPLANE](GUID-9B1EEA63-BEC1-413E-B69F-541B5865F1A1.htm "Specifies the current isometric plane. ")
* [ORTHO](GUID-128AC5D7-72B0-498F-958D-7F619A73EC5F.htm "Constrains cursor movement to the horizontal or vertical direction. ")
* [SNAP](GUID-F47F4AAF-4859-45D4-846C-3742268834A9.htm "Restricts cursor movement to specified intervals. ")

#### System Variables

* [SNAPISOPAIR](GUID-10E95216-5E3C-45F2-A6B9-79E7660A1F60.htm "Controls the isometric plane for the current viewport. ")
* [SNAPSTYL](GUID-E04B7A7B-8232-44C3-BD74-20BCFEC07C2E.htm "Sets the snap style, rectangular or isometric, for the current viewport. ")
