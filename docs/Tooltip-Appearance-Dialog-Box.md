# Tooltip Appearance Dialog Box

Controls the appearance of tooltips.

DSETTINGS (Command)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Drafting Settings:
Dynamic Input tab ![](../images/ac.menuaro.gif) Drafting Tooltip Appearance.

## Summary

Use the
[TOOLTIPMERGE](TOOLTIPMERGE-System-Variable.md) system variable to combine drafting tooltips into a single tooltip.

## List of Options

The following options are displayed.

Previews

Displays an example of the current tooltip appearance settings.

Size

Specifies a size for tooltips. The default size is 0. Use the slider to make tooltips larger or smaller.

Transparency

Controls the transparency of tooltips. The lower the setting, the less transparent the tooltip. A value of 0 sets the tooltip
to opaque.

Apply To

Specifies whether the settings apply to all drafting tooltips or only to Dynamic Input tooltips. ([DYNTOOLTIPS](DYNTOOLTIPS-System-Variable.md) system variable)

Override OS Settings for All Drafting Tooltips
:   Applies the settings to all tooltips, overriding the settings in the operating system.

Use Settings Only for Dynamic Input Tooltips
:   Applies the settings only to the drafting tooltips used in Dynamic Input.
