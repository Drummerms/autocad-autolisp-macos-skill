# COMPAREEXPORT (Command)

Exports the results of the comparison into a new drawing file, called a snapshot drawing.

## Access Methods

*DWG Compare Visor*:
![](../images/GUID-D32B0E1E-6551-4A10-8CBD-E8258B398BE8-low.png)

NOTE:This command is available only during a drawing comparison.

COMPAREEXPORT creates a new drawing file that combines the compared drawing files into two overlapping blocks. The result maintains the
visual appearance of the comparison. The file name for the new drawing uses the following format:

Compare\_*current drawing name* vs
*compared drawing name*.dwg.

The exported snapshot drawing opens automatically as your current drawing and displays the following visor.

![](../images/GUID-1847AF1B-D44D-4351-B2B3-CC39C043DCCE-low.png)

The visor has controls for controlling the display settings and for zooming to successive change sets. It always displays
when you open a snapshot drawing.
