# SHOWNEWSTATE (System Variable)

Controls whether new and changed features in product updates are highlighted in the user interface with an orange dot.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 1 |

| Value | Description |
| 0 | Off |
| 1 | On |

This setting is accessible from the Help drop-down menu only for updates.
