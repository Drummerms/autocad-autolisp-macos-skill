# Custom Paper Sizes Dialog Box

PLOT (Command)

*Toolbar*:
![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png): Paper Size: Manage Custom Sizes

![](../images/GUID-13A05E14-6B55-4C53-A565-379B04E45FB5-low.png)

Paper Size List
:   Lists the existing custom paper sizes by name.

    * ![](../images/GUID-40517BFA-4A82-4F1B-9580-59197EDD94CD-low.png). Adds a new paper size.
    * ![](../images/GUID-36652FED-6611-41B0-95B6-14B33F2D8DB1-low.png). Removes the selected paper size.
    * *Duplicate*. Copies the selected paper size to create a custom paper size that you can edit.

Paper Size
:   Defines the width and height for the custom paper size.

Non-Printable Area
:   Defines the margins for the custom paper size.
