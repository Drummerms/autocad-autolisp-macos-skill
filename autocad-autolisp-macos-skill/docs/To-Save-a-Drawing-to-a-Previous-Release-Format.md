# To Save a Drawing to a Previous Release Format

1. Click File menu ![](../images/ac.menuaro.gif)Save As.At the Command prompt, enter SAVEAS.
2. In the Save Drawing As dialog box, in the Save As box, enter a new drawing name.
3. Under Files Format type, select the required file format you want.
4. Click OK.
