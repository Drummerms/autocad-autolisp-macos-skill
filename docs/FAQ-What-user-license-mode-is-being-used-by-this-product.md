# FAQ: What user license mode is being used by this product?

The system variables \_PKSER, ONLINEUSERID and ONLINEUSERNAME can be used to determine the user license mode used by the product.

* If \_PKSER doesn't equal "000-00000000", then the user is using a perpetual license.
* If \_PKSER equals "000-00000000" and both ONLINEUSERID and ONLINEUSERNAME don't equal "", then the user is on subscription.
