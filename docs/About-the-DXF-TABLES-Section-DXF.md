# About the DXF TABLES Section (DXF)

The group codes described in this chapter are found in DXF™ files and used by applications. The TABLES section contains several
tables, each of which can contain a variable number of entries. These codes are also used by AutoLISP ®  and ObjectARX ®  applications in entity definition lists.
