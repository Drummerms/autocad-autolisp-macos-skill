# To Restore a Saved Viewport Arrangement

NOTE: Viewport arrangements can be
*saved*only in model space, but you can restore saved viewport arrangements either in model space, as model space viewports, or in
paper space, as layout viewports.

1. In model space, click
   View ![](../images/ac.menuaro.gif) Viewports ![](../images/ac.menuaro.gif) Named Viewports.
2. In the Viewports dialog box, Named Viewports tab, select the name of the viewport configuration from the list.
