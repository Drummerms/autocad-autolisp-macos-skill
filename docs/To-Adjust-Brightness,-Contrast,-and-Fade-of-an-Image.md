# To Adjust Brightness, Contrast, and Fade of an Image

1. Click Modify ![](../images/ac.menuaro.gif) Object ![](../images/ac.menuaro.gif) Image ![](../images/ac.menuaro.gif) Adjust.At the Command prompt, enter IMAGEADJUST.
2. Select the image to modify.
3. In the Image Adjust dialog box, to adjust brightness, contrast, and fade, use the appropriate slider or enter a value.

   The default value for both brightness and contrast is 50. You can adjust to a maximum brightness of 100 or to a minimum of
   0. The default fade value is 0. You can adjust to a maximum fade of 100.
4. Click OK.
