# To Create a Pull-down Menu

1. From the Mac OS menu bar, click Tools menu ![](../images/ac.menuaro.gif) Customize ![](../images/ac.menuaro.gif) Interface (CUI) or at the Command prompt, enter *cui* and press Enter.
2. In the Customize dialog box, Menus tab, Menus List, select the menu in which you want to create the new menu after.
3. Click the Create Menu Element (+) button below the Menus list and click Add Menu.

   A new menu (named Menu1) is created. The default name changes based on the number of menus you previously created.
4. Enter a name for the new menu.

   The name entered is what will appear on the Mac OS menu bar. Click a menu name twice to rename it.
5. In the Commands list, drag a command to the new menu.
6. Release the button on the pointer device when the menu or sub-menu is highlighted, or the desired location for the command
   is indicated by a horizontal line.
