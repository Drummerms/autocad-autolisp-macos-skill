# To Set Vertical Orientation in a Text Style

1. Click Format![](../images/ac.menuaro.gif)Text Style.At the Command prompt, enter *style*.
2. In the Text Style dialog box, select Vertical.

   NOTE:Vertical Orientation is available only if the selected font supports dual orientation. It is not available for TrueType fonts.
3. To update existing text that uses this text style, click Apply.
4. Click Close.
