# REFPATHTYPE (System Variable)

Controls whether reference files are attached using full, relative or no paths when they are first attached to a host drawing
file.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | Uses no path as default |
| 1 | Uses relative path as default |
| 2 | Uses full path as default |

NOTE:This system variable has no effect on references that are already attached to the host drawing.
