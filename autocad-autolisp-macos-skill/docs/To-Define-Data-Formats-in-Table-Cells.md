# To Define Data Formats in Table Cells

Table cells can use data formats such as currency, numbers, dates, and more.

1. Select the cells you want to redefine.
   * To select a single cell, click once inside the cell.
   * To select multiple cells, click inside one cell, hold the Shift key down, and click inside another cell. All cells between
     the two cells are selected.
   * To select a row, click the row label.
   * To select a column, click the column label.
2. If the Properties Inspector isn't open, click
   Window menu ![](../images/ac.menuaro.gif) Properties Inspector.
3. On the Properties Inspector, select a data type from the drop-down menu.

   ![](../images/GUID-3EA73834-77EC-4592-9FB9-C8F77F63A1A8-low.png)

   Different formatting options are shown depending on the data format selected.
4. Define the format.
5. If available, click Additional Format to display the Table Cell Format dialog box with more formatting options.

   ![](../images/GUID-25C8FE8B-D828-4F06-ABB7-42FEB50245EA-low.png)
6. Select from the available options.
7. Click OK.
