# To Set or Change the Block Unit of a Block Definition

While defining a new block definition, you can set its default block unit. If the wrong or no block unit is assigned to a
block definition, it can be changed in the Block Editor.

## Set the Block Unit for a New Block Definition

Windows
:   1. Click
       Insert tab ![](../images/ac.menuaro.gif) Block Definition panel ![](../images/ac.menuaro.gif) Create Block.
       ![](../images/GUID-1A9FC061-FB9C-4DD1-AD8D-CAEB369EC286-low.png) Find
    2. In the Block Definition dialog box, do the following:
       * Enter a name.
       * Select the source objects to visually describe the block.
       * Specify a base point.
    3. Under the Settings section, from the Block Unit drop-down list, choose a block unit for the block.
    4. Optionally, set any of the available Behavior settings and enter a description.
    5. Click OK.

Mac
:   1. Click
       Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block, Make.
       ![](../images/GUID-B2E92148-A285-4542-BEF5-AD10E48A0DE5-low.png)
    2. In the Define Block dialog box, do the following:
       * Enter a name.
       * Select the source objects to visually describe the block.
       * Specify a base point.
    3. Under the Units section, from the Block Unit drop-down list, choose a block unit for the block.
    4. Optionally, set any of the available Block Behavior settings and enter a description.
    5. Click Create Block.

    For additional details on creating a block definition, see
    [To Define a Block for the Current Drawing](To-Define-a-Block-for-the-Current-Drawing.md).

## Change the Block Unit an Existing Block Definition

Windows
:   1. Click
       Insert tab ![](../images/ac.menuaro.gif) Block Definition panel ![](../images/ac.menuaro.gif) Block Editor.
       ![](../images/GUID-C38A8D25-E8C5-4381-AF0B-753233C6EA2A-low.png) Find
    2. In the Edit Block Definition dialog box, select the block to edit and click OK.
    3. On the Properties palette, under the Block section, click the Units field.
    4. From the Units drop-down list, choose a new block unit.
    5. Click
       Block Editor tab ![](../images/ac.menuaro.gif) Close panel ![](../images/ac.menuaro.gif) Close Block Editor.
       ![](../images/GUID-292D3F59-71E1-4399-BCD1-58F167A11674-low.png) Find
    6. In the Block - Changes Not Saved dialog box, click Save the Changes.

Mac
:   1. Click
       Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block Editor.
       ![](../images/GUID-E3DDBE51-C92A-4D07-A51D-C7C685E4F578-low.png)
    2. In the Edit Block Definition dialog box, select the block to edit and click Edit Block.
    3. On the Properties Inspector, under the Block section, click the Units field.
    4. From the Units drop-down list, choose a new block unit.
    5. On the Block Editor visor, click Exit.
       ![](../images/GUID-F3CD8D62-2BA2-42C6-99E8-681C38D935DD-low.png)
    6. In the Block - Changes Not Saved dialog box, click Save the Changes.
