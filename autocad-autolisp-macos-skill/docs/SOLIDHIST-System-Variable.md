# SOLIDHIST (System Variable)

Controls whether new composite solids retain a history of their original components.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

When set to 1, all subsequently created composite solids retain a history of the original component objects. These component
objects can be accessed to modify the solids

| 0 | Sets the History property to None for new solids. No history is retained. |
| 1 | Sets the History property to Record for new solids. Solids retain a history of their original objects. |
