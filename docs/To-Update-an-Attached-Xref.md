# To Update an Attached Xref

1. Click
   Window menu ![](../images/ac.menuaro.gif) Reference Manager.
2. In the Reference Manager palette, select the reference name that you want to reload.

   NOTE:An xref that needs to be updated is indicated in the Reference Manager by a lightning bolt.
   ![](../images/GUID-01C5868A-145A-4905-9D47-7708999E8B4A-low.png)
3. Optional: Compare the xref to the reference drawing.
   * Right-click, click
     Compare ![](../images/ac.menuaro.gif) Recent Changes.
   * Review the changes as highlighted on the current drawing by revision clouds and color.
   * Close the comparison from the Xref Compare visor.
     ![](../images/GUID-9002B3B8-BF54-447F-B816-6EA0CDF91EFC-low.png)
4. Right-click, and click Reload.

If the drawing you selected has been changed since you opened your drawing, the xref is reloaded.

NOTE:Xrefs reload automatically when a drawing is opened.
