# acet-reg Functions

#### Topics in this section

* [acet-reg-del (AutoLISP/Express Tools)](acet-reg-del-AutoLISP-Express-Tools.md)

  Deletes a key or value from the Registry.
* [acet-reg-get (AutoLISP/Express Tools)](acet-reg-get-AutoLISP-Express-Tools.md)

  Returns a value from the Registry.
* [acet-reg-machine-prodkey (AutoLISP/Express Tools)](acet-reg-machine-prodkey-AutoLISP-Express-Tools.md)

  Returns the AutoCAD Windows registry path in the HKLM (HKEY\_LOCAL\_MACHINE).
* [acet-reg-prodkey (AutoLISP/Express Tools)](acet-reg-prodkey-AutoLISP-Express-Tools.md)

  (Obsolete) Returns the current product key from the Registry.
* [acet-reg-put (AutoLISP/Express Tools)](acet-reg-put-AutoLISP-Express-Tools.md)

  Writes a Registry key or value.
* [acet-reg-user-prodkey (AutoLISP/Express Tools)](acet-reg-user-prodkey-AutoLISP-Express-Tools.md)

  Returns the AutoCAD Windows registry path.
