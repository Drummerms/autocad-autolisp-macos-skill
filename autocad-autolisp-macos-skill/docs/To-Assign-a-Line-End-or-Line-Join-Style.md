# To Assign a Line End or Line Join Style

You can set the line end and join styles for objects that have lineweight assigned, either as an object property or as a plot
style override.

1. Click
   Format menu ![](../images/ac.menuaro.gif) Plot Style.
2. Select the Plot Style Table that you want to modify. Click Edit.
3. In the Plot Style Table Editor, click the Line End Style or Line Join Style field for the plot style you want to change.
4. Select an option from the drop-down list.
