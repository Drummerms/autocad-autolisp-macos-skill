# To View or Change Program-Related Folder Locations

1. Right-click the drawing area and choose Preferences.
2. On the Application tab, click the arrow to the left of the files path you want to change.
3. Double click the path you want to change.
4. In the Open dialog box, browse to the desired location and select the file.
5. Click Open.
