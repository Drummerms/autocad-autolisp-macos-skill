# FIELDDISPLAY (System Variable)

Controls whether fields are displayed with a gray background.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

The background is not plotted.

| 0 | Fields are displayed with no background |
| 1 | Fields are displayed with a gray background |
