# To Snap to a Geometric Location on an Object

1. At any prompt for a point, hold down Shift and right-click in the drawing area. Choose the object snap that you want to use.
2. Move your cursor over the desired object snap location.

   By default, your cursor automatically locks onto the object snap location you specified, and a marker and tooltip indicate
   the object snap location. If more than one object snap is available, press Tab to cycle through the choices.
3. Select the object.

   The cursor snaps to the specified geometric feature closest to where you selected the object.
