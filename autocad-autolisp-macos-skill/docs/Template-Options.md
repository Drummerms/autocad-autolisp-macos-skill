# Template Options

When a drawing is saved as a drawing template, you are prompted for the measurement to use when a new drawing is created from
the drawing template and a description for the template.

## List of Prompts

The following prompts are displayed.

Enter template measurement [English/Metric] <*current*>: *Enter* *e* *or* *m* *to specify the measurement to assign to the template and press* Enter

Enter a template description: *Enter a description and press* Enter
