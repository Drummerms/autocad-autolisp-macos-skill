# To Work with Pressing or Pulling 3D Solid Faces

How to extrude or offset a face or closed area using a press or pull operation.

Pressing or pulling a face offsets the face and adjusts adjacent geometry. You can press the Ctrl key to switch between offset
and extrusion modes.

![](../images/GUID-80ECE1BF-5CEF-4D1A-AC96-4D0D4840A5F5-low.png)

## Offset a 3D Solid Face by Pressing or Pulling

Use this method to adjust adjacent faces to the new position of the offset face.

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Presspull.
   ![](../images/GUID-7BAD4DA5-0A85-410F-A3A4-00F1A7483B7B-low.png)
2. Press Ctrl+click a bounded area on a 3D solid object and move the cursor to establish an offset direction.

   Edge highlighting indicates which areas will be offset.
3. Enter a value or click in the drawing area to set the offset distance.

## *Extrude a 3D Solid Face by Pressing or Pulling*

Use this method if you want adjacent faces to remain unaffected by the new position of the offset face.

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Presspull.
   ![](../images/GUID-7BAD4DA5-0A85-410F-A3A4-00F1A7483B7B-low.png)
2. Click a face on a 3D solid object and move the cursor to establish an offset direction.

   Highlighting indicates which faces can be offset
3. Enter a value or click in the drawing area to set the extrusion distance.

## *Offset Multiple Faces by Pressing or Pulling*

You can use PRESSPULL to simultaneously offset planar and non-planar faces and areas on the same or different objects. All
items in the selection set must be of the same type as the first (either a face or bounded area).

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Presspull.
   ![](../images/GUID-7BAD4DA5-0A85-410F-A3A4-00F1A7483B7B-low.png)
2. Enter
   *m* (Multiple).
3. Select the faces or areas to be modified and press Enter.
4. Enter a value or click in the drawing area to set the extrusion distance.
