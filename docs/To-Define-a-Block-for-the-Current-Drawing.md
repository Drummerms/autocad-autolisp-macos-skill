# To Define a Block for the Current Drawing

1. Create the objects you want to use in the block definition.
2. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block.
   ![](../images/GUID-B2E92148-A285-4542-BEF5-AD10E48A0DE5-low.png)
3. In the Define Block dialog box, enter a block name in the Name box.
4. Under Source Objects, select Convert to Block.

   NOTE:If Delete Objects is selected, the original objects are erased from the drawing when the block is created. If necessary, you
   can use OOPS to restore them.
5. Click Select Objects. This closes the Block Definition dialog box temporarily. Use your pointing device to select the objects
   to be included in the block definition. Press Enter to complete object selection.
6. In the Define Block dialog box under Base Point, specify the block insertion point using one of these methods.
   * Click Pick Point to specify a point using the pointing device.
   * Enter the
     *X,Y,Z* coordinate values of the point.
7. In the Description box, enter a description for the block definition.
8. Click OK.

   The block is defined in the current drawing and can be inserted at any time.
