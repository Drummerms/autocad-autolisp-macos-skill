# Search Review Toolbar

Displays various options and controls that allows you to navigate and manage the sets of searched objects.

## Toolbar Commands and Options

When you're in the review mode, the Search Review toolbar displays at the top of the drawing area.

![](../images/GUID-BBE501FA-E4F5-443E-9A1D-50C2095785CA-low.png)

The following controls are available.

## Filter Options

Applies filter to exclude or include existing blocks and text variations.

## Set Index

Displays the current set number along with the total number of searched sets.

## Previous or Next

Highlights the previous or next set, allowing an efficient review of the searched objects.

## Convert

Displays the Convert to Block dialog box, which you can use to convert the highlighted instances into blocks.

## End Review

Closes the Search toolbar and ends the review mode.
