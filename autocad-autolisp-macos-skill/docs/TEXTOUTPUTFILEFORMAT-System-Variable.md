# TEXTOUTPUTFILEFORMAT (System Variable)

Provides Unicode options for log files.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | ANSI format |
| 1 | UTF-8 (Unicode) |
| 2 | UTF-16LE (Unicode) |
| 3 | UTF-16BE (Unicode) |
