# PDFSHXLAYER (System Variable)

Controls what layer is assigned to newly created text objects when converting SHX geometry to text objects.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

This system variable provides direct access to the Create Text On setting in the PDF Text Recognition Settings dialog box
of the PDFSHXTEXT command.

| Value | Description |
| 0 | Creates text on same layer as the geometry being converted. |
| 1 | Creates text on the current layer. |
