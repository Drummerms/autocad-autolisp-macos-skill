# GRIDMODE (System Variable)

Specifies whether the grid is turned on or off.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| 0 | Turns the grid off |
| 1 | Turns the grid on |
