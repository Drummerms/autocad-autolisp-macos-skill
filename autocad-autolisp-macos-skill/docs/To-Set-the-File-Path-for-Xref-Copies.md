# To Set the File Path for Xref Copies

1. Right-click the drawing area and choose Preferences
2. On the Application tab, expand Temporary External Reference File Location.

   The expanded tree view displays the path where copies of xref files are placed. If no path is specified, the temporary copies
   are placed in the location specified by Temporary Drawing File Location.
3. Enter a new path, or double-click the path to browse for a location.
4. Click OK to set the path.
