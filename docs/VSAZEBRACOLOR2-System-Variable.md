# VSAZEBRACOLOR2 (System Variable)

Sets the second (contrasting) color of the zebra stripes displayed during zebra analysis (ANALYSISZEBRA).

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | RGB: 0, 0, 0 |

Values 1-255 designate an AutoCAD Color Index (ACI) color.

Values for RGB True Color are represented as a comma-separated string of integers 000 to 255. The default value is RGB:0,0,0.

NOTE:When a color book is installed, any colors defined in the book can be used.
