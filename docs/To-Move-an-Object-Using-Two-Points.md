# To Move an Object Using Two Points

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Move.
   ![](../images/GUID-DE17DBB8-D05E-46A6-A54D-BF9B6E37D8BC-low.png)
2. Select the objects to move.
3. Specify a base point for the move.
4. Specify a second point.

   The objects you selected are moved to a new location determined by the distance and direction between the first and second
   points.
