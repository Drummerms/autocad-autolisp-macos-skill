# Select Similar Settings Dialog Box

Controls which properties must match for an object of the same type to be selected.

SELECTSIMILAR (Command)

*Toolbar*:
![](../images/GUID-CEC6378A-3D4C-4061-9808-14D3D6B65A93-low.png): Specify the settings option at the prompt.

## List of Options

The following options are displayed.

Color
:   Considers objects with matching colors to be similar.

Linetype
:   Considers objects with matching linetypes to be similar.

Linetype scale
:   Considers objects with matching linetype scales to be similar.

Lineweight
:   Considers objects with matching lineweights to be similar.

Layer
:   Considers objects on matching layers to be similar.

Name
:   Considers referenced objects (such as blocks, xrefs, and images) with matching names to be similar.

Plot style
:   Considers objects with matching plot styles to be similar.

Object style
:   Considers objects with matching styles (such as text styles, dimension styles, and table styles) to be similar.
