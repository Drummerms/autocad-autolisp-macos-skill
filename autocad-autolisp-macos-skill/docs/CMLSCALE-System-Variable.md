# CMLSCALE (System Variable)

Controls the overall width of a multiline.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 1.0000 (imperial) or 20.0000 (metric) |

A scale factor of 2.0 produces a multiline twice as wide as the style definition. A zero scale factor collapses the multiline
into a single line. A negative scale factor flips the order of the offset lines (that is, the smallest or most negative is
placed on top when the multiline is drawn from left to right).
