# MBUTTONPAN (System Variable)

Controls the behavior of the third button or wheel on the pointing device.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | Supports the action defined in the customization file |
| 1 | Supports panning when you hold and drag the button or wheel |
