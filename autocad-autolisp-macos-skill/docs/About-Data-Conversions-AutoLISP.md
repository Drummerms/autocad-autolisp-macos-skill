# About Data Conversions (AutoLISP)

The listed topics are related to utilities for converting data types and units. See in AutoLISP Function Synopsis (AutoLISP),
for a complete list of conversion functions.

#### Topics in this section

* [About String Conversions (AutoLISP)](About-String-Conversions-AutoLISP.md)

  Numeric values can be converted to string values for use in output or textual data.
* [About Angular Conversion (AutoLISP)](About-Angular-Conversion-AutoLISP.md)

  Angular values returned by most AutoLISP functions and those stored in a drawing are expressed in radians, while angular input
  is commonly provided in degrees or another angular format than radians.
* [About ASCII Codes (AutoLISP)](About-ASCII-Codes-AutoLISP.md)

  ASCII codes are integer values that represent alphanumeric characters.
* [About Unit Conversion (AutoLISP)](About-Unit-Conversion-AutoLISP.md)

  Values that represent distances, volumes, or other forms of measurement can be converted from one real-world unit to another.
