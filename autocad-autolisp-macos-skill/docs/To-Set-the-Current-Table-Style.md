# To Set the Current Table Style

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/ac.menuaro.gif) ![](../images/GUID-EA6A1B89-9B6F-4552-AD8E-694113F12EA9-low.png) , or enter TABLESTYLE at the command prompt.

   Existing table styles for the current drawing are listed in the Table Style Manager dialog box.
2. Right-click a table style name and select Set Current.

   New tables will use this table style.
