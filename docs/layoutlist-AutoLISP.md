# layoutlist (AutoLISP)

Returns a list of all paper space layouts in the current drawing

*Supported Platforms:* Windows and Mac OS only

## Signature

```
(layoutlist)
```

No arguments.

## Return Values

*Type:* List

A list of strings.

## Examples

```
(layoutlist)
("Layout1" "Layout2")
```
