# DIMFXLON (System Variable)

Controls whether extension lines are set to a fixed length.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Drawing |
| Initial value: | Off |

When DIMFXLON is on, extension lines are set to the length specified by DIMFXL.
