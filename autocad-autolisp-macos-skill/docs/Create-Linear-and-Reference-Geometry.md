# Create Linear and Reference Geometry

#### Topics in this section

* [About Lines](About-Lines.md)
* [About Polylines](About-Polylines.md)

  A polyline is a connected sequence of line segments created as a single object. You can create straight line segments, arc
  segments, or a combination of the two.
* [About Multilines](About-Multilines.md)

  Multilines are composed of parallel lines, called elements.
* [About Freehand Sketches](About-Freehand-Sketches.md)

  Sketching is useful for creating irregular boundaries or for tracing with a digitizer.
* [About Reference Geometry](About-Reference-Geometry.md)

  With large or complex drawings, you can create and preserve geometric objects that serve as references for creating new objects
  and checking existing objects.
