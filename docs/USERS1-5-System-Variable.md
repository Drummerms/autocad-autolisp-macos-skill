# USERS1-5 (System Variable)

Provides storage and retrieval of text string data.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Not-saved |
| Initial value: | "" |

There are five system variables: USERS1, USERS2, USERS3, USERS4, and USERS5.
