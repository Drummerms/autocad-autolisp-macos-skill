# Work with the VS Code AutoLISP Extension

#### Topics in this section

* [Getting Started with Visual Studio Code (AutoLISP/VS Code)](Getting-Started-with-Visual-Studio-Code-AutoLISP-VS-Code.md)

  Before you can start developing AutoLISP programs with Visual Studio (VS) Code, there are some steps that you will need to
  complete first. This topic outlines the various steps needed to install, configure, and use Visual Studio (VS) Code and the
  AutoCAD AutoLISP Extension to develop and debug AutoLISP programs for AutoCAD.
* [What is and Why Visual Studio Code? (AutoLISP/VS Code)](What-is-and-Why-Visual-Studio-Code-AutoLISP-VS-Code.md)

  Visual Studio Code, commonly referred to as
  VS Code, is a modern lightweight cross platform source code editor that supports scripting and traditional programming languages.
* [Configuring VS Code (AutoLISP/VS Code)](Configuring-VS-Code-AutoLISP-VS-Code.md)

  Before you start editing and debugging AutoLISP source (LSP) files, you will want to configure the AutoCAD AutoLISP Extension.
* [Creating and Opening AutoLISP Files (AutoLISP/VS Code)](Creating-and-Opening-AutoLISP-Files-AutoLISP-VS-Code.md)

  VS Code allows you to work many different types of files that are saved in plain text; this includes AutoLISP source (LSP),
  AutoLISP Menu source (MNL), and Dialog Control Language (DCL) files.
* [Managing AutoLISP Files (AutoLISP/VS Code)](Managing-AutoLISP-Files-AutoLISP-VS-Code.md)

  VS Code provides you with several ways to manage your AutoLISP source code files.
* [Editing AutoLISP Files (AutoLISP/VS Code)](Editing-AutoLISP-Files-AutoLISP-VS-Code.md)

  The AutoCAD AutoLISP Extension offers several features that make it easier to write AutoLISP programs and DCL files, such
  as IntelliSense and code snippets.
* [Formatting AutoLISP Files (AutoLISP/VS Code)](Formatting-AutoLISP-Files-AutoLISP-VS-Code.md)

  AutoLISP expressions can be formatted to make them easier to read.
* [Debugging AutoLISP Files (AutoLISP/VS Code)](Debugging-AutoLISP-Files-AutoLISP-VS-Code.md)

  The AutoCAD program ships with a utility called the
  AutoLISP Debug Adapter which allows VS Code and AutoCAD to communicate with each other.
