# To Set the Number of Custom Grips to Display for a Dynamic Block Parameter

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block Editor.
   ![](../images/GUID-E3DDBE51-C92A-4D07-A51D-C7C685E4F578-low.png)
2. In the Edit Block Definition dialog box, select the block whose grips you want to modify
3. In the Block Editor drawing area, select a parameter icon.
4. On the Properties Inspector, under Misc, click Number of Grips, and then select the number of grips you want to display for
   the parameter.
5. Click
   Block Editor visor
   ![](../images/ac.menuaro.gif) Save.
   ![](../images/GUID-5E815E0C-AED9-4DCE-AC3D-3AA0C6FF0DA6-low.png)
6. Click
   Block Editor visor
   ![](../images/ac.menuaro.gif) Exit.
   ![](../images/GUID-F3CD8D62-2BA2-42C6-99E8-681C38D935DD-low.png)
