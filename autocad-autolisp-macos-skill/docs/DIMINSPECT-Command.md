# -DIMINSPECT (Command)

## List of Prompts

The following prompts are displayed.

[Add inspection data](DIMINSPECT-Command.md) or [[Remove](DIMINSPECT-Command.md)] <Add>: *Enter an option or press* Enter

Add

Adds an inspection dimension to the selected dimensions.

Round
:   Creates a frame with semi-circles on the two ends; the fields within the frame are separated by vertical lines.

Angular
:   Creates a frame with lines that form a 90-degree angle on the two ends; the fields within the frame are separated by vertical
    lines.

None
:   Specifies that no frame is drawn around the values; the fields are not separated by vertical lines.

None
:   No label is displayed with the inspection dimension.

Remove

Removes the inspection dimension from the selected dimensions.
