# DIVMESHSPHEREAXIS (System Variable)

Sets the number of radial subdivisions around the axis endpoint of the mesh sphere.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 12 |

![](../images/GUID-8C8FAB0F-47EA-4A83-9267-EFDE7517219F-low.png)

This setting affects new mesh spheres that are created with the [MESH](MESH-Command.md) command.
