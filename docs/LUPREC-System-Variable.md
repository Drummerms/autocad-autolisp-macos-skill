# LUPREC (System Variable)

Sets the display precision for linear units and coordinates.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 4 |

However, the internal precision of distance, area, and volume values, and linear coordinates is always maintained, regardless
of the display precision. LUPREC does not affect the display precision of dimension text (see [DIMSTYLE](DIMSTYLE-Command-2.md)).

Valid values are integers from 0 to 8.
