# ISAVEBAK (System Variable)

Controls whether BAK files are created when drawing files are saved.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | No BAK file is created |
| 1 | A BAK file is created |
