# To Extend Objects in 3D Wireframe Models

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Extend.
   ![](../images/GUID-74AC1D5E-0799-450B-8370-58FEE9FF962E-low.png)
2. Select the boundary edge for extending (1).
3. Enter
   *e* (Edge).
4. Enter
   *e* (Extend).
5. Enter
   *p* (Project).
6. Enter
   *u* (UCS).
7. Select the object to extend (2).

   ![](../images/GUID-908224D2-643A-4F08-A6C7-26EC6CB6E97F-low.png)
