# About Providing Help (DCL)

You should provide a Help facility. How much Help you provide depends on how complex your application is and how self-explanatory
your dialog boxes are. At the very least, it is recommended that the main dialog box of your application have a Help button
that displays a single dialog box describing important information. In most cases, the Help button should call the Help facility
using the
help function.

If your application is more sophisticated, consider using a Help button in each dialog box.
