# About Making Objects Transparent

You can control the transparency for selected objects, or for all objects on a layer. You can also specify a default transparency
value for new hatch and fill objects.

Set the transparency level of selected objects or layers to enhance drawings or reduce the visibility of areas that are included
for reference only.

Transparency can be set to ByLayer, ByBlock, or to a specific value.

IMPORTANT:For performance reasons, transparency when plotting is disabled by default. To plot transparent objects, click the Print Transparency
option in either the Plot dialog box or Page Setup dialog box.
