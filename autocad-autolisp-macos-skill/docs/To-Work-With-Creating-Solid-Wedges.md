# To Work With Creating Solid Wedges

## Create a Solid Wedge Based on Two Points and a Height

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Solid drop-down ![](../images/ac.menuaro.gif) Wedge.
   ![](../images/GUID-E6323DF7-45B1-4E99-B6D8-F23C52A62264-low.png)
2. Specify the first corner of the base
3. Specify the opposite corner of the base
4. Specify the height of the wedge.

## Create a Solid Wedge With Equal Length, Width, and Height

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Solid drop-down ![](../images/ac.menuaro.gif) Wedge.
   ![](../images/GUID-E6323DF7-45B1-4E99-B6D8-F23C52A62264-low.png)
2. Specify the first corner or enter c (Center) to set the center point of the base.
3. At the Command prompt, enter c (Cube). Specify the length of the wedge and a rotation angle. The length value sets both the
   width and height of the wedge.
