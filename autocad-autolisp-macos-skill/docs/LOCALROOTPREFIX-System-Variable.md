# LOCALROOTPREFIX (System Variable)

Stores the full path to the root folder where local customizable files were installed.

(Read-only)

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Registry |
| Initial value: | Varies |

The
*Template* and
*Textures* folders are in this location, and you can add any customizable files that you do not want to roam on the network. See
[ROAMABLEROOTPREFIX](ROAMABLEROOTPREFIX-System-Variable.md) for the location of the roamable files.
