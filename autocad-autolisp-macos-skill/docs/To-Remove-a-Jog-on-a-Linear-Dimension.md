# To Remove a Jog on a Linear Dimension

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Dimension panel ![](../images/ac.menuaro.gif) Dimension, Dimjogline.
   ![](../images/GUID-3026EBD8-95BC-47CB-8200-DCF4EDD5DDD3-low.png)

   NOTE:Hidden by default. Click
   ![](../images/GUID-78DED619-51B3-49C1-AD71-F928B5F69C0A-low.png) to display the icon on the tool set panel.
2. Enter
   *r* (Remove) and press Enter.
3. Select the linear dimension.
