# To Set Block Insertion Scale for a Drawing

Blocks are scaled during insertion based on the units of the block and drawing being inserted and the insertion scale units
of the current drawing.

NOTE:If you do not want blocks or drawings to be scaled during insertion, specify Unitless. When the Insertion Scale is set to
Unitless in either the source block or the target drawing, the Source Content Units and Target Drawing Units settings are
referenced to determine the scaling ratio. See the
[Set Block Insertion Scale for Unitless Drawings](GUID-410E4A05-F789-4416-9D63-3D24A46DD740.htm#SECTION_A4CFF0FBBF44409591AE18B7AB353313) section on setting the Source Content and Target Drawing units.

Windows
:   1. Click the
       Application menu ![](../images/ac.menuaro.gif) Drawing Utilities ![](../images/ac.menuaro.gif) Units.
       ![](../images/GUID-B18D6899-835E-4817-B1A9-1F1F292C54CD-low.png) Find
    2. In the Drawing Units dialog box, under Insertion Scale, click the drop-down list to set the insertion scale units.
    3. Click OK.

Mac
:   1. Click
       Format ![](../images/ac.menuaro.gif) Units.
       ![](../images/GUID-EE300CC0-76B4-494E-B511-C48189B99F65-low.png)
    2. In the Drawing Units dialog box, under Insertion Scale, click the drop-down list to set the insertion scale units.
    3. Click OK.

## Set Block Insertion Scale for Unitless Drawings

For unitless drawings, you must set the Source Content and Target Drawing units to allow for the automatic scaling of blocks
during insertion.

Windows
:   1. Click the Application menu ![](../images/ac.menuaro.gif) Options.
    2. In the Options dialog box, User Preferences tab, under Insertion Scale, click the following drop-down lists and change the
       current units:
       * Source Content Units
       * Target Drawing Units
    3. Click OK.

Mac
:   1. From the Mac OS menu bar, click
       *<Product Name>* ![](../images/ac.menuaro.gif) Preferences.

       NOTE:A drawing must be open to see the Preferences menu item.
    2. In the Application Preferences dialog box, Units & Guides tab, under Insertion Scale, click the following drop-down lists
       and change the current units:
       * Source Content Units
       * Target Drawing Units
    3. Click OK.
