# User Interface Customization

#### Topics in this section

* [About User Interface Customization on Mac OS](About-User-Interface-Customization-on-Mac-OS.md)

  The user interface includes commands, menus, tool sets, and a toolbar. Methods to customize each element varies, depending
  on the element.
* [About Command Customization](About-Command-Customization.md)

  You can easily create, edit, and reuse commands. The Commands tab of the Customize dialog box allows you to create and modify
  existing commands that can then be added to a user interface element.
* [About Command Macro Strings](About-Command-Macro-Strings.md)

  Command macro strings are used to instruct AutoCAD which commands and system variables to execute along with any expected
  input allowed at the Command prompt. Special characters, DIESEL expressions and AutoLISP programming code can also included
  in a command macro string.
* [About Creating and Customizing of Shortcut Keys](About-Creating-and-Customizing-of-Shortcut-Keys.md)

  Shortcut keys are key combinations that start commands.
* [About Creating Tooltips for Commands](About-Creating-Tooltips-for-Commands.md)

  Tooltips are descriptive messages that are displayed near the cursor when it hovers over a tool set tool.
* [About Assigning Images to a Command](About-Assigning-Images-to-a-Command.md)

  Standard and custom images can be assigned to a command.
* [About Creating and Managing Pull-down Menus](About-Creating-and-Managing-Pull-down-Menus.md)

  You can add and customize pull-down menus on the Mac OS menu bar.
* [About Managing Tool Sets](About-Managing-Tool-Sets.md)

  The Tool Sets provide efficient access to AutoCAD commands.
