# To Match the Layer Setting Between Selected Objects

1. Click
   Format menu![](../images/ac.menuaro.gif) Layers.
2. Click Layer Match.
   ![](../images/GUID-BD3247A5-4A0D-4151-B000-B802DC09511D-low.png)
3. Select the objects that you want to change to a different layer, and press Enter.
4. Select an object on the destination layer.
