# SURFACEMODELINGMODE (System Variable)

Controls whether surfaces are created as procedural surfaces or NURBS surfaces.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Not-saved |
| Initial value: | 0 |

| 0 | Creates a *procedural surface* when creating surfaces. |
| 1 | Creates a *NURBS surface* when creating surfaces. |
