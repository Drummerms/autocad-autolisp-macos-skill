# LAYERMANAGERDOCKSTATE (System Variable)

Indicates whether the Layers palette is docked or floating.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 1 |

| Value | Description |
| 0 | Floating |
| 1 | Docked |
