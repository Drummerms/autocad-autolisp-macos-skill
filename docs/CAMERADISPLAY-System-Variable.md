# CAMERADISPLAY (System Variable)

Turns the display of camera objects on or off.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

Camera objects can only be created with the Windows releases of AutoCAD for Mac.

| 0 | Camera glyphs are not displayed |
| 1 | Camera glyphs are displayed |
