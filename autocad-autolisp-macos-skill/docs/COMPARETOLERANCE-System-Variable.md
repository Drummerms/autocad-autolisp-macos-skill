# COMPARETOLERANCE (System Variable)

Specifies the tolerance used when comparing two drawing files. Objects are considered identical if they are below or equal
to a specified decimal point value.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 6 |

The valid values are 0 to 14.
