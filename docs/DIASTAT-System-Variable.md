# DIASTAT (System Variable)

Stores the exit method of the most recently used dialog box.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 0 |

| 0 | Cancel |
| 1 | OK |

NOTE:The DIASTAT system variable does not apply to standard file selection dialog boxes, such as the Select File and the Save Drawing
As dialog boxes.
