# PROJMODE (System Variable)

Sets the current Projection mode for trimming or extending.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | True 3D mode (no projection) |
| 1 | Project to the *XY* plane of the current UCS |
| 2 | Project to the current view plane |
