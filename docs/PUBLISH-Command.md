# PUBLISH (Command)

Publishes a set of drawings to PDF files or printers.

## Access Methods

*Toolbar*:
![](../images/GUID-4C887873-7EE8-497C-B87D-0D8DFF393268-low.png)

*Menu:
File ![](../images/ac.menuaro.gif) Batch Publish*

The Batch Publish dialog box is displayed. You can assemble a collection of drawings, and create a PDF file or hardcopy of
the selected drawings. PDF files can be viewed with a PDF viewer.

If you enter +PUBLISH at the Command prompt, you can select an existing Drawing Set Descriptions (DSD) file from a standard
file selection dialog box. The Publish dialog box opens and displays the drawing set saved in the DSD file.
