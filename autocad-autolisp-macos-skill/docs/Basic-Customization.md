# Basic Customization

#### Topics in this section

* [About Customization](About-Customization.md)

  AutoCAD and AutoCAD-based products, including AutoCAD LT, can be customized to improve your productivity with the product
  and help enforce CAD standards.
* [About Program and Support Files](About-Program-and-Support-Files.md)

  You can change the default directory structure for the program and support files to suit your needs.
* [About Defining Custom Commands](About-Defining-Custom-Commands.md)

  Custom command aliases and external commands can be defined to improve access to the commands and applications you use most
  frequently.
