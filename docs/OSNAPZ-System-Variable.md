# OSNAPZ (System Variable)

Controls whether object snaps are automatically projected onto a plane parallel to the XY plane of the current UCS at the
current elevation.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 0 |

| 0 | Osnap uses the *Z*-value of the specified point |
| 1 | Osnap substitutes the *Z*-value of the specified point with the elevation ([ELEV](ELEV-Command.md)) set for the current UCS |
