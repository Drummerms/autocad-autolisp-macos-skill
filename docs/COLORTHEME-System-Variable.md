# COLORTHEME (System Variable)

Sets the color theme of the toolsets, palettes, and several other interface elements to dark or light.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| Value | Description |
| 0 | Displays the dark theme. |
| 1 | Displays the light theme. |
