# To Edit Attribute Values and Properties for a Block Reference

Edits attributes in a block reference.

Use this procedure to modify attributes in a block reference without affecting the block or attribute definition.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Attribute, Single.
   ![](../images/GUID-91BCBDF8-1AB3-4876-B9D5-4959DE7B6ABD-low.png)
2. Select the block to edit.
3. In the Enhanced Attribute Editor, select the attribute you want to edit. You can change the attribute value or choose another
   tab and edit other attribute properties.
4. Make the attribute changes you want, and click OK.
