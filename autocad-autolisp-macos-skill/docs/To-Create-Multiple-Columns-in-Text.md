# To Create Multiple Columns in Text

1. Double-click a multiline text object to open the In-Place Text Editor.
2. Adjust the column height by moving the arrows located on the bottom left of the first column.
3. Adjust the column width by moving the arrows located on the ruler along the top.

   NOTE:

   The arrows on the ruler on the upper right side only adjust gutter width, not column width.
4. Right click In-Place Text Editor, to select a column option and suboption from shortcut menu. You have a choice between Dynamic
   or Static columns.

   You have two suboptions with Dynamic Columns - Auto height or Manual height. Static Columns allows you to choose the number
   of columns
