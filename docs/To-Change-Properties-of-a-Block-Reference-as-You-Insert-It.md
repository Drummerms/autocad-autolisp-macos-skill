# To Change Properties of a Block Reference as You Insert It

1. Click
   Window menu ![](../images/ac.menuaro.gif) Properties Inspector.
2. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block.
   ![](../images/GUID-6857E103-73A1-42F1-AA87-3265189DB33B-low.png)
3. In the Insert Block dialog box, in the Name box, select a name from a list of block definitions. Select Specify On-Screen
   to use the pointing device to specify one or more of the following:
   * Insertion Point
   * Scale
   * Rotation
4. Click OK.
5. In the Properties window, change the properties of the block (or dynamic block) reference.
