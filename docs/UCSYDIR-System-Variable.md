# UCSYDIR (System Variable)

Stores the Y direction of the current UCS for the current viewport in the current space.

(Read-only)

|  |  |
| --- | --- |
| Type: | 3D-point |
| Saved in: | Drawing |
| Initial value: | 0.0000,1.0000,0.0000 |

The setting of this system variable is viewport specific.
