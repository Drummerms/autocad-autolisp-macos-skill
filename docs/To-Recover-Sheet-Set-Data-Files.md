# To Recover Sheet Set Data Files

In the unlikely event of DST file corruption or a major user error, the previously saved sheet set data file can be recovered.
Every time the sheet set data file is opened, the current sheet set data file is copied to a backup file (DS$). This backup
file has the same file name and is located in the same folder as the current sheet set data file.

1. Make sure that no one else on your network is working on the sheet set.
2. (Recommendation) Copy the existing DST file to another file name.
3. Rename the backup DST file from the DS$ file extension to the DST file extension.

TIP:The data stored in the sheet set data file represents a significant amount of work, so you should take the same care to create
backups of DST files as you do for drawing files.
