# To Work With Applying Materials

## Apply a Material From the Materials Browser

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Render panel ![](../images/ac.menuaro.gif) Materials.
   ![](../images/GUID-C875B4CB-F18D-4D3C-808C-1FA47D276FE4-low.png)
2. Select the object.
3. Right-click a material and choose Assign to Selection.

   The material is applied to the object.

## Remove a Material From an Object

1. Select the object, right-click and select Properties.
2. Select Global from the Materials drop down.

   The material is removed from the object and replaced by the Global material.
