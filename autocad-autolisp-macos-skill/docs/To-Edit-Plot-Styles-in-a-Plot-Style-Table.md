# To Edit Plot Styles in a Plot Style Table

1. Click
   File menu ![](../images/ac.menuaro.gif) Plot Styles.
2. Double-click the plot style table you want to change.
3. In the Plot Style Table Editor, Plot Styles tab, select a plot style and edit the settings.
4. Click Save & Close.
