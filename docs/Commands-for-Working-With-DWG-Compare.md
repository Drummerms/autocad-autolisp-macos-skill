#### System Variables

* [COMPARECOLORCOMMON (System Variable)](GUID-57BFAA05-BA15-4E17-976B-A40A96A31FBB.htm "Inserts the information about the two compared drawing files in to your current drawing.
  		")
* [COMPARECOLOR1 (System Variable)](GUID-EE825A2E-F54B-4653-9AE0-DDA7AD4A04CA.htm "Sets the color of the objects that only exists in the first drawing in the resultant compared drawing file.
  		")
* [COMPARECOLOR2 (System Variable)](GUID-C8F5443E-1D02-41A7-8948-0A3EA23CEFB4.htm "Sets the color of the objects that only exists in the second drawing in the resultant compared drawing file.
  		")
* [COMPAREFRONT (System Variable)](GUID-A0F50F24-2D9F-4B18-897F-492A0C6F16F2.htm "Controls the default display order of overlapping objects in the compare result drawing.
  		")
* [COMPAREHATCH (System Variable)](GUID-BBB5E4A0-B607-4898-9A6B-A65C51551EE5.htm "Controls whether hatch objects are included in the drawing comparison.
  		")
* [COMPARERCMARGIN (System Variable)](GUID-7A230058-048B-4EE6-949D-105AF6AC8E73.htm "Specifies the offset distance from the bounding box of a revision cloud that contains the object differences in the compare result drawing.
  		")
* [COMPARETEXT (System Variable)](GUID-1BE58261-FA5F-4914-BAC6-C1DF7E3D1E9C.htm "Controls whether text objects are included in the drawing comparison.
  		")
* [COMPAREPROPS (System Variable)](GUID-FC52193A-3801-42D1-B5C3-873B192B36B2.htm "Controls whether a change in a non-geometric (display) property is identified as a change between two drawings revisions.
  		")
* [COMPARERCSHAPE (System Variable)](GUID-D1560319-05D9-4556-876B-A59D9E1419B9.htm "Controls whether individual changes are merged as a single large rectangle or a series of smaller rectangle in the compare result drawing.
  		")
* [COMPARESHOWCOMMON (System Variable)](GUID-07FE3435-EC0F-4473-AF79-5D1E5A7B9DAE.htm "Displays the objects that are identical in both the drawings that are being compared.
  		")
* [COMPARESHOW1 (System Variable)](GUID-6DCC6EB9-F301-4854-BAFD-B1E553C7BC63.htm "Displays the objects that exists only in the first drawing.
  		")
* [COMPARESHOW2 (System Variable)](GUID-8E3C5248-A2AE-4C26-A8A3-FCC9E1A846EF.htm "Displays the objects that exists only in the second drawing.
  		")
* [COMPARESHOWRC (System Variable)](GUID-2F394248-AE3F-41AD-A45D-7CCDC4EFE3D0.htm "Shows a revision cloud around the difference in the compare result drawing.
  		")
