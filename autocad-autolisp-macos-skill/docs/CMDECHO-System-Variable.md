# CMDECHO (System Variable)

Controls whether prompts and input are echoed during the AutoLISP command function.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 1 |

| 0 | Turns off echoing |
| 1 | Turns on echoing |
