# DIVMESHWEDGESLOPE (System Variable)

Sets the number of subdivisions in the slope that extends from the apex of the wedge to the edge of the base.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 3 |

![](../images/GUID-1325DAF3-D21C-47C3-9F52-D49E29676AF5-low.png)

This setting affects new mesh wedges that are created with the [MESH](MESH-Command.md) command.
