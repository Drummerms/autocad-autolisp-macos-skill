# S Functions

#### Topics in this section

* [set (AutoLISP)](set-AutoLISP.md)

  Sets the value of a quoted symbol name to an expression
* [setcfg (AutoLISP)](setcfg-AutoLISP.md)

  Obsolete. Writes application data to the AppData section of the
  acad20xx.cfg
  file
* [setenv (AutoLISP)](setenv-AutoLISP.md)

  Sets a system environment variable to a specified value
* [setfunhelp (AutoLISP)](setfunhelp-AutoLISP.md)

  Registers a user-defined command with the Help facility so the appropriate Help file and topic are called when the user requests
  help on that command
* [setpropertyvalue (AutoLISP)](setpropertyvalue-AutoLISP.md)

  Sets the property value for an entity
* [setq (AutoLISP)](setq-AutoLISP.md)

  Sets the value of a symbol or symbols to associated expressions
* [setvar (AutoLISP)](setvar-AutoLISP.md)

  Sets an AutoCAD system variable to a specified value
* [setview (AutoLISP)](setview-AutoLISP.md)

  Establishes a view for a specified viewport
* [showhtmlmodalwindow (AutoLISP)](showhtmlmodalwindow-AutoLISP.md)

  Displays a modal dialog box with a specified URI (Uniform Resource Identifier)
* [sin (AutoLISP)](sin-AutoLISP.md)

  Returns the sine of an angle as a real number expressed in radians
* [snvalid (AutoLISP)](snvalid-AutoLISP.md)

  Checks the symbol table name for valid characters
* [sqrt (AutoLISP)](sqrt-AutoLISP.md)

  Returns the square root of a number as a real number
* [ssadd (AutoLISP)](ssadd-AutoLISP.md)

  Adds an object (entity) to a selection set, or creates a new selection set
* [ssdel (AutoLISP)](ssdel-AutoLISP.md)

  Deletes an object (entity) from a selection set
* [ssget (AutoLISP)](ssget-AutoLISP.md)

  Creates a selection set from the selected object
* [ssgetfirst (AutoLISP)](ssgetfirst-AutoLISP.md)

  Determines which objects are selected and gripped
* [sslength (AutoLISP)](sslength-AutoLISP.md)

  Returns an integer containing the number of objects (entities) in a selection set
* [ssmemb (AutoLISP)](ssmemb-AutoLISP.md)

  Tests whether an object (entity) is a member of a selection set
* [ssname (AutoLISP)](ssname-AutoLISP.md)

  Returns the object (entity) name of the indexed element of a selection set
* [ssnamex (AutoLISP)](ssnamex-AutoLISP.md)

  Retrieves information about how a selection set was created.
* [sssetfirst (AutoLISP)](sssetfirst-AutoLISP.md)

  Sets which objects are selected and gripped
* [startapp (AutoLISP)](startapp-AutoLISP.md)

  Starts an external application
* [strcase (AutoLISP)](strcase-AutoLISP.md)

  Returns a string where all alphabetic characters have been converted to uppercase or lowercase
* [strcat (AutoLISP)](strcat-AutoLISP.md)

  Returns a string that is the concatenation of multiple strings
* [strlen (AutoLISP)](strlen-AutoLISP.md)

  Returns an integer that is the number of characters in a string
* [subst (AutoLISP)](subst-AutoLISP.md)

  Searches a list for an old item and returns a copy of the list with a new item substituted in place of every occurrence of
  the old item
* [substr (AutoLISP)](substr-AutoLISP.md)

  Returns a substring of a string
