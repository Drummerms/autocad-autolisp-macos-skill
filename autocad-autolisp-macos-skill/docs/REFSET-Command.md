# REFSET (Command)

Adds or removes objects from a working set during in-place editing of a reference, either an xref or a block definition.

## Access Methods

![](../images/ac.mouse.gif) Toolbar: Reference Editor visor ![](../images/ac.menuaro.gif) Add to Working Set

![](../images/ac.mouse.gif) Toolbar: Reference Editor visor ![](../images/ac.menuaro.gif) Remove from Working Set

## Summary

Objects that are part of the working set are visually distinct from other objects in the current drawing. All objects in the
current drawing, except objects in the working set, appear faded.

## List of Prompts

The following prompts are displayed.

Transfer objects between the Refedit working set and host drawing...

Enter an option [[Add](GUID-B441D1B3-A1F6-49F7-A43F-B74DB39FAAA2.htm#GUID-FDAB9A0C-2B45-4C30-A9FF-4C09C2E158DC)/[Remove](GUID-B441D1B3-A1F6-49F7-A43F-B74DB39FAAA2.htm#GUID-F8285FEF-47D9-4A9A-A8F9-DD12BBB890A1)] <Add>: *Enter an option or press* Enter

Add ![](../images/GUID-35C5C9D2-D442-4FAC-99D3-1A85BC2CEAEC-low.png)
:   Adds objects to the working set.

    An object that is part of the working set is added to the reference when changes are saved back, and the object is removed
    from the current drawing.

Remove ![](../images/GUID-1B23D72E-2498-44BF-B886-532F49C5DEB6-low.png)
:   Removes objects from the working set.

    An object that is removed from the working set is removed from the reference when changes are saved back; the object is also
    removed from the current drawing.
