# DCCONVERT (Command)

Converts associative dimensions to dimensional constraints.

## Summary

This command is equivalent to the Convert option in
[DIMCONSTRAINT](DIMCONSTRAINT-Command.md).

Objects other than associative dimensions are ignored and filtered from the selection set.

## List of Prompts

The following prompts are displayed.

Select associative dimensions to convert:
*Select the associative dimensions to be converted to dimensional constraints*
