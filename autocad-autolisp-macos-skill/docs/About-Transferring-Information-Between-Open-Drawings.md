# About Transferring Information Between Open Drawings

Copy objects between drawings that are open in a single session.

When you open several drawings in a single session, you can perform operations that can copy the objects between the open
drawings.

* Reference other drawings with the XREF command.
* Use the Copy with Basepoint (COPYBASE) or Cut with Basepoint (CUTBASE), and Paste to Original Coordinates (PASTEORIG) commands
  to copy and paste between drawings.
* Use WBLOCK to save drawing information into a separate drawing file, which can be subsequently inserted into a different drawing.
  This command is the equivalent of performing a SAVEAS on selected objects.

IMPORTANT:The Copy-Paste feature in Windows can be used to transfer objects between various applications, but it does not maintain the
highest level of precision within AutoCAD. To maintain precision, use COPYBASE or CUTBASE rather than COPYCLIP (Ctrl+C).
