# Create Array Visor - Rectangular Array

Modifies the properties of the rectangular array as it is being created.

![](../images/GUID-872F0116-375D-4422-91BC-685A4A720454-low.png)

## Summary

The Create Array visor is displayed after selecting the objects to array while using the Rectangular option of the ARRAY command
or the ARRAYRECT command.

## List of Options

The following options are displayed.

Base Point

Specifies a base point for the array.

Columns

Specifies the number of columns.

Rows

Specifies the number of rows.

Save Changes

Saves the changes to the array and exits the command.
