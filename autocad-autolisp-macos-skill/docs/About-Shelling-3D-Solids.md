# About Shelling 3D Solids

Convert a 3D solid to a hollow wall, or shell.

When you can convert a solid object to a shell, new faces are created by offsetting existing faces inside or outside their
original positions.

![](../images/GUID-80438ED2-46E9-4625-AAB0-63B6FDEBA019-low.png)

Continuously tangent faces are treated as a single face when they are offset.
