# DIMLUNIT (System Variable)

Sets units for all dimension types except Angular.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 2 |

| 1 | Scientific |
| 2 | Decimal |
| 3 | Engineering |
| 4 | Architectural (always displayed stacked) |
| 5 | Fractional (always displayed stacked) |
| 6 | System Format |
