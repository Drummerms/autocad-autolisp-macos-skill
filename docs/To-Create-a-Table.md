# To Create a Table

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/ac.menuaro.gif) Table.
   ![](../images/GUID-2E433CD0-2201-4BE7-97D5-5242CC65AF43-low.png)

   The Insert Table dialog box displays.

   ![](../images/GUID-AD4EAE6C-41A4-4A6C-8411-112837470369-low.png)
2. Select the table style to use for the table.
3. Select an option for the table insertion:
   * *Specify Insertion Point*: After clicking OK, you are prompted for an insertion point which is used for the upper-left corner of the table.
   * *Specify Window*: After clicking OK, you are prompted to specify the upper left and lower right corners for the table.

   The cursor is placed in the top cell for entering the text and the Text Editor visor displays.

   ![](../images/GUID-4FFEE9DF-FE91-45E3-8936-917470A8EACF-low.png)
4. Enter text as needed. Use the Tab key to move from one cell to the next cell.
5. Click Save on the visor to save your changes and exit the table.

NOTE:The current table style is assigned to the table.
