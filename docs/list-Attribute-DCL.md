# list Attribute (DCL)

Specifies the initial set of lines (choices) to be placed in the
popup\_list or
list\_box.

*Supported Platforms:* Windows, Mac OS, and Web

```
list = "string";
```

Possible value is a quoted string (no default). Lines are separated by a new line symbol (\
). Tab characters (\t) can occur within each line.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
