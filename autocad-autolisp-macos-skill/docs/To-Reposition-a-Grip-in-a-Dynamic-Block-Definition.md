# To Reposition a Grip in a Dynamic Block Definition

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block Editor.
   ![](../images/GUID-E3DDBE51-C92A-4D07-A51D-C7C685E4F578-low.png)
2. In the Edit Block Definition dialog box, select the block whose grips you want to modify.
3. In the Block Editor drawing area, select a custom grip (not the standard grip associated with the key point of the parameter).
4. Do one of the following to reposition the grip:
   * Drag the grip to another position in the block space.
   * On the Properties Inspector, under Geometry, enter values for the Base X and Base Y properties. (These values are relative
     to the base point of the parameter.)
5. Click
   Block Editor visor
   ![](../images/ac.menuaro.gif) Save.
   ![](../images/GUID-5E815E0C-AED9-4DCE-AC3D-3AA0C6FF0DA6-low.png)
6. Click
   Block Editor visor
   ![](../images/ac.menuaro.gif) Exit.
   ![](../images/GUID-F3CD8D62-2BA2-42C6-99E8-681C38D935DD-low.png)
