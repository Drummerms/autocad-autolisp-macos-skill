# ANNOSCALEZOOM (System Variable)

Determines whether the mouse wheel zoom in paperspace viewports is controlled by specific zoom scales or independent of viewport
scales (legacy behavior).

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| Value | Description |
| 0 | Off - mouse wheel zoom scales independent of viewport scales (legacy behavior) |
| 1 | On - mouse wheel zoom uses the scales of the current viewport |
