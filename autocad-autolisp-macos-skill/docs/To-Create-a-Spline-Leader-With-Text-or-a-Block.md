# To Create a Spline Leader With Text or a Block

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Leader panel ![](../images/ac.menuaro.gif) Multileader.
   ![](../images/GUID-31CC8661-13A6-4D2B-968D-39BE64E908F8-low.png)
2. At the Command prompt, enter
   *o* to select options.
3. Enter
   *l* to specify leaders.
4. Enter
   *t* to specify the leader type.
5. Enter
   *p* to specify a spline leader.
6. In the drawing, click a start point for the leader head.
7. Click the end point for the leader.
8. Enter your MTEXT content.
9. In the Text Formatting toolbar, click OK.

   ![](../images/GUID-9B7F4F95-141D-4B06-88B8-AA6820A5DA75-low.png)
