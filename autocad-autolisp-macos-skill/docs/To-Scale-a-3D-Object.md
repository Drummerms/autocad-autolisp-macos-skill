# To Scale a 3D Object

Use this method to resize the objects only along a plane that you specify.

1. At the Command prompt, enter 3DSCALE.
2. Select the objects and subobjects you want to scale using the following methods:
   * Press and hold Ctrl to select subobjects (faces, edges, and vertices).
   * Release Ctrl to select entire objects.
3. When you have selected all objects, press Enter.

   The 3D Scale gizmo is displayed at the center of the selected object or objects.
4. Do one of the following:
   * *To scale along a plane:* Hover the cursor over one of the bars found between each of the axes of the gizmo until it turns yellow. Click the yellow
     bar.
   * *To scale uniformly:* Hover the cursor over the triangular area closest to the center point of the gizmo until it turns yellow. Click the yellow
     area.
   * *To scale along an axis:* Hover the cursor over one of the axes of the gizmo until it turns yellow. Click the yellow axis.

   ![](../images/GUID-6A75B160-09FE-4B4D-95CC-42C535E7572E-low.png)
5. To resize the selection, drag and release, or enter a scale factor while holding the mouse button.
