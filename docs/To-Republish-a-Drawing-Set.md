# To Republish a Drawing Set

After drawings have been updated, you may want to republish a drawing set for viewing or plotting.

1. Click
   ![](../images/GUID-4C887873-7EE8-497C-B87D-0D8DFF393268-low.png).

   If you have a drawing open, all the layouts in your drawing are shown in the sheet list in the Batch Publish dialog box. Right-click
   in the sheet list. Click Remove All to delete the layouts from the list of drawing sheets.
2. In the page setup column, click the Add Sheets in the shortcut menu. In the Select Drawings dialog box, select drawings.
   Click Select to add them to the list of sheets in the Batch Publish dialog box.

   The drawing sheets in the saved drawing set are displayed in the sheet list in the Publish dialog box.
3. In the Publish to group, do one of the following:
   * To publish to a PDF file — click PDF format, and select PDF File.
   * To publish to a plotter or printer — click Plotter Named in Page Setup.
4. Click Publish.
