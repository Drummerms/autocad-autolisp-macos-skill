# ID (Command)

Displays the UCS coordinate values of a specified location.

## Access Methods

![](../images/ac.mouse.gif) Menu:   Tools
 ![](../images/ac.menuaro.gif)  Inquiry
 ![](../images/ac.menuaro.gif)  ID Point Not available in menus in the current workspace

![](../images/ac.keyboard.gif) Command entry:  *'id* for transparent use

## Summary

ID lists the
*X*,
*Y*, and
*Z* values of the specified point and stores the coordinate of the specified point as the last point. You can reference the last
point by entering
*@* at the next prompt that requests a point.

## List of Prompts

The following prompts are displayed.

Specify Point:
*Use the pointing device to specify a point*

The UCS coordinate values of the location are displayed at the Command prompt.

If you snap to an object in 3D space, the
*Z* coordinate value is the same as that of the selected feature of the object.
