# Select Linetype Dialog Box

Displays linetypes available in the current drawing.

LAYER (Command)

*Menu*:
Format
![](../images/ac.menuaro.gif) Layers:
 Click the layer's current linetype setting

![](../images/GUID-6DA97F68-1EDB-4FC7-97DD-9FD1E5ADC731-low.png)

## List of Options

The following options are displayed.

Loaded Linetypes
:   Displays the list of linetypes loaded in the current drawing.

Load
:   Displays the
    [Load or Reload Linetypes dialog box](GUID-8B84AA03-B157-4C5D-8B30-AB94C8890B2B.htm#WSC30CD3D5FAA8F6D813D93F4FFC2D60BB4-7FA3), in which you can load selected linetypes into the drawing and add them to the list of linetypes.
