# To Set a True Color for All New Objects

1. Click
    Format menu ![](../images/ac.menuaro.gif) Color.
2. In the Object Color drop-down list, click Select Colors.
3. In the Select Color dialog box, True Color tab, click the Color Model drop-down list, and do one of the following:
   * Choose the HSL color model, and specify a color by entering a color value in the Color box, or by specifying values in the
     Hue, Saturation, and Luminance boxes.
   * Choose the RGB color model, and specify a color by entering a color value in the Color box, or by specifying values in the
     Red, Green, and Blue boxes.
