# -VPORTS (Command)

Creates multiple viewports in model space or paper space.

## Summary

The command prompts available depend on whether you are configuring [model viewports](VPORTS-Model-Space-Viewports.md) (on the Model tab) or [layout viewports](VPORTS-Layout-Viewports.md) (on a layout).
