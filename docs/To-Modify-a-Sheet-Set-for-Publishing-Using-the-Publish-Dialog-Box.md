# To Modify a Sheet Set for Publishing Using the Publish Dialog Box

1. Click
   Window ![](../images/ac.menuaro.gif) Sheet Set Manager.
2. In the Sheet Set Manager, select the node or name of a sheet set, subset, or an individual sheet.
3. In the upper-right corner of the Sheet Set Manager, click Publish.
   ![](../images/GUID-B9A20323-6D41-4E5E-81BC-2640792C3822-low.png)

   The Batch Publish dialog box is displayed and lists the sheets you selected in the Sheet Set Manager.
4. In the Batch Publish dialog box, modify the sheet list and settings as required.
5. Click Publish to publish the modified sheet set.
