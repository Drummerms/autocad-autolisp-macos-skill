# Data Processing for Markup Import and Markup Assist

AutoCAD Markup Import and Markup Assist are cloud-native features that process your imported markups and your drawing files
in the Autodesk cloud (servers owned or controlled by Autodesk) with the use of machine learning technologies. To provide
functionality, these features depend on analyzing and processing your data. For example, text and object recognition, predictions,
or suggestions, and the continuous improvement of the features is only possible with continuous learning from the data.

NOTE:Markup Import and Markup Assist are not available on AutoCAD LT or AutoCAD LT for Mac.

## What customer data does Markup Import and Markup Assist process?

These features process customer markup data, such as an image or PDF of a drawing with markup, and the underlying drawing
for reference. AutoCAD's use of markup data is limited to providing and improving these product features and your data is
not used for any other purposes.

## How to disable Markup Import and Markup Assist

If you have concerns about importing data to the Autodesk cloud, you can choose not to use these features, or your administrator
can disable the features in the Autodesk Account portal.
