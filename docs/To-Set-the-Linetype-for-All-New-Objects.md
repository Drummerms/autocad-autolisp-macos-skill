# To Set the Linetype for All New Objects

1. Click
   Format menu ![](../images/ac.menuaro.gif) Linetype.
2. If the linetype that you want to use is displayed in the list, double-click the linetype to make it current.
3. If the linetype that you want to use is not displayed in the list, click + and do one of the following:
   * Choose a linetype and click Add, then double-click on the linetype to make it current.
   * Choose BYLAYER to draw new objects in the linetype assigned to the current layer.
   * Choose BYBLOCK to draw new objects in the Continuous linetype until they are grouped into a block. When the block is inserted
     into a drawing, the objects in the block display the current linetype.

NOTE:You can hold down Ctrl to load several linetypes or hold down Shift to load a range of linetypes.
