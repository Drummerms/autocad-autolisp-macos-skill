# SELECTSIMILAR (Command)

Adds similar objects to the selection set based on selected objects.

## Access Methods

*Toolbar*:
![](../images/GUID-CEC6378A-3D4C-4061-9808-14D3D6B65A93-low.png)

*Shortcut Menu*: Select the object that you want to use as the template object to select similar objects. Right-click in the drawing area
and click Select Similar.

## Summary

Selects similar objects of the same type based on specified matching properties, such as the color or block name.

## List of Prompts

The following prompt is displayed.

Select objects or [SEttings]: *Use an object selection method and press* Enter

The Settings option displays the Select Similar Settings dialog box.
