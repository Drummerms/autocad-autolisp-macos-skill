Saves, restores, and manages sets of layer settings that are called layer states.

## Summary

Displays a list of saved layer states in the drawing. You can create, rename, edit, and delete layer states.

## List of Options

The following options are displayed.

Layer States
:   Lists the layer states that have been saved in the drawing.

Do Not List Layer States in Xrefs
:   Controls whether layer states in xrefs are displayed.

Save & Close
:   Saves the settings and closes the Layer States Manager.

More Options

Layer Properties to Restore

Applies only the specified layer property settings when the specified layer state is restored. Click the Layer Properties
to Restore drop-down menu to view the Select All and Clear All options.

Select All
:   Selects all the layer property settings.

Clear All
:   Clears all the layer property settings.
