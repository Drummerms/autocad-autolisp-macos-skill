# Commands for Editing Specific Objects

#### Commands

* [3DEDITBAR](GUID-5A1D1B60-B9DF-447F-BA38-188EE797EF9B.htm "Reshapes splines and NURBS surfaces, including their tangency properties.
  		")
* [BREAK](GUID-7F63FF08-BF72-4975-8AFA-B6B3DC058F35.htm "Breaks the selected object between two points.
  		")
* [COPY](GUID-A1003C74-BAF0-4145-B057-C63D465A920E.htm "Copies objects a specified distance in a specified direction.
  		")
* [COPYBASE](GUID-59113CD3-B5EC-404B-989C-F98F4B70EDB5.htm "Copies selected objects to the Clipboard along with a specified base point.
  		")
* [COPYCLIP](GUID-8C693775-7BB2-48D6-9A6B-770F8A870707.htm "Copies selected objects to the Clipboard.
  		")
* [COPYHIST](GUID-2B86E8F7-B846-4113-B9EE-1F5010A8515D.htm "Copies the text in the command line history to the Clipboard.
  		")
* [EXPLODE](GUID-B38FD713-AD2F-4D9E-AD68-60E3F7E32D88.htm "Breaks a compound object into its component objects.
  		")
* [HELIX](GUID-5B6FB3FE-F66D-4B09-8744-C8B33E435C6D.htm "Creates a 2D spiral or 3D spring.
  		")
* [JOIN](GUID-09013620-4CD5-4FBC-AF75-AA7A92AE5CA0.htm "Joins the endpoints of linear and curved objects to create a single object.
  		")
* [MIRROR](GUID-C14BACE4-EB78-4AC4-AA17-26858CA5ABA9.htm "Creates a mirrored copy of selected objects.
  		")
* [MLEDIT](GUID-BDA85006-C833-4165-85A0-1788A92BC912.htm "Edits multiline intersections, breaks, and vertices.
  		")
* [PEDIT](GUID-C0E37352-51BE-4B67-B984-069BD5341CCD.htm "Edits polylines, objects to be joined to polylines, and related objects.
  		")
* [REVERSE](GUID-F6FA41D9-6460-4294-BA33-AA7DE4CCB6CC.htm "Reverses the vertices of selected lines, polylines, splines, and helixes, which is useful for linetypes with included text, or wide polylines with differing beginning and ending widths.
  		")
* [SPLINEDIT](GUID-216FEC99-80A1-4D1E-BC5B-010690C6FED1.htm "Modifies the parameters of a spline or converts a spline-fit polyline to a spline.
  		")

#### System Variables

* [PEDITACCEPT](GUID-ED35EB02-53AD-4725-97DD-346C7E01974D.htm "Automatically converts selected objects into polylines without a prompt when using PEDIT.
  		")
* [PLINECONVERTMODE](GUID-AB5B75E5-2302-454E-B4D5-5C0DBD7EF1F6.htm "Specifies the fit method used when converting splines to polylines.
  		")
* [PLINEGCENMAX](GUID-98FF3F46-C887-461C-AF0C-A3E6EB0C73A9.htm " Sets the maximum number of segments that a polyline can have for the application to calculate the geometric center.
  		")
* [PLINEGEN](GUID-352844E4-FBEA-44E0-B8F3-41C4187AD9C7.htm "Controls how linetype patterns generate around the vertices of a 2D polyline when it is drawn.
  		")
* [PLINEREVERSEWIDTHS](GUID-412EECFC-6AEB-4D44-9D74-DCF9BFB4D96C.htm " Controls the appearance of a polyline when its direction is reversed.
  		")
* [PLINETYPE](GUID-F6027128-2707-4BBE-A0A4-C757435D4874.htm "Specifies whether optimized 2D polylines are used.
  		")
* [PLINEWID](GUID-35EEB892-15C9-442F-847D-2F4DC52E9690.htm "Stores the default polyline width.
  		")
* [SPLINESEGS](GUID-938743E7-E438-4E93-9A9B-1514E0BADB47.htm "Sets the number of line segments to be generated for each spline-fit polyline generated by the Spline option of the PEDIT command.
  		")
* [SPLINETYPE](GUID-BDD2D1EF-9A72-47A6-9AAB-8877C76F05CF.htm "Sets the type of curve generated by the Spline option of the PEDIT command.
  		")
* [SPLKNOTS](GUID-134FD0A1-4396-4748-9FF4-C82B7BC24E28.htm "Stores the default knot option for the SPLINE command when specifying fit points.
  		")
* [SPLMETHOD](GUID-300EB1EF-E101-421C-A8B8-6D91E2873787.htm "Stores whether the default method used for the SPLINE command is fit points or control vertices.
  		")
* [SPLPERIODIC](GUID-60D7953C-E22D-4CF3-B779-F776592A5F23.htm "Controls whether closed splines and NURBS surfaces are generated with periodic properties to maintain the smoothest continuity at the closure point or seam.
  		")
* [SURFU](GUID-651DFAD4-8A22-4C48-9DD5-E5751BDBE055.htm "Sets the surface density for PEDIT Smooth in the M direction and the U isolines density on surface objects.
  		")
* [SURFV](GUID-13E30946-F85A-4E74-8B2D-5A627C21093E.htm "Sets the surface density for PEDIT Smooth in the N direction and the V isolines density on surface objects.
  		")
