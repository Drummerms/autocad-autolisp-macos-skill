# ETRANSMIT (Command)

Packages a set of drawings and dependency files into one unit for sharing.

## Access Methods

*Toolbar*:
![](../images/GUID-D5ECD674-84D1-4B3A-AB6A-F2264AA36BC2-low.png)

*Menu*:
File ![](../images/ac.menuaro.gif) Package Drawing.

## Summary

The
Package Drawing dialog box is displayed.

If you enter *-ETRANSMIT* at the Command prompt, options are displayed.

When you select a set of drawing files in a transmittal package, it automatically includes all related dependent files such
as xrefs and font files.
