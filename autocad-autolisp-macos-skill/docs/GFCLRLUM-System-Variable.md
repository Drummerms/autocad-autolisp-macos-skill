# GFCLRLUM (System Variable)

Controls the tint or shade level in a one-color gradient fill.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Not-saved |
| Initial value: | 1.0000 |

A tint gradient transitions toward white and a shade gradient transitions toward black. Valid values range from 0.0 (darkest)
to 1.0 (lightest).
