# DWGCHECK (System Variable)

Checks drawings for potential problems when opening them.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | If a drawing that you try to open has a potential problem, you are warned before the drawing is opened. |
| 1 | If a drawing that you try to open has a potential problem, or if it was saved by an application other than one released by Autodesk or based on RealDWG, you are warned before the drawing is opened. |
| 2 | If a drawing that you try to open has a potential problem, you are notified at the Command prompt. |
| 3 | If a drawing that you try to open has a potential problem, you are notified at the Command prompt. If the drawing has a potential problem and the drawing was saved by an application other than one released by Autodesk or based on RealDWG, you are warned before the drawing is opened. |
