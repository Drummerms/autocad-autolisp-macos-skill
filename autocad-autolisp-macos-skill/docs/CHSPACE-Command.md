# CHSPACE (Command)

On a layout, transfers selected objects between model space and paper space.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Change Space.
![](../images/GUID-1F36D443-FC78-43E3-88C7-6CD372388B90-low.png)

*Menu*:
Modify ![](../images/ac.menuaro.gif) Change Space.

The selected objects are scaled automatically in the new space.

The following prompts are displayed.

Select objects
:   Select the objects that you want to transfer to a different space.

Set the TARGET viewport active (current)
:   Specifies model space in a layout viewport as the destination for the transferred paper space objects. Click inside the layout
    viewport to make it the current one and press Enter. The transferred objects align visually with their former location.

Set the SOURCE viewport active (current)
:   Specifies paper space as the destination for the transferred objects. Click inside a layout viewport to make it the current
    one and press Enter. The transferred objects align visually with their former location.
