# IMAGEDEF\_REACTOR (DXF)

The following group codes are used by IMAGEDEF\_REACTOR objects.

| IMAGEDEF\_REACTOR group codes | |
| Group code | Description |
| 0 | Object name (IMAGEDEF\_REACTOR) |
| 5 | Handle |
| 100 | Subclass marker (AcDbRasterImageDefReactor) |
| 90 | Class version 2 |
| 330 | Object ID for associated image object |
