# To Change the Display Settings for Constraint Bars

1. Select a constrained object.
2. Right-click the constraint bar, and click Constraint Bar Settings.
3. In the Constraint Settings dialog box, on the Geometric tab, select the appropriate check boxes. Use the slider, or enter
   a value, to set the transparency level of constraint bars. Use a lower value for fainter, more transparent constraint bars.
