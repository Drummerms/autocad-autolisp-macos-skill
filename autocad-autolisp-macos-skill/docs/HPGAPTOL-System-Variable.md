# HPGAPTOL (System Variable)

Treats a set of objects that almost enclose an area as a closed hatch boundary.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Registry |
| Initial value: | 0.0000 |

The default value, 0, specifies that the objects enclose the area, with no gaps. Enter a value, in drawing units, from 0 to
5000 to set the maximum size of gaps that can be ignored when the objects serve as a hatch boundary.
