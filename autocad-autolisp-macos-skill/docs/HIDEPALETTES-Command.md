# HIDEPALETTES (Command)

Hides all currently displayed palettes, including the command window.

## Access Methods

*Menu*:
Window
![](../images/ac.menuaro.gif) Hide Palettes.

*Command Entry*: Tab

## Summary

Hides all currently displayed palettes such as the Command Line, Layers, and Properties Inspector.
