# Collection Manipulation Functions Reference (AutoLISP/ActiveX)

NOTE:ActiveX support in AutoLISP is limited to Windows only; not available on Mac OS or Web.

The following table provides summary descriptions of the AutoLISP ActiveX collection manipulation functions.

| Collection manipulation functions | | Platforms | | | | |
| Windows | | Mac OS | | Web |
| Function | Description | AutoCAD | AutoCAD LT | AutoCAD | AutoCAD LT | AutoCAD |
| [(vlax-for symbol collection [expression1 [expression2 ...]])](vlax-for-AutoLISP-ActiveX.md) | Iterates through a collection of objects, evaluating each expression (Visual LISP Function) | ✓ | ✓ | -- | -- | -- |
| [(vlax-map-collection obj function)](vlax-map-collection-AutoLISP-ActiveX.md) | Applies a function to all objects in a collection | ✓ | ✓ | -- | -- | -- |
