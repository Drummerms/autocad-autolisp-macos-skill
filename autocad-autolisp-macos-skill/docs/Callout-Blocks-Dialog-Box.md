# Callout Blocks Dialog Box

Lists, adds, or removes callout blocks for use in the current sheet set.

SHEETSET (Command)

*Menu:* Window ![](../images/ac.menuaro.gif) Sheet Set Manager:
Right-click a layout in Sheet Set Manager ![](../images/ac.menuaro.gif) Place Callout Block

![](../images/GUID-E0F758E8-F687-411A-A22D-A15506765707-low.png)

## List of Options

The following options are displayed.

### Preview

Displays a preview image when a block is selected.

### List of Blocks

Displays the list of blocks available for use with the current sheet set.

### Add Block (+)

Displays the Select Drawing dialog box. Select the drawing file to use as a callout block or a drawing that contains the blocks
you want to use.

### Remove Block (-)

Removes the selected block from the list of callout blocks.
