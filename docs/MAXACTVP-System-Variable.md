# MAXACTVP (System Variable)

Sets the maximum number of viewports that can be active at one time in a layout.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 64 |

MAXACTVP has no effect on the number of viewports that are plotted.
