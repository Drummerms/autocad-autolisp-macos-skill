# SPACEPANTIMEOUT (System Variable)

Sets the duration that the Spacebar must be held down before the PAN command is started transparently.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 250 |

Duration is measured in milliseconds. Valid range is 100 to 1000.
