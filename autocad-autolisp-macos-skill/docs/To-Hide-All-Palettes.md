# To Hide All Palettes

Use this method to hide and restore palettes.

1. At the Command prompt, enter HIDEPALETTES.
2. To turn on palettes that you've hidden, at the Command prompt, enter SHOWPALETTES.

   If a palette has been turned back on manually and moved, it is not affected by SHOWPALETTES.

   NOTE:You can also press Shift+CMD+H to show and hide palettes.
