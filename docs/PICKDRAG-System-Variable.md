# PICKDRAG (System Variable)

Controls the method of drawing a selection window.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 2 |

| 0 | Create a selection window using two points. Click once to begin a selection window, click again to complete the selection. |
| 1 | Create a selection window clicking and dragging. Release the mouse button to complete the selection. |
| 2 | Create a selection window using either of the methods above. |
