# DIVMESHBOXWIDTH (System Variable)

Sets the number of subdivisions for the width of a mesh box along the *Y* axis.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 3 |

![](../images/GUID-0132D51B-196C-4E2C-BE51-F9CD4C9360FA-low.png)

This setting affects new mesh boxes that are created with the [MESH](MESH-Command.md) command.
