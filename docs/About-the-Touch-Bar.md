# About the Touch Bar

For computers with the touch bar, the function keys are replaced with icons. The default touch bar contains the following
icons and corresponds to these function keys.

| Icon | Feature | Control Type | Function Key | Description |
| ![](../images/GUID-7C9407BB-CA9C-4A1C-B08D-85C448693909-low.png) | Help | Button | F1 | Displays Help for the active tooltip, command, palette or dialog box. |
| ![](../images/GUID-AFC1C102-0167-43DB-AB98-6A562F708082-low.png) | Expanded command line | Button | F2 | Displays an expanded command history in the Command window. |
| ![](../images/GUID-DEAF1244-966F-460B-AA47-3CF33E3E4148-low.png) | Object snap | Toggle | F3 | Turns object snap on and off. |
| ![](../images/GUID-331A1EC3-12FA-4D18-840B-CAFD4E7234CC-low.png) | Dynamic UCS (AutoCAD only) | Toggle | F6 | Turns automatic UCS alignment with planar surfaces on and off. |
| ![](../images/GUID-CD7987DD-AB67-4027-AD87-149DA630BFFB-low.png) | Grid display | Toggle | F7 | Turns the grid display on and off. |
| ![](../images/GUID-62FC01CB-D303-45C4-8D59-589FD9828446-low.png) | Ortho mode | Toggle | F8 | Locks cursor movement to horizontal or vertical. |
| ![](../images/GUID-BEBC9838-A4E4-4AEC-8612-5FE13E3058BC-low.png) | Grid snap | Toggle | F9 | Restricts cursor movement to specified grid intervals. |
| ![](../images/GUID-D32C1929-DF71-46A8-B5AD-D0D46289A3FC-low.png) | Polar tracking | Toggle | F10 | Guides cursor movement to specified angles. |
| ![](../images/GUID-9A63943B-D8CE-40DA-B4C7-487DE42335F9-low.png) | Object snap tracking | Toggle | F11 | Tracks the cursor horizontally and vertically from object snap locations. |
| ![](../images/GUID-598052FD-166B-48E0-9A64-DBB598CD8039-low.png) | Dynamic input | Toggle | F12 | Displays distances and angles near the cursor and accepts input as you use Tab between fields. |

NOTE:Press
*fn* to revert to the standard F1-F12 function keys.
