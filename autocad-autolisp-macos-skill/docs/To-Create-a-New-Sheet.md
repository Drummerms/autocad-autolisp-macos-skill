# To Create a New Sheet

As an alternative to importing existing layouts, you can create a new sheet.

1. In the Sheet Set Manager, right-click the sheet set node (at the top of the list), or an existing subset node.
2. Choose New Sheet.
3. In the New Sheet dialog box, enter the sheet information.
4. (Optional) Specify a different template.
5. Click Create.
6. In the Save Drawing As dialog box, click Save.
7. In the Sheet Set Manager, select the new sheet, if necessary.
8. In the Details panel, expand Sheet, and enter the name and other properties of the new sheet.
