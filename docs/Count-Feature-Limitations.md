# Count Feature Limitations

Some object types are not supported and are excluded from the count.

The Count feature does not support all object types in AutoCAD. The following are excluded from the count:

* Text
* Hatches
* 3D objects such as solids and meshes
* Images
* OLE objects
* Lights and cameras
* Non-DWG underlays
* Geographic data
* Attribute definitions
* External references
* Construction lines (xlines)
* Rays

NOTE:Only visible objects in the model space are displayed in the Count palette.
