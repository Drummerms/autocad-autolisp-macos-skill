# DTTEXT (System Variable)

Enables TrueType font performance improvement.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Disabled TrueType font performance improvement |
| 1 | Enabled TrueType font performance improvement |
