# Set Up the Drawing Environment

#### Topics in this section

* [About Setting Up the Drawing Area](About-Setting-Up-the-Drawing-Area.md)

  You can adjust the color and display schemes used in the application and drawing windows, and control the behavior of general
  features such as grip editing behavior.
* [About Tooltips](About-Tooltips.md)

  Several types of tooltips provide pop-up information for interaction with toolbars, object snaps, and drafting operations.
* [About Customizing Startup](About-Customizing-Startup.md)

  Command line switches can be used to control how the program is started from the Terminal window or a shell script.
* [About Rollover Tooltips](About-Rollover-Tooltips.md)

  Rollover tooltips display current values of certain properties when the cursor hovers over an object in the drawing area.
* [Interface Themes and Background Color](Interface-Themes-and-Background-Color.md)

  Many options are provided for customizing the look and feel of the product, including the color of the icons and the background
  color of the drawing area.
* [Model Space and Layouts](Model-Space-and-Layouts.md)

  There are two working environments, or spaces, in which you can work, model space and paper space layouts.
* [The UCS Icon](The-UCS-Icon.md)

  The drawing area displays an icon representing the XY axis of a rectangular coordinate system called the User Coordinate System,
  or UCS.
* [About the Touch Bar](About-the-Touch-Bar.md)
