# Special Unicode Characters

When entering text, you can create special characters, including the degree symbol, plus/minus tolerance symbol, and the diameter
symbol, by entering the following Unicode character strings.

## List of Options

The following options are displayed.

*\U+00B0*
:   Degrees symbol (°)

*\U+00B1*
:   ±±Tolerance symbol (±)

*\U+2205*
:   Diameter symbol (![](../images/GUID-B31171CF-EFE8-448B-8403-EB53D95702C3-low.png))
