# VPCONTROL (System Variable)

Controls whether the menus for viewport tools, views, and visual styles that are located in the upper-left corner of every
viewport are displayed.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | ON (or 1) |

| OFF (or 0) | Hides viewport controls. |
| ON (or 1) | Displays viewport controls. |
