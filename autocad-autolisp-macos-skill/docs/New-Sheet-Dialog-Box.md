# New Sheet Dialog Box

Creates a new sheet in the current sheet set by creating a new drawing (DWG) file that includes a named layout with the same
name as the drawing.

SHEETSET (Command)

*Menu:* Window ![](../images/ac.menuaro.gif) Sheet Set Manager:
Right-click sheet set name ![](../images/ac.menuaro.gif) New Sheet

![](../images/GUID-379366D7-CFCF-4687-821B-5B2145DD810E-low.png)

## List of Options

The following options are displayed.

### Sheet Number

Specifies the number for the sheet in the current sheet set. It is recommended that you do not fill in the number when you
first create a sheet. By default, the drawing name and layout name of the new sheet will be the number together with sheet
title.

You can remove the number from the file name when you create the new sheet, but the number will still be in the sheet name.

### Sheet Title

Specifies the name of the named layout in the new drawing.

### Sheet Creation Template

Specifies the drawing template (DWT) file and the named layout to use when creating a new sheet in the sheet set.

Named Layout
:   Displays the name of the layout in the specified drawing template (DWT) file to use when creating a new drawing file.

Template File Location
:   Displays the default drawing template (DWT) file to use for creating the new drawing file.

    The default drawing template (DWT) file used for creating new drawings in a subset is specified in the Details panel when
    the subset is selected. The default drawing template (DWT) file for creating new drawings in a sheet set is specified in the
    Details panel when the sheet set name is selected.

    Click Browse to select a drawing template (DWT) file in Finder.

### Open Drawing

When checked, allows you to open the drawing after creation and edit directly within the file.
