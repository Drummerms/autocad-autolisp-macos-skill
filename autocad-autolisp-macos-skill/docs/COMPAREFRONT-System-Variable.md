# COMPAREFRONT (System Variable)

Controls the default display order of overlapping objects in the comparison drawing.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| Value | Description |
| 1 | Sets the new or changed objects in the first drawing in front of those in the second drawing |
| 2 | Sets the new or changed objects in the second drawing in front of those in the first drawing |
