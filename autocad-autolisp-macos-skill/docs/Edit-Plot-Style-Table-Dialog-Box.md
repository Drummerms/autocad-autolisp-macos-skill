# Edit Plot Style Table Dialog Box

Create or edit plot style table files.

PLOTSTYLE (Command)

*Menu*:
Format ![](../images/ac.menuaro.gif) Plot Style.

![](../images/GUID-60986DD8-9A5D-4CC4-B482-E34DE683E2D8-low.png)

## List of Options

The following options are displayed.

Plot Style Table
:   Displays a list of the available plot style files that can be edited. A plot style is a collection of property settings used
    in plotting. Along with selecting a plot style to edit from the list, you can also select one of the following:

    * *Reveal in Finder.* Opens Finder to the folder that contains the listed plot styles. The location in which
      AutoCAD 2026 locates plot styles can be changed under Printer Support File Path ![](../images/ac.menuaro.gif) Plot Style Table Search Path on the Application tab of the Application Preferences dialog box ([OPTIONS](OPTIONS-Command.md) command).
    * *New.* Displays the
      [Plot Style Table Editor](Plot-Style-Table-Editor.md) with the default properties for a new plot style. The type of new plot style created depends on the type of plot style being
      used in the current drawing.

Edit
:   Displays the Plot Style Table Editor with the properties of the selected plot style.
