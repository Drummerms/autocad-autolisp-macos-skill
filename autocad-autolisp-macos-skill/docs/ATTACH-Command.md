# ATTACH (Command)

Inserts an external reference, raster image, or underlay (PDF files) in the current drawing.

## Access Methods

*Toolbar*:
Attach.
![](../images/GUID-46BA50D9-80A7-4F79-B6BA-AE4BD757DB7F-low.png)

*Menu*:
File ![](../images/ac.menuaro.gif) Attach.

## Summary

The Select Reference File dialog box (a standard file selection dialog box) is displayed. To select multiple DWG files for
attach, set the Files of Type to Drawing. You can select one file only for all other file formats.
