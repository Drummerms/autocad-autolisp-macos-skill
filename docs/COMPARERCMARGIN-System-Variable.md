# COMPARERCMARGIN (System Variable)

Specifies the offset distance between the boundary of a change set and the rectangle/polygonal revision cloud in the comparison
drawing.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 5 |

Enter a valid value from 1 to 25. The higher the margin value, the bigger is the boundary that contains the differences, which
results in a bigger revision cloud.
