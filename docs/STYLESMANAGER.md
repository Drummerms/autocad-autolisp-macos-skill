# STYLESMANAGER

Displays the folder in Finder that contains the plot style files used when plotting or publishing a drawing.

## Access Methods

![](../images/ac.mouse.gif) Menu: File ![](../images/ac.menuaro.gif) Plot Styles

## Summary

The location in which
AutoCAD 2026 locates plot styles can be changed under Printer Support File Path ![](../images/ac.menuaro.gif) Plot Style Table Search Path on the Application tab of the Application Preferences dialog box (OPTIONS command).

Double-click a plot style (STB or CTB) file in Finder to start the Plot Style Table Editor.
