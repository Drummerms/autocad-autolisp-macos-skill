# HPNAME (System Variable)

Sets the default hatch pattern name.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Not-saved |
| Initial value: | ANSI31 (imperial) or ANGLE (metric) |

Valid names can contain up to 34 characters without spaces. The value "" indicates that there is no default. Enter a period
(.) to reset HPNAME to the default value.
