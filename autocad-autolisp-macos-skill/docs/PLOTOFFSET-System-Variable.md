# PLOTOFFSET (System Variable)

Controls whether the plot offset is relative to the printable area or to the edge of the paper.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Sets the plot offset relative to the printable area. |
| 1 | Sets the plot offset relative to the edge of the paper |
