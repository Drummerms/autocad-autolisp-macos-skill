# To Work With Options for Plotted Objects

You can select options that affect the way in which objects are plotted.

## Hide Paper Space Objects

1. Click Plot.
   ![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png)
2. Under Print Options, select Hide Paperspace Objects.
3. Click Plot to print the drawing.

## Print Lineweights

1. Click Plot.
   ![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png)
2. Under Print Options, select Print Object Lineweights.
3. Click Plot to plot the drawing.

## Print Transparency

1. Click Plot.
   ![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png)
2. Under Print Options, select Print Transparency.
3. Click Plot to plot the drawing.

IMPORTANT:This setting can be overridden by the PLOTTRANSPARENCYOVERRIDE system variable.

## Turn off Plot Styles

1. Click Plot.
   ![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png)
2. Under Plot Style Table, clear the Plot with Plot Styles option.
3. Click Plot to plot the drawing.

NOTE:Plot styles are not available for objects with the Jitter edge modifier applied (VISUALSTYLES).

## Change the Plot Order

You can specify to print the objects in model space before printing in paper space.

1. Click Plot.
   ![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png)
2. Under Print Options, select Print Paperspace Last.

   Clear the Plot Paperspace Last option to plot paper space first.
3. Click Plot to plot the drawing.
