# SNAPGRIDLEGACY (System Variable)

Controls whether the cursor snaps to the snap grid only when an operation is in progress.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | Makes the cursor snap to the snap grid only when SNAP is turned on, and an operation is in progress |
| 1 | Makes the cursor snap to the snap grid at all times when SNAP is turned on (legacy behavior) |
