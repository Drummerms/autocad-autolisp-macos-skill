# LONGITUDE (System Variable)

Specifies the longitude of the drawing model in decimal format.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | -122.3940 |

The default is the longitude of San Francisco, California. The valid range is -180 to +180. Positive values represent east
longitudes.

This value is affected by the settings of the LUPREC system variable.

This value is not affected by the settings of the AUNITS and AUPREC system variables.
