# C Functions

#### Topics in this section

* [caddr (AutoLISP)](caddr-AutoLISP.md)

  Returns the third element of a list
* [cadr (AutoLISP)](cadr-AutoLISP.md)

  Returns the second element of a list
* [car (AutoLISP)](car-AutoLISP.md)

  Returns the first element of a list

  Returns a list containing all but the first element of the specified list
* [chr (AutoLISP)](chr-AutoLISP.md)

  Closes an open file

  Executes an AutoCAD command
* [command-s (AutoLISP)](command-s-AutoLISP.md)

* [cond (AutoLISP)](cond-AutoLISP.md)
