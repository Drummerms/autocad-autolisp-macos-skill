# To Control Automatic Notification for New Layers

1. Click
   Window menu ![](../images/ac.menuaro.gif) Layers.
2. In the Layer Properties Manager, click Settings. The button is located near the upper right corner.
3. In the Layer Settings dialog box, make sure that the New Layer Notification box is checked, and click Evaluate New Layers
   Added to Drawing.
4. Click Notify When New Layers are Present, and click one or more notification options.
