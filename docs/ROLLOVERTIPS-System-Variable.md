# ROLLOVERTIPS (System Variable)

Controls the display of rollover tooltips when the cursor hovers over an object.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | Object rollover tooltips are not displayed |
| 1 | Object rollover tooltips are displayed |
