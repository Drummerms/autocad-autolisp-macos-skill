# TABLE (Command)

Creates an empty table object.

## Access Methods

*Tool Sets*:
Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/ac.menuaro.gif) Table.
![](../images/GUID-2E433CD0-2201-4BE7-97D5-5242CC65AF43-low.png)

*Menu*:
Draw ![](../images/ac.menuaro.gif) Table.

## Summary

A table is a compound object that contains data in rows and columns. It can be created using a specific table style.

The Insert Table dialog box is displayed.

If you enter *-table* at the Command prompt,
[options are displayed](TABLE-Command.md).
