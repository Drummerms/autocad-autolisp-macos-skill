# REVCLOUDARCVARIANCE (System Variable)

Controls whether revcloud arcs are created with varying or generally uniform chord lengths.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | Arcs are created with generally uniform chord lengths |
| 1 | Arcs are created with varying chord lengths resulting in a more hand-drawn appearance. |

NOTE:Even uniform chord lengths will vary a little depending on the size, shape, and perimeter of the revcloud.
