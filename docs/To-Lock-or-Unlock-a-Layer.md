# To Lock or Unlock a Layer

You can lock or unlock a layer in the Layers palette or by selecting an object.

## In the Layers Palette

1. Click
   Window menu ![](../images/ac.menuaro.gif) Layers.
2. Select the layers you want to lock or unlock.
3. Click in the Lock column to set the status of the selected layers. The icon becomes visible when you mouse over the area.

   ![](../images/GUID-B24A6B16-F29B-48B2-88A9-1497DC1A173E-low.png) = locked

   blank = unlocked

NOTE:You can lock or unlock individual layers using the Properties Inspector.

## By Selecting an Object

1. Click the Model tab.
2. Click
   Window menu ![](../images/ac.menuaro.gif) Properties Inspector.
3. Do either of the following:
   * Click
     Lock Layer.
     ![](../images/GUID-BE5FC5DB-6874-4A05-9276-CB3D91E9CFC6-low.png)
   * Click
     Unlock Layer.
     ![](../images/GUID-7DD8DDF3-44D1-4044-80A4-27F6DAC8534B-low.png)
4. Select an object on the layer that you want to lock or unlock.

Objects on locked layers appear faded and a small lock icon is displayed when you hover over the object. Set the fade level
for locked layers with the system variable LAYLOCKFADECTL.
