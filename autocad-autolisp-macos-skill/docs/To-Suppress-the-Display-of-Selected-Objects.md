# To Suppress the Display of Selected Objects

1. Select the objects that you want to make temporarily invisible.
2. Right-click and choose *Isolate*![](../images/ac.menuaro.gif)*Hide Objects*.

To restore all suppressed objects, right-click in the drawing area and choose *Isolate*![](../images/ac.menuaro.gif)*End Object Isolation*.
