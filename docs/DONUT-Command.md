# DONUT (Command)

Creates a filled circle or a wide ring.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Donut.
![](../images/GUID-CFEBBE80-DA08-4094-8650-EA70BA259027-low.png)

NOTE:Hidden by default. Click
![](../images/GUID-78DED619-51B3-49C1-AD71-F928B5F69C0A-low.png) to display the icon on the tool set panel.

*Menu*:
Draw ![](../images/ac.menuaro.gif) Donut.

A donut consists of two arc polylines that are joined end-to-end to create a circular shape. The width of the polylines is
determined by the specified inside and outside diameters. If you specify an inside diameter of 0, the donut is a filled circle.

![](../images/GUID-C260D806-EE68-45E1-9326-B077BC487A0C-low.gif)

The following prompts are displayed.

Inside Diameter
:   Specifies the inside diameter of the donut as shown.

    ![](../images/GUID-40FEB73F-075E-40D8-9005-1A74B89A2B13-low.png)

Outside Diameter
:   Specifies the outside diameter of the donut.

Center of Donut
:   Specifies the location of the donut based on its center point. A donut is drawn at each point specified until you press Enter
    to end the command.

    ![](../images/GUID-6A8FFF9C-8F92-4C1D-9424-CC010EA8AA37-low.png)
