# To Apply Scaling to Non-ISO Linetypes and Fill Patterns

You can apply a global scale factor to non-ISO linetypes and fill patterns in plot styles.

1. Click
   Format menu ![](../images/ac.menuaro.gif) Plot Style.
2. Select the Plot Style Table that you want to modify. Click Edit.
3. In the Plot Style Table Editor, click Details to expand the dialog box.
4. Select Apply Global Scale Factor to Non-ISO LineTypes.

   This option scales line types and fill patterns according to a value that you specify.
5. In the Scale Factor box, enter a scale factor to apply.
6. Click Save & Close.
