# Add Custom Property Dialog Box

Creates a custom property to be associated with the current sheet set or with each sheet.

SHEETSET (Command)

*Menu:* Window ![](../images/ac.menuaro.gif) Sheet Set Manager:
Select a sheet set or sheet
![](../images/ac.menuaro.gif) click Add Property in Details panel.

NOTE:If the Details panel is not visible, click
![](../images/GUID-6364B815-1C7B-485B-A6D5-1D3B961D3025-low.png).

![](../images/GUID-AB8812E4-74AD-41A5-A29C-343635A1D050-low.png)

## List of Options

The following options are displayed.

### Owner

Specifies whether the custom property will belong to the sheet set or sheet.

### Name

Specifies the name of the new custom property.

### Default Value

Specifies a value for the custom property.
