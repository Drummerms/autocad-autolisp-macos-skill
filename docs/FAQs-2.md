# FAQs

#### Topics in this section

* [FAQ: What is the Difference Between Lightweight Polylines and "Legacy" Polylines? (AutoLISP)](FAQ-What-is-the-Difference-Between-Lightweight-Polylines-and-Legacy-Polylines-AutoLISP.md)

  A lightweight polyline (LWPOLYLINE) is defined in the drawing database as a single graphic entity unlike the "legacy" polyline
  (POLYLINE), which is defined as a group of subentities. A "legacy" polyline can be either a 2D or 3D polyline, while a lightweight
  polyline can only be a 2D polyline.
* [FAQ: What Symbol Table Entries Cannot Be Renamed or Modified? (AutoLISP)](FAQ-What-Symbol-Table-Entries-Cannot-Be-Renamed-or-Modified-AutoLISP.md)

  Most entries in the symbol tables can be renamed or modified with a few exceptions.
* [FAQ: How Do I Process Curve-Fit and Spline-Fit Polylines? (AutoLISP)](FAQ-How-Do-I-Process-Curve-Fit-and-Spline-Fit-Polylines-AutoLISP.md)

  A "legacy" polyline might contain vertices that were not created explicitly; these auxiliary vertices were inserted automatically
  by the AutoCAD PEDIT command's Fit and Spline options.
