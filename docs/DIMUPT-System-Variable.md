# DIMUPT (System Variable)

Controls options for user-positioned text.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Drawing |
| Initial value: | Off |

| Off | Cursor controls only the dimension line location |
| On | Cursor controls both the text position and the dimension line location |
