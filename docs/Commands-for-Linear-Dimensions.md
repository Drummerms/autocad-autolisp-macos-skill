# Commands for Linear Dimensions

#### Commands

* [DIM](GUID-45C1A271-9650-4927-858F-B3BDB19B3E6C.htm " Creates multiple types of dimensions within a single command session. ")
* [DIMALIGNED](GUID-D7C05A85-8985-4576-B22C-5A795DD5B179.htm "Creates an aligned linear dimension. ")
* [DIMBASELINE](GUID-19EA3161-D4A0-4280-9707-0A003C5B8A90.htm "Creates a linear, angular, or ordinate dimension from the baseline of the previous or selected dimension. ")
* [DIMCONTINUE](GUID-27CF5248-EECB-4F5E-9A25-39ED399EA1EC.htm "Creates a dimension that starts from an extension line of the previous or selected dimension. ")
* [DIMEDIT](GUID-4C422870-32A1-457B-8D1E-BD4AF7967FF1.htm "Edits dimension text and extension lines. ")
* [DIMLINEAR](GUID-E908848F-4621-43D2-B377-FD3FBC2B8848.htm "Creates a linear dimension. ")
* [DIMSTYLE](GUID-39D6E54F-E12F-46A4-9DD1-68D9357289E4.htm "Creates and modifies dimension styles. ")
* [QDIM](GUID-1FB628B7-C2AF-4B6D-B83E-20ADDD70614D.htm "Creates a series of dimensions quickly from selected objects. ")

#### System Variables

* [DIMDLI](GUID-D2D1AF3D-004B-4C47-BABE-22B7327EE128.htm "Controls the spacing of the dimension lines in baseline dimensions. ")
* [DIMSTYLE](GUID-98D575CA-C7FE-4442-83F3-C2BFD7D9A112.htm "Displays the unit type (imperial/standard or ISO-25/metric) used by dimensions in the drawing. ")
