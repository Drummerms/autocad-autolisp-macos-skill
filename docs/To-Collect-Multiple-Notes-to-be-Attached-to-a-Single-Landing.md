# To Collect Multiple Notes to be Attached to a Single Landing

1. ClickDrafting tab ![](../images/ac.menuaro.gif) Leader panel ![](../images/ac.menuaro.gif) Multileader Collect.
   ![](../images/GUID-F8A58FDF-0502-4595-A405-A50D175FEC82-low.png)
2. On the Multileader toolbar, click Collect Multileaders.
3. Select multileaders in the order you want them to be collected. The last multileader selected retains its landing. Press ENTER.
