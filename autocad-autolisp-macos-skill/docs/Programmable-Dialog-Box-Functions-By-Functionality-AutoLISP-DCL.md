# Programmable Dialog Box Functions By Functionality (AutoLISP/DCL)

Listing of AutoLISP functions that are used to implement dialog boxes.

The programmable dialog box functions perform dialog box opening and closing, tile and attribute handling, list box and pop-up
list handling, image tile handling, and application-specific data handling. This section lists each PDB function available
in AutoLISP, grouping them by the type of task each performs.
