# To Modify Drafting Settings

1. On the Status Bar, right-click any of the following and choose Settings.
   * Snap Mode
   * Grid Display
   * Polar Tracking
   * Object Snap
   * 3D Object Snap
   * Object Snap Tracking
   * Dynamic Input
2. In the Drafting Settings dialog box, modify settings as needed.
