# FONTALT (System Variable)

Specifies the alternate font to be used when the specified font file cannot be located.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Registry |
| Initial value: | simplex.shx |

When a drawing file with a defined text style is opened and an alternate font is not specified, the Alternate Font dialog
box is displayed.
