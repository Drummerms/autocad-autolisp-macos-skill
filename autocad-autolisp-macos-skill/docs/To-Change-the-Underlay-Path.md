# To Change the Underlay Path

1. Click
   Window menu ![](../images/ac.menuaro.gif) Reference Manager.
2. In the Reference Manager dialog box, select an underlay whose path you want to change. Click
   ![](../images/GUID-B50A0504-05B5-44A6-A31D-2E72778B8CB3-low.png) at the right side of the Saved Path box.
3. In the Select File dialog box, select a new path. Click Open.

   The new path is displayed in the Saved Path field.
