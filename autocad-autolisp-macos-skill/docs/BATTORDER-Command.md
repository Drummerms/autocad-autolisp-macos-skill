# BATTORDER (Command)

Specifies the order of attributes for a block.

## Summary

Displays the [Attribute Order dialog box](Attribute-Order-Dialog-Box.md), which controls the order in which attributes are listed and prompted for when you insert or edit a block reference. You
can only use the BATTORDER command in the [Block Editor](Block-Editor.md).
