# GTDEFAULT (System Variable)

Controls whether the 3D Move, 3D Rotate, or 3D Scale operation starts automatically when you start the MOVE, ROTATE, or SCALE
command in a viewport with a 3D visual style.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Does not automatically start the 3DMOVE, 3DROTATE, or 3DSCALE commands when the MOVE, ROTATE, or SCALE commands are started in a 3D view. |
| 1 | Automatically starts the 3DMOVE, 3DROTATE, or 3DSCALE commands when the MOVE, ROTATE, or SCALE commands are started in a 3D view. |
