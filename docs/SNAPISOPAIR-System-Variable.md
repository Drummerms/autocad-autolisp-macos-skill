# SNAPISOPAIR (System Variable)

Controls the isometric plane for the current viewport.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | Left |
| 1 | Top |
| 2 | Right |
