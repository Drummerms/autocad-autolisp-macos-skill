# To Modify Layout Viewport Properties Using the Properties Inspector

1. Click the border of the layout viewport whose property you want to change and right-click Properties.
2. In the Properties Inspector, select the value for the property you want to modify. Enter a new value or select a new setting
   from the list provided.

   The new property setting or value is assigned to the selected layout viewport.
