# About Color Books

When assigning colors to objects, you can choose colors from color books that are loaded on your system, which include third-party
or user-defined files.

Color book files use an *.acb* file extension and must be copied to specified file folders. On the Applications tab of the Application Preferences dialog
box, you can define one or more folder paths where color book files are located.

Once a color book file is available, you can access it from the Select Color dialog box for any feature that provides color
options, including objects, layers, hatches, the drawing area background color, and so on.
