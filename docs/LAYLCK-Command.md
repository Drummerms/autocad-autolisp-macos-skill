# LAYLCK (Command)

Locks the layer of a selected object.

## Access Methods

*Menu*:
Window ![](../images/ac.menuaro.gif) Properties Inspector ![](../images/ac.menuaro.gif) Lock Layer.
![](../images/GUID-BE5FC5DB-6874-4A05-9276-CB3D91E9CFC6-low.png)

*Menu*:
Format
![](../images/ac.menuaro.gif) Layer Tools
![](../images/ac.menuaro.gif) Lock Layer.

## Summary

Using this command, you can prevent objects on a layer from being accidentally modified. You can also fade the objects on
a locked layer using the LAYLOCKFADECTL system variable.
