# To Break an Object

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Break drop-down ![](../images/ac.menuaro.gif) Break.
   ![](../images/GUID-80FF8BCE-417E-49B8-B945-3EE246031EC0-low.png)
2. Select the object to break.

   By default, the point at which you select the object is the first break point. To select a different pair of break points,
   enter
   *f*(First) and specify the first break point.
3. Specify the second break point.

   To break an object without creating a gap, enter
   *@0,0* to specify the previous point or use the method below.

## Break at a Single Point

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Break drop-down ![](../images/ac.menuaro.gif) Break at Point.
   ![](../images/GUID-6FD61209-7835-4DA3-9F54-EE2F3ADB87D5-low.png)
2. Select the object to break.
3. Specify the break point.
