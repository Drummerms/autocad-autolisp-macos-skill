# ALIASEDIT (Command)

Creates, modifies, and deletes AutoCAD command aliases. A command alias is an abbreviation of a command name, which you can
enter at the Command prompt instead of entering the entire command name.

## Summary

Opens the Customize dialog box, Aliases tab where you can edit, add, and remove command aliases.
