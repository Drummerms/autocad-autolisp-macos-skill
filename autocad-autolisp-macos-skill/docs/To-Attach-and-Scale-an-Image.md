# To Attach and Scale an Image

1. Click Insert Raster Image Reference.![](../images/GUID-D114BD17-9696-4B50-BCC0-630B8C4C0B84-low.png)At the Command prompt, enter IMAGEATTACH. In the Select Image File dialog box, select a file name from the list or enter the name of the image file in the File Name
   box. Click Open.
2. In the Image dialog box, use one of the following methods to specify insertion point, scale, or rotation:
   * Select Specify On-Screen to use the pointing device to insert the image at the location, scale, or angle you want.
   * Clear Specify On-Screen and enter values under Insertion Point, Scale, or Rotation.

     To view the unit of measurement for the image, click Details.
3. Click OK.
