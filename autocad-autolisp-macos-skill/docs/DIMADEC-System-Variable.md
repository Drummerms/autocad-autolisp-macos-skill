# DIMADEC (System Variable)

Controls the number of precision places displayed in angular dimensions.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| -1 | Angular dimensions display the number of decimal places specified by  [DIMDEC](DIMDEC-System-Variable.md). |
| 0-8 | Specifies the number of decimal places displayed in angular dimensions (independent of DIMDEC) |
