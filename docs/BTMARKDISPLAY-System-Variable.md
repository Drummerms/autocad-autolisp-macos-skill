# BTMARKDISPLAY (System Variable)

Controls whether or not value set markers are displayed for dynamic block references.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Specifies that value set markers are not displayed |
| 1 | Specifies that value set markers are displayed |
