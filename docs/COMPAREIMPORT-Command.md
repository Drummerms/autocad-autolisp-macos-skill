# COMPAREIMPORT (Command)

Imports objects from the compared drawing into the current drawing.

## Access Methods

*DWG Compare Visor*:
![](../images/GUID-A25D47BB-899C-49B4-B969-D5E21B7071A1-low.png)

NOTE:This command is available only during a drawing comparison.

When you select objects at the prompt, only those objects that are not already in your
*current* drawing are selectable. This provides a reliable way of limiting the objects being imported to the objects that are unique
to the
*comparison* drawing.

You can select individual objects to import or use the Crossing or Window selection options to import objects from an area
of interest.
