# OFFSETDIST (System Variable)

Sets the default offset distance.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Not-saved |
| Initial value: | -1.0000 |

| <0 | Offsets an object through a specified point |
| ≥0 | Sets the default offset distance |
