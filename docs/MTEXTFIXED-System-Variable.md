# MTEXTFIXED (System Variable)

Sets the display size and orientation behavior of the In-Place Text Editor for multiline text.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 2 |

If MTEXTED is set to "Internal," MTEXTFIXED has the following results:

| Value | Description |
| 0 or 1 | Displays the In-Place Text Editor and the multiline text within it at the size, position, and rotation of the multiline text object in the drawing. |
| 2 | Displays the In-Place Text Editor, but the multiline text that would otherwise be difficult to read, such as if it's very small, very large, or rotated, is temporarily displayed at a legible size and is oriented horizontally so that you can easily read and edit it. |
