# To Enable Big Fonts

Windows
:   1. Click Annotate tab![](../images/ac.menuaro.gif)Text panel ![](../images/ac.menuaro.gif) ![](../images/GUID-5EED0390-225C-4C5D-819B-5AB2C7964B14-low.png) FindText Style ![](../images/GUID-3F0D12C4-29B1-4140-871D-A2A3B8DE57AD-low.png) Find

    Or at the Command prompt, enter *style*.

    2. In the Text Style dialog box, Styles list, select a text style to change.
    3. From the Font Name drop-down list, select a SHX file.
    4. Click Use Big Font.
    5. From the Big Font drop-down list, select a Big Font SHX file.
    6. Click Apply.
    7. Click Close.

Mac OS
:   1. Click Format menu ![](../images/ac.menuaro.gif) Text Style, or at the Command prompt, enter *style*.
    2. In the Text Style dialog box, Styles list, select a text style to change.
    3. From the Family list box, select a SHX file.
    4. From the Asian Set list box, select a Big Font SHX file.
    5. Click Apply.
    6. Click Close.
