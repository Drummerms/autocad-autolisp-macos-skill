# HATCHEDIT (Command)

Modifies an existing hatch or fill.

## Access Methods

*Tool Sets*:
Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Hatch Edit.
![](../images/GUID-E7FEF72F-5B54-4EB8-8166-F7BCE20877E3-low.png)

NOTE:Hidden by default. Click
![](../images/GUID-78DED619-51B3-49C1-AD71-F928B5F69C0A-low.png) to display the icon on the tool set panel.

Modifies hatch-specific properties, such as pattern, scale, and angle for an existing hatch or fill.

The following options are displayed.

Select hatch object
:   Use an object selection method and choose a hatch object to edit. Displays the Hatch Edit dialog box.

If you enter -HATCHEDIT at the Command prompt, the associated options are displayed in the command line.
