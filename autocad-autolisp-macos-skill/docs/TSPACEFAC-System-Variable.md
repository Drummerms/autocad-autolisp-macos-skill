# TSPACEFAC (System Variable)

Controls the multiline text line-spacing distance measured as a factor of text height.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Not-saved |
| Initial value: | 1.0000 |

Valid values are 0.25 to 4.0.
