# To Create a Donut

Donuts are filled rings or solid-filled circles that are actually closed polylines with width.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Donut.
   ![](../images/GUID-CFEBBE80-DA08-4094-8650-EA70BA259027-low.png)

   NOTE:Hidden by default. Click
   ![](../images/GUID-78DED619-51B3-49C1-AD71-F928B5F69C0A-low.png) to display the icon on the tool set panel.
2. Specify the inside diameter (1).
3. Specify the outside diameter (2).
4. Specify the center of the donut (3).
5. Specify the center point for another donut, or press Enter to complete the command.

   ![](../images/GUID-9FFA8B85-2F34-495C-91E2-777F82BC2CF6-low.png)
