# To Calculate the Area of a Selected Object

1. Click
   Tools menu ![](../images/ac.menuaro.gif) Inquiry ![](../images/ac.menuaro.gif) Area.
2. At the prompt, enter
   *o* (Object).
3. Select an object.

## Quick Mode

1. Click
   Tools menu
   ![](../images/ac.menuaro.gif) Inquiry ![](../images/ac.menuaro.gif) Quick.
2. Move the cursor to an enclosed area and click once.

   The area and perimeter values are shown in the command window.
