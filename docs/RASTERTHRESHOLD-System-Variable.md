# RASTERTHRESHOLD (System Variable)

Specifies a raster threshold in megabytes.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 20 |

If the plotted raster image exceeds this threshold, the availability of system memory is checked. The plot is aborted if the
image is too big for the available memory.
