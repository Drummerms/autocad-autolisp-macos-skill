# About Superscripts and Subscripts in SHX Files

You can modify shape definition files to improve their ability to display superscripts and subscripts.

SHX fonts have limited superscript and subscript capabilities. However, it is relatively easy to modify shape definition files
to improve superscript and subscript capability.

Creating superscripts and subscripts requires two steps. First, the “imaginary pen” that is creating the text, vector by
vector, on your screen needs to be shifted up or down. Then, the font “scale” needs to be reduced. In addition, the reverse
process has to take place to return to the normal font. The font needs to recognize four new keys: two for superscripts and
two for subscripts. To avoid altering the existing font definitions, you can access these with the numeric keypad on your
keyboard.
