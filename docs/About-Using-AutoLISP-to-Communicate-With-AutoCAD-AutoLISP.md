# About Using AutoLISP to Communicate With AutoCAD (AutoLISP)

Query and command functions provide direct access to AutoCAD commands and drawing services.

AutoLISP ®  provides various functions for examining the contents of the currently loaded drawing. This chapter introduces these functions
and describes how to use them in conjunction with other functions.
