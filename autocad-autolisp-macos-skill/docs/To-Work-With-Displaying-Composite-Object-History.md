# To Work With Displaying Composite Object History

## Set a 3D Solid to Retain the History of Its Original Forms

1. If the Properties palette is not displayed, right-click the drawing area and select Properties.
2. Select a composite 3D solid.
3. On the Properties palette, Solid History area, under History, select Record.

## Display the Original Components of a Composite Solid

1. If the Properties palette is not displayed, select any object. Right-click the object and select Properties.
2. Select a composite 3D solid.
3. On the Properties palette, Solid History area, under Show History, select Yes.

## Remove the History of a Composite Object

1. If the Properties palette is not displayed, select any object. Right-click the object and select Properties.
2. Select a composite 3D solid.
3. On the Properties palette, Solid History area, under History, select None.
