# SNAPUNIT (System Variable)

Sets the snap spacing for the current viewport.

|  |  |
| --- | --- |
| Type: | 2D-point |
| Saved in: | Drawing |
| Initial value: | 0.5000,0.5000 (imperial) or 10.0000,10.0000 (metric) |

If SNAPSTYL is set to 1, the X value of SNAPUNIT is adjusted automatically to accommodate the isometric snap.

Changes to this system variable are not reflected in the grid until the display is refreshed.
