# To Draw a Multiline

1. At the command prompt, enter MLINE.
2. At the Command prompt, enter
   *st* to select a style.
3. To list available styles, enter the style name or enter
   *?*.
4. To justify the multiline, enter
   *j* and select top, zero, or bottom justification.
5. To change the scale of the multiline, enter
   *s* and enter a new scale.

   Now draw the multiline.
6. Specify the starting point.
7. Specify a second point.
8. Specify additional points or press Enter.

   If you specify three or more points, you can enter
   *c* to close the multiline.
