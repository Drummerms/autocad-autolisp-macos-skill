# 3Dconnexion Device Support

You can use a 3Dconnexion device to reorient and navigate a model's view.

![](../images/GUID-CB3EAD3E-9070-42F6-8DD6-1EC56AA59BDC-low.png)

The device has a pressure sensitive controller cap designed to flex in all directions. When a view change occurs with the
3Dconnexion 3D mouse, the ViewCube tool is reoriented to reflect the current view.

NOTE:The 3Dconnexion device only works in a 3D visual style.

Install the latest device driver for macOS found at
[3Dconnexion.com](https://3dconnexion.com) to use the buttons on the device.
