# About Multiple Configurations

If you use more than one pointing device, you can set up more than one configuration file to make it easy to switch between
devices.

When you configure pointing device drivers, the information you supply is recorded in a configuration file.

Typically, only a single configuration is necessary, but you may need multiple configurations. For example, if you use a
mouse for most of your work but occasionally require a large digitizing tablet, you can set up your system to handle multiple
configurations rather than reconfiguring each time you change a device.

Use the */c* command line switch to specify alternative configuration files at startup.
