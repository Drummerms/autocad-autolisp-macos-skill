# BSAVEAS (Command)

Saves a copy of the current block definition under a new name.

## Access Methods

![](../images/ac.mouse.gif) Toolbar: Block Editor visor ![](../images/ac.menuaro.gif) Save New.
![](../images/GUID-CC0474D2-6119-40CE-95BF-48775ED556DC-low.png)

## Summary

Displays the
[Save Block Definition As dialog box](Save-Block-Definition-As-Dialog-Box.md).

You can only use the BSAVEAS command in the
[Block Editor](Block-Editor.md).
