# atof (AutoLISP)

Converts a string into a real number

*Supported Platforms:* Windows, Mac OS, and Web

## Signature

```
(atof str)
```

*str*
:   *Type:* String

    A string to be converted into a real number.

## Return Values

*Type:* Real

A numeric value.

## Examples

```
(atof "97.1")
97.1

(atof "3")
3.0

(atof "3.9")
3.9
```
