# ARRAYCLASSIC (Command)

Creates copies of objects arranged in a 2D pattern using the legacy dialog box.

## Summary

You can create copies of objects in a regularly spaced rectangular or polar array in the
[Array dialog box](Array-Dialog-Box.md).

If you enter *-array* at the Command prompt,
[options are displayed](ARRAY-Command-2.md).

NOTE:This legacy dialog box does not support array associativity or path arrays. To use these features, use the ARRAY command.
