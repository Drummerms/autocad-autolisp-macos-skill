# EATTEDIT (Command)

Edits attributes in a block reference.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Attribute drop-down ![](../images/ac.menuaro.gif) Attribute, Single.
![](../images/GUID-91BCBDF8-1AB3-4876-B9D5-4959DE7B6ABD-low.png)

*Menu*:
Modify ![](../images/ac.menuaro.gif) Object ![](../images/ac.menuaro.gif) Attribute ![](../images/ac.menuaro.gif) Single.

Edits the values, text options, and properties of each attribute in a block.

The following prompt is displayed.

Select Block
:   After you select a block with attributes, the Enhanced Attribute Editor is displayed.
