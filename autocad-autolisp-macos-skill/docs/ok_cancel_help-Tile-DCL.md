# ok\_cancel\_help Tile (DCL)

This tile is the
ok\_cancel cluster combined with the Help button.

*Supported Platforms:* Windows, Mac OS, and Web

```
ok_cancel_help;
```

![](../images/GUID-FC9F3085-5E97-4C90-84A8-6D7D31C2A294-low.png)

The key of the Help button is
"help". Help buttons are recommended for the main dialog box of an application and for complex dialog boxes. The function that handles
the Help button can display the standard AutoCAD Help dialog box by invoking the AutoLISP
help function.

The
ok\_cancel\_help tile is defined in the
*base.dcl* file.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
