# TRACEPALETTECLOSE (Command)

Closes the Trace palette where you view and manage traces in the current drawing.

![](../images/GUID-87ACDEF3-FA42-4C02-8AE9-DA283CEDFB7B-low.png) Find

A trace is created in the AutoCAD web app or the AutoCAD mobile app to provide feedback, comments, markups, and design exploration
of the drawing without altering the content of the drawing.
