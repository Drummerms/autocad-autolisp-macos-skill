# 3DORBITCTR (Command)

Sets the center of rotation in 3D Orbit view.

## Summary

Starts 3D Orbit view and uses a center of rotation that you specify with your pointing device. If you specify a point outside
the current view, 3DORBITCTR ignores the specified point and uses the default center of rotation.

3DORBITCTR overrides the AutoTarget option in the [3DORBIT](3DORBIT-Command.md) command.
