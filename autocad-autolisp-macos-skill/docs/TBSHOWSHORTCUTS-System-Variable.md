# TBSHOWSHORTCUTS (System Variable)

Specifies if shortcut keys are displayed in tooltips.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Registry |
| Initial value: | yes |

| no | Display shortcut keys |
| yes | Do not display shortcut keys |
