# FILEDIA (System Variable)

Suppresses display of file navigation dialog boxes.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Does not display dialog boxes. You can still request a file dialog box to appear by entering a tilde (~) in response to the command's prompt. The same is true for AutoLISP and ObjectARX. |
| 1 | Displays dialog boxes. However, if a script or AutoLISP/ObjectARX ™  program is active, an ordinary prompt is displayed. |
