# To Work With the Display of Light Glyphs

A point light or spotlight is represented in a drawing by a light glyph. You can control the appearance and visibility of
such glyphs.

## Control the Appearance of Light Glyphs

Change the color or size of glyphs that represent lights in the current viewport.

1. Select a point light or a spotlight.
2. In the Lighting Properties Tab of the Properties Inspector, specify color from the Filter Color list. Or click Select Color
   to open the Select Color dialog box.
3. To change the size of the glyph in the drawing, use the Glyph Size slider.

## Show or Hide Light Glyphs

Turn on or off the display of glyphs that represent lights in the current viewport.

1. Click View ![](../images/ac.menuaro.gif) Render ![](../images/ac.menuaro.gif) Light ![](../images/ac.menuaro.gif) Light Glyph![](../images/GUID-FE149ECF-44D1-4A20-8130-6A0832BE2D81-low.png)

NOTE:

The display setting does not control plotting of light glyphs.

## Plot Light Glyphs

Specify whether light glyphs will be visible when plotting a drawing.

1. Select a point light or a spotlight.
2. In the Lighting Properties Tab of Properties Inspector, check the Plot Glyph checkbox to make it visible while plotting.
