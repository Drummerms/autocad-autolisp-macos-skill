# TINSERT (Command)

Inserts a block in a table cell.

## Access Methods

![](../images/ac.mouse.gif) Toolbar: With a table and table cell selected, click Insert Block on the Table Cell visor.

## Summary

The [Insert Block in a Table Cell dialog box](Insert-Block-in-Table-Cell-Dialog-Box.md) is displayed.
