# DIMDLE (System Variable)

Sets the distance the dimension line extends beyond the extension line when oblique strokes are drawn instead of arrowheads.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 0.0000 |
