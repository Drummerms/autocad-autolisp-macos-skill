# Initialization File and Registry Functions Reference (AutoLISP/Express Tools)

NOTE:Express Tools are supported in AutoCAD for Windows only; not available in AutoCAD LT for Windows, or on Mac OS and Web.

The following table provides a summary of the AutoLISP Express Tools initialization file and registry functions.

| Initialization file and registry functions | | Platforms | | | | |
| Windows | | Mac OS | | Web |
| Function | Description | AutoCAD | AutoCAD LT | AutoCAD | AutoCAD LT | AutoCAD |
| [(acet-ini-get inifile [section [key [default]]])](acet-ini-get-AutoLISP-Express-Tools.md) | Retrieves data from an INI file | ✓ | -- | -- | -- | -- |
| [(acet-ini-set inifile section [key [value]])](acet-ini-set-AutoLISP-Express-Tools.md) | Writes data to an INI file | ✓ | -- | -- | -- | -- |
| [(acet-reg-del key [name])](acet-reg-del-AutoLISP-Express-Tools.md) | Deletes a key or value from the Registry | ✓ | -- | -- | -- | -- |
| [(acet-reg-get key [name])](acet-reg-get-AutoLISP-Express-Tools.md) | Returns a value from the Registry | ✓ | -- | -- | -- | -- |
| [(acet-reg-machine-prodkey)](acet-reg-machine-prodkey-AutoLISP-Express-Tools.md) | Returns the AutoCAD Windows registry path in the HKLM (HKEY\_LOCAL\_MACHINE) | ✓ | -- | -- | -- | -- |
| [(acet-reg-prodkey)](acet-reg-prodkey-AutoLISP-Express-Tools.md) | (Obsolete) Returns the current product key from the Registry | ✓ | -- | -- | -- | -- |
| [(acet-reg-put key [name value])](acet-reg-put-AutoLISP-Express-Tools.md) | Writes a Registry key or value | ✓ | -- | -- | -- | -- |
| [(acet-reg-user-prodkey)](acet-reg-user-prodkey-AutoLISP-Express-Tools.md) | Returns the AutoCAD Windows registry path | ✓ | -- | -- | -- | -- |
