# EXPORT (Command)

Saves the objects in a drawing to a different file format.

## Access Methods

*Toolbar*:
![](../images/GUID-0F2B4668-69AA-4C41-9C2C-EF732C3B9441-low.png)

*Menu*:
File ![](../images/ac.menuaro.gif) Export.

## Summary

The Export Data dialog box (a
[standard file selection dialog box](Standard-File-Selection-Dialog-Boxes.md)) is displayed.

If you do not see the file format that you need in the Files of Type drop-down list, also check the PLOT command for other
file types, including PDF.

NOTE:The Export Data dialog box records the last used file format selection and stores it for use during the current drawing session
and between drawing sessions.

The following output types are available:

* *ACIS (\*.sat):* ACIS solid object file (see
  [ACISOUT](ACISOUT-Command.md))
* *Bitmap (\*.bmp):* Device-independent bitmap file (see
  [BMPOUT](BMPOUT-Command.md))
* *Block (\*.dwg):* Drawing file (see
  [WBLOCK](WBLOCK-Command.md))
* *DXX Extract (\*.dxx):*Attribute extract DXF ™  file (see
  [-ATTEXT](ATTEXT-ATTEXT-Command.md))
* *Encapsulated PS (\*.eps):* Encapsulated PostScript file
* *Lithography (\*.stl):* Solid object stereolithography file (see
  [STLOUT](STLOUT-Command.md))

NOTE:Use the PLOT command to output a drawing to PDF.
