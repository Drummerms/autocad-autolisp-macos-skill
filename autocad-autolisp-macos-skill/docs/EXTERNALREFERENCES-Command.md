# EXTERNALREFERENCES (Command)

Opens the Reference Manager palette.

*Menu*:
Window ![](../images/ac.menuaro.gif) Reference Manager.

## Summary

The EXTERNALREFERENCES command opens the
[Reference Manager palette](Reference-Manager-Palette.md) when the palette is closed.

NOTE:The
 [FILEDIA](FILEDIA-System-Variable.md) system variable is ignored when attaching files from the Reference Manager palette.
