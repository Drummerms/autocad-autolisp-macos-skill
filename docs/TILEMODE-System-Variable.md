# TILEMODE (System Variable)

Makes the Model tab or the last layout tab current.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| 0 | Makes the last active layout tab (paper space) active |
| 1 | Makes the Model tab active |
