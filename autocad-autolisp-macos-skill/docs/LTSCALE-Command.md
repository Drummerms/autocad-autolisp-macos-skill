# LTSCALE (Command)

Sets the global linetype scale factor.

## Access Methods

![](../images/ac.keyboard.gif) Command entry:  *'ltscale* for transparent use

## Summary

Use LTSCALE to change the scale factor of linetypes for all objects in a drawing. Changing the linetype scale factor causes
the drawing to be regenerated.
