# L Functions Reference (AutoLISP)

These AutoLISP functions all start with 'L'.

## Functions

* [lambda](lambda-AutoLISP.md)

  Defines an anonymous function
* [last](last-AutoLISP.md)

  Returns the last element in a list
* [layerstate-addlayers](layerstate-addlayers-AutoLISP.md)

  Adds or updates a series of layers to a layer state
* [layerstate-compare](layerstate-compare-AutoLISP.md)

  Compares a layer state to the layers in the current drawing
* [layerstate-delete](layerstate-delete-AutoLISP.md)

  Deletes a layer state
* [layerstate-export](layerstate-export-AutoLISP.md)

  Exports a layer state to a specified file
* [layerstate-getlastrestored](layerstate-getlastrestored-AutoLISP.md)

  Returns the name of the last restored layer state in the current drawing
* [layerstate-getlayers](layerstate-getlayers-AutoLISP.md)

  Returns the layers saved in a layer state
* [layerstate-getnames](layerstate-getnames-AutoLISP.md)

  Returns a list of the layer state names
* [layerstate-has](layerstate-has-AutoLISP.md)

  Checks if a layer state is present
* [layerstate-import](layerstate-import-AutoLISP.md)

  Imports a layer state from a specified file
* [layerstate-importfromdb](layerstate-importfromdb-AutoLISP.md)

  Imports a layer state from a specified drawing file
* [layerstate-removelayers](layerstate-removelayers-AutoLISP.md)

  Removes a list of layers from a layer state
* [layerstate-rename](layerstate-rename-AutoLISP.md)

  Renames a layer state
* [layerstate-restore](layerstate-restore-AutoLISP.md)

  Restores a layer state into the current drawing
* [layerstate-save](layerstate-save-AutoLISP.md)

  Saves a layer state in the current drawing
* [layoutlist](layoutlist-AutoLISP.md)

  Returns a list of all paper space layouts in the current drawing
* [length](length-AutoLISP.md)

  Returns an integer indicating the number of elements in a list
* [list](list-AutoLISP.md)

  Takes any number of expressions and combines them into one list
* [listp](listp-AutoLISP.md)

  Verifies that an item is a list
* [load\_dialog](load_dialog-AutoLISP-DCL.md)

  Loads a DCL file
* [load](load-AutoLISP.md)

  Evaluates the AutoLISP expressions in a file
* [log](log-AutoLISP.md)

  Returns the natural log of a number as a real number
* [logand](logand-AutoLISP.md)

  Returns the result of the logical bitwise AND of a list of integers
* [logior](logior-AutoLISP.md)

  Returns the result of the logical bitwise inclusive OR of a list of integers
* [lsh](lsh-AutoLISP.md)

  Returns the logical bitwise shift of an integer by a specified number of bits
