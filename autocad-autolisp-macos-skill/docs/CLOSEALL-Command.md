# CLOSEALL (Command)

Closes all currently open drawings.

## Access Methods

*Shortcut Menu*: right-click on any File tab.

## Summary

All open drawings are closed. A message box is displayed for each unsaved drawing, in which you can save any changes to the
drawing before closing it.
