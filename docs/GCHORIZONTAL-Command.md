# GCHORIZONTAL (Command)

Causes lines or pairs of points to lie parallel to the X axis of the current coordinate system.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Parametric panel ![](../images/ac.menuaro.gif) Geometric Constraint drop-down ![](../images/ac.menuaro.gif) Horizontal.
![](../images/GUID-AC8ADAFD-3EF5-43CE-B08A-32EF0862CE56-low.png)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Parametric ![](../images/ac.menuaro.gif) Geometric Constraints ![](../images/ac.menuaro.gif) Horizontal.

## Summary

This command is equivalent to the Horizontal option in
[GEOMCONSTRAINT](GEOMCONSTRAINT-Command.md).

You can select different constraint points on the same object or different objects.

The following are the valid constraint objects and points:

* Line
* Polyline segment
* Ellipse
* Multiline text
* Two valid constraint points

## List of Prompts

The following prompts are displayed.

Select an object or [[2Points](GCHORIZONTAL-Command.md)] <2Points>:
*Select an object or two constraint points*

2Points

Selects two constraint points instead of an object.

The second selected point on an object is made horizontal to the first selected point.
