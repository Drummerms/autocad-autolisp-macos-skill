# Repair, Restore, and Recover Drawings

#### Topics in this section

* [About Repairing Damaged Drawing Files](About-Repairing-Damaged-Drawing-Files.md)

  If a drawing file is damaged, you can recover some or all of the data by using commands to find and correct errors.
* [About Creating and Restoring from Backup Files](About-Creating-and-Restoring-from-Backup-Files.md)

  Backup files help ensure the safety of your drawing data. If a problem occurs, you can restore a drawing backup file.
* [About Recovering from a System Failure](About-Recovering-from-a-System-Failure.md)

  A hardware problem, power failure, or software problem can cause this program to terminate unexpectedly. If this happens,
  you can restore the drawing files that were open.
