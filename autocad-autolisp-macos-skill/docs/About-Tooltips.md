# About Tooltips

Several types of tooltips provide pop-up information for interaction with toolbars, object snaps, and drafting operations.

Tooltips are displayed for buttons and options in Tool Sets palette and dialog boxes. When the cursor is over a tool button,
for example, a basic tooltip is displayed. If you continue to hover, the tooltip expands to display additional information.
You can customize the tooltip appearance in the Look & Feel tab of the Application Preferences dialog box.
