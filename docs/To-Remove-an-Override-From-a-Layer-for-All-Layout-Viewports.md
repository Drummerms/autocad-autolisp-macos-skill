# To Remove an Override From a Layer for All Layout Viewports

1. On a layout, double-click inside a layout viewport to make it the
   *current* viewport.
2. Click
   Window menu ![](../images/ac.menuaro.gif) Layers.
3. In the Layer Properties Manager, right-click the property override that you want to remove. Property overrides are in columns
   labeled VP Freeze, VP Color, and so on.
4. Click Remove Viewport Overrides For![](../images/ac.menuaro.gif) [
   *property name* ] ![](../images/ac.menuaro.gif)In All Viewports.
