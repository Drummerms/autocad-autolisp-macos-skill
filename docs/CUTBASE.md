# CUTBASE

The new CUTBASE command copies selected objects to the Clipboard, along with a specified base point, and removes them from
the drawing.

![](../images/GUID-B7E7C147-BA54-4B60-A6A2-45C90BD83261-low.png)

1. Click
   Edit menu ![](../images/ac.menuaro.gif) Cut with Base Point.
2. Specify the base point on the drawing.
3. Select the objects you want to cut and press Enter.

   Use
   [PASTECLIP](PASTECLIP-Command.md) to specify the insertion point when pasting the objects into other drawings.
