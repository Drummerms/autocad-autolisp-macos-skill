# big\_increment Attribute (DCL)

Specifies the value used by the slider's incremental controls.

*Supported Platforms:* Windows only

```
big_increment = integer;
```

The default value of
big\_increment is one-tenth of the total range. The value must be within the range specified by
min\_value and
max\_value.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
