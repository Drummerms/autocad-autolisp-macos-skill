# To Deactivate a Layout Viewport

Displaying views in a large number of active layout viewports can negatively impact your system's performance. You can save
time by temporarily turning them off.

1. Select the border of a layout viewport.
2. Right-click, and choose Display Viewport Objects![](../images/ac.menuaro.gif)No.
