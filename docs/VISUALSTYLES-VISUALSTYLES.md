# VISUALSTYLES (-VISUALSTYLES)

Creates and modifies visual styles from the command line.

## List of Prompts

The following prompts are displayed.

Enter an option [[set Current](VISUALSTYLES-VISUALSTYLES.md)/[Saveas](VISUALSTYLES-VISUALSTYLES.md)/[Rename](VISUALSTYLES-VISUALSTYLES.md)/[Delete](VISUALSTYLES-VISUALSTYLES.md)/[?](VISUALSTYLES-VISUALSTYLES.md)]:

Set Current

These options are the same as the options in [VSCURRENT](VSCURRENT-Command.md).

Save As

Saves the visual style with the name you specify.

NOTE:You must be in model space to save a visual style. If you enter a name that is already in use for a visual style, you can
either replace the existing visual style or enter a different name.

Rename

Renames a visual style.

Delete

Deletes a visual style.

?—List Visual Styles

Lists the visual styles in the drawing.
