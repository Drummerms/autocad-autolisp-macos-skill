# ACADVER (System Variable)

Returns the product's version number.

(Read-only)

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Not-saved |
| Initial value: | Varies |

This variable differs from the DXF file $ACADVER header variable, which contains the drawing database level number.
