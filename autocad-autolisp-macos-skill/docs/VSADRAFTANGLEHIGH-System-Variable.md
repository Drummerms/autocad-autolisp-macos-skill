# VSADRAFTANGLEHIGH (System Variable)

Sets the value at which a model displays as green during draft analysis ([ANALYSISDRAFT](ANALYSISDRAFT-Command.md)).

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 3 |

Draft angle is the angle in degrees between the surface normal and the UCS plane. High value is set by this system variable.

| -90 | Surface is parallel to the UCS with surface normal facing the opposite direction as the construction plane |
| 0 | Surface is perpendicular to the construction plane |
| 90 | Surface is parallel to the construction plane with surface normal facing the same direction as the UCS |
