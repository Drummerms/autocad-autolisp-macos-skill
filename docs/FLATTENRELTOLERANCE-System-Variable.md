# FLATTENRELTOLERANCE (System Variable)

Controls the precision of a curved polyline during FLATTEN. The smaller the number, the more precise.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Registry |
| Initial value: | 0.01 |
