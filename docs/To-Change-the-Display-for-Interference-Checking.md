# To Change the Display for Interference Checking

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Interference Checking.
   ![](../images/GUID-6A71F6D9-C943-4022-92E5-8DF55C683927-low.png)
2. Enter
   *s* (Settings). Press Enter.
3. In the Interference Settings dialog box, change any of the settings and click OK.
