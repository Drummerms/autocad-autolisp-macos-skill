# SPACEPAN (System Variable)

Controls whether pressing and holding the Spacebar starts the PAN command transparently.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | PAN command does not start transparently when the Spacebar is held. |
| 1 | PAN command starts transparently after the Spacebar is held for the duration specified in the SPACEPANTIMEOUT system variable. |
