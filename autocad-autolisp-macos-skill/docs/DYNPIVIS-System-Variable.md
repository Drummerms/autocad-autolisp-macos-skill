# DYNPIVIS (System Variable)

Controls when pointer input is displayed.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Only when you type at a prompt for a point |
| 1 | Automatically at a prompt for a point |
| 2 | Always |

The
[DYNMODE](DYNMODE-System-Variable.md) system variable turns Dynamic Input features on and off.
