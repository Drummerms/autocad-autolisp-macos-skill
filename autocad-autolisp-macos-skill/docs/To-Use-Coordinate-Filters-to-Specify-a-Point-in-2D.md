# To Use Coordinate Filters to Specify a Point in 2D

1. At the prompt for a point, enter a coordinate filter (*.x* or *.y*).

   For example, enter *.x* to specify the *X* value first.
2. To extract the first coordinate value, specify a point, or enter a value.

   For example, if you entered *.x* in step 1, the *X* value is extracted from this point.
3. To extract the next coordinate value, specify a different point or value.

   The new point location combines the coordinate values extracted from the points or values that you specified.
