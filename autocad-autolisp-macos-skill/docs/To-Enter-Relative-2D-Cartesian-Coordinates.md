# To Enter Relative 2D Cartesian Coordinates

* At a prompt for a point, enter UCS coordinates using the following format:

  *@* *x* *,* *y*
* Turn off dynamic input. At a prompt for a point, enter WCS coordinates using the following format:

  *@ \** *x* *,* *y*
