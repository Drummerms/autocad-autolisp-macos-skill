# To Add a View to a Sheet

1. Click
   Window menu ![](../images/ac.menuaro.gif) Sheet Set Manager.
2. In the Sheet Set Manager, double-click a sheet to open it. Or, create a new sheet and open it.
3. At the Command prompt, enter
   *-view* to create a named view.

   The specified view is added to the sheet. If a view label block is defined in the sheet set properties, a view label that
   displays view-specific information is automatically placed on the sheet.
