# Zebra Tab (Analysis Options Dialog Box)

Changes the display settings for the ANALYSISZEBRA command.

ANALYSISOPTIONS (Command)

*Tool Set*:
Modeling tab ![](../images/ac.menuaro.gif) Surface panel ![](../images/ac.menuaro.gif) Analysis Options.
![](../images/GUID-E43E5959-FA91-43AA-B25D-2754135C2B20-low.png)

## List of Options

The following options are displayed.

Select Objects to Analyze

Prompts you to select the surface objects to analyze. When you are done selecting objects, press Enter to return to the dialog
box.

Stripe Display

Stripe Direction
:   Specifies the display angle. ([VSAZEBRADIRECTION](VSAZEBRADIRECTION-System-Variable.md)) This option is only available when Type is set to Cylinder. If you are using the Chrome Ball type, change the stripe directions
    with the
    [VSAZEBRADIRECTION](VSAZEBRADIRECTION-System-Variable.md) system variable.

Type
:   Sets analysis display type. ([VSAZEBRATYPE](VSAZEBRATYPE-System-Variable.md))

Size
:   Sets the width of the zebra stripes. ([VSAZEBRASIZE](VSAZEBRASIZE-System-Variable.md))

Color 1
:   Sets the first color for zebra stripes. ([VSAZEBRACOLOR1](VSAZEBRACOLOR1-System-Variable.md))

Color 2
:   Sets the second color for zebra stripes. ([VSAZEBRACOLOR2](VSAZEBRACOLOR2-System-Variable.md))

Clear Zebra Analysis

Removes the zebra display from all objects in the current drawing.
