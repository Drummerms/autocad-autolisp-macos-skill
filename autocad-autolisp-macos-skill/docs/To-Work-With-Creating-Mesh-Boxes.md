# To Work With Creating Mesh Boxes

## Create a Mesh Box Based on a Diagonal and a Height

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Mesh panel ![](../images/ac.menuaro.gif) Primitive drop-down ![](../images/ac.menuaro.gif) Box.
   ![](../images/GUID-420D13B2-59AB-420C-A8A2-CFAE1D5EBC66-low.png)
2. Specify the first corner of the base.
3. Specify the opposite corner of the base.
4. Specify the height.

## Create a Mesh Cube

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Mesh panel ![](../images/ac.menuaro.gif) Primitive drop-down ![](../images/ac.menuaro.gif) Box.
   ![](../images/GUID-420D13B2-59AB-420C-A8A2-CFAE1D5EBC66-low.png)
2. Specify the first corner or enter
   *c* (Center) and specify the center point of the base.
3. At the Command prompt, enter
   *c* (Cube). Specify the length of the cube and a rotation angle.

## Create a Mesh Box Based on Length, Width, and Height

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Mesh panel ![](../images/ac.menuaro.gif) Primitive drop-down ![](../images/ac.menuaro.gif) Box.
   ![](../images/GUID-420D13B2-59AB-420C-A8A2-CFAE1D5EBC66-low.png)
2. Specify the first corner of the base.
3. At the Command prompt, enter
   *L* (Length). Specify the length of the base.
4. Specify the width of the base.
5. Specify the height.

## Create a Mesh Box Based on a Center Point, Corner of Base, and Height

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Mesh panel ![](../images/ac.menuaro.gif) Primitive drop-down ![](../images/ac.menuaro.gif) Box.
   ![](../images/GUID-420D13B2-59AB-420C-A8A2-CFAE1D5EBC66-low.png)
2. At the Command prompt, enter
   *c* (Center). Specify the center point of the base.
3. To set the location of the corner of the base, use one of the following methods:
   * To set the length and width simultaneously: Specify the location of one corner of the base.
   * To set the length and width separately: At the Command prompt, enter
     *L* (Length) and specify the length. Then specify the width.
   * Specify the height.
