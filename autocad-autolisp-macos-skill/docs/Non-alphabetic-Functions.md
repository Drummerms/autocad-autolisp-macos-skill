# Non-alphabetic Functions

#### Topics in this section

* [+ (add) (AutoLISP)](+-add-AutoLISP.md)

  Returns the sum of all numbers
* [/ (divide) (AutoLISP)](divide-AutoLISP.md)

  Divides the first number by the product of the remaining numbers and returns the quotient
* [> (greater than) (AutoLISP)](greater-than-AutoLISP.md)

  Returns
  T if each argument is numerically greater than the argument to its right; otherwise returns
  nil
* [>= (greater than or equal to) (AutoLISP)](=-greater-than-or-equal-to-AutoLISP.md)

  Returns
  T if each argument is numerically greater than or equal to the argument to its right; otherwise returns
  nil
* [< (less than) (AutoLISP)](less-than-AutoLISP.md)

  Returns
  T if each argument is numerically less than the argument to its right; otherwise
  nil
* [<= (less than or equal to) (AutoLISP)](=-less-than-or-equal-to-AutoLISP.md)

  Returns
  T if each argument is numerically less than or equal to the argument to its right; otherwise returns
  nil
* [\* (multiply) (AutoLISP)](multiply-AutoLISP.md)

  Returns the product of all numbers
* [\*error\* (AutoLISP)](error-AutoLISP.md)

  A user-definable error-handling function
* [\*pop-error-mode\* (AutoLISP)](pop-error-mode-AutoLISP.md)

  Error-handling function that ends the previous call to
  \*push-error-using-command\* or
  \*push-error-using-stack\*
* [\*push-error-using-command\* (AutoLISP)](push-error-using-command-AutoLISP.md)

  Error-handling function that indicates the use of the
  command function within a custom
  \*error\* handler
* [\*push-error-using-stack\* (AutoLISP)](push-error-using-stack-AutoLISP.md)

  Error-handling function that indicates the use of variables from the AutoLISP stack within a custom
  \*error\* handler
* [- (subtract) (AutoLISP)](subtract-AutoLISP.md)

  Subtracts the second and following numbers from the first and returns the difference
* [/= (not Equal to) (AutoLISP)](=-not-Equal-to-AutoLISP.md)

  Compares arguments for numerical inequality
* [= (equal to) (AutoLISP)](=-equal-to-AutoLISP.md)

  Compares arguments for numerical equality
* [~ (bitwise NOT) (AutoLISP)](~-bitwise-NOT-AutoLISP.md)

  Returns the bitwise NOT (1's complement) of the argument
* [1- (decrement) (AutoLISP)](1-decrement-AutoLISP.md)

  Decrements a number by 1
* [1+ (increment) (AutoLISP)](1+-increment-AutoLISP.md)

  Increments a number by 1
