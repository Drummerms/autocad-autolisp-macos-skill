# To Dock, Undock, or Auto-Hide Palettes

## Dock a Palette

Palettes are docked at the right side of the drawing area. If you dock multiple palettes, click the palette's tab to switch
to that palette.

![](../images/GUID-E490283A-0592-4286-B5D7-5C473BB4DE5E-low.png) - dock a palette

## Undock a Palette

Floating palettes can be moved and resized. Floating palettes can't be set to auto-hide.

![](../images/GUID-ED0D56F6-458D-44A3-A482-8B5EF683322E-low.png) - undock a palette

## Auto-hide Docked Palettes

Auto-hide affects all docked palettes. The title bar for each docked palette remains visible. Mouse over the title bar for
the palette you want to show, and the palette automatically expands.

![](../images/GUID-E345AE0B-F4AC-44E4-B673-8376FDCD554C-low.png) - set auto-hide on

![](../images/GUID-62E726F6-80EC-46BE-A730-C2532D4726FB-low.png) - set auto-hide off

NOTE:The tool sets and command line palettes are fixed and can't be floating. Sheet Set Manager and Materials Browser can't be
docked.
