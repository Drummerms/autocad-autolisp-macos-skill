# SURFU (System Variable)

Sets the surface density for PEDIT Smooth in the M direction and the U isolines density on surface objects.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 6 |

Valid values are 0 through 200. Meshes are always created with a minimum surface density of 2.
