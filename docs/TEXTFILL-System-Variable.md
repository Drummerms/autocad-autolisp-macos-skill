# TEXTFILL (System Variable)

Controls the filling of TrueType fonts while printing.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Displays text as outlines |
| 1 | Displays text as filled images |
