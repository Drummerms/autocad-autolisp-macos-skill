# To Preview an Image and View File Details

1. Click Insert ![](../images/ac.menuaro.gif) References Manager.At the Command prompt, enter externalreferences.
2. In the Reference Manager palette, select an image file.

   The file preview is displayed.
3. To view file details, click the Details and Preview Panel.

   The file details (image size) are displayed.
