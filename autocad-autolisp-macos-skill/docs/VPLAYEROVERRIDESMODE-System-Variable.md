# VPLAYEROVERRIDESMODE (System Variable)

Controls whether layer property overrides for layout viewports are displayed and plotted.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Layer property overrides are not displayed in layout viewports or plotted |
| 1 | Layer property overrides are displayed in layout viewports and plotted |
