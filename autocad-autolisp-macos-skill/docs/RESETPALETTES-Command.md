# RESETPALETTES (Command)

Resets all palettes to their default placement settings.

## Summary

*Menu*:
Window ![](../images/ac.menuaro.gif) Reset Palettes.

## Summary

The Reset Palettes message box is displayed. Resetting palettes closes and restarts
AutoCAD 2026.

## List of Options

The following prompts are displayed.

Reset Palettes
:   Closes and restarts
    AutoCAD 2026. After
    AutoCAD 2026 restarts, the placement of all palettes is reset to their default settings.
