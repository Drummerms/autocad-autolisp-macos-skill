# To Automatically Fill Cells With Incremented Data

1. Set the data type for the cell, for example currency, date, whole number.
   1. Click inside the cell.
   2. If the Properties Inspector isn't open, click
      Window menu ![](../images/ac.menuaro.gif) Properties Inspector.
   3. On the Properties Inspector, select a data type from the drop-down menu and set any format options.
2. Double-click inside the cell.
3. Enter a numeric value; for example, 1 or 01/01/2000.
4. On the Text Editor visor, click Save.
5. Select the cell and Click the grip in the lower right corner.
6. Drag the grip through the cells you would like to automatically increment.

   A preview of the value for each cell will display to the right of the selected grip.
7. Click the last cell to fill the cells.
