# To Create a 3D Solid Shell

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Solid Edit - Face drop-down ![](../images/ac.menuaro.gif) Taper Faces.
   ![](../images/GUID-08D599BE-62B5-4351-8EA7-3D4E5C4B6B7E-low.png)
2. Select the 3D solid object.
3. Select one or more faces to be excluded from shelling and press Enter.
4. Specify the shell offset distance.

   A positive offset value creates a shell wall in a positive direction from the face. A negative value creates a shell wall
   in a negative direction from the face.
5. Press Enter to complete the command.
