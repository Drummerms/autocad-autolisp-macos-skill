# Rename Dialog Box

Changes the names assigned to named objects such as layers and dimension styles.

RENAME (Command)

*Menu*:
Format menu ![](../images/ac.menuaro.gif) Rename

![](../images/GUID-5921ACBA-DE91-483A-AAAB-7E9169CBF5DC-low.png)

## List of Options

The following options are displayed.

Named Objects
:   Lists named objects in a drawing by category.

Items
:   Displays items of the type specified in Named Objects.
