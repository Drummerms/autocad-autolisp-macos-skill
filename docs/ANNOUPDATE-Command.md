# ANNOUPDATE (Command)

Updates existing annotative objects to match the current properties of their styles.

## Summary

When a non-annotative text object is updated to an annotative text style, the object becomes annotative and supports the current
 . If the text style has a fixed Paper Height, the object is set to that height. If the text style’s Paper Height is set to
0, the size of the text does not change. The Paper Height value is inferred by dividing the model height by the current annotation
scale.

If an annotative object is updated to a non-annotative style, the object becomes non-annotative and any alternate   are removed.
