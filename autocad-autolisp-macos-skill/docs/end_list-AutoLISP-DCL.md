# end\_list (AutoLISP/DCL)

Ends processing of the currently active dialog box list

*Supported Platforms:* Windows, Mac OS, and Web

## Signature

```
(end_list)
```

No arguments.

## Return Values

*Type:* nil

Always returns
nil.

## Remarks

This function is the complement of
start\_list.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
