# DIVMESHCYLAXIS (System Variable)

Sets the number of subdivisions around the perimeter of the mesh cylinder base.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 8 |

![](../images/GUID-2C0F5B69-E875-4595-B192-49F8D80AC9D6-low.png)

This setting affects new mesh cylinders that are created with the [MESH](MESH-Command.md) command.
