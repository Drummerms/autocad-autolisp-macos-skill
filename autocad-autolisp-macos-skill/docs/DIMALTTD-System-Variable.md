# DIMALTTD (System Variable)

Sets the number of decimal places for the tolerance values in the alternate units of a dimension.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 2 (imperial) or 3 (metric) |
