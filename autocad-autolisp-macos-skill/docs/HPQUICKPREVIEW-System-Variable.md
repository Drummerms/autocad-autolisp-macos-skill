# HPQUICKPREVIEW (System Variable)

Controls whether a hatch preview is displayed when specifying a hatch area.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | User-settings |
| Initial value: | 0 |

| 0 or OFF | No preview |
| 1 or ON | Quick preview |
