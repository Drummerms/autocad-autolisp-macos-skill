# VSACURVATUREHIGH (System Variable)

Sets the value at which a surface displays as green during curvature analysis ([ANALYSISCURVATURE](ANALYSISCURVATURE-Command.md)).

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 1 |

The default value for this system variable is 1.
