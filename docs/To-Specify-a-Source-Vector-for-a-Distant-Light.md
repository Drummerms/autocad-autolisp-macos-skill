# To Specify a Source Vector for a Distant Light

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Render panel ![](../images/ac.menuaro.gif) Light drop-down ![](../images/ac.menuaro.gif) New Distant Light.
   ![](../images/GUID-7F345E12-A694-45AA-930C-B18D4104EA58-low.png)
2. Specify light direction FROM <0,0,0> or [Vector]: Specify a point on the screen
3. Specify light direction TO <1,1,1>: Specify a point on the screen.
4. If you enter V for the Vector option, the following prompt is displayed:

   Specify vector direction <0.0000,-0.0100,1.0000>: Enter a vector
5. Select options (*Name, Intensity Factor, Status, Shadow, Filter color*) to change or select Exit.
