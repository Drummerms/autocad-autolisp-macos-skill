# To Create a Sheet Set

1. Click
   File menu ![](../images/ac.menuaro.gif) New Sheet Set.
2. Do any of the following:

   ## Create a Blank Sheet Set

   1. In the New Sheet Set dialog box, enter the name, location, and description select of the sheet set.
   2. Select Create Blank Project.

   Once the sheet set is created, you can modify its properties, add sheets from existing drawings, and create new sheets or
   subsets.

   ## Use a Template

   * In the New Sheet Set dialog box, select Use Template and browse to the desired template.

     The template provides the organizational structure and desired settings for the new sheet set.

   After you create an empty sheet set with this option, you can import layouts or create sheets individually.

   ## Copy an Existing Sheet Set Structure

   When you choose to create a sheet set from an existing DST file, the sheet set provides the organizational structure only
   for the new sheet set. The properties are set to their default values, and no sheets are added from the existing sheet set.

   1. In the New Sheet Set dialog box, select Copy Existing Sheet Set Structure.
   2. Click Browse to select a sheet set data (DST) file.
   3. Specify one or more folders that contain drawing files.
   4. Specify the subset organization for the sheet set.

      The layouts from these drawings can be imported into the sheet set automatically.
3. Click Create.
4. In the Save As dialog box, click Save.
