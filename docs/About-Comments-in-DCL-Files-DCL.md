# About Comments in DCL Files (DCL)

A statement preceded by two forward slashes (//) is treated as a comment in a DCL file. Anything that appears between the
// and the end of the line is ignored. DCL also allows C language style comments. These have the form
/\* *comment text*  \*/. The starting
/\* and ending
\*/ can be on separate lines.
