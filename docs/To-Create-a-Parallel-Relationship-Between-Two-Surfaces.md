# To Create a Parallel Relationship Between Two Surfaces

1. Click
   Tools menu ![](../images/ac.menuaro.gif) Parametric ![](../images/ac.menuaro.gif) Geometric Constraints ![](../images/ac.menuaro.gif) Parallel.
2. Select the first and second profile objects. Be careful not to select the surface or the edge subobject.

If the surfaces are not already parallel, they are moved into a parallel position and the parallel constraint symbol is displayed.
