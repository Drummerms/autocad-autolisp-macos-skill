# DELCONSTRAINT (Command)

Removes all geometric and dimensional constraints from a selection set of objects.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Parametric panel ![](../images/ac.menuaro.gif) Delete Constraints.
![](../images/GUID-72CB99BF-8435-4B37-8801-3BF001E8AA1E-low.png)

NOTE:Hidden by default. Click
![](../images/GUID-78DED619-51B3-49C1-AD71-F928B5F69C0A-low.png) to display the icon on the tool set panel.

*Menu*:
Tools ![](../images/ac.menuaro.gif) Parametric ![](../images/ac.menuaro.gif) Delete Constraints.

## Summary

The number of constraints removed are displayed on the command line.

Removes all geometric and dimensional constraints from the selected objects.
