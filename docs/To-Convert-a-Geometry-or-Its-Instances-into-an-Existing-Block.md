# To Convert a Geometry or Its Instances into an Existing Block

You can convert the source object or the found instances into an existing block.

1. At the Command prompt, enter
   *BSEARCH*.
2. Select the geometry you want to convert and press Enter.

   The instances of the selected geometry are highlighted in the drawing.
3. (Optional) To exclude certain instances from the conversion, click them individually.
4. At the prompt, choose whether to convert only the source object or include all found instances.
5. Press Enter.
6. In the Convert dialog box, under Convert instances to, click Existing block.
7. Do one of the following:
   * Pick a block from the current drawing
   * Select a block from the list of blocks in the current drawing
8. Click Convert.
9. In the drawing, accept the default placement or adjust the scale and rotation to determine how the selected block definition
   replaces the found instances.
