# Modify Dimensions

#### Topics in this section

* [About Modifying Dimension Geometry](About-Modifying-Dimension-Geometry-2.md)

  You can modify dimensions with the editing commands and with grip editing.
* [About Modifying Dimension Text](About-Modifying-Dimension-Text.md)

  After you create a dimension, you can move, rotate, or replace the dimension text.
* [About Adjusting the Spacing Between Dimensions](About-Adjusting-the-Spacing-Between-Dimensions.md)

  You can automatically adjust existing parallel linear and angular dimensions in a drawing so they are equally spaced or aligned
  at the dimension line with each other.
* [About Creating Breaks in Dimensions](About-Creating-Breaks-in-Dimensions.md)

  With dimension breaks, you can keep the dimension, extension, or leader lines from appearing as if they are a part of the
  design.
* [About Converting Dimensions Into Inspection Dimensions](About-Converting-Dimensions-Into-Inspection-Dimensions.md)

  Inspection dimensions allow you to effectively communicate how frequently manufactured parts should be checked to ensure that
  the dimension value and tolerances of the parts are within the specified range.
