# COMPARESHOWCONTEXT (System Variable)

Controls the visibility of objects that are not used in the xref comparison.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

NOTE:This system variable is available only during an xref comparison.

| Value | Description |
| 0 | Hidden |
| 1 | Displayed |
