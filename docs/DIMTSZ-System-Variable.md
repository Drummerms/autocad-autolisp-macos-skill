# DIMTSZ (System Variable)

Specifies the size of oblique strokes drawn instead of arrowheads for linear, radius, and diameter dimensioning.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 0.0000 |

| 0 | Draws arrowheads. |
| >0 | Draws oblique strokes instead of arrowheads. The size of the oblique strokes is determined by this value multiplied by the  [DIMSCALE](DIMSCALE-System-Variable.md) value. |
