# GRIDMAJOR (System Variable)

Controls the frequency of major grid lines compared to minor grid lines.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 5 |

Valid values range from 1 to 100.

Grid lines are displayed in any visual style.
