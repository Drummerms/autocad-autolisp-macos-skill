# GCPARALLEL (Command)

Causes selected lines to lie parallel to each other.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Parametric panel ![](../images/ac.menuaro.gif) Geometric Constraint drop-down ![](../images/ac.menuaro.gif) Parallel.
![](../images/GUID-F9CB63B4-27C0-4119-B630-81BE60AB9205-low.png)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Parametric ![](../images/ac.menuaro.gif) Geometric Constraints ![](../images/ac.menuaro.gif) Parallel.

## Summary

This command is equivalent to the Parallel option in
[GEOMCONSTRAINT](GEOMCONSTRAINT-Command.md).

Following are the valid constraint objects and points:

* Line
* Polyline segment
* Ellipse
* Multiline text
