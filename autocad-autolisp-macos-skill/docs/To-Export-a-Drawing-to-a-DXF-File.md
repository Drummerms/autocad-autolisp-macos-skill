# To Export a Drawing to a DXF File

1. Click File menu ![](../images/ac.menuaro.gif)Save As.At the Command prompt, enter SAVEAS.
2. In the Save Drawing As dialog box, File Format pane, select a DXF format.
3. Select a folder and enter a file name.
4. Click Save.
