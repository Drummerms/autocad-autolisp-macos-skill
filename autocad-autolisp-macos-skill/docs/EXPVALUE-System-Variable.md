# EXPVALUE (System Variable)

Specifies the exposure value to apply during rendering.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 8.8000 |

The exposure value controls how much light is applied to a rendered image.

Decrease the value to brighten or increase the value to darken the rendered image. Valid value range is -6 to 21.
