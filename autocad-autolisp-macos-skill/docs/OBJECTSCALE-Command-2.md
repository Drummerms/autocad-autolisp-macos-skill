# OBJECTSCALE (Command)

Adds or deletes supported scales for annotative objects.

## Access Methods

*Menu*:
Modify
![](../images/ac.menuaro.gif) Annotative Object Scale
![](../images/ac.menuaro.gif) Add/Delete Scales.

Shortcut menu: Select an annotative object. Right-click in the drawing area and click Annotative Object Scale ![](../images/ac.menuaro.gif) Add/Delete Scales.

![](../images/ac.keyboard.gif) Command entry:  *'objectscale* for transparent use

## Summary

An annotative object can support several annotation scales for views at different scales. You can add or delete scales from
the list of scales assigned to the selected annotative objects.

If you enter
*objectscale*, you are prompted to select annotative objects.

The
[Annotation Object Scale List dialog box](Annotation-Object-Scale-List-Dialog-Box.md) is displayed.

If you enter
*-objectscale* at the Command prompt,
[options are displayed](OBJECTSCALE-Command.md).
