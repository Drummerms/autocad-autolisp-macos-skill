# Commands for Working With Trace

#### Commands

* [TRACE](GUID-D5D7D88B-6C2F-46F9-84F0-FD7B438BDCED.htm " Opens and manages traces from the command prompt.
  		")
* [TRACEBACK](GUID-8B2037E1-D1E3-496F-BF67-ACBC4155AE89.htm " Displays the host drawing with full saturation, while dimming the trace geometry.
  		")
* [TRACEEDIT](GUID-B830702F-25E9-4548-BE97-BFF2152934B9.htm " Changes the active trace to edit mode so you can contribute to the trace.
  		")
* [TRACEFRONT](GUID-BA5EED80-4DBB-478E-802B-B1BA933E3FA1.htm " Displays the active trace with full saturation, while dimming the host drawing geometry.
  		")
* [TRACEPALETTECLOSE](GUID-189DED05-AFC1-4F83-A616-937D3D50C293.htm "Closes the Trace palette where you view and manage traces in the current drawing.
  		")
* [TRACEPALETTEOPEN](GUID-13413FA0-13B1-4323-BD6A-CEEE3A688C09.htm " Opens the Trace palette where you can view and manage traces in the current drawing.
  		")
* [TRACEVIEW](GUID-DDDD9D4D-36BC-421F-AA95-1C5846A1FC7A.htm " Changes the active trace to view mode so you can edit the parent drawing while the trace is still visible.
  		")

#### System Variables

* [TRACECURRENT](GUID-E1AB2A1C-672A-4946-A21B-4B9C341E8F20.htm " Displays the name of the active trace when TRACEMODE=1 or 2.
  		")
* [TRACEDISPLAYMODE](GUID-BE8D20A7-4F37-4272-8B0D-B9ACBD0EDA83.htm " Indicates whether the tracing paper effect is displayed (front) or not (back) while a trace is active.
  		")
* [TRACEFADECTL](GUID-03F37CD9-53F6-4A93-87B2-868BA11D217B.htm " Controls the amount of fading when TRACEMODE is active.
  		")
* [TRACEMODE](GUID-B026D045-3154-4989-9711-B31159CCF5AB.htm " Indicates whether Trace is active and which mode is current - editing or viewing.
  		")
* [TRACEOSNAP](GUID-4B220A84-9185-4213-9C07-EE36E0411546.htm " Controls whether object snaps apply to trace geometry while viewing a trace.
  		")
* [TRACEPALETTESTATE](GUID-017F3481-442D-410A-8BB7-5A79195327B0.htm " Reports whether the Trace palette is open or closed.
  		")
* [TRACEPAPERCTL](GUID-B2D2101F-F8E4-468A-9FAE-DC7BBE909985.htm " Controls the opaqueness of the tracing paper effect.
  		")
