# About AutoLISP Basics (AutoLISP)

You can use number, string, and list-handling functions to customize AutoCAD.

The following links introduce the basic concepts of the AutoLISP ®  programming language. It describes the core components and data types used in AutoLISP, and presents examples of simple number-,
string-, output-, and list-handling functions.

AutoLISP code does not need to be compiled, so you can enter the code at a Command line and immediately see the results.

#### Topics in this section

* [About Expressions (AutoLISP)](About-Expressions-AutoLISP.md)

  An expression is the basic structure that is used when working with AutoLISP.
* [About Data Types (AutoLISP)](About-Data-Types-AutoLISP.md)
* [About Source Code Files (AutoLISP)](About-Source-Code-Files-AutoLISP.md)

  Although you can enter AutoLISP code at the AutoCAD Command prompt or Visual LISP Console window prompt (AutoCAD for Windows
  only), any functions you define are lost when you close the drawing or session they were created in.
* [About Variables (AutoLISP)](About-Variables-AutoLISP.md)

  Variables are used to store a value or list of values in memory.
* [About Number Handling (AutoLISP)](About-Number-Handling-AutoLISP.md)

  AutoLISP provides functions for working with integers and real numbers.
* [About Strings and String Handling (AutoLISP)](About-Strings-and-String-Handling-AutoLISP.md)

  A string is a group of characters surrounded by quotation marks. Within quoted strings the backslash (\) character allows
  control characters (or escape codes) to be included.
* [About Basic Output Functions (AutoLISP)](About-Basic-Output-Functions-AutoLISP.md)
* [About Equality and Conditional (AutoLISP)](About-Equality-and-Conditional-AutoLISP.md)
* [About List Handling (AutoLISP)](About-List-Handling-AutoLISP.md)
* [About Symbol and Function Handling (AutoLISP)](About-Symbol-and-Function-Handling-AutoLISP.md)
* [About Error Handling (AutoLISP)](About-Error-Handling-AutoLISP.md)

  The AutoLISP language provides several functions for handling errors.
