# To Open a Drawing Saved as a Block (Block Editor)

1. Click the Application button, and click Open ![](../images/ac.menuaro.gif) Drawing.
2. Open the drawing file that is saved as a block.
3. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block Editor.
   ![](../images/GUID-E3DDBE51-C92A-4D07-A51D-C7C685E4F578-low.png)
4. In the Edit Block Definition dialog box, double-click <Current Drawing>.
