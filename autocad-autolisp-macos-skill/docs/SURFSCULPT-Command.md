# SURFSCULPT (Command)

Trims and combines surfaces that bound a watertight area to create a solid.

*Menu*:
Modify ![](../images/ac.menuaro.gif) Surface Editing ![](../images/ac.menuaro.gif) Sculpt.

## Summary

The SURFSCULPT command automatically combines and trims a collection of surfaces that enclose a watertight area to create
a solid.

![](../images/GUID-F78D3F8D-A630-41E1-96B8-90DB31E8AEF6-low.gif)

NOTE:The area enclosed by the surfaces must be watertight and the surfaces must have a continuity of G0 or the SURFSCULPT command
cannot complete.

The SURFSCULPT command also works with solid and mesh objects. If you are working with meshes, the operation uses the
[SMOOTHMESHCONVERT](SMOOTHMESHCONVERT-System-Variable.md) setting.
