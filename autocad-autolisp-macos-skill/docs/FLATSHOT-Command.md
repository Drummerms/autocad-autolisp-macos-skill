# FLATSHOT (Command)

Creates a 2D representation of all 3D objects based on the current view.

## Access Methods

*Tool Set*:
Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Section Plan drop-down ![](../images/ac.menuaro.gif) Flatshot.
![](../images/GUID-64E3DC35-6542-45DB-87C3-DDEAD129B03C-low.png)

## Summary

The
[Flatshot dialog box](Flatshot-Dialog-Box.md) is displayed.

The edges of all 3D solids, surfaces, and meshes are projected line-of-sight onto a plane parallel to the viewing plane. The
2D representations of these edges are inserted as a block on the
*XY* plane of the UCS. This block can be exploded for additional changes.

![](../images/GUID-06863BDE-8120-4A34-A410-83B6828BB8E6-low.gif)
