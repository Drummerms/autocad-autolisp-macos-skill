# Commands for Layers

#### Commands

* [CHANGE](GUID-0D1BC415-5CDC-449D-8019-885F654FB1B5.htm "Changes the properties of existing objects.
  		")
* [CHPROP](GUID-05074362-FC4B-4582-A7A4-B3F6170BB4A7.htm "Changes the properties of an object.
  		")
* [LAYCUR](GUID-EA22341E-62F1-4C7D-9B14-F83526433C64.htm "Changes the layer property of selected objects to the current layer.
  		")
* [LAYER](GUID-9123091A-2DCB-4DE8-983C-F7CA38FA67BE.htm "Manages layers and layer properties.
  		")
* [LAYERCLOSE](GUID-9D0D75C7-3717-4358-9987-66B90DA1F093.htm " Closes the Layer Properties Manager.
  		")
* [LAYERP](GUID-F266407A-0894-4930-9544-8D4087FD66B9.htm "Undoes the last change or set of changes made to layer settings.
  		")
* [LAYERPMODE](GUID-249DBD51-2A59-4932-9E63-591C293D7FE9.htm "Turns on and off the tracking of changes made to layer settings for use by the LAYERP command.
  		")
* [LAYERSTATE](GUID-2BD44182-CA38-40E4-8ECD-F7D478D4B650.htm "Saves, restores, and manages sets of layer settings that are called layer states.
  		")
* [LAYERSTATESAVE](GUID-82A67E3E-0DA4-48B3-B268-BF5367D7DA22.htm "Displays the Add New Layer State dialog box, where you can provide a name and a description for a new layer state.
  		")
* [LAYFRZ](GUID-E7DB1E23-EB54-4DB7-B168-7725DFE6F100.htm "Freezes the layer of selected objects.
  		")
* [LAYISO](GUID-E24B9866-9538-43BF-A3DF-AA7E2341C624.htm "Hides or locks all layers except those of the selected objects.
  		")
* [LAYMCH](GUID-73F158B0-8401-4CAE-9204-94C36DF6D409.htm "Changes the layer of a selected object to match the destination layer.
  		")
* [LAYMCUR](GUID-E44FA50E-DB98-48C4-AE6B-052ED7287497.htm "Sets the current layer to that of a selected object.
  		")
* [LAYMRG](GUID-7D66940F-EC67-4DEC-89BC-82B887EABD6E.htm "Merges selected layers into a target layer, removing them from the drawing.
  		")
* [LAYOFF](GUID-BAE680D0-9AD5-4952-A139-B81732D7B5FB.htm "Turns off the layer of a selected object.
  		")
* [LAYON](GUID-B06C7A70-FA9D-49E7-A138-AD9A3ED07309.htm "Turns on all layers in the drawing.
  		")
* [LAYTHW](GUID-46F9F770-B0D1-4223-AEBF-C58A5F1E0B69.htm "Thaws all layers in the drawing.
  		")
* [LAYULK](GUID-6B23C58C-F9D1-4926-A691-0447EC506E1F.htm "Unlocks the layer of a selected object.
  		")
* [LAYUNISO](GUID-0795CBC9-7A9D-4A36-B49E-244C146FF6EA.htm "Restores all layers that were hidden or locked with the LAYISO command.
  		")
* [PROPERTIES](GUID-DC3674C5-A4C7-4CF6-9148-9B124DF29B78.htm "Controls properties of existing objects.
  		")
* [PURGE](GUID-76FC2D0A-5BAE-4889-A416-92537EEEEEB1.htm "Removes unused items, such as block definitions and layers, from the drawing.
  		")
* [RENAME](GUID-99C5072C-C71E-4BF7-9B3B-69B3BEA07940.htm "Changes the names assigned to items such as layers and dimension styles.
  		")
* [ULAYERS](GUID-168F1CF4-8E69-49A4-9E83-DE50FC61DBCB.htm "Controls the display of layers in a DWF, DWFx, PDF, or DGN underlay.
  		")
