# About Using an Alternate Text Editor

You can use any text editor, such as Microsoft Notepad, that saves files in ASCII format.

You can elect to use an alternate editor by specifying the editor with the MTEXTED system variable.

If you use an alternate text editor for multiline text, you specify the properties of the multiline text object at the Command
prompt first. Then the text editor opens for entering text. When you close the text editor, the text is inserted within the
width limit you specified.

If you use an alternate editor, you must enter special codes to apply formatting.

To edit text using an alternate text editor, use the same format codes. To avoid losing format information when you make changes
to the text, use the same text editor you used to create the text.
