# WORLDUCS (System Variable)

Indicates whether the UCS is the same as the WCS.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 1 |

| 0 | UCS differs from the WCS |
| 1 | UCS matches the WCS |
