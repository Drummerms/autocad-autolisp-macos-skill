# SNAPANG (System Variable)

Sets the snap and grid rotation angle for the current viewport relative to the current UCS.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 0.0000 |

When SNAPANG is set to a value other than 0, the lined grid will not display.
