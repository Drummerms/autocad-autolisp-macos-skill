# To Insert a Block Defined in the Current Drawing

## From the Blocks Palette

1. Click
   Window menu ![](../images/ac.menuaro.gif) Blocks.
2. Select to display the blocks in the current drawing.
   ![](../images/GUID-0AAC56FF-83EA-4D4F-84E3-260C5F78A5C2-low.png)
3. Set the insertion point, scale, and rotation options. When checked, you specify the option when inserting the block, using
   either your pointing device or manually entering a value.
4. Set the Repeat Placement option. When checked, you will be prompted automatically for additional insertion points until you
   press Esc to cancel the command. When cleared, the block you specified is inserted once.
5. Set the Explode option. When checked, the component objects in the block are inserted as individual objects and revert to
   their original properties.

   NOTE:As an indication that the block will be exploded on insertion, the preview of the block at the cursor is automatically suppressed.
6. From the palette, double-click the drawing file.
7. Move the mouse to the location you want to place the drawing and click.

## Classic Insert Command

1. At the Command prompt, enter CLASSICINSERT.
2. In the Insert Block dialog box, in the Blocks box, select a name from a list of block definitions.
3. Expand Show Insertion Options.
4. If you want to use the pointing device to specify the insertion point, scale, and rotation, select Specify On-Screen. Otherwise,
   enter values in the Insertion Point, Scale, and Rotation boxes.
5. If you want the objects in the block to be inserted as individual objects instead of as a single block, select Explode Block.
