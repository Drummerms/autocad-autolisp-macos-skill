# RAY (Command)

Creates a linear object that starts at a point and continues to infinity.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Xline drop-down ![](../images/ac.menuaro.gif) Ray.
![](../images/GUID-52F08D0F-9896-4302-BA99-6B846E1B0AE6-low.png)

*Menu*:
Draw ![](../images/ac.menuaro.gif) Ray.

The ray is extended to the edge of the display in the direction defined by the starting point and the through point. The
prompt for a through point is redisplayed so you can create multiple rays. Press Enter to end the command.

![](../images/GUID-058102AD-C9E3-4971-805C-A3EDE4B5219C-low.png)

Lines that extend to infinity in one direction, known as rays, can be used as references for creating other objects.

![](../images/GUID-693FC845-DF1C-4CD0-BA0A-20C865F8FC0A-low.gif)

The following prompts are displayed.

* Specify start point.
* Specify through point.
