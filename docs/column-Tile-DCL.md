# column Tile (DCL)

Grouping tile used to organize tiles into a column without a border around it.

*Supported Platforms:* Windows, Mac OS, and Web

```
: column {
   alignment children_alignment
   children_fixed_height children_fixed_width
   fixed_height fixed_width height label width
}
```

![](../images/GUID-7F1547F9-FA52-48A6-AE42-E2DFFC4A7525-low.png)

Tiles in a column are laid out vertically in the order in which they appear in the DCL file. A column can contain any kind
of tile (except for solitary radio buttons), including rows and other columns.

A column without a box has no additional attributes beyond the standard layout attributes.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
