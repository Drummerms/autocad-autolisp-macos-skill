# To Restore a Drawing Using the Files Recovered Dialog Box

1. Open the Files Recovered dialog box.
2. Click a drawing node to list all available drawing and backup files.
3. Click a file to open it.

   If the drawing file is damaged, the drawing is automatically repaired, if possible.
