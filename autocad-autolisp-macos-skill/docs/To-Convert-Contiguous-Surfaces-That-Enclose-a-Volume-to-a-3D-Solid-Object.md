# To Convert Contiguous Surfaces That Enclose a Volume to a 3D Solid Object

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Surface panel ![](../images/ac.menuaro.gif) Surface Sculpt.
   ![](../images/GUID-2CFF6B1B-2050-4D14-A427-01792D75E399-low.png)
2. Select surfaces that enclose a watertight volume (that is, there must be no gaps between the surfaces).

   A solid object is created.
