# FAQ: What's the fastest way to access the Help topic for a command or system variable?

If you start typing the name of the command or system variable on the command line, the autocomplete feature displays your
choices. For example, if you enter *dim*, all commands and system variables that begin with those letters are listed as shown.

![](../images/GUID-19D5C038-E8B5-43A7-BD6E-DCB324D35801-low.png)

Keep typing to narrow your choices, or use the arrow keys to move up or down the list. Press Fn + F1 to display the Help topic
for the command or system variable that you choose. You can also hover over a tool on a toolbar and press Fn + F1.
