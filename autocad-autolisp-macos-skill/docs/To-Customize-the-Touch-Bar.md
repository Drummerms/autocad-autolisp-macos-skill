# To Customize the Touch Bar

Select from the list of available commands to customize the touch bar.

1. Select
   View menu ![](../images/ac.menuaro.gif) Customize Touch Bar.
2. Select from the list of available commands and drag to the bottom of the screen to drop on to the touch bar.

   ![](../images/GUID-A9BF96F8-B3D4-49E4-B2ED-D3ACC1E89B0A-low.png)
3. On the touch bar, drag a command to change its position.
4. Drag the command off the touch bar to remove it.
5. Click Done to exit.
