# To Display Underlay Layer Override Status

1. Select the underlay of which you want to view the layer override status.
2. Click
   Window menu ![](../images/ac.menuaro.gif) Properties Inspector.
3. In the Properties Inspector, scroll to Misc and view the status of the layer display in the Layer Display Override option.
