# VSAZEBRASIZE (System Variable)

Controls the width of the zebra stripes displayed during zebra analysis (ANALYSISZEBRA).

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 45 |

Valid values are from 1 to 100.
