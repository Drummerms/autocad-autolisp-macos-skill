# Format and Style Dimensions

#### Topics in this section

* [About Dimension Styles](About-Dimension-Styles.md)

  A dimension style is a named collection of dimension settings that controls the appearance of dimensions, such as arrowhead
  style, text location, and lateral tolerances.
* [About Applying Dimension Styles](About-Applying-Dimension-Styles.md)

  You can modify dimension styles individually or by changing the current settings.
* [About Comparing Dimension Styles](About-Comparing-Dimension-Styles.md)

  You can view all the settings in a dimension style. Dimension styles used in externally referenced drawings are differentiated
  from those defined in your current drawing.
* [About Overriding a Dimension Style](About-Overriding-a-Dimension-Style.md)

  With dimension style overrides, you can temporarily change a dimensioning system variable without changing the current dimension
  style.
