# DIMSTYLE (System Variable)

Stores the name of the current dimension style.

(Read-only)

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | Standard (imperial) or ISO-25 (metric) |

This system variable has the same name as a command. Use the SETVAR command to access this system variable. The DIMSTYLE system
variable is read-only; to change the current dimension style, use the DIMSTYLE command.
