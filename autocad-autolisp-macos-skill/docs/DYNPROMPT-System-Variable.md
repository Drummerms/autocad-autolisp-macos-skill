# DYNPROMPT (System Variable)

Controls display of prompts in Dynamic Input tooltips.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Off |
| 1 | On |

The  [DYNMODE](DYNMODE-System-Variable.md) system variable turns Dynamic Input features on and off. When DYNMODE is set to 0 (off), prompts are not displayed regardless
of the DYNPROMPT setting.
