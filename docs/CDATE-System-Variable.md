# CDATE (System Variable)

Stores the current date and time in decimal format.

(Read-only)

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Not-saved |
| Initial value: | Varies |

The date and time displays in a decimal format starting with the year. After the decimal point, CDATE displays the time using
a 24-hour clock. For example, the ninth day of February in the year 2006 at 3:05 pm displays as 20060209.150500.
