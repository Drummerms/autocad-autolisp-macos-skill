# To Rotate a 3D Object Around an Axis

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) 3D Rotate.
   ![](../images/GUID-A09A2043-CFA8-4806-B587-BDDFF9C4AE62-low.png)
2. Select the object to rotate (1).
3. Specify the start point and endpoint of the axis about which the objects are to be rotated (2 and 3).

   The positive axis direction is from the start point to the end point, and the rotation follows the right-hand rule.
4. Specify the angle of rotation.

   ![](../images/GUID-9BABEB06-E562-4269-9D6B-018D75301149-low.png)
