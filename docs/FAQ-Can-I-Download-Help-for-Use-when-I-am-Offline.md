# FAQ: Can I Download Help for Use when I am Offline?

Yes, an offline version of the AutoCAD-based product Help can be installed when you do not have access to the Internet.

You download and install offline Help from the Help menu after you launch the product or by using the following URL:

*AutoCAD for Mac*:
<https://www.autodesk.com/acdmac-helpdownload-enu>

*AutoCAD LT for Mac*:
<https://www.autodesk.com/acdltmac-helpdownload-enu>

Once the offline Help is downloaded, run the installer and accept the default values. Once installed, the offline Help displays
by default if access to the internet is not available.
