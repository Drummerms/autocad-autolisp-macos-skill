# To Work With Path Arrays

How to create and edit path arrays.

## Create a Path Array

The easiest way to work with path arrays is to create them and then use the tools on the Create Array Visor or Properties
Inspector to make adjustments.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Array drop-down ![](../images/ac.menuaro.gif) Path Array.
   ![](../images/GUID-FA3890C9-C1EF-4A16-926B-11DD2BEECE29-low.png)
2. Select the objects to array and press Enter.
3. Select an object such as a line, polyline, 3D polyline, spline, helix, arc, circle, or ellipse to serve as a path for the
   array.
4. In the Create Array Visor - Path Array, specify a method for distributing the objects along the path:
   * Click Divide from the drop-down list
     ![](../images/GUID-EE2C6656-C837-4936-B55B-690C1D069AED-low.png) to evenly distribute a specified number of items evenly along the length of the path.
   * Click Measure from the drop-down list to distribute objects along the path at specified intervals.
     ![](../images/GUID-CB0DD3A3-2CBE-43A0-9F33-4C51218A75A2-low.png)
5. Move the cursor along the path to make adjustments.
6. Press Enter to complete the array.

## To Specify the Distance Between Objects Arrayed on a Path

Use this method if the path array is associative.

1. Click an item in the array.
2. In the Create Array Visor - Path Array, Item Spacing
   ![](../images/GUID-67CBEA8F-7CC6-440D-9281-689975D79840-low.png), enter a distance.

## Fill in Arrayed Objects After the Path Length Changes (Properties window)

Use this method if the path array is associative and the Properties window is displayed.

1. Click an item in the array.
2. In the Properties Inspector, under Misc, make sure the Method setting equals Measure.
3. In the Create Array Visor - Path Array, select Fill Entire Path box.

## Toggle the Object Alignment on Path Arrays

Use this method if the path array is associative.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Array drop-down ![](../images/ac.menuaro.gif) Path Array.
   ![](../images/GUID-FA3890C9-C1EF-4A16-926B-11DD2BEECE29-low.png)
2. Click an item in the array.
3. Enter A to align items.

## Adjust the Spacing Between Items in a Path Array

1. Click an item in the array.
2. In the Properties Inspector, set the Method property to Measure.
3. Click the Item Spacing grip.

   The
   *Item Spacing*grip is displayed on the second item in the first row of the path array. This grip is displayed only when there are three
   or more items in the array.

   ![](../images/GUID-4C725ADF-8963-4D03-8883-9DD4E29D2304-low.png)
4. Move the cursor along the path to increase or decrease the distance between the items and then click.
