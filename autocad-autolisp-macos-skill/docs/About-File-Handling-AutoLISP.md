# About File Handling (AutoLISP)

AutoLISP provides functions for handling files and data I/O. See File-Handling Functions (AutoLISP) in AutoLISP Function
Synopsis (AutoLISP) topic, for a complete list of file-handling functions.

#### Topics in this section

* [About Searching for Files (AutoLISP)](About-Searching-for-Files-AutoLISP.md)

  An application can use the
  findfile function to search for a particular file name.
* [About Accessing and Assigning Help to a Command (AutoLISP)](About-Accessing-and-Assigning-Help-to-a-Command-AutoLISP.md)

  The help and setfunhelp functions provide access to the product and your custom help files.
