# XLOADPATH (System Variable)

Creates a path for storing temporary copies of demand-loaded xref files.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Registry |
| Initial value: | Varies |

For more information, see [XLOADCTL](XLOADCTL-System-Variable.md).
