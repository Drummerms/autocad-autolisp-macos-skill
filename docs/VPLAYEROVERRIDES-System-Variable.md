# VPLAYEROVERRIDES (System Variable)

Indicates if there are any layers with viewport (VP) property overrides for the current layout viewport.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | Current viewport does not have any associated layer property overrides |
| 1 | Current viewport has associated layer property overrides |
