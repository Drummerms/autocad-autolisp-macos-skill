# To Change the Linetype Display Style of Existing 2D Polylines

1. Select the 2D polylines that you want to change.
2. Right-click in the drawing area, and choose Properties.
3. In the Properties Inspector, click Linetype Generation, and then click the down arrow.
4. Do one of the following:
   * Choose Enabled to display the linetype without regard to the polyline segments.
   * Choose Disabled to display the linetype centered on each segment, starting and ending with a dash or dot.
5. Press Esc to remove the selection.
