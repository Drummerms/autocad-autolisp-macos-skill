# To Display Dimension Values in Sub-units

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Dimension panel ![](../images/ac.menuaro.gif) Dimension Style.
   ![](../images/GUID-89D5BC05-30EE-4627-B80B-8680F08A01A9-low.png)
2. In the Dimension Style Manager, select the style you want to change. Click Options
   ![](../images/GUID-0888F513-2A7D-45B1-AD99-683EBA803A86-low.png) ![](../images/ac.menuaro.gif) Modify.
3. In the Modify Dimension Style dialog box, Primary Units tab or Alternate Units tab, under Zero Suppression, select Leading.
   * *Sub-units factor:* Sets the number of sub-units to a unit. It is used to display the dimension distance in a sub-unit for distances less than
     one unit. For example, enter
     *100* if the suffix is m (meters) and the sub-unit suffix is to display in cm (centimeters).
   * *Sub-units suffix:* Includes a suffix to the dimension text sub unit. You can enter text or use control codes to display special symbols. For
     example, enter cm to for .96m to display as 96cm.
4. Click OK.
5. Click Close to exit the Dimension Style Manager.
