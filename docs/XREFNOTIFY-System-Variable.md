# XREFNOTIFY (System Variable)

Controls the notification for updated or missing xrefs.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 2 |

| 0 or 1 | Disables xref notification |
| 2 | Enables xref notification and balloon messages. |
