# To Locate and View the Migration Report File

After you migrate your custom settings and files, a message is displayed from which you can open the migration report. The
report shows all the files that were successfully migrated, and provides specific information about any settings or files
that were not successfully migrated. To open it later:

1. In Finder, select
   Go ![](../images/ac.menuaro.gif) Go To Folder and enter the following location:

   */Users/<user profile>/library/logs/com.autodesk.admigrator<product version>*
2. Double-click to open the
   *MigrationReport\_<date>.html* file and view the details of your migration.
