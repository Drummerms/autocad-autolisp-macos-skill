# SSFOUND (System Variable)

Displays the sheet set path and file name if a search for a sheet set is successful.

(Read-only)

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Not-saved |
| Initial value: | "" |

SSLOCATE must be set to 1 and the drawing file must be open for a successful search.
