# DIMASZ (System Variable)

Controls the size of dimension line and leader line arrowheads. Also controls the size of hook lines.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 0.1800 (imperial) or 2.5000 (metric) |

Multiples of the arrowhead size determine whether dimension lines and text should fit between the extension lines. DIMASZ
is also used to scale arrowhead blocks if set by  [DIMBLK](DIMBLK-System-Variable.md). DIMASZ has no effect when  [DIMTSZ](DIMTSZ-System-Variable.md) is other than zero.
