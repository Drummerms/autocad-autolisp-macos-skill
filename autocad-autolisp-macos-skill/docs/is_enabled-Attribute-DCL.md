# is\_enabled Attribute (DCL)

Specifies whether or not the tile is initially available.

*Supported Platforms:* Windows, Mac OS, and Web

```
is_enabled = true-false;
```

Possible values are
true or
false (default:
true). If
false, the tile is unavailable and appears grayed out.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
