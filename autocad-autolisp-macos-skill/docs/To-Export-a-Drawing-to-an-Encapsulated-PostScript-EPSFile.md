# To Export a Drawing to an Encapsulated PostScript (EPS) File

1. Click File ![](../images/ac.menuaro.gif) Export
2. In the Export Data dialog box, specify a location and file name for the EPS file.
3. Under Files Format drop-down menu, select Encapsulated PS (\*.eps)
4. Click Save.
