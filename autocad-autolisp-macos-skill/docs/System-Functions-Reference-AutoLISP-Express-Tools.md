# System Functions Reference (AutoLISP/Express Tools)

NOTE:Express Tools are supported in AutoCAD for Windows only; not available in AutoCAD LT for Windows, or on Mac OS and Web.

The following table provides a summary of the AutoLISP Express Tools system functions.

| System functions | | Platforms | | | | |
| Windows | | Mac OS | | Web |
| Function | Description | AutoCAD | AutoCAD LT | AutoCAD | AutoCAD LT | AutoCAD |
| [(acet-sys-beep sound)](acet-sys-beep-AutoLISP-Express-Tools.md) | Makes a noise | ✓ | -- | -- | -- | -- |
| [(acet-sys-command shell-command)](acet-sys-command-AutoLISP-Express-Tools.md) | Runs a command in a shell and waits for the shell to complete execution before returning | ✓ | -- | -- | -- | -- |
| [(acet-sys-foreground)](acet-sys-foreground-AutoLISP-Express-Tools.md) | Forces AutoCAD's application window to be the topmost and receive focus | ✓ | -- | -- | -- | -- |
| [(acet-sys-keystate keycode)](acet-sys-keystate-AutoLISP-Express-Tools.md) | Checks the state of a virtual key | ✓ | -- | -- | -- | -- |
| [(acet-sys-lasterr)](acet-sys-lasterr-AutoLISP-Express-Tools.md) | Reports the last error | ✓ | -- | -- | -- | -- |
| [(acet-sys-procid)](acet-sys-procid-AutoLISP-Express-Tools.md) | Returns the current process identifier | ✓ | -- | -- | -- | -- |
| [(acet-sys-sleep msecs)](acet-sys-sleep-AutoLISP-Express-Tools.md) | Delay to allow other processes to run | ✓ | -- | -- | -- | -- |
| [(acet-sys-spawn flags command [argument ...])](acet-sys-spawn-AutoLISP-Express-Tools.md) | Executes an external program | ✓ | -- | -- | -- | -- |
| [(acet-sys-term processID)](acet-sys-term-AutoLISP-Express-Tools.md) | Terminates the given process | ✓ | -- | -- | -- | -- |
| [(acet-sys-wait processID [time])](acet-sys-wait-AutoLISP-Express-Tools.md) | Waits for a process to complete | ✓ | -- | -- | -- | -- |
