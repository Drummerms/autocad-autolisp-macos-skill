# COORDINATION MODEL (DXF)

The following group codes apply to coordination model entities.

| Coordination Model group codes | |
| Group code | Description |
| 0 | Object name (Coordination Model) |
| 5 | Handle |
| 330 | Soft-pointer ID/handle to owner dictionary |
| 100 | Subclass marker (AcDbNavisworksModel) |
| 340 | Soft-pointer ID/handle to the AcDbNavisworksModelDef object |
| 40, 40, 40, 40  40, 40, 40, 40  40, 40, 40, 40  40, 40, 40, 40 | Model transform in WCS coordinates (4x4 matrix) |
| 40 | Insertion unit factor |
