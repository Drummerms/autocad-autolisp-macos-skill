# About Publishing Sheet Sets

From the Sheet Set Manager, you can easily publish an entire sheet set, a subset, or a single sheet. It is quicker to publish
a sheet set in the Sheet Set Manager rather than using the Batch Publish dialog box to add drawings individually and then
publish them as a single set.

When you publish from the Sheet Set Manager, you can publish an electronic sheet set by publishing to a PDF file, or you
can publish a paper set by publishing to the plotter named in the page setup that is associated with each sheet. You can also
publish your sheet sets using a page setup that is saved in the page setup overrides drawing template (DWT) file associated
with the sheet set. This page setup overrides the current page setup settings for the individual publish job.
