# To Set the Transparency for All New Objects

1. Click Modify ![](../images/ac.menuaro.gif) Properties.
2. If any objects are currently selected, press Esc to clear the selection.
3. In the Properties Inspector, Transparency box, enter a number from 0 to 90 percent.
