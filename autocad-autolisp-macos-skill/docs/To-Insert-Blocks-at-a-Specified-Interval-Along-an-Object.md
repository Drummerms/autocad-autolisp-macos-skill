# To Insert Blocks at a Specified Interval Along an Object

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Point drop-down ![](../images/ac.menuaro.gif) Measure.
   ![](../images/GUID-D6E62826-5FF1-43A8-86EA-4D9B5EEE53F5-low.png)
2. Select a line, arc, spline, circle, ellipse, or polyline.
3. Enter
   *b* (Block).
4. Enter the name of the block.
5. Enter
   *y* (Yes) to align the blocks with the selected object. Enter
   *n* (No) to use a rotation angle of 0 degrees.
6. Specify the length of the interval between blocks.
