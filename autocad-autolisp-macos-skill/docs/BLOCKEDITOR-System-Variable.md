# BLOCKEDITOR (System Variable)

Indicates whether or not the Block Editor is open.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 0 |

| 0 | Indicates that the Block Editor is not open |
| 1 | Indicates that the Block Editor is open |
