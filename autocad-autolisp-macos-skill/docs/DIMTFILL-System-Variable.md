# DIMTFILL (System Variable)

Controls the background of dimension text.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | No background |
| 1 | The background color of the drawing |
| 2 | The background specified by [DIMTFILLCLR](DIMTFILLCLR-System-Variable.md) |
