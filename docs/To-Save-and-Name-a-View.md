# To Save and Name a View

1. Do one of the following:
   * If you have more than one viewport in model space, click inside the viewport that contains the view you want to save.
   * If you are working in a layout, select the viewport.
2. At the command prompt, enter VIEW.
3. Enter Save from the options.

   This saves the display in the current viewport using the name you supply.

   The current value of the UCSVIEW system variable is displayed when you save a view. To change the setting and turn this option
   on or off, use the UCS option of VIEW.
