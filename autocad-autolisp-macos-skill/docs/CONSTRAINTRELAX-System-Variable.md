# CONSTRAINTRELAX (System Variable)

Indicates whether constraints are enforced or relaxed when editing an object.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 0 |

| 0 | Maintains constraints |
| 1 | Relaxes constraints |

Use this system variable when you run scripts and want to relax any constraints.
