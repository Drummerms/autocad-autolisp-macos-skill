# CONSTRAINTSETTINGS (Command)

Controls the display of geometric constraints on constraint bars.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Parametric panel ![](../images/ac.menuaro.gif) Constraint Settings.
![](../images/GUID-7E0A1C95-D022-48ED-BF18-D6A1EA5C33AA-low.png)

NOTE:Hidden by default. Click

*Menu*:
Tools ![](../images/ac.menuaro.gif) Parametric ![](../images/ac.menuaro.gif) Constraint Settings.
