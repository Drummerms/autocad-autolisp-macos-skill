# To Work With Layout Tabs

![](../images/GUID-3E8AA9D2-3455-4DAA-9194-F785BFE8F4AC-low.png)

## Rename Layout

* Right-click any layout tab, and choose Rename.

## Delete Layout

* Right-click any layout tab, and choose Delete.

## Rearrange the Layout Tabs

* Click and drag any layout tab to a different location.

## Scroll through Layout Tabs

* When there are more layout tabs than can be displayed, click the left and right arrows to the right of the rightmost layout
  tab to scroll through the available layout tabs.

## Next or Previous Layout Tab

* Press
  *command+]* to switch to the next tab or
  *command +[* to switch to the previous tab.

## Edit the Page Setup for the Layout

* Right-click any layout tab, and choose Edit Page Setup.

## Print a Layout

* Right-click any layout tab, and choose Print.
