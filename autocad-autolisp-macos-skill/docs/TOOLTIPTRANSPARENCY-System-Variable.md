# TOOLTIPTRANSPARENCY (System Variable)

Sets the transparency for drafting tooltips.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

Valid range is 0 to 100. When a value of 0 is used, the drafting tooltip is fully opaque. The greater the value entered, the
more transparent the drafting tooltip will appear.
