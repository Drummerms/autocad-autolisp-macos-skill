# Insert Table Dialog Box

Create a table object using the new Insert Table dialog box where you can select table and cell styles, specify the number
of rows and columns, and more.

![](../images/GUID-BD7A4A78-7D8E-4B4F-8269-25C534509018-low.png)

The Insert Table dialog box has been added to the TABLE command workflow. You can still insert a table by specifying a window,
but now you can predefine the number of rows, columns, cell styles, and more.
