# TRACE (Command)

Opens and manages traces from the command prompt.

A trace is created in the AutoCAD web app, the AutoCAD mobile app, or AutoCAD desktop (Windows or Mac) to provide feedback,
comments, markups, and design exploration of the drawing without altering the content of the drawing.

You can also use the Trace palette to open and manage the traces available in the current drawing.

The following options are available.

## New

Creates a trace.

## Open

Opens an existing trace.

## Delete

Deletes an existing trace.

## Remove Contribution

Removes the objects in the active trace that you added.

## Rename

Renames an existing trace. Enter the current name of the trace and the new name when prompted.

## Close

Closes the active trace.

## ?

Lists the existing traces.
