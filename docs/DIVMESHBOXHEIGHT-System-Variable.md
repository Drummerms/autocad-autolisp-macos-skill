# DIVMESHBOXHEIGHT (System Variable)

Sets the number of subdivisions for the height of a mesh box along the *Z* axis.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 3 |

![](../images/GUID-77E97F54-DBC6-45E9-8B8D-3F4D6AB2B07B-low.png)

This setting affects new mesh boxes that are created with the [MESH](MESH-Command.md) command.
