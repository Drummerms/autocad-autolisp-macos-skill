# small\_increment Attribute (DCL)

Specifies the value used by the slider's incremental controls.

*Supported Platforms:* Windows, Mac OS, and Web

```
small_increment = integer;
```

Default value of
small\_increment is one one-hundredth the total range. The value must be within the range specified by
min\_value and
max\_value. This attribute is optional.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
