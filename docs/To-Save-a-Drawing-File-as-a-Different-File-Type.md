# To Save a Drawing File as a Different File Type

1. Click File menu ![](../images/ac.menuaro.gif) Save As.
2. In the Save Drawing As dialog box, select a location, and enter a file name.
3. In the Files of Type drop-down list, choose one of the following types and save:
   * DWG drawing format versions (\*.dwg)
   * a drawing standards file (\*.dws)
   * a drawing template file (\*.dwt)
   * Drawing Interchange Format versions (\*.dxf)
