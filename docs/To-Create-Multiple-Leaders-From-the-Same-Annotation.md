# To Create Multiple Leaders From the Same Annotation

1. Select the multileader.
2. Do one of the following:
   * Hover over the landing grip, and choose Add Leader from the grip menu.
   * Click
     Drafting tab ![](../images/ac.menuaro.gif) Leader panel ![](../images/ac.menuaro.gif) Add Leader.
     ![](../images/GUID-CEFE5B90-CCDB-4419-835A-9AF11F85694E-low.png)
3. Specify the endpoint for the new leader.
