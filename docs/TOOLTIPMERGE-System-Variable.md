# TOOLTIPMERGE (System Variable)

Combines drafting tooltips into a single tooltip.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Off |
| 1 | On |
