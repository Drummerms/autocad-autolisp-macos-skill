# To Remove List Formatting From Multiline Text

1. Double-click a multiline text object to open the In-Place Text Editor.
2. Select the list items.
3. Right-click in the In-Place Text Editor, clear Bullets and Lists ![](../images/ac.menuaro.gif) Allow Bullets and Lists to turn off bullets and lists.

   When Allow Bullets and Lists is not checked, you cannot create new formatted lists in the multiline text object.
4. To save your changes, click in the drawing outside the editor, or Press CMD + ENTER.
