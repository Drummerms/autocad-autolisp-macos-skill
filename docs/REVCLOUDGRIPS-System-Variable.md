# REVCLOUDGRIPS (System Variable)

Controls the number of grips displayed on a revision cloud.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | Displays a grip for each arc segment on a revision cloud (legacy behavior) |
| 1 | Displays limited number grips on a revision cloud |
