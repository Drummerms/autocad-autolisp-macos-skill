# ACTIVITYINSIGHTSSUPPORT (System Variable)

Enables or disables the Activity Insights feature.

|  |  |
| --- | --- |
| Type: | Bitcode |
| Saved in: | Registry |
| Initial value: | 3 |

| Value | Description |
| 0 | Disables Activity Insights. The palette and all access points within the ribbon and command line interface are removed. |
| 1 | Enables Activity Insights. Turns on the Activity Insights palette and displays Version History from Autodesk Docs and supported cloud storage providers.  NOTE:Autodesk Docs is available on Windows only. |
| 2 | Enables Activity Insights. Turns on the Activity Insights palette and displays Activity Insights activities.  NOTE:Only available in AutoCAD. |

NOTE:AutoCAD LT only supports values of 0 and 1. The initial value is 1.
