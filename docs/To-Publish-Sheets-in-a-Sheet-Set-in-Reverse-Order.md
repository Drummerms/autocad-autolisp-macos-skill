# To Publish Sheets in a Sheet Set in Reverse Order

Use the Publish feature to output the sheets in a sheet set to a printer or PDF in either normal or reverse order.

1. Click Publish.
   ![](../images/GUID-4C887873-7EE8-497C-B87D-0D8DFF393268-low.png)
2. In the Batch Publish dialog box, right-click the Sheet Name list to specify order in which it must be printed.
3. Select the required Output Options.
4. Click Publish.
