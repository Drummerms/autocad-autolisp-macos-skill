# To Change the Behavior of the Pointer Input Tooltips for Dynamic Input

1. At the Command prompt, enter
   *dsettings*.
2. In the Drafting Settings dialog box, Dynamic Input tab, under Pointer Input, click Settings.
3. In the Pointer Input Settings dialog box, choose the desired options.
4. Click OK to close each dialog box.
