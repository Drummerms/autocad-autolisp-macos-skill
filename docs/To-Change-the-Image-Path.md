# To Change the Image Path

1. Click Insert ![](../images/ac.menuaro.gif) Reference Manager![](../images/GUID-6EC866D6-17B4-4BF1-81D1-9AB5FB49A6CB-low.png). In the Reference Manager palette, select an image whose path you want to change.
2. In the Details Pane, click the Browse button for Found At.
3. In the Select Image File dialog box, browse to the new path. Click OK.

   The new path is displayed in the Found At properties.
