# REDRAW (Command)

Refreshes the display in the current viewport.

## Access Methods

![](../images/ac.mouse.gif) Menu:   View  ![](../images/ac.menuaro.gif)  Redraw Not available in menus in the current workspace

![](../images/ac.keyboard.gif) Command entry:  *'redraw* for transparent use

## Summary

Removes temporary graphics left by some operations in the current viewport. To remove stray pixels, use the [REGEN](REGEN-Command.md)  command.
