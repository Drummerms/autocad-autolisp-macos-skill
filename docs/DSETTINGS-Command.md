# DSETTINGS (Command)

Sets grid and snap, polar and object snap tracking, object snap modes, and Dynamic Input.

## Access Methods

![](../images/ac.mouse.gif) Menu:   Tools
 ![](../images/ac.menuaro.gif)  Drafting Settings Not available in menus in the current workspace

Shortcut menu: Right-click Snap Mode, Grid Display, Polar Tracking, Object Snap, Object Snap Tracking, or Dynamic Input on the status bar.
Choose Settings.

## Summary

The Drafting Settings dialog box is displayed.
