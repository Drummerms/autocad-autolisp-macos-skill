# To Unload or Reload an Underlay

1. Click Insert menu ![](../images/ac.menuaro.gif)Reference Manager.
2. In the Reference Manager pane, select a DGN underlay.
3. Right-click, and click Unload or Reload.

   When unloaded, the DGN underlay is still linked to the drawing file, but it is not displayed in the drawing.
