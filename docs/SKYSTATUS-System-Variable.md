# SKYSTATUS (System Variable)

Determines if the sky illumination is computed at render time.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

This has no impact on the viewport illumination or the background. It only makes the sky available as a gathered light source
for rendering.

| 0 | Sky off |
| 1 | Sky background |
| 2 | Sky background and illumination |
