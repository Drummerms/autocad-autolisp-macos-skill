# CONSTRAINTINFER (System Variable)

Controls whether the geometric constraints are inferred while drawing and editing geometry.

|  |  |
| --- | --- |
| Type: | Bitcode |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Geometric constraints are not inferred |
| 1 | Geometric constraints are inferred |
