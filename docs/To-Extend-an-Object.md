# To Extend an Object

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Extend.
   ![](../images/GUID-74AC1D5E-0799-450B-8370-58FEE9FF962E-low.png)
2. Select the objects to serve as boundary edges.

   To select all displayed objects as potential boundary edges, press Enter without selecting any objects.
3. Select the objects to extend.

## Quick Mode

Instead of having to first select boundaries for the Extend command, the Quick mode selects all potential boundaries by default.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Extend.
   ![](../images/GUID-74AC1D5E-0799-450B-8370-58FEE9FF962E-low.png)
2. Enter O and select Quick.

   NOTE:Once Quick mode is enabled, it remains enabled until you change to Standard mode. The Quick mode setting is stored in the
   TRIMEXTENDMODE system variable.
3. Select the objects to extend.

NOTE: Use the cuTting edges option to temporarily allow you to select the object to use as the boundary edge.
