# To Save and Insert a 2D or 3D Section as a Block

1. Enter SECTIONPLANETOBLOCK at the command prompt.
2. Select the live section object.
3. Right-click in the drawing area and click Generate 2D/3D Section.
4. In the Generate Section/Elevation dialog box, click 2D Section/Elevation or 3D Section.
5. In the expanded dialog box, select Include All Objects.
6. Under Destination, click Insert as New Block.
7. Click Create.
8. In the drawing area, specify an insertion point for the new block.

   An unnamed block consisting of the 2D or 3D section is inserted .
