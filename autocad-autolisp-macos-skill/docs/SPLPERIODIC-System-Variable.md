# SPLPERIODIC (System Variable)

Controls whether closed splines and NURBS surfaces are generated with periodic properties to maintain the smoothest continuity
at the closure point or seam.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 1 |

| 0 | Creates closed splines and NURBS surfaces with the method used in AutoCAD 2026 2011 and earlier releases. |
| 1 | Creates periodic closed splines and closed NURBS surfaces, for the smoothest (C2) continuity. (Recommended) |
