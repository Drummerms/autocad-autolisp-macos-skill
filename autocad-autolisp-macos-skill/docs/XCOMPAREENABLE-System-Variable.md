# XCOMPAREENABLE (System Variable)

Enables the comparison between an xref and the referenced drawing file.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | Off |
| 1 | On |
