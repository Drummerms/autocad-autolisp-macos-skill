# FILLETRAD3D (System Variable)

Stores the current fillet radius for 3D objects.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 1.0000 |

Stores the default radius for 3D solid and surface fillets created with the FILLETEDGE and SURFFILLET commands. If you change
the radius while using the FILLETEDGE or SURFFILLET commands, the FILLETRAD system variable does *not* update to this new value.
