# TEXTTOATTRIBUTE (System Variable)

In the BSEARCH command, if text objects are selected, controls whether text variations will be searched for conversion into
block attributes.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | 1 |

| Value | Description |
| 0 | Search for only identical text to convert into normal text |
| 1 | In addition to identical text, search for text variations to convert them into block attributes |
