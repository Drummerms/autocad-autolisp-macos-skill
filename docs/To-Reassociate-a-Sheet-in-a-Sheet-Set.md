# To Reassociate a Sheet in a Sheet Set

*Reassociate a sheet.* If you move a sheet to a different folder, you should reassociate the sheet to the sheet set with the properties listed in
the Sheet Set Manager to correct the path.

NOTE:You can quickly confirm whether a sheet is in the expected folder by looking at the path information in the Sheet Set Manager
(File Locations).

1. In the Sheet Set Manager, open the sheet that you want to reassociate.
2. In the Details panel, expand File Locations.
3. Click the ellipsis button in Expected Layout Location property and specify the new location of the drawing.
4. Update the sheet property as needed.
