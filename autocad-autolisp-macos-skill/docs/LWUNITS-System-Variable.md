# LWUNITS (System Variable)

Controls whether lineweight units are displayed in inches or millimeters.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Inches |
| 1 | Millimeters |
