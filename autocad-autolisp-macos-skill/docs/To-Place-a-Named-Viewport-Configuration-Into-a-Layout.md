# To Place a Named Viewport Configuration Into a Layout

1. Click a layout tab.
2. Click View ![](../images/ac.menuaro.gif) Viewports ![](../images/ac.menuaro.gif) Named Viewports.At the Command prompt, enter VPORTS. In the Viewports dialog box, Named Viewports tab, select the named viewport configuration from the list. Click OK.
3. In the layout, specify a location for the named viewport configuration.
