# What's New in AutoCAD for Mac 2023

Learn about new features in this release of AutoCAD for Mac 2023 release.

## Updates

## 2023.1 Highlights

*3D Graphics*: Below is a list of improvements that have been made to the 3D Graphics system.

* TrueType font support
* Line weights
* Locked layer dimming
* Xref dimming
* And other performance improvements

## 2023 Release

## Highlights

* [Trace](Trace-2.md): Trace provides a safe space to collaborate on drawing changes without fear of altering the existing drawing. The analogy
  of trace is a virtual, collaborative tracing paper that's laid over the drawing, allowing collaborators to add feedback right
  in the drawing.
* [Count](Count.md): Quickly and accurately count the instances of blocks or other selected objects in a drawing. You can insert a table with
  the block count data into the current drawing.
* [Export and Import Settings](Export-and-Import-Settings.md): Transfer settings from one machine to another. Settings can include user profile, toolbar and toolset customizations, command
  aliases, plot settings, templates, and the My Properties settings for the Property palette.
* [Table Styles](Table-Styles.md): Create and modify table styles using the new Table Style manager. Table styles control the appearance of a table and the
  table cells.
* [Insert Table Dialog Box](Insert-Table-Dialog-Box-2.md): Create a table object using the new Insert Table dialog box where you can select table and cell styles, specify the number
  of rows and columns, and more.
* [Spotlight Search](Spotlight-Search.md): The Spotlight search in macOS now supports searching for drawings based on drawing content. You can search on mtext, mleader,
  field, or block attribute values. You can also search on named objects such as blocks, layers, or layouts, and system attributes
  such as date or author.
* [Share](Share-Current-Drawing.md): Share a link to a copy of the current drawing to view or edit in the AutoCAD web app. All related DWG xrefs and images are
  included.
* [3D Graphics](3D-Graphics.md): This release includes the new cross platform 3D graphics system, leveraging all the power of modern GPUs and multi-core
  CPUs to offer a smooth navigation experience for much larger drawings.
* [2D Graphics Display](2D-Graphics-Display.md): This release of AutoCAD for Mac includes a new graphic engine, which provides a better display effect on some high-end GPUs.
* [Save Publish List](Save-Publish-List.md): When using the PUBLISH command you can now save the current list of drawings as a DSD file for reuse or load a list of drawings
  from a saved DSD file.
* [CUTBASE Command](CUTBASE.md): The new CUTBASE command copies selected objects to the Clipboard, along with a specified base point, and removes them from
  the drawing.
* [Polyline Extend](Polyline-Extend.md): A new grip option, Extend Vertex, adds a new vertex to your polyline extending from the selected end grip. You no longer
  have to reverse the polyline direction.
* [MLEADER Enhancement](MLEADER-Enhancement.md) - The MLEADER command now has an option to select an existing mtext object to use for the new leader.
