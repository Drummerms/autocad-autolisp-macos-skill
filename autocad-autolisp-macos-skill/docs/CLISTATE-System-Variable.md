# CLISTATE (System Variable)

Indicates whether the command line is open or closed.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 1 |

| Value | Description |
| 0 | Closed |
| 1 | Open |
