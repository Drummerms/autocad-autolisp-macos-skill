# To Match the Orientation of a Block to a Layout

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block.
   ![](../images/GUID-B2E92148-A285-4542-BEF5-AD10E48A0DE5-low.png)
2. In the Name drop-down list, select the block you want to modify.
3. Under Behavior, click Annotative and then click Match Block Orientation to Layout.
4. Click OK.
5. In the Block – Redefine Block dialog box, click Redefine Block.
