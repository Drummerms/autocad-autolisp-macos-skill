# expt (AutoLISP)

Returns a number raised to a specified power

*Supported Platforms:* Windows, Mac OS, and Web

## Signature

```
(expt number power)
```

*number*
:   *Type:* Integer or Real

    Any number.

*power*
:   *Type:* Integer or Real

    The power to raise
    *number* to.

## Return Values

*Type:* Integer or Real

If both arguments are integers, the result is an integer; otherwise, the result is a real.

## Examples

```
(expt 2 4)
16

(expt 3.0 2.0)
9.0
```
