# APERTURE (System Variable)

Sets the display size for the object snap target box, in pixels.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 10 |

This system variable has the same name as a command. Use the SETVAR command to access this system variable.

Enter a value (1-50). The higher the number, the larger the target box.

You can also change this setting on the [Application Preferences dialog box, Cursor & Selection tab](Cursor-&-Selection-Tab-Application-Preferences-Dialog-Box.md).

APERTURE controls the object snap target box, not the pickbox displayed at the Select Objects prompt. The object selection
pickbox is controlled by the [PICKBOX](PICKBOX-System-Variable.md) system variable.
