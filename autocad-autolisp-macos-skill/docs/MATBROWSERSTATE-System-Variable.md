# MATBROWSERSTATE (System Variable)

Indicates whether the Materials Browser is open or closed.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 0 |

| Value | Description |
| 0 | Closed |
| 1 | Open |
