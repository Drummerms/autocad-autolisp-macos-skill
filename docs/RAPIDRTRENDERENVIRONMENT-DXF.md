# RAPIDRTRENDERENVIRONMENT (DXF)

The following group codes are used by RAPIDRTRENDERENVIRONMENT objects.

Stored in the ACAD\_BACKGROUND dictionary.

| RAPIDRTRENDERENVIRONMENT group codes | |
| Group code | Description |
| 0 | Object name (RAPIDRTRENDERENVIRONMENT) |
| 5 | Handle |
| 102 | Start of persistent reactors group; always “{ACAD\_REACTORS” |
| 330 | Soft-pointer ID/handle to owner dictionary |
| 102 | End of persistent reactors group; always “}” |
| 330 | Soft-pointer ID/handle to owner dictionary |
| 100 | Subclass marker (AcDbIBLBackground) |
| 90 | Subclass version |
| 290 | Environment flag  0 = Disabled  1 = Enabled |
| 1 | Image-based lighting (IBL) map file name |
| 40 | Rotation angle of the IBL map, in degrees (-180 to 180) |
| 290 | IBL map as background flag  0 = Custom background  1 = IBL map as background |
| 340 | Soft-pointer ID/handle to the custom background |
