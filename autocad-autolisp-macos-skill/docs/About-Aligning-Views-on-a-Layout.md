# About Aligning Views on a Layout

You can arrange the elements of your drawing by aligning the view in one layout viewport with the view in another viewport.

For angled, horizontal, and vertical alignments, you can move each layout viewport relative to distances defined by the model-space
geometry displayed.

![](../images/GUID-56C5BFF1-6633-4C4C-B189-FF454BA4430B-low.png)

To adjust the views on a layout with precision, you can create construction geometry, use object snaps on the model space
objects displayed in layout viewports, or use one of the drafting aids on the status bar.
