# To Use a Custom Arrowhead

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Dimension panel ![](../images/ac.menuaro.gif) Dimension Style.
   ![](../images/GUID-89D5BC05-30EE-4627-B80B-8680F08A01A9-low.png)
2. In the Dimension Style Manager, select the style you want to change. Click Options
   ![](../images/GUID-0888F513-2A7D-45B1-AD99-683EBA803A86-low.png) ![](../images/ac.menuaro.gif) Modify.
3. In the Dimension Style Manager, Symbols and Arrows tab, under Arrowheads, select User Arrow from the First arrowhead list.
4. In the Select Custom Arrow Block dialog box, enter the name of the block that you want to use as an arrowhead. Click OK.
5. To choose a different custom arrowhead for the second arrowhead, repeat steps 3 and 4, choosing User Arrow from the Second
   arrowhead list (optional).
6. Click OK.
7. Click Close to exit the Dimension Style Manager.
