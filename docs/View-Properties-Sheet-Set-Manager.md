# View Properties (Sheet Set Manager)

Displays information that is specific to the selected view.

SHEETSET (Command)

*Menu:* Window ![](../images/ac.menuaro.gif) Sheet Set Manager: Select a saved view in Sheet Set Manager

![](../images/GUID-FFF446CC-9889-4FEA-A6B0-56543196D6E9-low.png)

## Summary

Views represent named views that have been placed on a sheet in the current sheet set using the Sheet Set Manager. Views cannot
be modified or placed using the Sheet Set Manager.

## List of Options

The following options are displayed.

### View

Displays information about the view.

View Name
:   Displays the name of the view.

Sheet Category
:   Assigns a category to the view.

    The category list is populated by the current value of each model space view's View Category property when it was placed on
    a sheet in the sheet set.

### File Locations

Displays information about the drawing file that the view is located.

Drawing Name
:   Displays the name of the drawing that contains the named view.

Drawing Location
:   Displays the path of the drawing where the view is saved.

Reveal in Finder
:   Displays the location of the drawing (DWG) file associated with the view in Finder.

### Preview

Displays a preview image of the selected view.
