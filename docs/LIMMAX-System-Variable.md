# LIMMAX (System Variable)

Stores the upper-right grid limits for the current space, expressed as world coordinates.

|  |  |
| --- | --- |
| Type: | 2D-point |
| Saved in: | Drawing |
| Initial value: | 12.0000,9.0000 (imperial) or 420.0000,297.0000 (metric) |

LIMMAX is read-only when paper space is active and the paper background or printable area is displayed.
