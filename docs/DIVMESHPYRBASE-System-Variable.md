# DIVMESHPYRBASE (System Variable)

Sets the number of radial subdivisions between the center of the mesh pyramid base and its perimeter.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 3 |

![](../images/GUID-FB648B26-A8D1-485A-A87D-04B54D690F6E-low.png)

This setting affects new mesh pyramids that are created with the [MESH](MESH-Command.md) command.
