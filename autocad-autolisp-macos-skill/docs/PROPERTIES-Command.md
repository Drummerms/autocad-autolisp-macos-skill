# PROPERTIES (Command)

Controls properties of existing objects.

## Access Methods

*Menu*:
Window
![](../images/ac.menuaro.gif) Properties Inspector.

*Shortcut Menu*: Select the objects whose properties you want to view or modify, right-click in the drawing area, and click Properties.

## Summary

The
[Properties Inspector palette](Properties-Inspector.md) is displayed and lists the properties of the selected objects.

When more than one object is selected, only those properties common to all selected objects are displayed.

When no objects are selected, only the current settings of general properties are displayed.

You can also view or modify properties of third-party application objects that are based on AutoCAD application programming
interface (API) standards.

There are several
[general properties](General-Properties-of-Objects.md) common to all objects. All other object properties are specific to the type of object.
