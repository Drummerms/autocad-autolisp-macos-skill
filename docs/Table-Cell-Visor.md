# Table Cell Visor

Insert or delete table rows and columns, merge and unmerge cells, and insert a block, field, or formula in a table cell.

![](../images/GUID-61CA3BD0-5B80-4C84-AEAB-29EA39DE0330-low.png)

## Summary

The Table Cell visor is displayed when you select a table cell or group of cells.

## List of Options

The following options are displayed.

Insert Row Above ![](../images/GUID-88B9E66C-9D2E-48CA-82F4-B576858FDD01-low.png)
:   Inserts a row above the currently selected cell or row.

Insert Row Below ![](../images/GUID-D45420A0-7B6B-474A-B498-5084564018C3-low.png)
:   Inserts a row below the currently selected cell or row

Delete Row ![](../images/GUID-956D77F6-71CF-4C68-9986-061F13FEC05C-low.png)
:   Deletes the currently selected rows.

Insert Column Left ![](../images/GUID-62E6395B-8321-45D7-8D5D-62556EDE4758-low.png)
:   Inserts a column to the left of the currently selected cell or row.

Insert Column Right ![](../images/GUID-4F46F5FE-8CF7-4266-BA23-3B61A0B1547F-low.png)
:   Inserts a column to the right of the currently selected cell or row.

Delete Column ![](../images/GUID-320F35A4-5A7F-4FC5-8C99-9D14B78CD657-low.png)
:   Deletes the currently selected columns.

Merge Cells ![](../images/GUID-DE1C599E-C8FC-43DA-A7F0-7EF06B993813-low.png)
:   Merges the selected cells into one large cell. Choose Merge All, Merge By Row, or Merge By Column.

Unmerge Cells ![](../images/GUID-EB1E0BC9-5166-40C7-BD69-AD16BC6BD2BB-low.png)
:   Unmerges cells that were previously merged.

Match Cell ![](../images/GUID-D6937B88-9F92-4985-80AA-2DBD72942E65-low.png)
:   Applies the properties of a selected cell to other cells.

Insert Block ![](../images/GUID-D56BE860-2727-47B0-B5B3-621281F6EBAB-low.png)
:   The Insert Block in Table Cell dialog box is displayed, where you can insert a block into the currently selected table cell.

Insert Field ![](../images/GUID-BA6D7FAE-6E00-4D97-B917-16C3827BA436-low.png)
:   The Insert Field dialog box is displayed, where you can insert a field into the currently selected table cell.

Insert Formula ![](../images/GUID-2DDDD11D-9FC9-45A4-A301-49FA78ECA3BE-low.png)
:   Inserts a formula into the currently selected table cell. A formula must start with an equal sign (=). The formulas for sum,
    average, and count ignore empty cells and cells that do not resolve to a numeric value. Other formulas display an error (#)
    if any cell in the arithmetic expression is empty or contains nonnumeric data.
