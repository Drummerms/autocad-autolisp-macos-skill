# To Modify a Lofted Solid or Surface by Changing the Surface Normal Settings

1. If the Properties palette is not displayed, select any object. Right-click the object and select Properties.
2. In a drawing, select a lofted solid or surface that was created with cross sections.
3. On the Properties palette, Geometry area, change the Surface Normals setting.
