# COMPASS (System Variable)

Controls whether the 3D compass is on or off in the current viewport.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 0 |

| 0 | Turns off the 3D compass |
| 1 | Turns on the 3D compass |
