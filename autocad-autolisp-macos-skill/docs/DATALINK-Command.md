# DATALINK (Command)

Displays the Select Reference File box.

## Access Methods

*Menu*:
Insert
![](../images/ac.menuaro.gif) Data Link.

## Summary

A table can be linked to data in a Microsoft Excel (XLS, XLSX, or CSV) file. You can link to an entire spreadsheet, individual
row, column, cell, or cell range in Excel.

After you select a reference file, the New Excel Data Link is displayed.
