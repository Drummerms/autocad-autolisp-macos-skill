# VIEW (Command)

Saves and restores named model space views, layout views, and preset views.

## Summary

The
[options are displayed](VIEW-Command.md) at the Command prompt, . If you enter
*-view* at the Command prompt, the options are displayed at the Command prompt as well.

NOTE:The VIEW command cannot be used transparently.
