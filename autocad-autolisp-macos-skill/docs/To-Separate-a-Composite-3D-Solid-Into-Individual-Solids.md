# To Separate a Composite 3D Solid Into Individual Solids

This operation separates only
*non-intersecting* objects that have been combined by the union operation.

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Solid Edit - Body drop-down ![](../images/ac.menuaro.gif) Solid, Separate.
   ![](../images/GUID-1A44467C-21B8-47D3-AA3C-3933AFA908CD-low.png)
2. Select the 3D solid object.
