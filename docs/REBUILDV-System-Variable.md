# REBUILDV (System Variable)

Sets the number of grid lines in the V direction when rebuilding a NURBS surface.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 6 |

Valid values are 2 to 32767 to increase or decrease the number of grid lines in the V direction used to rebuild the NURBS
surface.
