# To Work With Visual Styles

Apply, save, import, and restore viewport visual styles.

## Apply a Visual Style to a Viewport

1. Click View tab ![](../images/ac.menuaro.gif)Visual Styles panel ![](../images/ac.menuaro.gif)![](../images/GUID-6EC866D6-17B4-4BF1-81D1-9AB5FB49A6CB-low.png)![](../images/ac.menuaro.gif) Visual Styles Manager.![](../images/GUID-22E5288E-A842-44A4-84C2-168ADD3EF085-low.png)
2. Click in the viewport to make it current.
3. In the Visual Styles Manager, double-click the sample image of the visual style.

   The selected visual style is applied to the model in the viewport.

   An icon in the sample image indicates that the visual style is in use in the current viewport. When you change viewports,
   the icon changes to indicate that the visual style is in use in the current drawing.

## Save a Visual Style in the Drawing

1. Click View tab ![](../images/ac.menuaro.gif)Visual Styles panel ![](../images/ac.menuaro.gif) Visual Styles drop-down ![](../images/ac.menuaro.gif) Save As A New Visual Style.
2. Enter a name for the new style.

TIP:Enter ? to view a list of existing visual styles. Predefined styles cannot be overwritten.

## Restore the Default Visual Style Settings

1. Click View tab![](../images/ac.menuaro.gif)Visual Styles panel![](../images/ac.menuaro.gif)![](../images/GUID-6EC866D6-17B4-4BF1-81D1-9AB5FB49A6CB-low.png)![](../images/ac.menuaro.gif)Visual Styles Manager.![](../images/GUID-22E5288E-A842-44A4-84C2-168ADD3EF085-low.png)
2. Right-click a predefined visual style, identified by a drawing icon at bottom-right, and choose Reset To Default.

### Use a Visual Style from Another Drawing

1. Open the drawing that has the visual style that you want to use.
2. Click View tab![](../images/ac.menuaro.gif)Visual Styles panel![](../images/ac.menuaro.gif)![](../images/GUID-6EC866D6-17B4-4BF1-81D1-9AB5FB49A6CB-low.png)![](../images/ac.menuaro.gif)Visual Styles Manager.![](../images/GUID-22E5288E-A842-44A4-84C2-168ADD3EF085-low.png)
3. Click View tab ![](../images/ac.menuaro.gif)Palettes panel ![](../images/ac.menuaro.gif)Tool Palettes.![](../images/GUID-70E7C9E4-8738-4A18-A026-30197403ABF7-low.png)
4. In the Tool Palettes window, click the Visual Styles tab.
5. In the Visual Styles Manager, select the sample image of the visual style.
6. Below the images, click the Export the Selected Visual Style to the Tool Palette button.
7. Open the drawing in which you want to use the visual style.
8. In the Tool Palette window, right-click the visual style and click Add to Current Drawing.

   The visual style is added to the sample images in the Visual Styles Manager and the ribbon.

#### Store a Visual Style on a Tool Palette

1. Click View tab![](../images/ac.menuaro.gif)Visual Styles panel![](../images/ac.menuaro.gif)![](../images/GUID-6EC866D6-17B4-4BF1-81D1-9AB5FB49A6CB-low.png)![](../images/ac.menuaro.gif)Visual Styles Manager.![](../images/GUID-22E5288E-A842-44A4-84C2-168ADD3EF085-low.png)
2. Click View tab ![](../images/ac.menuaro.gif)Palettes panel ![](../images/ac.menuaro.gif)Tool Palettes.![](../images/GUID-70E7C9E4-8738-4A18-A026-30197403ABF7-low.png)
3. In the Tool Palettes window, click the Visual Styles tab.
4. In the Visual Styles Manager, select the image of a visual style.
5. Below the images, click the Export The Selected Visual Style To The Tool Palette button.

WARNING:Visual styles, when exported, are placed on the active tool palette. Be sure the Visual Styles palette, or a desired custom
palette, is active.
