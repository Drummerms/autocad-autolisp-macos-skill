# HANDLES (System Variable)

Reports whether object handles can be accessed by applications.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | On |

Because handles can no longer be turned off, has no effect except to preserve the integrity of scripts.
