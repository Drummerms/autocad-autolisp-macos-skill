# To Set Up a New Visibility State in a Dynamic Block

You can only use this procedure after a visibility parameter has been added to the block definition.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block Editor.
   ![](../images/GUID-E3DDBE51-C92A-4D07-A51D-C7C685E4F578-low.png)
2. In the Edit Block Definition dialog box, select the block you want to modify and click Edit Block.
3. On the Block Editor visor,click the
   Add Visibility
    button.At the Command prompt, enter BVSTATE.
4. Specify parameter location as asked in the prompt.
5. Click the Visibility State drop down menu in the Block Editor, and click the Visibility State Manager option. The Visibility
   States dialog box appears that displays the default visibility state of 0. You can double-click the visibility state to rename
   it.
6. Click the
   *+* button to add a new visibility state In the Visibility States dialog box. The New Visibility State dialog box appears.
7. Enter a name for the new visibility state in the Visibility State Name field.
8. Select one of the following:
   * Hide All Existing Objects in New State
   * Show All Existing Objects in New State
   * Leave Visibility of Existing Objects Unchanged in New State
9. Click OK twice.
