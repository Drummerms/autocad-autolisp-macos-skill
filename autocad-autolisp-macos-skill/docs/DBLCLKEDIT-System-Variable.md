# DBLCLKEDIT (System Variable)

Controls the double click editing behavior in the drawing area.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

The system variable can accept the values of On and Off in place of 1 and 0.

| 0 | Disabled |
| 1 | Enabled |
