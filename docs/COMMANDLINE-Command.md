# COMMANDLINE (Command)

Displays the Command Line window.

## Access Methods

*Menu*:
Window ![](../images/ac.menuaro.gif) Command Line.

*Command entry*: Cmd-3

## Summary

Displays the command line when it has been hidden.

You can display commands, system variables, options, messages, and prompts in a dockable and resizable window called the
*Command Line*.
