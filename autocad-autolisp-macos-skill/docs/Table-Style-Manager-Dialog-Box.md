# Table Style Manager Dialog Box

Creates and modifies table styles, which control the appearance of tables.

## Access Methods

TABLESTYLE (Command)

Format ![](../images/ac.menuaro.gif) Table Style.

## List of Options

The following options are displayed.

Current Table Style
:   Displays the name of the table style that is applied to tables you create. Right-click a table style name and select Set Current
    to change the current table style.

Styles
:   Displays a list of table styles. The current style is highlighted.

Preview Of
:   Displays a preview image of the style that is selected in the Styles list.

Set Current
:   Sets the table style selected in the Styles list as the current style. All new tables are created using this table style.

New ![](../images/GUID-6B850EA9-1427-42F5-8E66-E967AD5445C4-low.png)
:   *Coming soon*: Displays the Create New Table Style dialog box, in which you can define new table styles.

Delete ![](../images/GUID-36652FED-6611-41B0-95B6-14B33F2D8DB1-low.png)
:   Deletes the table style selected in the Styles list. A style that is being used in the drawing cannot be deleted.

Options ![](../images/GUID-4AC4C87F-0D68-4B11-AD8D-D11171DD08B0-low.png)
:   Displays the Modify Table Style dialog box, in which you can modify table styles.
