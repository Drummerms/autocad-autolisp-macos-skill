# About Changing Raster Image Brightness, Contrast, and Fade

You can change several display properties of raster images in a drawing for easier viewing or special effects.

You can adjust brightness, contrast, and fade for the display of an image as well as for plotted output without affecting
the original raster image file and without affecting other instances of the image in the drawing. Adjust brightness to darken
or lighten an image. Adjust contrast to make poor-quality images easier to read. Adjust fade to make drawing geometry easier
to see over images and to create a watermark effect in your plotted output.

Bitonal images cannot be adjusted for brightness, contrast, or fade. Images fade to the current screen background when displayed,
and they fade to white when plotted.
