# MESHCAP (Command)

Creates a mesh face that connects open edges.

## Access Methods

*Tool Set*:
Modeling tab ![](../images/ac.menuaro.gif) Mesh panel ![](../images/ac.menuaro.gif) Close Hole.
![](../images/GUID-D7C0A668-44F4-43D1-BF18-5654D40B3748-low.png)

*Menu*:
Modify
![](../images/ac.menuaro.gif) Mesh Editing
![](../images/ac.menuaro.gif) Close Hole.

## Summary

You can close gaps in mesh objects by selecting the edges of the surrounding mesh faces. For best results, the faces should
be on the same plane.

![](../images/GUID-3A5AC39C-52DE-474E-B40B-323DF3EEE58B-low.gif)

## List of Prompts

The following prompts are displayed.

Select connecting mesh edges to create a new mesh face
:   Specifies which mesh edges form the boundary for the closure. Click the edges to select them.

Select edges or [CHain]

Edges
:   Selects individual edges and adds them to the selection set.

Chain
:   Selects contiguous edges of mesh objects connected end-to-end.
