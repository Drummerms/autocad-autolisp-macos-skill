# Hide Lines in 3D Objects

To remove all hidden lines from a view

* Click
  View menu ![](../images/ac.menuaro.gif) Hide.

  The hidden lines reappear when the drawing is regenerated.

To restore hidden lines in a view

* Click
  View menu ![](../images/ac.menuaro.gif) Regen.

  Hidden objects in the current viewport are restored to their normal display.

To hide lines in layout views

1. Select the viewport in which you want to apply hidden lines.
2. Click
   Window menu ![](../images/ac.menuaro.gif) Properties Inspector.
3. On the Properties Inspector, Misc area, Shade Plot list, select Hidden.

To show hidden lines in the current viewport

1. Select the viewport in which you want to show hidden lines.
2. Click
   Window menu ![](../images/ac.menuaro.gif) Properties Inspector.
3. On the Properties Inspector, Misc area, Shade Plot list, select As Displayed.

To turn hidden lines on or off for printing objects not in layout views

1. Click
   File menu ![](../images/ac.menuaro.gif) Page Setup Manager.
2. In the Page Setup Manager, select a page setup and click
   ![](../images/GUID-3811A988-85ED-4568-A59C-A09C6074AEC6-low.png).
3. Click Edit.
4. In the Page Setup Edit dialog box under Print Options, select Hide Paperspace Objects.
