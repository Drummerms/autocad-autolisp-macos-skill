# CENTEREXE (System Variable)

Controls the length of the centerline extensions.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 0.1200 (imperial) or 3.5000 (metric) |

The CENTEREXE system variable applies only to the length of centerline extensions created with the CENTERMARK and CENTERLINE
commands. This system variable only accepts positive real numbers.
