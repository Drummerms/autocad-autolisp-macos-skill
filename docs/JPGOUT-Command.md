# JPGOUT (Command)

Saves selected objects to a file in JPEG file format.

## Summary

The Create Raster File dialog box (a [standard file selection dialog box](Standard-File-Selection-Dialog-Boxes.md)) is displayed. Enter the file name in the dialog box.

## List of Prompts

The following prompts are displayed.

Select objects or <all objects and viewports>: *Press* Enter *to select all objects and viewports or use an object selection method and press* Enter

A JPEG file is created that contains the objects you select. Shade Plot options are preserved in the file when you use this
command.

Light glyphs that are displayed in the drawing appear in the new file, even if the Plot Glyph property of the lights is set
to No.

NOTE:When the [FILEDIA](FILEDIA-System-Variable.md) system variable is set to 0 (Off), command prompts are displayed.
