# DIMTEDIT (Command)

Moves and rotates dimension text and relocates the dimension line.

## Summary

The companion command that edits the dimension text and changes the extension line angle is DIMEDIT.

## List of Prompts

The following prompts are displayed.

Select dimension:
*Select a dimension object*

You are prompted for the new location of the dimension text.

Specify
[new location for dimension text](DIMTEDIT-Command.md) or [[Left](DIMTEDIT-Command.md)/[Right](DIMTEDIT-Command.md)/[Center](DIMTEDIT-Command.md)/[Home](DIMTEDIT-Command.md)/[Angle](DIMTEDIT-Command.md)]:
*Specify a point or enter an option*

Location for Dimension Text
:   Updates the location of the dimension text dynamically as you drag it. To determine whether text appears above, below, or
    in the middle of the dimension line, use the Text tab in the New, Modify, and Override Dimension Style dialog box.

Left
:   Left-justifies the dimension text along the dimension line.

    This option works only with linear, radius, and diameter dimensions.

    ![](../images/GUID-D2EEC4D9-62EF-4BD8-B731-9A23839F6BB2-low.gif)

Right
:   Right-justifies the dimension text along the dimension line.

    This option works only with linear, radius, and diameter dimensions.

    ![](../images/GUID-10BE5692-F4CD-49F0-830D-7866F8FCC1AD-low.gif)

Center
:   Centers the dimension text on the dimension line.

    This option works only with linear, radius, and diameter dimensions.

    ![](../images/GUID-430E4ACD-CD1D-4D24-97E6-804C36F55D33-low.gif)

Home
:   Moves the dimension text back to its default position.

    For example:

    ![](../images/GUID-A3D640AA-6CE5-4355-BE50-7BCD6FC23E0E-low.gif)

Angle
:   Changes the angle of the dimension text.

    The center point of the text does not change. If the text moves or the dimension is regenerated, the orientation set by the
    text angle is retained. Entering an angle of 0 degrees puts the text in its default orientation.

    The text angle is measured from the
    *X* axis of the UCS.

    ![](../images/GUID-E139CBC7-FDA8-4E9C-B760-35D6D9A437A8-low.gif)
