# About Assigning Images to a Command

Standard and custom images can be assigned to a command.

When a command is added to a tool set, the assigned image is displayed on the tool.

AutoCAD comes with a library of standard images that are used for the standard commands; these can be assigned to your own
custom commands. You can create your own custom commands using an external image editor.
