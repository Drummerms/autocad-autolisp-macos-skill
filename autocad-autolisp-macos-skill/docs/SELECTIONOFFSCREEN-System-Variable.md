# SELECTIONOFFSCREEN (System Variable)

Controls the selection of objects that are off-screen.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

With SELECTIONOFFSCREEN set to on, objects that are off screen can now participate in the selection.

| Value | Description |
| 0 | Off-screen selection is off |
| 1 | Off-screen selection is on with the limitations of mouse input support only. Also, zoom/pan will hit some limitations during the selection. |
| 2 | Off-screen selection on without limitation  NOTE:When you set SELECTIONOFFSCREEN to 2, there may be a possibility that the selection results are incorrect. This is meant for experimental purposes only. |
