# SPLDEGREE (System Variable)

Stores the last-used degree setting for splines and sets the default degree setting for the SPLINE command when specifying
control vertices.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 3 |

Enter a value from 1 to 5.

NOTE:SPLDEGREE defaults to 3 when AutoCAD 2026 starts.
