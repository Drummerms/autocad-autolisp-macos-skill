# About Named Page Setups in Sheet Sets

Use named page setups to specify the same output options for all the sheets in a sheet set.

You can create a sheet from scratch using the Sheet Set Manager, and then apply a named page setup to the sheet.

You can also apply named page setups that are stored in the sheet set's page setup overrides drawing template (DWT) file to
a single sheet or to an entire sheet set for a one-time publish operation.
