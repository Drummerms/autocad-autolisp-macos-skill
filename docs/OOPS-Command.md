# OOPS (Command)

Restores erased objects.

## Summary

OOPS restores objects erased by the last
[ERASE](ERASE-Command.md) command.

![](../images/GUID-A36BF908-62FF-4BEA-99B8-12F057DE5DB0-low.png)

You can also use OOPS after
[BLOCK](BLOCK-Command.md) or
[WBLOCK](WBLOCK-Command.md) because these commands can erase the selected objects after creating a block.

You cannot use OOPS to restore objects on a layer that has been removed with the
[PURGE](PURGE-Command.md) command.
