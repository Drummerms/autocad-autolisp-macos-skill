# To Include a Plot Stamp on Published Sheets

1. Click Publish.
   ![](../images/GUID-4C887873-7EE8-497C-B87D-0D8DFF393268-low.png)
2. In the Output Options, click Include Plot Stamp.

   A plot stamp will be included on a specified corner of each drawing.
3. To change the plot stamp settings, click Configure Plot Stamp.
