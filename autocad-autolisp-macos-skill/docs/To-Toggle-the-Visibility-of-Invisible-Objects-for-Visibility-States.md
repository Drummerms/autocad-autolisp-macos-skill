# To Toggle the Visibility of Invisible Objects for Visibility States

Use this procedure to show and hide objects that are invisible in the current state. This change affects the Block Editor
display but does not affect the inserted blocks.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block Editor.
   ![](../images/GUID-E3DDBE51-C92A-4D07-A51D-C7C685E4F578-low.png)
2. In the Edit Block Definition dialog box, select the block you want to modify and click Edit Block.
3. On the Block Editor visor, click Visibility Mode.
   ![](../images/GUID-2008FDE4-C581-4F17-9103-EEFDC1CA73EE-low.png)
