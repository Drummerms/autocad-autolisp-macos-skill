# MIRRTEXT (System Variable)

Controls how MIRROR reflects text.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| Value | Description |
| 0 | Retains text direction |
| 1 | Mirrors the text |
