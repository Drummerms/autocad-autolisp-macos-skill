# To Edit Objects in Model Space in a Maximized Layout Viewport

If you are in paper space, and you need to make significant changes to the model, you can work in a maximized layout viewport
(or switch to the Model tab).

1. Select the border of a layout viewport.
2. Right-click and choose Maximize Viewport.

   TIP:You can restore the viewport and maximize another viewport by clicking one of the arrows next to the Maximize Viewport button
   in the status bar ( ![](../images/GUID-2F46C154-C427-4DC6-A3BF-670035847A5B-low.png) Find).
3. To return to the layout, click the Minimize Viewport button on the status bar ( ![](../images/GUID-9921F438-4750-4549-B68D-BE607915A6BE-low.png) Find).

NOTE:

* You can maximize a locked viewport and modify objects. When you restore the viewport, it is locked again.
* If you use PLOT while a viewport is maximized, the layout is restored before the Plot dialog box is displayed.
* If you save and close the drawing while a viewport is maximized, the drawing opens with the named layout restored.
