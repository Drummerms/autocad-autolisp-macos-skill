# UPDATETHUMBSNOW (Command)

Manually updates thumbnail previews for named views, drawings, and layouts.

## Summary

The
[UPDATETHUMBNAIL](UPDATETHUMBNAIL-System-Variable.md) system variable controls how the thumbnail previews are updated. The UPDATETHUMBSNOW command updates the following:

* Sheets, sheet views, and model space views thumbnails in the Sheet Set Manager
* Quick View images
