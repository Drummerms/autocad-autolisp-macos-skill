# DELAY (Command)

Provides a timed pause within a script.

## Access Methods

![](../images/ac.keyboard.gif) Command entry:  *'delay* for transparent use

## Summary

Specifies the duration of a pause. Entering *delay 1000* in your script delays the start of execution of the next command for about one second. The longest delay available is 32767,
which is slightly less than 33 seconds.
