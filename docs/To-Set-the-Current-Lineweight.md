# To Set the Current Lineweight

1. Click
   Format menu ![](../images/ac.menuaro.gif) Lineweight.
2. In the Lineweight Settings dialog box, choose a lineweight.

NOTE: To display or hide lineweights, click the Show/Hide Lineweight button on the status bar.
![](../images/GUID-1181E2DA-4C54-47A5-B32E-D79762F0CE7C-low.png)
