# Commands for Basic Blocks

These commands and system variables are used to work with basic blocks.

#### Commands

* [BASE](GUID-B65911C2-3E36-458E-8430-015053D9F9FF.htm "Sets the insertion base point for the current drawing.
  		")
* [BCLOSE](GUID-C63C7855-B666-4B99-B371-F58786E1E5A8.htm "Closes the Block Editor.
  		")
* [BEDIT](GUID-DE7688D9-67B1-4C4F-B530-26FD7F0DBE0B.htm "Opens the block definition in the Block Editor.
  		")
* [BLOCK](GUID-C638FC08-E63A-41DE-A204-558CC2E58DE5.htm "Creates a block definition from selected objects.
  		")
* [CONTENT](GUID-6EE002FC-A0E1-477B-879B-11C81ADD556C.htm " Opens the Content palette where you can insert blocks from the current drawing or a custom library of blocks.
  		")
* [COLOR](GUID-F12EF623-7CD3-4864-8130-A3BF718D730D.htm "Sets the color for new objects.
  		")
* [EXPLODE](GUID-B38FD713-AD2F-4D9E-AD68-60E3F7E32D88.htm "Breaks a compound object into its component objects.
  		")
* [EXPORT](GUID-A72DB257-3410-4792-B548-6B9FC1DED72B.htm "Saves the objects in a drawing to a different file format.
  		")
* [INSERT](GUID-9DE33560-3013-4BF6-801C-A93A8B2B7D1A.htm "Inserts a block or drawing into the current drawing.
  		")
* [LAYER](GUID-9123091A-2DCB-4DE8-983C-F7CA38FA67BE.htm "Manages layers and layer properties.
  		")
* [LINETYPE](GUID-1D3817A0-C5C7-4D52-99A9-F0922F09D4CD.htm "Loads, sets, and modifies linetypes.
  		")
* [OOPS](GUID-0E72CDCD-9ECA-453A-8A04-CA2921740270.htm "Restores erased objects.
  		")
* [PASTEBLOCK](GUID-B5717F8B-096E-4880-9C35-745168992DE8.htm "Pastes objects from the Clipboard into the current drawing as a block.
  		")
* [PROPERTIES](GUID-DC3674C5-A4C7-4CF6-9148-9B124DF29B78.htm "Controls properties of existing objects.
  		")
* [PURGE](GUID-76FC2D0A-5BAE-4889-A416-92537EEEEEB1.htm "Removes unused items, such as block definitions and layers, from the drawing.
  		")
* [WBLOCK](GUID-1C5F8AF0-4A46-4AAC-8965-867123E4C5C8.htm "Saves selected objects or converts a block to a specified drawing file.
  		")
* [XPLODE](GUID-FC4139AB-D527-4743-8E68-F83DA6E5D192.htm "Breaks a compound object into its component objects, with specified properties for the resulting objects.
  		")

#### System Variables

* [ATTDIA](GUID-DD7AE7D3-D9D7-49E7-948E-079D8451FFA2.htm "Controls whether the INSERT command uses a dialog box for attribute value entry.
  		")
* [BLOCKEDITLOCK](GUID-BBBAEABC-E8D3-4E32-8D03-81067ADF4AD0.htm "Disallows opening of the Block Editor and editing of dynamic block definitions.
  		")
* [BLOCKMRULIST](GUID-55AEFADF-6A03-4F5F-B901-54B8D727D483.htm "Controls the number of most recently used blocks displayed in the Recent pane of the Content palette.
  		")
* [BLOCKREDEFINEMODE](GUID-77BE7DBF-7390-47DB-936F-CCBE44065157.htm "Controls whether a dialog box displays when inserting a block from the Content palette with the same name as an existing block definition.
  		")
* [EXPLMODE](GUID-113BD30D-2F46-4E23-9B49-034AB50064A7.htm "Controls whether the EXPLODE command supports nonuniformly scaled (NUS) blocks.
  		")
* [EXTNAMES](GUID-8EC065EC-D551-4E02-9C5A-A33D1DB80B05.htm "Controls the characters accepted for the names of blocks, dimension styles, layers, and other named objects.
  		")
* [INSNAME](GUID-1654B78D-A11A-44D6-9790-28C756929EFB.htm "Sets a default block name for the INSERT command.
  		")
* [INSUNITS](GUID-A58A87BB-482B-4042-A00A-EEF55A2B4FD8.htm "Specifies a drawing-units value for automatic scaling of blocks, images, or xrefs when inserted or attached to a drawing.
  		")
* [INSUNITSDEFSOURCE](GUID-4C21BD16-8C17-4469-A91C-F7CC105B770F.htm "Sets source content units value when INSUNITS is set to 0.
  		")
* [INSUNITSDEFTARGET](GUID-5E9D42F6-CAE4-49BD-8850-E665455EF5CB.htm "Sets target drawing units value when INSUNITS is set to 0.
  		")
* [MAXSORT](GUID-7AD30968-4982-4E1C-9847-69A5EAEB7C63.htm "Sets the maximum number of items such as file names, layer names, and block names that are sorted alphabetically in dialog boxes, drop-down lists, and palettes.
  		")
