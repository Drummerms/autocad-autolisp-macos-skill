# VSEDGEOVERHANG (System Variable)

Makes edges on 3D objects extend beyond their intersection for a hand-drawn effect.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | -6 |

The amount of overhang can be set between 1 and 100 pixels. Negative numbers store the value but turn off the effect.

![](../images/GUID-C0D0F1A3-D519-4434-9E6F-FB38C5CB42FE-low.gif)

NOTE:Existing visual styles are not changed when you enter a new value for this system variable. Any new value entered for this
system variable temporarily creates an unsaved new visual style.
