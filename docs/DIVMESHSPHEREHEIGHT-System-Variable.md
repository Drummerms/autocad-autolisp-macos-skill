# DIVMESHSPHEREHEIGHT (System Variable)

Sets the number of subdivisions between the two axis endpoints of the mesh sphere.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 6 |

![](../images/GUID-BC7695DF-734B-4EDB-95FE-2516CDFD5948-low.png)

This setting affects new mesh spheres that are created with the [MESH](MESH-Command.md) command.
