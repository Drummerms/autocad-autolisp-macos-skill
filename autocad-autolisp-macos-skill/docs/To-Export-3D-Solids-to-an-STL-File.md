# To Export 3D Solids to an STL File

1. Click File ![](../images/ac.menuaro.gif) Export.
2. In the Export Data dialog box, enter a file name.
3. Under Files of type, select Lithography (\*.stl). Click Save.
4. Select one or more solid objects. All objects must be entirely within the positive XYZ octant of the world coordinate system
   (WCS). That is, their *X*, *Y*, and *Z* coordinates must be greater than zero.

   The file extension .*stl* is automatically appended to the file name.
