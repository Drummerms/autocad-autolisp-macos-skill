# CENTERLTYPE (System Variable)

Specifies the linetype used by center marks and centerlines.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | "CENTER2" |

The CENTERLTYPE system variable applies only to the linetype used on the center marks and centerlines created with the CENTERMARK
and CENTERLINE commands. This system variable accepts the following values: "." (use current), ByLayer, or any valid linetype.
