# To Set Running Object Snaps

1. On the status bar, right-click Object Snap, and choose Settings.

   ![](../images/GUID-B889046D-C651-40E2-BC2F-087B2755B520-low.png)
2. In the Drafting Settings dialog box, Object Snap tab, select the object snaps you commonly want to use.
