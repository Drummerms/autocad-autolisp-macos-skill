# WINDOWAREACOLOR (System Variable)

Controls the color of the transparent selection area during window selection.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 150 |

The valid range is 1 to 255. [SELECTIONAREA](SELECTIONAREA-System-Variable.md) must be on.
