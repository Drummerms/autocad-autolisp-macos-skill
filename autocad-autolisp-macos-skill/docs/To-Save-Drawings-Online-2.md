# To Save Drawings Online

Save files to FTP sites and other online locations.

## Save to an FTP Site

1. First, connect to the FTP site outside of AutoCAD.
   1. Click
      Finder menu ![](../images/ac.menuaro.gif) Go ![](../images/ac.menuaro.gif) Connect to Server.
   2. Enter the server address in the box provided, ftp://{server}.
   3. Click Connect.
2. In AutoCAD, click
   File menu ![](../images/ac.menuaro.gif) Save As.
3. In the dialog box, select the server from the available locations.
4. Navigate to a folder.
5. Select a file format from the File Format list and click Save.

## Save to a Server

1. First, connect to the FTP site outside of AutoCAD.
   1. Click
      Finder menu ![](../images/ac.menuaro.gif) Go ![](../images/ac.menuaro.gif) Connect to Server.

   3. Click Connect.

   File menu ![](../images/ac.menuaro.gif) Save As.
3. In the dialog box, select the server from the available locations.
4. Navigate to a folder.
5. Select a file format from the File Format list and click Save.
