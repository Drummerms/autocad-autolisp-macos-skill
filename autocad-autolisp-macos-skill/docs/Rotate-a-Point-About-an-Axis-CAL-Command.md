# Rotate a Point About an Axis (CAL Command)

The
*rot* function rotates a point about an axis and returns the resulting point.

rot(p,origin,ang)
:   Rotates point *p*through angle *ang*about the
    *Z* axis passing through the point *origin,*as shown in the following example:

    ![](../images/GUID-90F4CCAE-E710-49DA-80D9-9C687168F9DB-low.png)

rot(p,AxP1,AxP2,ang)
:   Rotates point
    *p* through an angle
    *ang* about the axis passing through points
    *AxP1* and
    *AxP2*, as shown in the following example. The axis is oriented from the first point to the second point.

    ![](../images/GUID-B79F9D16-7B41-4DCA-ACE3-58204E1067E3-low.png)
