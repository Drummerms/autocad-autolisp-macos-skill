# CUTCLIP (Command)

Copies selected objects to the Clipboard and removes them from the drawing.

## Access Methods

![](../images/ac.mouse.gif) Menu:   Edit
 ![](../images/ac.menuaro.gif)  Cut Not available in menus in the current workspace

Shortcut menu: End any active commands, right-click in the drawing area, and choose Clipboard ![](../images/ac.menuaro.gif) Cut.

![](../images/ac.keyboard.gif) Command entry: Cmd-X

## Summary

CUTCLIP copies selected objects to the Clipboard, removing them from the drawing.

You can paste the contents of the Clipboard into a document (see
[PASTECLIP](PASTECLIP-Command.md)).

When you want to use objects from a drawing file in another application, you can cut these objects to the Clipboard and then
paste them into another application. You can also use Cut and Paste to transfer objects between drawings.
