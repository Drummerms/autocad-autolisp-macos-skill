# N Functions

#### Topics in this section

* [namedobjdict (AutoLISP)](namedobjdict-AutoLISP.md)

  Returns the entity name of the current drawing's named object dictionary, which is the root of all nongraphical objects in
  the drawing
* [nentsel (AutoLISP)](nentsel-AutoLISP.md)

  Prompts the user to select an object (entity) by specifying a point, and provides access to the definition data contained
  within a complex object
* [nentselp (AutoLISP)](nentselp-AutoLISP.md)

  Provides similar functionality to that of the
  nentsel function without the need for user input
* [not (AutoLISP)](not-AutoLISP.md)

  Verifies that an item evaluates to nil
* [nth (AutoLISP)](nth-AutoLISP.md)

  Returns the nth element of a list
* [null (AutoLISP)](null-AutoLISP.md)

  Verifies that an item is bound to nil
* [numberp (AutoLISP)](numberp-AutoLISP.md)

  Verifies that an item is a real number or an integer
