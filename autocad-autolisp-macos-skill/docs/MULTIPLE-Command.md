# MULTIPLE (Command)

Repeats the next command until canceled.

## Summary

The command that you enter is repeated until you press Esc. Because MULTIPLE repeats only the command name, any parameters
must be specified each time.

MULTIPLE does not repeat commands that display dialog boxes.

NOTE:You cannot use MULTIPLE as an argument to the AutoLISP ®  command function.
