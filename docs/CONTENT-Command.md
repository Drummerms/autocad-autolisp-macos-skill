# CONTENT (Command)

Opens the Blocks palette.

## Access

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Blocks Palette.
![](../images/GUID-6857E103-73A1-42F1-AA87-3265189DB33B-low.png)

*Menu*:
Window ![](../images/ac.menuaro.gif) Blocks.

*Command entry*: Cmd-2

The Blocks palette allows you to access and insert blocks from the current drawing, recently used, or stored in block libraries.
A block library can be a drawing that contains blocks or a folder that contains drawing files. A block library stored on the
cloud can be shared across devices and AutoCAD platforms. Blocks inserted or created are added automatically to the recent
block list.
