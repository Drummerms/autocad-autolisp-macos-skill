# NAVVCUBELOCATION (System Variable)

Identifies the corner in a viewport where the ViewCube tool is displayed.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Upper-right |
| 1 | Upper-left |
| 2 | Lower-left |
| 3 | Lower-right |
