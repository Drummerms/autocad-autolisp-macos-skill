# Configure the Plot Settings for Layouts

#### Topics in this section

* [About Named Page Setups](About-Named-Page-Setups.md)

  Use named page setups if you want to plot the same layout in several ways or specify the same output options for several
  layouts.
* [About Setting the Lineweight Scale for a Layout](About-Setting-the-Lineweight-Scale-for-a-Layout.md)

  You can scale lineweights proportionately in a layout with the plot scale.
* [About Setting the Plot Scale](About-Setting-the-Plot-Scale.md)

  When you specify a scale to output your drawing, you can choose from a list of real-world scales, enter your own scale, or
  select Fit to Paper to fit onto the selected paper size.
