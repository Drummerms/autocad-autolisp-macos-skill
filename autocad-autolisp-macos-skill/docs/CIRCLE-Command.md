# CIRCLE (Command)

Creates a circle.

## Access Methods

*Toolset*:
Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Circle drop-down
.
![](../images/GUID-7520F49C-B7B7-4285-8460-6FFC4EA34DF9-low.png)

*Menu*:
Draw ![](../images/ac.menuaro.gif) Circle.

## List of Prompts

The following prompts are displayed.

### Center Point

Creates a circle based on a center point and a radius or diameter value.

Radius ![](../images/GUID-7520F49C-B7B7-4285-8460-6FFC4EA34DF9-low.png)
:   Enter a value, or specify a point.

    For example:

    ![](../images/GUID-D00BF256-30FA-4BC7-B759-69ACB29579FD-low.gif)

Diameter ![](../images/GUID-CEE736F3-2FDE-466D-8A62-767EED3B741F-low.png)
:   Enter a value, or specify a second point.

    For example:

    ![](../images/GUID-4DFD8BC3-5653-4D0F-A9BE-C1DE5744788E-low.gif)

### 3P (Three Points) ![](../images/GUID-987F17B7-B898-4AEC-BE74-66668352AB28-low.png)

Creates a circle based on three points on the circumference.

For example:

![](../images/GUID-D259A90A-4761-4169-A3FF-E60746726414-low.gif)

### Tan, Tan, Tan ![](../images/GUID-1957EB8D-A922-4B0A-A74E-7F67D7593409-low.png)

Creates a circle tangent to three objects.

For example:

![](../images/GUID-092D1700-7089-445A-BB58-E55DE6173C15-low.gif)

### 2P (Two Points) ![](../images/GUID-13191C2B-3C7B-483A-A434-7500AE7383BD-low.png)

Creates a circle based on two endpoints of the diameter.

For example:

![](../images/GUID-E6001089-0358-48F3-8632-0B1599B0859D-low.gif)

### TTR (Tangent, Tangent, Radius) ![](../images/GUID-099AE5F3-6138-4FEE-89E8-6FA82676729E-low.png)

Creates a circle with a specified radius and tangent to two objects.

Sometimes more than one circle matches the specified criteria. The program draws the circle of the specified radius whose
tangent points are closest to the selected points.

For example:

![](../images/GUID-50A1C9A9-664C-4DC7-ACA9-55C54A1BC27A-low.gif)
