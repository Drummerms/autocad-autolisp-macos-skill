# About Improving Speed by Displaying Simplified Objects

You can improve the speed of large drawing files by displaying simplified versions of certain kinds of objects.

Display performance is improved when wide polylines and donuts, solid-filled polygons (2D solids), hatches, gradient fills,
and text are displayed in simplified form. Simplified display also increases the speed of creating test plots.

When you turn off Fill mode, the listed objects are displayed in outline form. Except for patterned hatches and gradient
fills, solid fill is automatically turned off for hidden and 3D views.

![](../images/GUID-E064CD1D-2881-476D-8D97-4B675151238B-low.png)

When you turn on Quick Text mode, only a rectangular frame that identifies the text area is displayed or plotted.

![](../images/GUID-996441B7-F2F1-4772-B52A-BF999F2BE634-low.png)

To update existing objects using these settings, you must use REGEN or REGENALL to regenerate the display.
