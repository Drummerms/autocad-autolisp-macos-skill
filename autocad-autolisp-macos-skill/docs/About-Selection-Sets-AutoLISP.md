# About Selection Sets (AutoLISP)

Selection sets are groups of one or more objects (entities). You can interactively add objects to, or remove objects from,
selection sets with AutoLISP routines.

The following example uses the ssget function to return a selection set containing all the objects in a drawing.

```
(ssget "X")

<Selection set: 1>
```
