# FILETAB (Command)

Displays the file tabs at the top of the drawing area.

File tabs provide an easy way for you to access all the open drawings in the application. Clicking the plus sign (+) on the
left end of the file tabs bar opens a new drawing based on the specified template file used by the QNEW command.

NOTE:If there is no specified template, clicking the plus sign (+) opens a new drawing based on the last used template.

Use Ctrl+TAB to navigate through the file tabs.

You can right-click on the file tabs or the file tab bar to access the shortcut menus that offer you options to open, close
or save drawings.
