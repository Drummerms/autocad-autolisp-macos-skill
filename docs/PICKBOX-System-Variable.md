# PICKBOX (System Variable)

Sets the object selection target height, in device independent pixels.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 3 |

NOTE: When PICKBOX is set to 0, selection previewing of objects is not available.
