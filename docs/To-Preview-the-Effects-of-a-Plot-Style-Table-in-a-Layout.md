# To Preview the Effects of a Plot Style Table in a Layout

1. Click the layout tab in which you want to preview the effects of the plot style table.
2. Click
   File ![](../images/ac.menuaro.gif) Page Setup Manager.
3. In Page Setup Manager, click Modify (\*). Click Edit.
4. In the Page Setup dialog box, under Plot Style Table, select the Display Plot Styles option.
5. Click OK.
6. In Page Setup Manager, click Close.

   The effects of the plot style table are previewed in the layout.
