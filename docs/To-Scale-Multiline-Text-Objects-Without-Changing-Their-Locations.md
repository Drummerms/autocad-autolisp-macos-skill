# To Scale Multiline Text Objects Without Changing Their Locations

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Scale.
   ![](../images/GUID-2E69AFE3-06D8-4B6B-AFAE-C8CB3FB009E8-low.png)
2. Select one or more multiline text objects and press ENTER.
3. Select the base point and press ENTER.
4. Enter the scale factor to be applied to each text object.
