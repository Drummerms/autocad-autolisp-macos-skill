# Commands for the Dynamic UCS

#### System Variables

* [OSOPTIONS](GUID-0D05BECE-0DC3-454D-999A-208C6DBC3C3E.htm "Controls whether objects snaps are suppressed on hatch objects, geometry with negative Z values when using dynamic UCS, or dimension extension lines.
  ")
* [UCSDETECT](GUID-7BFB1FF9-3498-47DE-A0DA-E0A860B62210.htm "Controls whether dynamic UCS acquisition is active when creating objects involving 3D planar surfaces.
  ")
