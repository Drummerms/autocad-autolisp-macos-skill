# To Set the Plot Area

1. Click the layout tab for which you want to set the plot area and adjust the display.
2. Click
   File ![](../images/ac.menuaro.gif) Page Setup Manager.
3. In the Page Setup Manager, Page Setups area, select the page setup that you want to modify.
4. Click
   ![](../images/GUID-78DED619-51B3-49C1-AD71-F928B5F69C0A-low.png) ![](../images/ac.menuaro.gif) Edit.
5. In the Page Setup dialog box, under Plot Settings, select one of the following options:
   * *Layout.* Prints all objects within the printable area of the paper. This option is only available from a layout tab.
   * *Limits.* Prints or plots the current grid limits. This option is only available from the Model tab.
   * *Extents.* Plots all objects in the drawing.
   * *Display.* Plots all objects displayed in the drawing area.
   * *View.* Prints or plots a saved view. Select a named view from the list provided.
   * *Window.* Plots objects in the area you define. Select the Window option, and then respond to the prompts to define the area. Click
     the Window button to edit the defined area.
6. Click OK.
7. In the Page Setup Manager, click Close.
