# To Change the Coordinate Display on the Status Bar

Use one of the following methods:

* At any prompt for a point, click the coordinate display located on the left end of the status bar ( ![](../images/GUID-44FB590E-5C3C-4CB5-BDCF-E48927891449-low.png) Find).
* Press Ctrl + i repeatedly.
* Set the COORDS system variable to 0 for static display, 1 for dynamic display, or 2 for distance and angle display.

NOTE:If coordinates are not displayed on the status bar, select Coordinates from the status bar settings.
