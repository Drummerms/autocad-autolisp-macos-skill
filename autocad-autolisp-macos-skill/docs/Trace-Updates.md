# Trace Updates

Updates to Trace include an improved toolbar and editing xrefs while in Edit Drawing mode.

## Trace Toolbar

The Trace toolbar now more clearly shows whether you're editing the trace (TRACEEDIT) or editing the drawing (TRACEVIEW).

![](../images/GUID-57CEB71D-9030-4975-8AA7-F51DA1697E95-low.png)

## Edit External References while in Edit Drawing mode

While a Trace is open, in edit drawing mode, REFEDIT is now enabled, meaning you can now make changes to xrefs.
