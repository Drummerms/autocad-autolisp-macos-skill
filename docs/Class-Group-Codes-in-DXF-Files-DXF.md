# Class Group Codes in DXF Files (DXF)

The following is an example of the CLASSES section of a DXF file:

| 0  SECTION  2  CLASSES | *Beginning of CLASSES section* |
| 0  CLASS  1  <class dxf record>  2  <class name>  3  <app name>  90  <flag>  280  <flag>  281  <flag> | *Repeats for each entry* |
| 0  ENDSEC | *End of CLASSES section* |
