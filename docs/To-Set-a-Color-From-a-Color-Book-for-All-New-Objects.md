# To Set a Color From a Color Book for All New Objects

1. Click
    Modify ![](../images/ac.menuaro.gif) Properties.
2. In the Object Color drop-down list, click Select Color.
3. In the Color Palette dialog box, Color Books tab, choose a color book from the Color Book drop-down list.
4. Click within the narrow strip of colors, and then use the up and down arrow keys to browse through the color book.
5. Click a color swatch to the left of the strip. You can then use the up and down arrow keys, or you can enter a color number
   or name in the Color box.
