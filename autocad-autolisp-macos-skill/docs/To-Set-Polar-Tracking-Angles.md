# To Set Polar Tracking Angles

1. On the status bar, right-click ![](../images/GUID-519E604B-11D3-41C7-BDD7-35141FA899C4-low.png) Find.
2. From the menu that displays, click Tracking Settings.
3. In the Drafting Settings dialog box, Polar Tracking tab, select Polar Tracking On.
4. In the Increment Angle list, select the polar tracking angle.
5. To set additional tracking angles, select Additional Angles. Click New. Enter the angle value in the text box.
6. Under Polar Angle Measurement, specify whether polar tracking increments are based on the UCS or relative to the last object
   you created.
7. Click OK.
8. On the status bar, right-click ![](../images/GUID-519E604B-11D3-41C7-BDD7-35141FA899C4-low.png). Click an available angle or Settings to set additional tracking angles.
