# CANNOSCALE (System Variable)

Sets the name of the current annotation scale for the current space.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | 1:1 |

You can only enter a named scale that exists in the drawing’s named scale list.
