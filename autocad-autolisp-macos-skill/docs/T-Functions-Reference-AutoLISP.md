# T Functions Reference (AutoLISP)

These AutoLISP functions all start with 'T'.

## Functions

* [tablet](tablet-AutoLISP.md)

  Retrieves and sets digitizer (tablet) calibrations
* [tblnext](tblnext-AutoLISP.md)

  Finds the next item in a symbol table
* [tblobjname](tblobjname-AutoLISP.md)

  Returns the entity name of a specified symbol table entry
* [tblsearch](tblsearch-AutoLISP.md)

  Searches a symbol table for a symbol name
* [term\_dialog](term_dialog-AutoLISP-DCL.md)

  Terminates all current dialog boxes as if the user had canceled each of them
* [terpri](terpri-AutoLISP.md)

  Prints a newline to the command line
* [textbox](textbox-AutoLISP.md)

  Measures a specified text object, and returns the diagonal coordinates of a box that encloses the text
* [textpage](textpage-AutoLISP.md)

  Switches focus from the drawing area to the text screen
* [textscr](textscr-AutoLISP.md)

  Switches focus from the drawing area to the text screen
* [trace](trace-AutoLISP.md)

  Aids in AutoLISP debugging
* [trans](trans-AutoLISP.md)

  Translates a point (or a displacement) from one coordinate system to another
* [type](type-AutoLISP.md)

  Returns the type of a specified item
