# Create Surfaces

#### Topics in this section

* [About Creating 3D Surfaces](About-Creating-3D-Surfaces.md)

  Surface modeling provides the ability to create and edit associative and NURBS surfaces.
* [About Surface Modeling Workflow](About-Surface-Modeling-Workflow.md)

  Surface modeling is a process that may require conversions between different 3D modeling types.
* [About Creating Procedural Surfaces](About-Creating-Procedural-Surfaces.md)

  There are many ways to create procedural surfaces from existing surface or solid objects and subobjects.
* [About Converting Objects to Procedural Surfaces](About-Converting-Objects-to-Procedural-Surfaces.md)

  Convert 3D solids, meshes, and 2D geometry to procedural surfaces.
* [About Creating Associative Surfaces](About-Creating-Associative-Surfaces.md)

  Associative surfaces automatically adjust to changes made to other, related objects.
* [About Creating NURBS Surfaces](About-Creating-NURBS-Surfaces.md)

  Create NURBS surfaces when you need a freeform way to sculpt a surface shape.
