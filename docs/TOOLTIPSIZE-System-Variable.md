# TOOLTIPSIZE (System Variable)

Sets the display size for drafting tooltips, and for automatic completion text at the Command prompt.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

Valid range is -3 to 6. Greater values result in larger drafting tooltips, and larger automatic completion text at the Command
prompt. Negative values represent smaller sizes than the default.
