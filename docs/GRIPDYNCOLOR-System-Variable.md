# GRIPDYNCOLOR (System Variable)

Controls the color of custom grips for dynamic blocks.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 140 |

The valid range is 1 to 255.
