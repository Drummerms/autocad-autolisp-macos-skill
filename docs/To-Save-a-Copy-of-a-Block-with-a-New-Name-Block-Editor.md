# To Save a Copy of a Block with a New Name (Block Editor)

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block Editor.
   ![](../images/GUID-E3DDBE51-C92A-4D07-A51D-C7C685E4F578-low.png)
2. In the Edit Block Definition dialog box, select the block. Click Edit Block.

   The Block Editor opens up. Edit the drawing.
3. Click
   Block Editor visor ![](../images/ac.menuaro.gif) Save drop-down ![](../images/ac.menuaro.gif) Save New.
4. In the Save Block Definition As dialog box, enter a name for the new block definition. Click Save Block.
5. To save the block definition in the drawing, click
   File ![](../images/ac.menuaro.gif) Save.
