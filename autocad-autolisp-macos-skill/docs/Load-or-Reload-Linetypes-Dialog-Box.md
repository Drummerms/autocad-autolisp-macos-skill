# Load or Reload Linetypes Dialog Box

Loads a linetype whose definition exists in a linetype library (LIN) file.

## Summary

The
 *acad.lin*  file contains the standard linetypes.

## List of Options

The following options are displayed.

File Name
:   Displays the name of the current LIN file. You can enter the name of another LIN file or click the File button to select a
    file from the Select Linetype File dialog box.

Browse
:   Displays the Select Linetype File dialog box, in which you can select a different linetype (LIN) file.

Available Linetypes
:   Displays the linetypes available to load. To select or clear all of the linetypes on the list, right-click and choose Select
    All or Clear All.
