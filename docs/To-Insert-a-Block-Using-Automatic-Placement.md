# To Insert a Block Using Automatic Placement

1. At the Command prompt, enter AUTOPLACEMENT.
2. Enter
   *1* to display placement suggestions as you insert a block.
3. Click
   Window menu ![](../images/ac.menuaro.gif) Blocks.
4. Click one of the following tabs:
   * Current drawing
   * Recent Blocks
   * Blocks from Libraries
5. Right-click a block and choose Insert in Drawing.

   Alternatively, double-click to insert the block reference.
6. As you move the block in the drawing, placement suggestions are displayed. Do one of the following:
   * Click to accept the block placement suggestion.
   * Press Cmd to switch to another placement suggestion.
   * Move the cursor away to ignore the placement suggestion.

NOTE:To turn off block placement suggestions temporarily, hold down Shift+W or Shift+[ while you insert the block.
