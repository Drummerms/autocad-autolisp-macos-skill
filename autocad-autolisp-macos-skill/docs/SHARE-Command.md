# SHARE (Command)

Shares a link to a copy of the current drawing to view or edit in the AutoCAD web app. The drawing copy includes all external
references and images.

## Access Methods

*Toolbar*:
![](../images/GUID-EFB46326-0826-4394-9848-96582A551AE5-low.png)

*Menu:
File ![](../images/ac.menuaro.gif) Share*

Anyone with the link can access the drawing in the AutoCAD web app. The link expires seven days after it's created. You can
choose between two permission levels for recipients:
*View only* and
*Edit and save a copy*.
