# N Functions Reference (AutoLISP)

These AutoLISP functions all start with 'N'.

## Functions

* [namedobjdict](namedobjdict-AutoLISP.md)

  Returns the entity name of the current drawing's named object dictionary, which is the root of all nongraphical objects in
  the drawing
* [nentsel](nentsel-AutoLISP.md)

  Prompts the user to select an object (entity) by specifying a point, and provides access to the definition data contained
  within a complex object
* [nentselp](nentselp-AutoLISP.md)

  Provides similar functionality to that of the nentsel function without the need for user input
* [new\_dialog](new_dialog-AutoLISP-DCL.md)

  Begins a new dialog box and displays it, and can also specify a default action
* [not](not-AutoLISP.md)

  Verifies that an item evaluates to nil
* [nth](nth-AutoLISP.md)

  Returns the *nth* element of a list
* [null](null-AutoLISP.md)

  Verifies that an item is bound to nil
* [numberp](numberp-AutoLISP.md)

  Verifies that an item is a real number or an integer
