# To Create a Dimension Substyle

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Dimension panel ![](../images/ac.menuaro.gif) Dimension Style.
   ![](../images/GUID-89D5BC05-30EE-4627-B80B-8680F08A01A9-low.png)
2. In the Dimension Style Manager, select the style from which you want to create a substyle. Click New.
3. In the Create New Dimension Style dialog box, select the type of dimension that will apply to the substyle from the Use For
   list. Click Continue.
4. In the New Dimension Style dialog box, select the appropriate tab and make changes to define the dimension substyle.
5. Click OK.
6. Click Close to exit the Dimension Style Manager.
