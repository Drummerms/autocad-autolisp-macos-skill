# JIGZOOMMAX (System Variable)

Controls the maximum percentage of the view dimensions that the block extents must fit when being inserted.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

Valid values range from 0 to 90 (percent).

NOTE:The JIGZOOMMAX value must be equal to or greater than the specified value in the JIGZOOMMIN system variable.
