# To Create a Boundary Polyline

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Hatch panel ![](../images/ac.menuaro.gif) Boundary.
   ![](../images/GUID-D54FC086-4D7A-4EAA-8243-41FF6509F5EC-low.png)
2. In the Boundary Creation dialog box, in the Object Type list, select Polyline.
3. Under Boundary Set, do one of the following:
   * To create a boundary set from all objects visible in the current viewport, select Current Viewport.
   * To specify which objects to include in the boundary set, click New. Select the objects that you want to use to create the
     boundary.
4. Click Pick Points.
5. Specify points within each area to form a boundary polyline.

   NOTE:This area must be totally enclosed: there can be no gaps between enclosing objects.
6. Press Enter to create the boundary polyline and end the command.

   NOTE:The polyline may not be visible as it overlaps the objects used to create it.
