# HPMAXLINES (System Variable)

Sets the maximum number of hatch lines that are generated in a hatch operation.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 100000 |

Valid values are from 100 to 10,000,000.
