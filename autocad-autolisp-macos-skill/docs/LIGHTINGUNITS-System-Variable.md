# LIGHTINGUNITS (System Variable)

Specifies the lighting units for the drawing.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 2 |

| Value | Description |
| 0 | No longer a valid setting starting with AutoCAD 2017-based products  NOTE:It is recommended to update legacy drawings to a 1 or 2. |
| 1 | American lighting units (foot-candles) are used and photometric lighting is enabled |
| 2 | International lighting units (lux) are used and photometric lighting is enabled |
