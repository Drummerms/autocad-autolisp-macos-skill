# To Specify a Point From a Temporary Reference Point

1. At a prompt for a point, enter *from*. Alternately, press Shift and right-click to display the object snap menu, and then choose From.
2. If you want to offset from a location on an existing object, specify an object snap, and select the object snap point.
3. Enter a relative coordinate such as @1,0.
