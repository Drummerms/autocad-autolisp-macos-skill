# VSFACESTYLE (System Variable)

Controls how faces are displayed in the current viewport.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | No style applied |
| 1 | Real: as close as possible to how the face would appear in the real world |
| 2 | Gooch: uses cool and warm colors instead of dark and light to enhance the display of faces that might be shadowed and difficult to see in a realistic display |

NOTE:Existing visual styles are not changed when you enter a new value for this system variable. Any new value entered for this
system variable temporarily creates an unsaved new visual style.
