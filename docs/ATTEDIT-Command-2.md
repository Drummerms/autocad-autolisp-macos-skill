# ATTEDIT (Command)

Changes attribute information in a block.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Attribute drop-down ![](../images/ac.menuaro.gif) Attribute, Global.
![](../images/GUID-91BCBDF8-1AB3-4876-B9D5-4959DE7B6ABD-low.png)

*Menu*:
Modify ![](../images/ac.menuaro.gif) Object ![](../images/ac.menuaro.gif) Attribute ![](../images/ac.menuaro.gif) Global.

The Edit Attributes dialog box is displayed to edit attribute values for a specific block.

If you enter -ATTEDIT at the Command prompt, you can modify both attribute values and attribute properties, independent of
a block.

The following prompt is displayed.

Select block reference
:   Select the block reference whose attribute values you want to update.
