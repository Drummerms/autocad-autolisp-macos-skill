# BLOCKREDEFINEMODE (System Variable)

Controls whether a dialog box displays when inserting a block from the Blocks palette with the same name as an existing block
definition.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

When you try to insert a different block with the same name, a dialog box displays that provides you the option of redefining
the existing block to the one you're inserting.

| Value | Description |
| 0 | Always uses the existing block definition rather than the new one being inserted |
| 1 | Displays a dialog box that allows you to redefine the existing block definition |
