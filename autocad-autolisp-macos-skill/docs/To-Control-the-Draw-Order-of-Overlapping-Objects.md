# To Control the Draw Order of Overlapping Objects

1. Click
   Tools menu ![](../images/ac.menuaro.gif) Draw Order.
2. Choose one of the options.
3. Select the objects that you want to modify, and press Enter.
4. For the Above Objects and Under Objects options, select the reference objects, and press Enter.
