# Shortcuts Tab (Customize Dialog Box)

Manages the available keyboard shortcuts. A keyboard shortcut is a combination of keystrokes, such as CTRL+S for SAVE.

CUI (Command)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Customize ![](../images/ac.menuaro.gif) Interface (CUI).

![](../images/GUID-68514F1C-8CE3-433D-A487-18BF686744D6-low.png)

The following options are displayed.

Command\Shortcut List
:   Displays the current list of keyboard shortcuts. Click a column header to sort the list by that column.

Add ![](../images/GUID-9174311E-7DC8-49C8-B39F-991EC3DAD0E2-low.png)
:   Displays the visor where you can record a shortcut and assign it to a command.

Remove ![](../images/GUID-A140F0E8-0D4A-41B4-8078-48722E2E2CDB-low.png)
:   Removes the selected shortcut.

Search
:   Enter text to filter the list of commands that have an assigned shortcut.

Shortcut drop-down
:   You can view all keyboard shortcuts or you can view just the custom keyboard shortcuts.

Reveal in Finder
:   Locates the CUI file in Finder. This is helpful if you want to copy the file to another machine.

Apply
:   Save and apply the changes made so far in the dialog box.

Reset to Default
:   Reset all keyboard shorts to the product's default state.
