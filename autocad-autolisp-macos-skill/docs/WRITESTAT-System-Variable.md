# WRITESTAT (System Variable)

Indicates whether a drawing file is read-only or can be revised.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 1 |

For developers who need to determine write status through AutoLISP.

| 0 | Cannot write to the drawing |
| 1 | Can write to the drawing |
