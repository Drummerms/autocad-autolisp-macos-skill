# About Controlling the Display of Fractions

You can control the format of the fraction displayed in dimensions.

You can set the fraction format in dimensions using the DIMFRAC system variable when the DIMLUNIT system variable is set to
4 (architecture) or 5 (fractional).

The following illustration shows the different fraction formats available.

![](../images/GUID-B1E81581-3E5D-4D69-9A8D-4B8EECE1ED08-low.png)

These settings are available on the Modify/New Dimension Style dialog box, Primary Units tab.
