# Decorative Tiles (DCL)

Image and Spacer tiles are used to provide messages to the user when a dialog definition is displayed.

The tiles listed below do not cause actions and cannot be selected. They are provided for visual emphasis, or to assist you
in laying out the dialog box.

| Decorative and Informative Tiles | | Platforms | | | | |
| Windows | | Mac OS | | Web |
| Tile name | Description | AutoCAD | AutoCAD LT | AutoCAD | AutoCAD LT | AutoCAD |
| [spacer](spacer-Tile-DCL.md) | Tile used to add space between other tiles in a dialog definition. | ✓ | ✓ | ✓ | -- | ✓ |
| [spacer\_0](spacer_0-Tile-DCL.md) | Tile used to add space between other tiles in a dialog definition. | ✓ | ✓ | ✓ | -- | ✓ |
| [spacer\_1](spacer_1-Tile-DCL.md) | Tile used to add space which has a width and height equal one between other tiles in a dialog definition. | ✓ | ✓ | ✓ | -- | ✓ |
