# GCVERTICAL (Command)

Causes lines or pairs of points to lie parallel to the Y axis of the current coordinate system.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Parametric panel ![](../images/ac.menuaro.gif) Geometric Constraint drop-down ![](../images/ac.menuaro.gif) Vertical.
![](../images/GUID-40851FA7-5805-4EA0-81A3-31000EC13910-low.png)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Parametric ![](../images/ac.menuaro.gif) Geometric Constraints ![](../images/ac.menuaro.gif) Vertical.

## Summary

This command is equivalent to the Vertical option in
[GEOMCONSTRAINT](GEOMCONSTRAINT-Command.md).

Following are the valid constraint objects and points:

* Line
* Polyline segment
* Ellipse
* Multiline text
* Two valid constraint points

You can select different constraint points on the same object or two separate objects.

## List of Prompts

The following prompts are displayed.

Select an object or [[2Points](GCVERTICAL-Command.md)] <2Points>:
*Select an object or two constraint points*

2Points

Selects two constraint points instead of an object.
