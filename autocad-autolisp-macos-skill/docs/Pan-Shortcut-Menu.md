# Pan Shortcut Menu

Pan and Zoom shortcut menu allows for switching between navigation tools and cancelling the current navigation tool.

To access the Pan shortcut menu, right-click in the drawing area while PAN is active.

## List of Options

The following options are displayed.

Exit
:   Cancels PAN or ZOOM.

Pan
:   Switches to PAN.

Zoom
:   Switches to ZOOM in real time.

3D Orbit
:   Switches to 3DORBIT.

Zoom Window
:   Zooms to display an area specified by a rectangular window.

Zoom Original
:   Restores the original view.

Zoom Extents
:   Zooms to display the drawing extents.
