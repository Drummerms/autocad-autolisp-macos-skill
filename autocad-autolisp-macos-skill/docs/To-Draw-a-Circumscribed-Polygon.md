# To Draw a Circumscribed Polygon

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Rectangle drop-down ![](../images/ac.menuaro.gif) Polygon.
   ![](../images/GUID-B4E2434C-55AA-4D6F-AA57-33F01CA9297F-low.png)
2. At the Command prompt, enter the number of sides.
3. Specify the center of the polygon (1).
4. Enter
   *c* to specify a polygon circumscribed about a circle.
5. Enter the radius length (2).

   ![](../images/GUID-6C2373DD-51F1-4D15-8EEE-3D0DDD7032CD-low.png)
