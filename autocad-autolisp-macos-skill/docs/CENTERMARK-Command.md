# CENTERMARK (Command)

Creates an associative, cross-shaped mark at the center of a selected circle or arc.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Centerline drop-down ![](../images/ac.menuaro.gif) Center Mark.
![](../images/GUID-B272B416-A3D9-4ABD-A4FB-92E2A91E8D94-low.png)

*Menu*:
Draw ![](../images/ac.menuaro.gif) Center Mark.

The CENTERMARK command automatically repeats after you select a circle or an arc.

The following prompt is displayed.

Select Circle or Arc
:   Specifies circles or arcs for drawing of center marks. You can add a center mark to one or more circles or arcs within the
    command duration.

Layer
:   Assigns new center marks to the specified layer, overriding the current layer. Enter
    *Use Current* or "*.*" to use the current layer. ([CENTERLAYER](CENTERLAYER-System-Variable.md) system variable)
