# To Modify a User Parameter in the Parameters Manager

1. Click Parametric tab![](../images/ac.menuaro.gif)Manage panel![](../images/ac.menuaro.gif)Parameters Manager. ![](../images/GUID-6C8237C9-7325-4FDE-9739-185DA7AE46E7-low.png) Find
2. Double-click the box of the variable that you want to edit, and change the value.
3. Press Enter or click in the drawing area to confirm the change.

NOTE:You can modify values only in the Name, Expression, and Description columns.
