# To Assign a TrueType Font to a Text Style

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Text panel ![](../images/ac.menuaro.gif) Text Style.
   ![](../images/GUID-A1A489F9-EDAC-453E-B777-A21DD0ADC60D-low.png)
2. Click New (+). Enter a style name for the new text style. Click OK.
3. Under Font Family, select a TrueType font from the list.

   TrueType fonts display a TrueType icon in front of their names.
4. To update text of the current style in the drawing, click Apply.
5. Click Close.
