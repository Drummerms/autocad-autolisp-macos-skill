# To Specify the Size of a Center Mark

Control the size for new and existing center marks.

## Specify the Default Size for New Center Marks

1. At the Command prompt, enter CENTERCROSSSIZE.
2. Specify a new value. You can enter any of the following:
   * Relative value - any positive real number followed by
     *x*.
   * Absolute value - any positive real number
   * *ByLineType*

## Specify the Size for Existing Center Marks

1. If the Properties Inspector is not displayed, select Menu ![](../images/ac.menuaro.gif) Properties.
2. Select one or more center marks.
3. In the Properties Inspector ![](../images/ac.menuaro.gif) All tab, under Geometry, select Cross Size.
4. Specify a new value.
