# View Gallery

Inserts or switches to a view by selecting from the thumbnail images of named views.

Views created using the NEWVIEW command are placed on the view gallery. You can also create views directly from the view gallery.

## Model Tab

If a layout is active, selecting a named view that was saved in model space places a viewport on to the current layout. Click
Insert New Model View to create a view and insert it on the active layout.

![](../images/GUID-7CC73D75-2548-44BF-A58E-AD8563F9EBC7-low.png)

If model space is active, selecting any named view switches to that view. Click New Model View to create a view and add it
to the view gallery.

![](../images/GUID-E808AB4B-C3F3-4034-B79C-09077ED1ADB8-low.png)

## Layout Tab

Selecting any view created from a layout, switches to that view. Click New Layout View to create a layout view and save it
to the gallery.

![](../images/GUID-D2F99ED3-07CF-4EB9-9B43-D1E6858A65DA-low.png)
