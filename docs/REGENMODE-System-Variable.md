# REGENMODE (System Variable)

Controls automatic regeneration of the drawing.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| 0 | Turns off the [REGENAUTO](REGENAUTO-Command.md) command |
| 1 | Turns on the [REGENAUTO](REGENAUTO-Command.md) command |
