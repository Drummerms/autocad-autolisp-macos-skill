# To Work with Counting Objects

## Count the Instances of an Object in a Drawing

1. Select an object, then right-click and choose Count.

All instances of the specified object are highlighted and the count result is displayed on the Count visor. This visor provides
several commands and options to review and manage the objects in the count.

![](../images/GUID-3951D146-B66F-4393-B135-8CEDEB6C08F1-low.png)

## Count Blocks Using the Count Palette

1. Click
   Window menu ![](../images/ac.menuaro.gif) Count or enter COUNTLIST at the command prompt.

   The Count palette displays all the blocks in the drawing with their corresponding count.
2. On the Count palette, select a block. The counted blocks are highlighted in the drawing area.
3. Click Select on the visor to select the highlighted objects.
   ![](../images/GUID-8CD7650C-9809-4C89-886A-A81B64959DE7-low.png)

## Count Objects Within an Area

1. Select Count from the toolbar.
   ![](../images/GUID-1FBDE4F0-CE0E-42FA-AEB8-F17779ECC144-low.png)
2. Specify the two corners for the area on the drawing.
3. Select specific objects or blocks or press Enter to count all blocks within the area.

   All instances that match the selected objects are highlighted in the drawing.

   NOTE:The specified area becomes the current area for the next count.
4. Click Details on the Count visor to open the palette for further filtering options.
   ![](../images/GUID-BF2DBC8C-3390-4661-BFD8-AF1CE0A36153-low.png)
5. Click Select on the Count visor to select the highlighted objects.
   ![](../images/GUID-8CD7650C-9809-4C89-886A-A81B64959DE7-low.png)

## Filter the Count Based on Block Properties

1. On the Count palette, right-click a block.
2. Under Expand By list, choose one or more property filters.

   Available filters include the general properties of the counted blocks such as the layer, scale, and mirror state. Additional
   filters are displayed for user-defined parameters and attributes of the counted blocks.

## View the Count Details

Do one of the following:

* On the Count palette, right-click a block and choose Review Count Details.
* On the Count visor, click the Count Details icon.
  ![](../images/GUID-BF2DBC8C-3390-4661-BFD8-AF1CE0A36153-low.png) or
  ![](../images/GUID-525D7C63-B386-4CE0-8067-475C63D0D592-low.png)

The Count palette displays the count details of the selected object or block and the error report, if any.

## Create a Count Table

1. On the Count palette, click Create Table.
2. Select the counted blocks you want to add in the table. You can also add the counted blocks with property filters to the table.
3. Click Insert.
4. Click and place the count table into the current drawing. The table uses the current table style. A field is used for the
   count.

   NOTE:Use REGEN to update the count field if the drawing changes.

## Insert a Count Field

1. Select an object in the drawing area, right-click and choose Count.
2. In the Count visor, click Insert Count Field.
   ![](../images/GUID-FF1AA845-FC4A-4242-BAE9-8E7DF095DB73-low.png)
3. Click and place the field into the current drawing.

The field displays the current count of the selected block.

NOTE:Use REGEN to update the count field.

## Navigate Between Objects in a Count

* In the Count visor, click the right and left arrows to zoom into the previous or next object in the count.

## End the Count

* In the Count visor, click End Count.
  ![](../images/GUID-61E6D764-AFA5-4A56-83DE-DF6A70741C41-low.png)
