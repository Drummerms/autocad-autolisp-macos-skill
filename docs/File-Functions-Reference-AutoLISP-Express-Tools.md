# File Functions Reference (AutoLISP/Express Tools)

NOTE:Express Tools are supported in AutoCAD for Windows only; not available in AutoCAD LT for Windows, or on Mac OS and Web.

The following table provides a summary of the AutoLISP Express Tools file functions.

| File functions | | Platforms | | | | |
| Windows | | Mac OS | | Web |
| Function | Description | AutoCAD | AutoCAD LT | AutoCAD | AutoCAD LT | AutoCAD |
| [(acet-file-attr filename [attr])](acet-file-attr-AutoLISP-Express-Tools.md) | Gets or sets the protection attributes of a specified file or directory | ✓ | -- | -- | -- | -- |
| [(acet-file-chdir directory)](acet-file-chdir-AutoLISP-Express-Tools.md) | Changes the current working directory | ✓ | -- | -- | -- | -- |
| [(acet-file-copy from to [force])](acet-file-copy-AutoLISP-Express-Tools.md) | Copies a file | ✓ | -- | -- | -- | -- |
| [(acet-file-cwd)](acet-file-cwd-AutoLISP-Express-Tools.md) | Returns the current working directory | ✓ | -- | -- | -- | -- |
| [(acet-file-dir pattern [attributes] [starting-directory])](acet-file-dir-AutoLISP-Express-Tools.md) | Locate files | ✓ | -- | -- | -- | -- |
| [(acet-file-mkdir directory)](acet-file-mkdir-AutoLISP-Express-Tools.md) | Creates a new directory | ✓ | -- | -- | -- | -- |
| [(acet-file-move from to)](acet-file-move-AutoLISP-Express-Tools.md) | Moves or renames files | ✓ | -- | -- | -- | -- |
| [(acet-file-remove filespec [force])](acet-file-remove-AutoLISP-Express-Tools.md) | Deletes one or more files | ✓ | -- | -- | -- | -- |
| [(acet-file-rmdir directory)](acet-file-rmdir-AutoLISP-Express-Tools.md) | Removes a directory | ✓ | -- | -- | -- | -- |
