# APBOX (System Variable)

Turns the display of the AutoSnap aperture box on or off.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

The aperture box is displayed in the center of the crosshairs when you snap to an object.

| 0 | Off |
| 1 | On |
