# UCSVIEW (System Variable)

Determines whether the current UCS is saved with a named view.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Does not save current UCS with a named view |
| 1 | Saves current UCS whenever a named view is created |
