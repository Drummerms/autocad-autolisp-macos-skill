# HPOBJWARNING (System Variable)

Sets the number of hatch boundary objects that can be selected before displaying a warning message.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 10000 |

The maximum value can vary, but is significantly larger than 100000000 (one hundred million).
