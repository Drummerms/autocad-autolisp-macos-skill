# HIGHLIGHT (System Variable)

Controls object highlighting; does not affect objects selected with grips.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 1 |

| Value | Description |
| 0 | Turns off object selection highlighting |
| 1 | Turns on object selection highlighting |
