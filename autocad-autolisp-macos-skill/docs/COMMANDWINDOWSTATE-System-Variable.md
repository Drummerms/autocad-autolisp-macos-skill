# COMMANDWINDOWSTATE (System Variable)

Indicates whether the command window is open or closed.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | Varies |

| 0 | Closed |
| 1 | Open |

Cmd-3 turns the command window on and off.
