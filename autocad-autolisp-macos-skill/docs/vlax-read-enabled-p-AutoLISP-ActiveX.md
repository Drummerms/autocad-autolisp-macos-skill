# vlax-read-enabled-p (AutoLISP/ActiveX)

Determines if an object can be read

*Supported Platforms:* Windows only; not available on Mac OS or Web

## Signature

```
(vlax-read-enabled-p obj)
```

*obj*
:   *Type:* VLA-object

    An object.

## Return Values

*Type:* T or nil

T, if the object is readable; otherwise
nil.
