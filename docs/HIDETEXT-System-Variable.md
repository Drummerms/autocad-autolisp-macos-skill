# HIDETEXT (System Variable)

Specifies whether text objects created by the TEXT or MTEXT command are processed during a HIDE command.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Drawing |
| Initial value: | On |

| Off | Text is not hidden and does not hide other objects |
| On | Text is hidden but does not hide other objects |
