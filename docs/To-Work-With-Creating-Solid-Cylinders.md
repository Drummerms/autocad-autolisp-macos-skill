# To Work With Creating Solid Cylinders

## Create a Solid Cylinder With a Circular Base

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Solid drop-down ![](../images/ac.menuaro.gif) Cylinder.
   ![](../images/GUID-B5FCBEFD-B5F8-4D76-AFA2-0CFA4C896D24-low.png)
2. Specify the center point of the base.
3. Specify the radius or diameter of the base.
4. Specify the height of the cylinder.

## Create a Solid Cylinder With an Elliptical Base

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Solid drop-down ![](../images/ac.menuaro.gif) Cylinder.
   ![](../images/GUID-B5FCBEFD-B5F8-4D76-AFA2-0CFA4C896D24-low.png)
2. At the Command prompt, enter
   *e* (Elliptical).
3. Specify the start point of the first axis.
4. Specify the endpoint of the first axis.
5. Specify the endpoint (length and rotation) of the second axis.
6. Specify the height of the cylinder.

## Create a Solid Cylinder With the Height and Rotation Specified (Axis Endpoint)

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Solid drop-down ![](../images/ac.menuaro.gif) Cylinder.
   ![](../images/GUID-B5FCBEFD-B5F8-4D76-AFA2-0CFA4C896D24-low.png)
2. Specify the center point of the base.
3. Specify the radius or diameter of the base.
4. At the Command prompt, enter
   *a* (Axis endpoint). Specify the axis endpoint of the cylinder.

   This endpoint can be located anywhere in 3D space.
