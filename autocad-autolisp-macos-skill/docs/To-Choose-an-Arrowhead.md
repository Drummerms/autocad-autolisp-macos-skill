# To Choose an Arrowhead

Set dimension arrowheads for each dimension style.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Dimension panel ![](../images/ac.menuaro.gif) Dimension Style.
   ![](../images/GUID-89D5BC05-30EE-4627-B80B-8680F08A01A9-low.png)
2. In the Dimension Style Manager, select the style you want to change. Click Options
   ![](../images/GUID-0888F513-2A7D-45B1-AD99-683EBA803A86-low.png) ![](../images/ac.menuaro.gif) Modify.
3. In the Modify Dimension Style dialog box, Symbols and Arrows tab, under Arrowheads, select the arrowhead type for the first
   end of the dimension line.

   The second arrowhead is automatically set to the same type.
4. To set the second end of the dimension line to a different arrowhead type, select an arrowhead type from the Second list.
5. In the Size box, enter a size for the arrowhead.
6. Click OK.
7. Click Close to exit the Dimension Style Manager.
