# To Save a Block as a Drawing

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block.
   ![](../images/GUID-B2E92148-A285-4542-BEF5-AD10E48A0DE5-low.png)
2. In the Define Block dialog box, Click Select Objects to select the block.
3. In the Name and Description box, enter the drawing name and description.

   The name can have up to 255 characters and can include letters, numbers, blank spaces, and any special character not used
   by the operating system or the program for other purposes.
4. The Block Editor opens up. Click Save As to save the copy of the block with a new name.
