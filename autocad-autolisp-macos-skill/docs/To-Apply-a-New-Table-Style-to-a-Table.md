# To Apply a New Table Style to a Table

## Table Visor

1. Click a gridline on the table to select the table.

   The Table visor displays thumbnails for each table style.

   ![](../images/GUID-CE48C351-3112-4794-88BD-79C31FC95237-low.png)
2. Select a table style.

   The table style is applied to the table.

## Properties Inspector

1. Click a gridline on the table to select the table.
2. If the Properties Inspector isn't open, click
   Window menu ![](../images/ac.menuaro.gif) Properties Inspector.
3. On the Properties Inspector, select a table style from the Table Style drop-down in the Table section.

   The table style is applied to the table.

## Context Menu

1. Click a gridline on the table to select the table.
2. Right-click and select Table Style.
3. On the Table Style flyout, select a table style from the list.

   The table style is applied to the table.
