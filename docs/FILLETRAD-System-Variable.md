# FILLETRAD (System Variable)

Stores the current fillet radius for 2D objects.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 0.0000 |

If you use the FILLET command to change the radius of a 2D fillet, the value of this system variable changes to match.
