# About Using Named UCS Definitions and Preset Orientations

When you need to switch between several UCS orientations, you can save and restore them by name with the UCS command. Each
UCS definition can have its own origin and *X*, *Y*, and *Z* axes. You can also choose from several preset orientations when you use the UCSMAN command.
