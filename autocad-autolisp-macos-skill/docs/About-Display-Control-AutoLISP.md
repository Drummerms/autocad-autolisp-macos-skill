# About Display Control (AutoLISP)

AutoLISP includes functions for controlling the AutoCAD display in both text and graphics windows. Some functions prompt
for, or depend on, input from the AutoCAD user.

The prompt, princ, prin1, and print functions are the primary text output functions.

#### Topics in this section

* [About Controlling Menus (AutoLISP)](About-Controlling-Menus-AutoLISP.md)

  The
  menucmd function controls the display of the menus on the menu bar, drawing area, or an image title menu.
* [About Controlling the Graphics and Text Windows (AutoLISP)](About-Controlling-the-Graphics-and-Text-Windows-AutoLISP.md)

  You can control the display of the graphics and text windows from an application.
* [About Controlling Low-Level Graphics (AutoLISP)](About-Controlling-Low-Level-Graphics-AutoLISP.md)

  Low-level graphics in the drawing area and application window can be controlled using AutoLISP functions.
