# GCPERPENDICULAR (Command)

Causes selected lines to lie 90 degrees to one another.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Parametric panel ![](../images/ac.menuaro.gif) Geometric Constraint drop-down ![](../images/ac.menuaro.gif) Perpendicular.
![](../images/GUID-47368B24-8DE6-419B-B572-045416045EF2-low.png)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Parametric ![](../images/ac.menuaro.gif) Geometric Constraints ![](../images/ac.menuaro.gif) Perpendicular.

## Summary

This command is equivalent to the Perpendicular option in
[GEOMCONSTRAINT](GEOMCONSTRAINT-Command.md).

Lines do not have to intersect to be perpendicular.

Following are the valid constraint objects and points:

* Line
* Polyline segment
* Ellipse
* Multiline text
