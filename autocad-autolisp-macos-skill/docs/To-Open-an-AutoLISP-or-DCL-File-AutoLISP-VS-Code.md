# To Open an AutoLISP or DCL File (AutoLISP/VS Code)

## Open a File in an AutoLISP Project (AutoLISP Project Manager)

1. In Visual Studio Code, on the Activity Bar, click AutoLISP Project Manager.
2. In the AutoLISP Project Manager view,
   [open an AutoLISP project (PRJ) file](GUID-C5E9BA22-258E-4091-9B65-9DD965990F36.htm#GUID-F39AF752-3C7A-4B35-A4C6-47B7290F9DCF), if one is not already open.
3. Under the current project, click the LSP file to open.

## Open a File Outside the Open Folder

1. In Visual Studio Code, click File menu > Open File.
2. In the Open File dialog box, browse to and select the file to open.
3. Click Open.

## Open a File from the Open Folder (Explorer View)

1. In Visual Studio Code, on the Activity Bar, click Explorer (or click View menu > Explorer).
2. In the Explorer view, expand the open folder and double-click the file to open.
