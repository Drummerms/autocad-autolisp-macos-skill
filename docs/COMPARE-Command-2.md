# -COMPARE (Command)

At the Command prompt, compares a specified drawing to the current drawing.

The following prompt is displayed.

Enter name of drawing to compare:
:   Specifies the location and name of the drawing to compare with the current drawing. The DWG Compare toolbar displays.
