# About Symbol and Function Handling (AutoLISP)

AutoLISP provides several functions for grouping one or more functions into a custom named function along with a number of
functions for handling symbols and variables . The function- and symbol-handling functions are listed in the AutoLISP Function
Synopsis (AutoLISP), under the heading Symbol-Handling Functions (AutoLISP).

#### Topics in this section

* [About Defining a Function (AutoLISP)](About-Defining-a-Function-AutoLISP.md)

  You can define your own functions.
* [About C:XXX Functions (AutoLISP)](About-C-XXX-Functions-AutoLISP.md)

  If an AutoLISP function is defined with a name of the form
  C:xxx, it can be issued at the AutoCAD Command prompt in the same manner as a built-in AutoCAD command.
* [About Local Variables in Functions (AutoLISP)](About-Local-Variables-in-Functions-AutoLISP.md)
* [About Functions with Arguments (AutoLISP)](About-Functions-with-Arguments-AutoLISP.md)

  With AutoLISP, many functions require you to pass them values. These values are known as arguments.
