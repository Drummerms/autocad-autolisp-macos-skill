# ok\_cancel\_help\_errtile Tile (DCL)

The
ok\_cancel\_help\_errtile tile provides a convenient way to specify the exit buttons and error tile all at once.

*Supported Platforms:* Windows, Mac OS, and Web

```
ok_cancel_help_errtile;
```

![](../images/GUID-5A352E6C-84EF-4A85-AA50-51D4C4184566-low.png)

The
ok\_cancel\_help\_errtile tile is defined in the
*base.dcl* file.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
