# Calculate, Query, and Measure Geometric Values

#### Topics in this section

* [About the Command Prompt Calculator](About-the-Command-Prompt-Calculator.md)

  By entering an expression in the Command prompt calculator, you can quickly solve a mathematical problem or locate points
  in your drawing.
* [About Finding Distances, Angles, and Point Locations](About-Finding-Distances,-Angles,-and-Point-Locations.md)

  You can extract geometric information from point locations on objects in a drawing.
* [About Finding Area and Mass Properties Information](About-Finding-Area-and-Mass-Properties-Information.md)

  You can create points or insert symbols (blocks) at equal intervals on an object.
