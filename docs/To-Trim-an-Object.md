# To Trim an Object

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Trim.
   ![](../images/GUID-53B8A7B8-0050-44CA-A923-870AF64E125A-low.png)
2. Select the objects to serve as cutting edges.

   To select all displayed objects as potential cutting edges, press Enter without selecting any objects.
3. Select the objects to trim.

## Quick Mode

Instead of having to first select boundaries for the Trim command, the Quick mode selects all potential boundaries by default.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Trim.
   ![](../images/GUID-53B8A7B8-0050-44CA-A923-870AF64E125A-low.png)
2. Enter O and select Quick.

   NOTE:Once Quick mode is enabled, it remains enabled until you change to Standard mode. The Quick mode setting is stored in the
   TRIMEXTENDMODE system variable.
3. Select the objects to trim.

NOTE: Use the cuTting edges option to temporarily allow you to select the object to use as the cutting edge.
