# To Work With Arcs

To create an arc, you can specify various combinations of center, endpoint, start point, radius, angle, chord length, and
direction values.

## Draw the Arc in a Clockwise Direction

By default, arcs are drawn in a counterclockwise direction. Use this method to draw them in a clockwise direction.

* Hold down the Ctrl key as you drag to draw the arc.
* Release Ctrl to draw in a counterclockwise direction again.

## Draw an Arc by Specifying Three Points

1. Click
   Draw
   ![](../images/ac.menuaro.gif) Arc drop-down
   ![](../images/ac.menuaro.gif) 3-Point.
   ![](../images/GUID-6E1BE1CC-A3C0-4FD0-A44D-8A242CAF780B-low.png)At the Command prompt, enter arc.
2. Specify the start point.
3. Specify a point on the arc.
4. Specify the endpoint.

## Draw an Arc Using a Start Point, a Center Point, and an Endpoint

1. Click
   Draw ![](../images/ac.menuaro.gif) Arc drop-down
   ![](../images/ac.menuaro.gif) Start, Center, End.
   ![](../images/GUID-2151E970-19A0-4BEE-96B2-9106687A191C-low.png)At the Command prompt, enter arc.
2. Specify a start point.
3. Specify the center point.
4. Specify the endpoint.

## Continue an Arc with a Tangential Line

1. Complete the arc.
2. Click
   Draw ![](../images/ac.menuaro.gif) Line.
   ![](../images/GUID-12971FA0-B529-418C-80D0-B2FD66874C09-low.png)At the Command prompt, enter line.
3. Press Enter at the first prompt.
4. Enter the length of the line and press Enter.

## Continue an Arc with a Tangential Arc

1. Complete the arc.
2. Click
    Draw ![](../images/ac.menuaro.gif) Arc drop-down
   ![](../images/ac.menuaro.gif) Continue.
   ![](../images/GUID-2A5192E6-DF4D-40F2-911A-07CC8F84F904-low.png)At the Command prompt, enter arc.
3. Specify the second endpoint of the tangent arc.
