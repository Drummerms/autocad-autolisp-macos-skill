# About Controlling Extension Lines

You can control extension line properties including color, lineweight, overshoot, and offset length.

You can

* Specify color and lineweight for visual effect and plotting
* Suppress one or both extension lines if they are unnecessary, or if there is not enough space

  ![](../images/GUID-95ECF2B5-EEEC-4AB3-8B36-466D74515DD2-low.png)
* Specify how far beyond from the dimension line the extension line extends (overshoot)

  ![](../images/GUID-C4BC322F-AA27-4A91-AAD3-E578506D5D57-low.png)
* Control the extension origin offset, the distance between the extension line origin, and the start of the extension line

  ![](../images/GUID-12CCCE19-B6BE-45BC-87BA-2CE5CD2EC4B4-low.png)
* Specify a fixed length for extension lines, as measured from the dimension line toward the extension line origin

  ![](../images/GUID-C4BC322F-AA27-4A91-AAD3-E578506D5D57-low.png)
* Specify a noncontinuous linetype, typically used for centerlines

  ![](../images/GUID-51D28751-C349-481F-B18F-51122A9BFEAF-low.png)
* Modify the angle of the extension lines of a selected dimension to make them oblique

  ![](../images/GUID-E8C84E43-D7F7-4D2E-A6A5-8DD414B124E7-low.png)

## Fixed-Length Extension Lines

You can specify a dimension style that sets the total length for extension lines starting from the dimension line toward
the dimension origin point.

![](../images/GUID-2A2572EA-507A-41ED-A592-5C73CDA730A7-low.png)

The extension line offset distance from the origin will never be less than the value specified by the DIMEXO system variable.
