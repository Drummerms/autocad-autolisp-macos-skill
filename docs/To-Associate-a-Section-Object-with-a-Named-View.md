# To Associate a Section Object with a Named View

1. Enter View at the command prompt.
2. From the options in the command prompt, select view > settings > live Section > enter View Name > enter Live Section Name
   > enter > enter.
3. The section object is now associated with a named view. When the named view is restored, live sectioning is turned on for
   the associated section object.
