# To Work with Tool Set Panels

## Change the Panel Order

1. Display the tool set tab that contains the panel you want to move.
2. Click Reorder Panels located at the bottom of the tool set.
   ![](../images/GUID-6721239D-925C-4C53-84C8-846E2D234300-low.png)
3. Drag the panel titles to set the order.
4. Press Esc to exit.

## Collapse or Expand a Panel

* Click the arrow next to the panel title.

## Display or Hide Tools on a Standard Panel

1. Click Customize Panel located at the upper right of the panel.
   ![](../images/GUID-3811A988-85ED-4568-A59C-A09C6074AEC6-low.png)
2. * Clear the check mark for any tools you want to hide.
   * Check any tools you want to show.

## Create a Custom Panel

1. Display the tool set tab that you want to add the panel.
2. Click Create a New Panel which is located at the bottom of the tool set.
   ![](../images/GUID-9174311E-7DC8-49C8-B39F-991EC3DAD0E2-low.png)

   The Panel interface opens which contains all the commands defined in the Customize interface.
3. Enter a panel title in the New Panel edit box.
4. Select the panel layout from the set of predefined layouts.
5. To add a command, drag it from the list on the right to the desired location on the left.
6. To add a drop-down,
   * Click
     ![](../images/GUID-9174311E-7DC8-49C8-B39F-991EC3DAD0E2-low.png) which is located at the bottom left of the interface, and select Add Drop-down.
   * Click to edit the drop-down name.
   * Drag commands to position them under the drop-down.
7. To remove a command or drop-down, select the tool and click
   ![](../images/GUID-A140F0E8-0D4A-41B4-8078-48722E2E2CDB-low.png) which is located at the bottom left of the interface.

## Edit a Custom Panel

1. Click Customize Panel located at the upper right of the panel.
   ![](../images/GUID-3811A988-85ED-4568-A59C-A09C6074AEC6-low.png)

   The Panel interface opens.
2. Follow the steps used to create the panel.

## Remove a Custom Panel

1. Hover over the upper right area of the custom panel.
2. Click Delete Panel when it appears.
   ![](../images/GUID-CB6168BA-3F5F-484B-9685-5E40611793BC-low.png)
3. Click Delete.
