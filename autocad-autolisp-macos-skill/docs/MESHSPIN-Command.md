# MESHSPIN (Command)

Spins the adjoining edge of two triangular mesh faces.

## Access Methods

*Tool Set*:
Modeling tab ![](../images/ac.menuaro.gif) Mesh panel ![](../images/ac.menuaro.gif) Spin Triangle Face.
![](../images/GUID-3ED1C747-EBDB-40AB-A4AC-BAD47D54C672-low.png)

*Menu*:
Modify
![](../images/ac.menuaro.gif) Mesh Editing
![](../images/ac.menuaro.gif) Spin Triangle Face.

## Summary

You can rotate the edge that joins two triangular mesh faces to modify the shapes of the faces. The edge shared by the selected
faces spins to intersect the apex of each face.

![](../images/GUID-E7527F8C-C53B-4692-A470-784EFC7A566C-low.gif)

NOTE:You can use
[MESHSPLIT](MESHSPLIT-Command.md) to divide a rectangular face into two triangular faces. If you plan to spin the edge between the triangular faces, use the
Vertex option to ensure that the split extends precisely from one vertex to the other.

## List of Prompts

The following prompts are displayed.

First triangular mesh face to spin
:   Specifies one of two faces to be modified. Click a triangular mesh face.

Second adjacent triangular mesh face to spin
:   Specifies the second of two faces to be modified. Click a triangular mesh face that is next to the first selected face.
