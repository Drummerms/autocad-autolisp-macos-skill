# HPPATHALIGNMENT (System Variable)

Specifies the alignment of new drawn paths to be hatched or filled.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 or CENTER |

Alignment is used to determine the directions in which the closed hatch boundary will be defined along the open or closed
path drawn.

| Value | Description |
| 1 or CENTER | Hatch boundary is centered equally along the path |
| 2 or INSIDE | Hatch boundary is defined towards the inside of the path when specifying points counterclockwise |
| 3 or OUTSIDE | Hatch boundary is defined towards the outside of the path when specifying points counterclockwise |
