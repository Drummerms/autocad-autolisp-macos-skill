# REDEFINE (Command)

Restores AutoCAD internal commands overridden by UNDEFINE.

## Summary

If a command has been undefined, you can still use it if you precede the command name with a period. Enter the name of an
AutoCAD command turned off by the
[UNDEFINE](UNDEFINE-Command.md) command.
