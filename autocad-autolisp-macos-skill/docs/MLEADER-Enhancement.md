# MLEADER Enhancement

The MLEADER command now has an option to select an existing mtext object to use for the new leader.

![](../images/GUID-6EAF2135-0BB6-4DC2-BC98-1064474D850E-low.png)

NOTE:If your mleader style has a default text value, the Mtext option is not available.
