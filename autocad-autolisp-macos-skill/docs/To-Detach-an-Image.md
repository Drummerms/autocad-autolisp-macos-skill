# To Detach an Image

1. Click Insert ![](../images/ac.menuaro.gif) References Manager.At the Command prompt, enter EXTERNALREFERENCES.
2. In the Reference Manager palette, right-click the image name.
3. Click Detach.

   The image is no longer linked to the drawing file, and all instances of the image are removed from the drawing.
