# OBSCUREDLTYPE (System Variable)

Specifies the linetype of obscured lines.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

An obscured line is a hidden line made visible by changing its color and linetype. OBSCUREDLTYPE is available only in 2D views.
In 3D views, the [VSOBSCUREDLTYPE](VSOBSCUREDLTYPE-System-Variable.md) system variable is used.

Obscured linetypes are independent of zoom level, unlike regular linetypes. The linetype values are defined as follows:
