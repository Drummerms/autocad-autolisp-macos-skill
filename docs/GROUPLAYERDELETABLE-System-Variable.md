# GROUPLAYERDELETABLE (System Variable)

Controls what happens to the layers referenced by a layer group when the group is deleted.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| Value | Description |
| 0 | Layers referenced by a layer group are not removed from the drawing when the group is deleted. |
| 1 | You are prompted on how to handle the deletion of layers referenced by the layer group being deleted.  When deleting the referenced layers, you will be prompted on how to handle layers that contain objects on them. |
