# Attach, Scale, and Detach Raster Images

#### Topics in this section

* [About Attaching Raster Images](About-Attaching-Raster-Images.md)

  You can attach a reference to a raster image file to a drawing file using a linked image path. The image file can be accessed
  from the Internet.
* [About Scaling Raster Images](About-Scaling-Raster-Images.md)

  You can control the size of a raster image in a drawing to match the scale of the drawing.
* [About Detaching Raster Images](About-Detaching-Raster-Images.md)

  You can detach the reference to an image file in a drawing.
