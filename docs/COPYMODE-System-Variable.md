# COPYMODE (System Variable)

Controls whether the COPY command repeats automatically.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Sets the COPY command to repeat automatically |
| 1 | Sets the COPY command to create a single copy |
