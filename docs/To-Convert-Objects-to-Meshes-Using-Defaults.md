# To Convert Objects to Meshes Using Defaults

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Mesh panel ![](../images/ac.menuaro.gif) Smooth Object.
   ![](../images/GUID-CC5A20F2-223C-4B04-84F8-C845F78C669B-low.png)
2. Select an object such as a 3D solid or surface.

   The objects are converted to a mesh using the default settings for the following system variables.

   * FACETERDEVNORMAL
   * FACETERDEVSURFACE
   * FACETERGRIDRATIO
   * FACETERMAXEDGELENGTH
   * FACETERMAXGRID
   * FACETERMESHTYPE
   * FACETERMINUGRID
   * FACETERMINVGRID
   * FACETERSMOOTHLEV
