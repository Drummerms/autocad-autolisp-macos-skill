# Add and Style Leaders with Callouts

#### Topics in this section

* [About Leader Objects](About-Leader-Objects.md)

  A leader object is a line or a spline with an arrowhead at one end and a multiline text object or block at the other.

* [About Leader Styles](About-Leader-Styles.md)

  The appearance of a leader is controlled by its multileader style. You can use the default multileader style, STANDARD, or
