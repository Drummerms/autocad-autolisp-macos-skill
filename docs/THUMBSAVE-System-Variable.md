# THUMBSAVE (System Variable)

Controls whether thumbnail preview images are created and saved with the drawing.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | No preview image is saved in the drawing |
| 1 | A preview image is created and saved in the drawing |
