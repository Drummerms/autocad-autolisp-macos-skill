# To Cut Objects to the Clipboard

1. Select the objects you want to cut.
2. Press Ctrl+X, or click
   Edit ![](../images/ac.menuaro.gif) Cut.

   The objects are available to be pasted into other Windows applications.

## To Cut Objects using a Specified Basepoint

1. Click
   Edit ![](../images/ac.menuaro.gif) Cut with Base Point.
2. Specify the base point on the drawing.
3. Select the objects you want to cut and press Enter.

   Use
   [PASTECLIP](PASTECLIP-Command.md) to specify the insertion point when pasting the objects into other drawings.
