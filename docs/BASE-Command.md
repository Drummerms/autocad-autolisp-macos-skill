# BASE (Command)

Sets the insertion base point for the current drawing.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block, Base.
![](../images/GUID-96E50FDE-0015-4C60-9253-DEC531B41F3C-low.png)

*Menu*:
Draw ![](../images/ac.menuaro.gif) Block ![](../images/ac.menuaro.gif) Base.

The base point is expressed as coordinates in the current UCS. When you insert or externally reference the current drawing
into other drawings, this base point is used as the insertion base point.
