# spacer\_0 Tile (DCL)

Tile used to add space between other tiles in a dialog definition.

*Supported Platforms:* Windows, Mac OS, and Web

```
spacer_0;
```

A
spacer\_0, demonstrated in the following figure, is a spacer that normally has no width. However, it indicates a point in a tile group
where you want space to be inserted, if the group has to be stretched during layout. If the
spacer\_0 tiles in a group are assigned a positive width, all of them are assigned an equal share of the spacing.

The
spacer\_0 tile is defined in the
*base.dcl* file.
