# FILETABSTATE (System Variable)

Indicates the display status of the file tabs at the top of the drawing area.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | The file tabs are hidden. |
| 1 | The file tabs are displayed. |

When file tabs are hidden, use Ctrl+TAB to navigate through the file tabs.
