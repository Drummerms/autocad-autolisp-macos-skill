# About the Privacy Settings

You can manage your privacy settings and the type of usage data Autodesk collects.

* Select Privacy Settings from the Help drop-down menu.
* Depending on the options selected, Autodesk automatically receives product-usage statistics, such as which features you use
  most, and system data, such as which operating system you use.
* Autodesk uses this real world data to help make better product development decisions.

Read the Autodesk Privacy Statement:
*<https://www.autodesk.com/cipprivacy>*
