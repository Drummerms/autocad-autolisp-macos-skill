# About Using Layers to Represent Multiple Annotation Scales

As an alternative to creating annotative object, you can use layers to control the display of annotation objects of different
sizes or scales in a viewport. This approach can be helpful when working with drawings that might need to be compatible with
legacy releases of AutoCAD or different products. For example, you might have two layers named Notes1-4 and Notes3-16 that
are used in viewports scaled to 1/4”=1’-0” and 3/16”=1’-0”. Each layer would contain a copy of each annotation object that
needed to be displayed at the appropriate size or scale. In the viewport, you would simply hide or show the annotation layer
by using the Viewport Freeze and Viewport Thaw settings.
