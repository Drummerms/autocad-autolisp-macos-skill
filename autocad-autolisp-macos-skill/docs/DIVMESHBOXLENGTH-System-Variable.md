# DIVMESHBOXLENGTH (System Variable)

Sets the number of subdivisions for the length of a mesh box along the *X* axis.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 3 |

![](../images/GUID-C8047EE4-6E9F-4A7F-A3FB-5725BEB99445-low.png)

This setting affects new mesh boxes that are created with the [MESH](MESH-Command.md) command.
