# To Select Objects Within a Polygonal Area

1. At the Select Objects prompt, do one of the following:
   * Enter *wp* (Window Polygon) and then specify points that define an area entirely enclosing the objects you want to select.
   * Enter *cp* (Crossing Polygon) and then specify points that define an area that encloses or *crosses* the objects you want to select.
2. Press Enter to close the polygon selection area and complete the selection.
