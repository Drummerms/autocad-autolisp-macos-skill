# External Reference Visor

Displays options for attaching external references.

![](../images/GUID-36E1E1B7-02D6-472F-A6F2-BB943CE7E72F-low.png)

## Summary

The External Reference visor is displayed when you select an external reference.

## List of Options

The following options are displayed.

Edit Reference in Place

Edits the selected xref directly within the current drawing window. (REFEDIT command)

Open Reference in New Window

Opens the selected drawing reference (xref) in a new window. (XOPEN command)

Create Clipping Boundary

Crops the display of the selected external reference to a specified boundary. (XCLIP command)

Delete the Clipping Boundary

Removes the clipping boundary from the selected external reference. (XCLIP command)

Show/Hide Reference Manager

Opens or closes the Reference Manager palette. (EXTERNALREFERENCES command)
