# DIMALT (System Variable)

Controls the display of alternate units in dimensions.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Drawing |
| Initial value: | Off |

See also  [DIMALTD](DIMALTD-System-Variable.md),  [DIMALTF](DIMALTF-System-Variable.md),  [DIMALTTD](DIMALTTD-System-Variable.md),  [DIMALTTZ](DIMALTTZ-System-Variable.md),  [DIMALTZ](DIMALTZ-System-Variable.md), and [DIMAPOST](DIMAPOST-System-Variable.md).

| Off | Disables alternate units |
| On | Enables alternate units |
