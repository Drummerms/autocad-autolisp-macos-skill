# To Create a Ruled Mesh

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Mesh panel ![](../images/ac.menuaro.gif) Modeling, Meshes, Ruled Surface.
   ![](../images/GUID-362C6D83-E86B-42FF-A45B-4E91BE875016-low.png)
2. Select an object to act as the first defining curve.
3. Select a second object as the second defining curve.

   Mesh segments are drawn between the defining curves. The number of segments equals the value set in the SURFTAB1 system variable.
4. Erase the original curves if necessary.
