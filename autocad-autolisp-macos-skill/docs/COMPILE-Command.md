# COMPILE (Command)

Compiles shape files and PostScript font files into SHX files.

## Summary

The Select Shape or Font File dialog box (a
[standard file selection dialog box](Standard-File-Selection-Dialog-Boxes.md)) is displayed. Enter the SHP or PFB file name in the dialog box. The compiled file is assigned this name with the file extension
.*shx*.
