# New Sheet Set Dialog Box

Creates a new blank sheet set, or a sheet set based on a template or one derived from an existing sheet set data structure.

NEWSHEETSET (Command)

*Menu*:
File ![](../images/ac.menuaro.gif) New Sheet Set

![](../images/GUID-5D4133F6-24E2-41F5-B61E-86722D1A0141-low.png)

## List of Options

The following options are displayed.

### Name

Specifies the name of the sheet set.

### Sheet Set Data File Location

Specifies the path of the sheet set data (DST) file. The name of the sheet set data (DST) file is determined by the text entered
for the sheet set name.

Click Browse to select a folder in Finder.

### Description

Specifies a description for the sheet set.

### Create Blank Sheet Set

Creates a new empty sheet set using the default settings and property values.

### Use Template

Creates a new sheet set based on one of the standard sheet set template files.

A sheet set template contains a standard set of subsets, label and callout blocks, and other property values specific to an
industry and unit of measurement.

### Copy Existing Sheet Set Structure

Creates a new sheet set based on an existing sheet set data (DST) file. The new sheet set contains all the structure, sheet
references, and property values of the specified sheet set data (DST) file.

Click Browse to select a sheet set data (DST) file in Finder.
