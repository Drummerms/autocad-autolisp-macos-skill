# Add Action Parameters to Dynamic Blocks

#### Topics in this section

* [Work With Action Parameters in Blocks](Work-With-Action-Parameters-in-Blocks.md)

  Use grips or the Properties palette to manipulate a block reference that contains action parameters.
* [About Adding Parameters to Dynamic Blocks](About-Adding-Parameters-to-Dynamic-Blocks.md)

  Parameters determine the geometry that will be affected by an action when you manipulate a block reference.
* [About Adding Actions to Dynamic Blocks](About-Adding-Actions-to-Dynamic-Blocks.md)

  Actions define how the geometry of a dynamic block reference will move or change when its grips are manipulated.
* [About Specifying Properties for Dynamic Blocks](About-Specifying-Properties-for-Dynamic-Blocks.md)

  Control the way parameters are displayed using the Properties palette.
