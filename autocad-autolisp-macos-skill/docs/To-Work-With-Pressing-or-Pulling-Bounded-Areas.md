# To Work With Pressing or Pulling Bounded Areas

How to create holes and solid objects by pressing and pulling closed objects and areas.

For best results, set the viewport to an isometric view before doing a press or pull operation.

NOTE:Running object snaps are turned off when using Presspull. However, you can still set object snap overrides to snap to specific
points on objects.

## Push a Hole Into a 3D Solid

![](../images/GUID-AE931FC5-6362-42E4-B1B0-147F5064A541-low.png)

1. Create a circle or other closed shape on the face of a 3D solid object.
2. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Presspull.
   ![](../images/GUID-7BAD4DA5-0A85-410F-A3A4-00F1A7483B7B-low.png)
3. Click inside the closed shape or object and drag the cursor through the interior of the 3D solid object to create the hole.
4. Enter a value or click in the drawing area to indicate the extrusion distance.

## Create a Solid Object by Pressing or Pulling a Closed Object

![](../images/GUID-D5D29F72-81B2-457E-A6BC-7504667CDEF0-low.png)

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Presspull.
   ![](../images/GUID-7BAD4DA5-0A85-410F-A3A4-00F1A7483B7B-low.png)
2. Click a closed object such as a circle and move the cursor to establish an extrusion direction.
3. Enter a value or click in the drawing area to indicate the extrusion distance.

## Create a Solid Object by Pressing or Pulling a Bounded Area

A bounded area is an area that contains no gaps. It can be formed by two or more overlapping objects.

![](../images/GUID-6CE67C08-E489-48D3-A672-EC49CBAEA715-low.png)

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Presspull.
   ![](../images/GUID-7BAD4DA5-0A85-410F-A3A4-00F1A7483B7B-low.png)
2. Click within any closed area that is bounded by coplanar objects or edges.
3. Drag the mouse to establish an extrusion direction.
4. Enter a value or click in the drawing area to indicate the extrusion distance.

## Press or Pull Multiple Areas or Objects

Objects used for a multiple press or pull operation must all support the same type of behavior (extrusion or offset). The
extrusion distance for all objects is based on the change in the last object you select.

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Presspull.
   ![](../images/GUID-7BAD4DA5-0A85-410F-A3A4-00F1A7483B7B-low.png)
2. Do one of the following:
   * Press and hold Shift as you select the object, face, or area.
   * Enter
     *m* (Multiple), select the objects and areas you want to extrude, and press Enter.
3. Enter a value or click in the drawing area to indicate the extrusion distance.

![](../images/GUID-523D410F-09C4-434E-B82E-5B04F48A12A0-low.png)
