# About Creating Network Surfaces

Create non-planar surfaces in the space between edge subobjects, splines, and other 2D and 3D curves.

Network surfaces created with SURFNETWORK are similar to lofted surfaces in that they are created in the space between several
curves in the U and V directions. The curves can be surface or solid edges. When you create the surface you can specify the
tangency and bulge magnitude.
