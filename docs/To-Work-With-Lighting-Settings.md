# To Work With Lighting Settings

## Set a Drawing for Photometric Lighting

1. At the command prompt, enter
    *lightingunits*.
2. Set the value to 1 to enable photometric lights in American lighting units (foot candles).
3. Set the value to 2 to enable photometric lights in International lighting units (lux).

NOTE:Starting with AutoCAD 2017-based products, you can no longer set a drawing to standard lighting.
