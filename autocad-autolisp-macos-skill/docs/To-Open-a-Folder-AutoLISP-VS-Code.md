# To Open a Folder (AutoLISP/VS Code)

NOTE:It is recommended to manage AutoLISP source (LSP) files with AutoLISP projects. For information on using AutoLISP projects,
see
[Managing AutoLISP Files](Managing-AutoLISP-Files-AutoLISP-VS-Code.md).

1. In Visual Studio Code, on the Activity Bar, click Explorer and then click Open Folder (or click File menu > Open Folder/Open).

   ![](../images/GUID-182FA9A4-E0BF-4393-971E-DF8C5E7C6A06-low.png)
2. In the Open Folder dialog box, browse to and select the folder you want to work from.

   If you don't have a folder to work from, create one on your local or network drive.

   NOTE:This folder can, but doesn't necessarily need to contain your AutoLISP source (LSP) files. You might want to consider organizing
   your files in folders to make them easier to access from VS Code.
3. Click Select Folder (or Open on Mac OS).
