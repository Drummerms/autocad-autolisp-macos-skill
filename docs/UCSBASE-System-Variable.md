# UCSBASE (System Variable)

Stores the name of the UCS that defines the origin and orientation of orthographic UCS settings.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | "" |

Valid values include any named UCS.
