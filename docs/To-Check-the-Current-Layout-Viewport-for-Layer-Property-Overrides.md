# To Check the Current Layout Viewport for Layer Property Overrides

This method is useful for drawings that have a large number of layers.

1. In a layout, double-click within a layout viewport to make it the current one.
2. At the Command prompt, enter VPLAYEROVERRIDES.

   If VPLAYEROVERRIDES displays 1, the current layout viewport includes layer property overrides. If it displays 0, no overrides
   have been assigned.
