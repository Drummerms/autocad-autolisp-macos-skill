# GRIPOBJLIMIT (System Variable)

Suppresses the display of grips when the selection set includes more than the specified number of objects.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 100 |

The valid range is 0 to 32,767. For example, when set to 1, grips are suppressed when more than one object is selected. When
set to 0, grips are always displayed.

NOTE:When GRIPBLOCK is set to 1 and the number of selected objects is exceeded, blocks display only their insertion point grip
rather than all the grips for each object within the block.
