# To Start a New Drawing

Do one of the following:

* On the Files tab, click the plus (+) sign. This creates a drawing based on the default drawing template file. If a default
  drawing template file is not specified, a selection dialog box is displayed, from which you can choose a drawing template
  file.
* On the Start tab, click New or the New drop-down menu and select a drawing template.
* Click
  File menu ![](../images/ac.menuaro.gif) New Drawing.
