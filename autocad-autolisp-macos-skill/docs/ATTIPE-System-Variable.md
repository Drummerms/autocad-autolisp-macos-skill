# ATTIPE (System Variable)

Controls if all formatting options are available with the in-place editor for modifying multiline attributes.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Limited formatting options with the in-place editor |
| 1 | Full formatting options with the in-place editor |

Use the limited formatting options with the in-place editor when modifying multiline attributes for best compatibility with
releases prior to AutoCAD 2008.
