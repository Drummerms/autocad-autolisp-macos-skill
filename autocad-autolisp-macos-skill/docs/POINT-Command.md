# POINT (Command)

Creates a point object.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Point drop-down.
![](../images/GUID-CD9AA02A-17DC-468C-A6FB-E0C3DD73DFFE-low.png)

*Menu*:
Draw ![](../images/ac.menuaro.gif) Point.

Points objects can act as nodes to which you can snap objects. You can specify 2D and 3D locations for a point. The current
elevation is assumed if you omit the
*Z* coordinate value. The PDMODE and PDSIZE system variables control the appearance of point objects.

You can use MEASURE and DIVIDE to create points along an object. Use PTYPE to specify point size and styles.

![](../images/GUID-6D61F0EF-098D-4750-9F6B-5AAFA388370A-low.gif)

The following prompt is displayed.

Specify a point
:   Specifies the location of the point object.
