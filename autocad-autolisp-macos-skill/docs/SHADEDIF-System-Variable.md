# SHADEDIF (System Variable)

Sets the ratio of diffuse reflective light to ambient light.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 70 |

The ratio is a percentage of diffuse reflective light when [SHADEDGE](SHADEDGE-System-Variable.md) is set to 0 or 1.
