# Tile- and Attribute-Handling Functions Reference (AutoLISP/DCL)

The following table provides summary descriptions of the DCL tile- and attribute-handling functions.

| Tile- and attribute-handling functions | | Platforms | | | | |
| Windows | | Mac OS | | Web |
| Function name | Description | AutoCAD | AutoCAD LT | AutoCAD | AutoCAD LT | AutoCAD |
| [(action\_tile key action-expression)](action_tile-AutoLISP-DCL.md) | Assigns an action to evaluate when the user selects the specified tile in a dialog box | ✓ | ✓ | ✓ | -- | ✓ |
| [(get\_attr key attribute)](get_attr-AutoLISP-DCL.md) | Retrieves the DCL value of a dialog box attribute | ✓ | ✓ | ✓ | -- | ✓ |
| [(get\_tile key)](get_tile-AutoLISP-DCL.md) | Retrieves the current runtime value of a dialog box tile | ✓ | ✓ | ✓ | -- | ✓ |
