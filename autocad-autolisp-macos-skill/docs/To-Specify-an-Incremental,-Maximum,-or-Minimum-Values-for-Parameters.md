# To Specify an Incremental, Maximum, or Minimum Values for Parameters

This procedure is used only for linear, polar, XY, or rotation parameters in a dynamic block.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block Editor.
   ![](../images/GUID-E3DDBE51-C92A-4D07-A51D-C7C685E4F578-low.png)
2. In the Edit Block Definition dialog box, select the block you want to modify and click Edit Block.
3. On the Block Editor visor,click the icon for linear, polar, XY, or rotation parameter and set the parameter for the block.
4. In the Properties Inspector palette, under Value Set, select Increment in the drop-down list next to one of the following
   items:
   * Dist type
   * Ang Type
   * Hor Type
   * Ver Type

For example, for Linear parameter, select Increment in the Dist type drop-down list. For Rotation parameter, select Increment
in the Ang Type.

5. Enter an incremental value for the parameter next to the Dist, Ang, Hor, or Ver Increment value set.
6. Enter a minimum value for the parameter next to Dist, Ang, Hor, or Ver Minimum.
7. Enter a maximum for the parameter next to Dist, Ang, Hor, or Ver Maximum.
8. Click the Save button in the Block Editor visor.
9. (Optional) If you are finished using the Block Editor, click Close Block Editor.
