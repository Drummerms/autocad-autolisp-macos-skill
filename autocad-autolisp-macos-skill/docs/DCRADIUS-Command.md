# DCRADIUS (Command)

Constrains the radius of a circle or an arc.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Parametric panel ![](../images/ac.menuaro.gif) Dimensional Constraint drop-down ![](../images/ac.menuaro.gif) Radius.
![](../images/GUID-0435E58A-9E43-49DE-9690-F2EB8882BBAD-low.png)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Parametric ![](../images/ac.menuaro.gif) Dimensional Constraints ![](../images/ac.menuaro.gif) Radius.

## Summary

This command is equivalent to the Radius option in
[DIMCONSTRAINT](DIMCONSTRAINT-Command.md).

The following table outlines the valid constraint objects and points:

| Valid Objects or Points | Characteristics |
| Circle  Arc | * Constrains the radius of the circle or arc. |

## List of Prompts

The following prompts are displayed.

Select arc or circle:
*Select an arc or a circle to be constrained*

Dimension Line Location

Determines where the dimension line is located on the constrained object.
