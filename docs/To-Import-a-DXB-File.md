# To Import a DXB File

1. Click Insert ![](../images/ac.menuaro.gif) Drawing Exchange Binary.At the Command prompt, enter DXBIN.
2. In the Select DXB File dialog box, find and select the DXB file you want to import.
3. Click Open.
