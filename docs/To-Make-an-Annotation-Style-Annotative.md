# To Make an Annotation Style Annotative

NOTE:After updating an existing annotation style to being annotative or non-annotative, existing objects that reference those styles
are not automatically updated to reflect the annotative property of the style or definition. Use the ANNOUPDATE command to
update the existing objects to the current annotative properties of the style.

## Text Style

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Text panel ![](../images/ac.menuaro.gif) Style.
   ![](../images/GUID-A1A489F9-EDAC-453E-B777-A21DD0ADC60D-low.png)
2. In the Styles list, click Annotative to select.

## Dimension Style

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Dimension panel ![](../images/ac.menuaro.gif) Dimension Style.
   ![](../images/GUID-89D5BC05-30EE-4627-B80B-8680F08A01A9-low.png)
2. In the Styles list box, select a style.
3. Click Modify under Action pop-up menu. On the Fit tab, under Scale for Dimension Features, click Annotative.

## Multileader Style

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Leader panel ![](../images/ac.menuaro.gif) Multileader Style Manager.
   ![](../images/GUID-35FD07B0-ECA0-4396-A6F4-B3DC82B3998C-low.png)
2. In the Styles list box, select a style.
3. Click Modify under Action pop-up menu. On the Leader Structure tab, under Scale, click Annotative.
