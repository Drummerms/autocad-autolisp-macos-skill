# DIMAUNIT (System Variable)

Sets the units format for angular dimensions.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | Decimal degrees |
| 1 | Degrees/minutes/seconds |
| 2 | Gradians |
| 3 | Radians |
