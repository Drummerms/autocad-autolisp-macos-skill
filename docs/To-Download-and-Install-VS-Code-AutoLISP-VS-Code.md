# To Download and Install VS Code (AutoLISP/VS Code)

1. Launch your Web browser and browse to
   <https://code.visualstudio.com/>.
2. On the Visual Studio Code page, click Download and then choose the download specific to your platform.

   ![](../images/GUID-AF2C7391-970B-468E-A25D-C9F92FDCFD05-low.png)

   NOTE:The AutoCAD AutoLISP Extension is only supported on Windows and Mac OS; make sure to download the 64-bit version of VS Code
   for Windows.
3. After you download the VS Code installer, start the installer and follow the on-screen instructions.
