# BCLOSE (Command)

Closes the Block Editor.

## Access Methods

![](../images/ac.mouse.gif) Toolbar: Block Editor visor ![](../images/ac.menuaro.gif) Close

## Summary

Closes the [Block Editor](Block-Editor.md). If you have modified the block definition since it was last saved, you are prompted to save or discard the changes.
