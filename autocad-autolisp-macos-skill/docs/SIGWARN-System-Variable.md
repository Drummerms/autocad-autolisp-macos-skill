# SIGWARN (System Variable)

Controls whether a warning is presented when a file with an attached digital signature is opened.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

If the system variable is on and you open a file with a valid signature, the digital signature status is displayed. If the
variable is off and you open a file, the digital signature status is displayed only if a signature is invalid.

| 0 | Warning is not presented if a file has a valid signature |
| 1 | Warning is presented |
