# About Applying Dimension Styles

You can modify dimension styles individually or by changing the current settings.

In most cases, a new dimension inherits the current dimension style. To change it, you can change the current dimension style
or set up dimension style overrides.

Some dimension types, such as continued and baseline dimensions use the style of their associated dimensions. To change the
default to use the current dimension style, use DIMCONTINUEMODE.

When you edit the properties of a dimension style, you can choose whether to update dimensions that already use that style.
If necessary, you can restore an existing dimension style or apply the current dimension style, including any dimension style
overrides, to selected dimensions.
