# FADEMARKUP (Command)

Fade individual markups so they are less visible on a trace.

The following prompts are displayed.

## Unfade Markup

Select which markup assist boxes to unfade.

## All

Fade all markup assist boxes.

NOTE:The -FADEMARKUP command provides the same command line options.
