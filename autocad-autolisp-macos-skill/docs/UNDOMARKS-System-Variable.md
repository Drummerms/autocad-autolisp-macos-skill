# UNDOMARKS (System Variable)

Stores the number of marks placed in the UNDO control stream by the Mark option.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 0 |

The Mark and Back options are not available if a group is currently active.
