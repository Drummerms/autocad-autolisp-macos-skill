# To Assign a Plot Style Table to a Layout

1. Click the Model tab or the layout tab to which you want to assign the plot style table.
2. Click
   File ![](../images/ac.menuaro.gif) Plot Styles.
3. Double-click Plot Style in the Plotter dialog box.
4. Double- click the ctb/stb file. In the Plot Style Table Editor, make the necessary settings.
5. Click Save and Close. The Save dialog box opens up. Save the Plot Style in the desired location.
6. Close the Plot Style dialog box.
