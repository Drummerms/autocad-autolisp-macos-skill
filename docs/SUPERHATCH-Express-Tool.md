# SUPERHATCH (Express Tool)

Hatches an area using a selected image, block, xref, or wipeout object.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Hatch panel ![](../images/ac.menuaro.gif) Super Hatch.
![](../images/GUID-580982C1-A03D-404E-8B6D-17994C73CCD2-low.png)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Express Tools ![](../images/ac.menuaro.gif) Super Hatch.

## Super Hatch Options

| Image | Opens a selection dialog box for specifying the image for the superhatch pattern. |
| Block | Opens the Insert Block dialog box for specifying the block for the superhatch pattern. |
| Xref Attach | Opens a selection dialog box for specifying an xref file for the superhatch pattern. |
| Wipeout | Uses a wipeout object for the superhatch pattern. |
| Object | Selects blocks, images and xreferences. |

## Prompts

*For images and Object:*

Insertion point <0,0>:
*Specify an insertion point*

Scale factor <1>:
*Specify a scale factor*

*For blocks and xrefs:*

Insertion point:
*Specify an insertion point*

Placement:
*Specify yes or no*

Specify a window around the block to define column and row tile distances.

Specify block [Extents] First corner <magenta rectang>:
*Specify a tiling (tesselation) frame, and press Enter*

Specify an option [Advanced options] <Internal point>:
*Specify an internal point for hatch boundary, or enter A for advanced options, and press Enter*
