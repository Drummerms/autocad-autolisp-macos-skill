# DRAGP1 (System Variable)

When hardware acceleration is used, controls how many vectors the system draws when you drag objects within a 2D viewport
before it checks for a new input sample from the mouse.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 5000 |

A larger number results in more graphical vectors being displayed as objects are dragged. For best performance, it is recommended
that you use hardware acceleration rather than software acceleration. DRAGP1 has no effect when dragging objects in a 3D viewport.
