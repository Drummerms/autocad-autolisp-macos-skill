# acet-sys-lasterr (AutoLISP/Express Tools)

Reports the last error.

*Supported Platforms:* AutoCAD for Windows only; not available in AutoCAD LT for Windows, or on Mac OS and Web

## Signature

```
(acet-sys-lasterr)
```

No arguments.

## Return Values

*Type:* Integer

Error code of the last error encountered by routines in the acetutil library. See
*winerror.h* for details.
