# SELECTIONEFFECT (System Variable)

Specifies the visual effect used when objects are selected.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | Dashed lines |
| 1 | When hardware acceleration is on, displays glowing line highlighting effect |
