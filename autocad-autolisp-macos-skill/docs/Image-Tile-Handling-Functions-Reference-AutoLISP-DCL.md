# Image Tile-Handling Functions Reference (AutoLISP/DCL)

The following table provides summary descriptions of the DCL image tile-handling functions.

| Image tile-handling functions | | Platforms | | | | |
| Windows | | Mac OS | | Web |
| Function name | Description | AutoCAD | AutoCAD LT | AutoCAD | AutoCAD LT | AutoCAD |
| [(dimx\_tile key)](dimx_tile-AutoLISP-DCL.md) | Retrieves the width of a tile in dialog box units | ✓ | ✓ | ✓ | -- | ✓ |
| [(dimy\_tile key)](dimy_tile-AutoLISP-DCL.md) | Retrieves the height of a tile in dialog box units | ✓ | ✓ | ✓ | -- | ✓ |
| [(end\_image)](end_image-AutoLISP-DCL.md) | Ends creation of the currently active dialog box image | ✓ | ✓ | ✓ | -- | ✓ |
| [(fill\_image x1 y1 wid hgt color)](fill_image-AutoLISP-DCL.md) | Draws a filled rectangle in the currently active dialog box image tile | ✓ | ✓ | ✓ | -- | ✓ |
| [(slide\_image x1 y1 wid hgt sldname)](slide_image-AutoLISP-DCL.md) | Displays an AutoCAD slide in the currently active dialog box image tile | ✓ | ✓ | -- | -- | -- |
| [(start\_image key)](start_image-AutoLISP-DCL.md) | Starts the creation of an image in the dialog box tile | ✓ | ✓ | ✓ | -- | ✓ |
| [(vector\_image x1 y1 x2 y2 color)](vector_image-AutoLISP-DCL.md) | Draws a vector in the currently active dialog box image | ✓ | ✓ | ✓ | -- | ✓ |
