# TRACEPAPERCTL (System Variable)

Controls the opaqueness of the tracing paper effect.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 20 |

| Value | Description |
| 0-50 | Controls opaqueness of tracing paper. The lower the number, the more transparent the tracing paper is. |
