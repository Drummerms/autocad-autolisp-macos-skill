# To Edit Attribute Data

Changes attribute information in a block.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Attribute, Single.
   ![](../images/GUID-91BCBDF8-1AB3-4876-B9D5-4959DE7B6ABD-low.png)
2. Select the block to edit.
3. In the Enhanced Attribute Editor dialog box, retype the attribute information as necessary.

   NOTE:You cannot edit attribute values on locked layers.
