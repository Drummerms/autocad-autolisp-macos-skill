# SELECTIONAREAOPACITY (System Variable)

Controls the transparency of the selection area during window and crossing selection.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 25 |

The valid range is 0 to 100. The lower the setting, the more transparent the area. A value of 100 makes the area opaque. The
[SELECTIONAREA](SELECTIONAREA-System-Variable.md) system variable must be on.
