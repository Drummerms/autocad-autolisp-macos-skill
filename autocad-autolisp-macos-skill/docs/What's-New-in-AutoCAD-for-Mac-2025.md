# What's New in AutoCAD for Mac 2025

Learn about new features in this release of AutoCAD for Mac 2025 release.

## Updates

## 2025.1 Highlights

*Activity Insights:* The palette has a new refresh button, as well as improved search capability. Xrefs are now linked in the palette, and Shared
activities also appear in the palette.

*Multileaders:* The new
[MLEADERLAYER](MLEADERLAYER-System-Variable.md) system variable assigns new multileaders to the specified layer, overriding the current layer.

*Dimensions:* Baseline and continued dimensions automatically inherit the annotation scales of the original dimension.

## 2025 Highlights

* *[Application Home (Start Tab)](Application-Home-Start-Tab.md)*: Replaces the Welcome screen and provides a consistent welcome experience across AutoCAD on macOS and Windows along with
  other Autodesk products.
* *[Activity Insights](Activity-Insights-2.md)*: Provides an understanding of the past actions that you or others have performed with regard to your drawings.
* *[Hatch Improvements](Hatch-Improvements.md)*: The HATCH command now makes drawing hatches easier and more convenient with new and improved options.
* *[Trace Updates](Trace-Updates.md)* - Updates to Trace include an improved toolbar and editing xrefs while in Edit Drawing mode.
* *[Markup Import and Markup Assist](Markup-Import-and-Markup-Assist-2.md)* - Markup Import and Markup Assist continue to improve, making it easier to bring drawing revisions from a markup file into
  a drawing.

## Additional enhancements

*3D Graphics*: In a continued effort to modernize the 3D graphics system, the 3D Wireframe visual style is now supported by Fast Shaded
mode.

*File Open Performance:* Improvements have been made to reduce the amount of time it takes to open 2D drawing files.
