# COUNTERRORCOLOR (System Variable)

Sets the highlighting color on objects that can cause potential errors in a count.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

Valid values are from 1 to 255 (AutoCAD Color Index (ACI) colors).

NOTE:COUNTERRORCOLOR has no effect when the current viewport is set to a 3D visual style.
