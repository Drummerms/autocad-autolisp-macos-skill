# BLOCKMRULIST (System Variable)

Controls the number of most recently used blocks displayed in the Recent pane of the Blocks palette.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 50 |

Valid values include integers from 0 to 100.

TIP:Set this system variable to 0 to clear the blocks displayed in the Recent pane.
