# UPDATEFIELD (Command)

Manually updates fields in selected objects in the drawing.

## Access Methods

*Tool Sets*:
Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/ac.menuaro.gif) Update Fields.
![](../images/GUID-28499997-8E4C-463B-ADAB-58358FC8CAF5-low.png)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Update Fields ![](../images/ac.menuaro.gif) Text.

## Summary

You can update a field manually when you want to see the latest value. With Date fields, you must update them manually, they
never update automatically.
