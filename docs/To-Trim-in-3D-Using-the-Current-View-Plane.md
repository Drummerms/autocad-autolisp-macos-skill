# To Trim in 3D Using the Current View Plane

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Trim.
   ![](../images/GUID-53B8A7B8-0050-44CA-A923-870AF64E125A-low.png)
2. Select the cutting edge for trimming (1).
3. Enter
   *p* (Project).
4. Enter
   *v* (View).
5. Select the object to trim (2).

   ![](../images/GUID-00B5D04B-C8C7-4B40-8F47-6D8129526CA8-low.png)
