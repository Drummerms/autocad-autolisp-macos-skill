# To Turn on and Turn off Running Object Snaps

* Click the Object Snap button on the status bar ( ![](../images/GUID-05E18A1E-3F0F-4117-96A7-137C885BBEF7-low.png) Find) or press the F3 key.

  The status bar is located in the lower right-hand corner of the application window.

NOTE:You can turn off running object snaps temporarily by holding down the F3 key as you work.
