# layout Attribute (DCL)

Specifies the orientation of a slider.

*Supported Platforms:* Windows, Mac OS, and Web

```
layout = position;
```

Possible values are
horizontal or
vertical (default:
horizontal). For horizontal sliders, the value increases from left to right. For vertical sliders, the value increases from bottom to
top.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
