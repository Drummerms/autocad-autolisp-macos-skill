# LIGHTGLYPHDISPLAY (System Variable)

Turns on and off the display of light glyphs.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

When this system variable is set to Off, the glyphs that represent lights in the drawing are not displayed.

A light glyph is a symbolic representation of a point lights and spotlights.

| 0 | Off |
| 1 | On |
