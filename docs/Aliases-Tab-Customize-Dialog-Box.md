# Aliases Tab (Customize Dialog Box)

Manages the available command aliases. A command alias is an abbreviation of a command name, which you can enter at the Command
prompt instead of entering the entire command name.

CUI (Command)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Customize ![](../images/ac.menuaro.gif) Interface (CUI).

![](../images/GUID-1D305E56-1C01-43B6-B824-04B62B46C8D1-low.png)

The following options are displayed.

Command\Alias List
:   Displays the current list of command aliases. Click a column header to sort the list by that column.

Add ![](../images/GUID-9174311E-7DC8-49C8-B39F-991EC3DAD0E2-low.png)
:   Displays the visor where you can assign an alias to a command.

Remove ![](../images/GUID-A140F0E8-0D4A-41B4-8078-48722E2E2CDB-low.png)
:   Removes the selected aliases.

Search
:   Enter text to filter the list of commands that have an assigned alias.

Alias drop-down
:   You can view all command aliases or you can view just the custom aliases.

Load
:   Browse to and load an existing PGP file to import command aliases.

Reveal in Finder
:   Locates the acaduser.pgp file in Finder. This is helpful if you want to copy the file to another machine.

Apply
:   Save and apply the changes made so far in the dialog box.

NOTE:Custom aliases are stored in the acaduser.pgp file. Program defined aliases are stored in the acad.pgp file. It is not recommended
that you edit the acad.pgp file directly.
