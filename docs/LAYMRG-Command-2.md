# LAYMRG (Command)

Merges selected layers into a target layer, removing the previous layers from the drawing.

## Summary

You can reduce the number of layers in a drawing by merging them. Objects on merged layers are moved to the target layer,
and the original layers are purged from the drawing.

If you enter
*laymrg* or
*-laymrg* at the Command prompt,
[options are displayed](LAYMRG-Command.md).
