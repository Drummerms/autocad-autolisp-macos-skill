# DATALINKNOTIFY (System Variable)

Controls the notification for updated or missing data links.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Disables data link update notification. |
| 1 | Enables data link update notification. |
