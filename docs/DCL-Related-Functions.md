# DCL Related Functions

#### Topics in this section

* [Programmable Dialog Box Functions By Functionality (AutoLISP/DCL)](Programmable-Dialog-Box-Functions-By-Functionality-AutoLISP-DCL.md)

  Listing of AutoLISP functions that are used to implement dialog boxes.
* [action\_tile (AutoLISP/DCL)](action_tile-AutoLISP-DCL.md)

  Assigns an action to evaluate when the user selects the specified tile in a dialog box
* [add\_list (AutoLISP/DCL)](add_list-AutoLISP-DCL.md)

  Adds or modifies a string in the currently active dialog box list
* [client\_data\_tile (AutoLISP/DCL)](client_data_tile-AutoLISP-DCL.md)

  Associates application-managed data with a dialog box tile
* [dimx\_tile (AutoLISP/DCL)](dimx_tile-AutoLISP-DCL.md)

  Retrieves the width of a tile in dialog box units
* [dimy\_tile (AutoLISP/DCL)](dimy_tile-AutoLISP-DCL.md)

  Retrieves the height of a tile in dialog box units
* [done\_dialog (AutoLISP/DCL)](done_dialog-AutoLISP-DCL.md)

  Terminates a dialog box
* [end\_image (AutoLISP/DCL)](end_image-AutoLISP-DCL.md)

  Ends creation of the currently active dialog box image
* [end\_list (AutoLISP/DCL)](end_list-AutoLISP-DCL.md)

  Ends processing of the currently active dialog box list
* [fill\_image (AutoLISP/DCL)](fill_image-AutoLISP-DCL.md)

  Draws a filled rectangle in the currently active dialog box image tile
* [get\_attr (AutoLISP/DCL)](get_attr-AutoLISP-DCL.md)

  Retrieves the DCL value of a dialog box attribute
* [get\_tile (AutoLISP/DCL)](get_tile-AutoLISP-DCL.md)

  Retrieves the current runtime value of a dialog box tile
* [load\_dialog (AutoLISP/DCL)](load_dialog-AutoLISP-DCL.md)

  Loads a DCL file
* [mode\_tile (AutoLISP/DCL)](mode_tile-AutoLISP-DCL.md)

  Sets the mode of a dialog box tile
* [new\_dialog (AutoLISP/DCL)](new_dialog-AutoLISP-DCL.md)

  Begins a new dialog box and displays it, and can also specify a default action
* [set\_tile (AutoLISP/DCL)](set_tile-AutoLISP-DCL.md)

  Sets the value of a dialog box tile
* [slide\_image (AutoLISP/DCL)](slide_image-AutoLISP-DCL.md)

  Displays an AutoCAD slide in the currently active dialog box image tile
* [start\_dialog (AutoLISP/DCL)](start_dialog-AutoLISP-DCL.md)

  Displays a dialog box and begins accepting user input
* [start\_image (AutoLISP/DCL)](start_image-AutoLISP-DCL.md)

  Starts the creation of an image in the dialog box tile
* [start\_list (AutoLISP/DCL)](start_list-AutoLISP-DCL.md)

  Starts the processing of a list in the list box or in the pop-up list dialog box tile
* [term\_dialog (AutoLISP/DCL)](term_dialog-AutoLISP-DCL.md)

  Terminates all current dialog boxes as if the user had canceled each of them
* [unload\_dialog (AutoLISP/DCL)](unload_dialog-AutoLISP-DCL.md)

  Unloads a DCL file
* [vector\_image (AutoLISP/DCL)](vector_image-AutoLISP-DCL.md)

  Draws a vector in the currently active dialog box image
