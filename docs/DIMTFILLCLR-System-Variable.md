# DIMTFILLCLR (System Variable)

Sets the color for the text background in dimensions.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

Color numbers are displayed in the Color Palette dialog box. For BYBLOCK, enter 0. For BYLAYER, enter 256.
