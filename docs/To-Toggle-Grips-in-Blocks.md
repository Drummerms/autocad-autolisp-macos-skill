# To Toggle Grips in Blocks

You can specify whether a selected block reference displays a single grip at its insertion point or displays multiple grips
associated with the objects grouped within the block.

1. Right click in the drawing area, and choose Preferences
2. In the Application Preferences dialog box, in the Cursor & Selection tab, select or clear Enable Grips With Blocks.
