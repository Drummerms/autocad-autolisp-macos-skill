# About Plotting Adobe PDF Files

You can create Adobe® Portable Document Format (PDF) files from drawings.

The Adobe® Portable Document Format (PDF) is a standard for electronic information exchange. PDF files can be easily distributed
for viewing and printing in the Adobe Reader available from the Adobe web site without cost. Using PDF files, you can share
drawings with virtually anyone.

PDF files are generated in a vector-based format to maintain precision. Drawings that are converted to PDF can be easily
distributed for viewing and printing in Adobe Acrobat Reader, versions 7 or later.
