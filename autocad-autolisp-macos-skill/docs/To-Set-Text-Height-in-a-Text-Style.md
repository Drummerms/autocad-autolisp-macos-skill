# To Set Text Height in a Text Style

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Text panel ![](../images/ac.menuaro.gif) Style.
   ![](../images/GUID-A1A489F9-EDAC-453E-B777-A21DD0ADC60D-low.png)
2. In the Text Style dialog box, select the text style to modify, and enter the text height (in drawing units) in the Height
   box.
3. To update existing text that uses this text style, click Apply.
4. Click Close.
