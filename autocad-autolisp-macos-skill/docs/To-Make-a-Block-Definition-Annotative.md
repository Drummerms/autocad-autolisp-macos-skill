# To Make a Block Definition Annotative

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block.
   ![](../images/GUID-B2E92148-A285-4542-BEF5-AD10E48A0DE5-low.png)
2. In the Define Block dialog box, Enter a name for the block.
3. Select the object you want to use as a block using Pick Point or Select Objects.
4. Under Block Behavior, click Annotative.
5. Click Create Block.
