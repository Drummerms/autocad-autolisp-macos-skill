# About Hatch Islands

Enclosed areas or text objects inside hatch boundaries are treated as *islands*.

There are four island detection styles available from the user interface:

* Normal island detection
* Outer island detection (recommended)
* Ignore island detection
* No island detection (legacy behavior that is similar to the Ignore style)

The three island detection styles are compared below:

Using Normal island detection, if you specify the internal pick point shown, islands remain unhatched and islands within islands
are hatched.

the text.
