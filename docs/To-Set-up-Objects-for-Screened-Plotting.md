# To Set up Objects for Screened Plotting

1. Click
   Format menu ![](../images/ac.menuaro.gif) Plot Style.
2. In the Edit Plot Style Table, elect a plot style to change. Click Edit.
3. In the Plot Style Table Editor, enter an intensity value between 1 and 100.
4. Select the objects whose plot style you want to change.
5. Click Save & Close.
6. Double-click within the layout viewport that contains the objects whose screening you want to change.
7. Right-click in the drawing area and click Properties.
8. In the Properties Inspector, select the plot style you edited in the Plot Style Table Editor.
9. If you are using color-dependent plot style tables, next to Color, select the color whose plot style you edited in the Plot
   Style Table Editor.
