# To Lock the Scale of a Layout Viewport

When you lock the display of a layout viewport, you can still perform a zoom, but the result of the zoom is applied to the
*entire layout*, leaving the viewport scale in the layout viewport unchanged.

1. Select the border of a layout viewport.
2. Right-click and choose Display Locked![](../images/ac.menuaro.gif)Yes.

NOTE:For non-rectangular layout viewports, select the *viewport* object rather than the associated geometric object.
