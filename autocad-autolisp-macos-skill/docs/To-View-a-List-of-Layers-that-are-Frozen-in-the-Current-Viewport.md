# To View a List of Layers that are Frozen in the Current Viewport

1. Click the layout tab.
2. Double-click within a layout viewport to make it current.
3. Click
   Window menu ![](../images/ac.menuaro.gif) Layers.

   The VP Freeze column shows the Freeze status for layers in the viewports.

TIP:If you want a layer to be frozen automatically in all new viewports, click the icon in the New VP Freeze column for that layer.
