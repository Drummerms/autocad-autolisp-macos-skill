# About Overriding Layer Properties in a Layout Viewport

In a layout, you can assign a set of unique display properties for each layout viewport. These settings override those in
the Model tab.

The following properties can be changed:

* Freeze\thaw
* Color
* Linetype
* Lineweight
* Transparency
* Plot style

In this example, the layer of the hatch object is frozen, and the color of the remaining objects are changed in the layout
viewport.

![](../images/GUID-B48A6787-7B9D-4371-B9DD-55ECCB73DD6B-low.png)

Layer property overrides are applied to the current layout viewport. When you double-click a layout viewport to make it current,
extra columns are shown on the Layer Properties Manager.

![](../images/GUID-866427BF-A28C-4272-846D-BDBED7E3428E-low.png)
