# TRANSPARENCYDISPLAY (System Variable)

Controls whether the object transparency is displayed.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Object transparency is not displayed |
| 1 | Object transparency is displayed |
