# Commands for Regions and Areas

Use these commands for drawing regions and areas.

#### Commands

* [BOUNDARY](GUID-D7A1EA7A-D1FD-4275-9742-BC077B601363.htm "Creates a region or a polyline from an enclosed area.
  		")
* [INTERSECT](GUID-904008EB-D92A-4B69-B79F-6C3A033DB3DF.htm " Creates a 3D solid, surface, or 2D region from overlapping solids, surfaces, or regions.
  		")
* [MASSPROP](GUID-CAA51229-293E-4A0C-BFF3-93226252CF13.htm "Calculates the mass properties of selected 2D regions or 3D solids.
  		")
* [REGION](GUID-8F8D5B88-3E66-450B-B582-A717877DA29A.htm "Converts objects that enclose an area into a 2D region object.
  		")
* [SUBTRACT](GUID-14872FC1-8827-4D3B-978E-20936F9A78E5.htm "Creates as a new object by subtracting one overlapping region or 3D solid from another.
  		")
* [UNION](GUID-C38426A3-B4CA-4788-A6B9-F132DD705CA0.htm "Combines two or more 3D solids, surfaces, or 2D regions into a single, composite 3D solid, surface, or region.
  		")

#### System Variables

* [AREA](GUID-CD078D73-FE99-49A0-8FFA-0D2FFA906167.htm "Stores the last area computed by the AREA command.
  		")
* [DELOBJ](GUID-CB64587F-611E-441E-AB07-14B415BF535F.htm "Controls whether geometry used to create other objects is retained or deleted.
  		")
