# SURFACEASSOCIATIVITYDRAG (System Variable)

Sets the dragging preview behavior of associative surfaces.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Registry |
| Initial value: | 1 |

Controls the preview behavior of surfaces and any associated surfaces when a selected surface is dragged. Disabling the preview
can increase performance.

| 0 | Displays no preview. The display refreshes after dragging is complete. |
| 1 | Previews only the movement of the first associated surface. All other associated surfaces do not move until after dragging is complete. |
| 2 | Previews the movement of all associated surfaces. |
