# DIMSE2 (System Variable)

Suppresses display of the second extension line.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Drawing |
| Initial value: | Off |

| Off | Extension line is not suppressed |
| On | Extension line is suppressed |
