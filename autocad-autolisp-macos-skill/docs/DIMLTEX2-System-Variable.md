# DIMLTEX2 (System Variable)

Sets the linetype of the second extension line.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | "" |

The value is BYLAYER, BYBLOCK, or the name of a linetype.
