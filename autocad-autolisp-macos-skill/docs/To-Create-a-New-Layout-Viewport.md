# To Create a New Layout Viewport

1. Click
   View
   ![](../images/ac.menuaro.gif) Viewports
   ![](../images/ac.menuaro.gif) New Viewports....At the Command prompt, enter VPORTS.
2. In the Viewports dialog box, enter the new name for the viewport.
3. Select the viewports from the Standard Viewports list.
4. Click OK to create the new Viewport.

## Create from Saved Named Views using the View Gallery

1. If not in Model space, select the Model tab in the lower left corner below the drawing area.
2. Click New View on the toolbar.
   ![](../images/GUID-B7C7C4DE-4C10-44D1-891F-C5529C6D3F41-low.png)
3. On the New View dialog box:
   * Enter a view name.
   * Click Define Window and select the area for the view.
   * Click OK.

This saves a view and adds it to the view gallery. The view gallery can be used to create a view on a layout.

4. Select the layout tab to switch to the layout where you want to create the new layout viewport.
5. Click the view gallery on the toolbar.
   ![](../images/GUID-34C2F3B8-1BB5-4257-AC72-8DBBE5B4CD87-low.png)

   The view gallery displays showing any previously saved views.

   ![](../images/GUID-A20FB4DA-74A6-4145-AE73-CCAE60A69C14-low.png)
6. Select the view you want to insert on the layout.
7. Specify a location for the view.
8. To change the view's scale:
   * Select the viewport.
   * Click the center triangle grip.
   * Select a scale from the list.

## Create from the View Gallery

1. Select the layout tab to switch to the layout where you want to create the new layout viewport.
2. Click the view gallery on the toolbar.
   ![](../images/GUID-34C2F3B8-1BB5-4257-AC72-8DBBE5B4CD87-low.png)
3. Click Insert New Model View.

   The drawing switches to model space temporarily so you can define the view.
4. Specify the corners for the view and press Enter.
5. Specify a location for the view.
