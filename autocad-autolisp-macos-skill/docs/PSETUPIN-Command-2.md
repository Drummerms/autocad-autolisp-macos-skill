# -PSETUPIN (Command)

If you enter *-psetupin* at the Command prompt, the following prompts are displayed.

Enter file name: *Enter the name of the drawing file that contains the pagesetup to import*

Enter user defined page setup to import or [?]: *Enter the name of the page setup to import*
