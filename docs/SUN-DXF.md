# SUN (DXF)

The following group codes apply to the sun entity.

| Sun group codes | |
| Group code | Description |
| 100 | Subclass marker (AcDbSun) |
| 90 | Version number |
| 290 | Status |
| 63 | Color |
| 40 | Intensity |
| 291 | Shadows |
| 91 | Julian day |
| 92 | Time (in seconds past midnight) |
| 292 | Daylight savings time |
| 70 | Shadow type  0 = Ray traced shadows  1 = Shadow maps |
| 71 | Shadow map size |
| 280 | Shadow softness |
