# To Add a Callout Block to a Sheet

1. Click
   Window menu ![](../images/ac.menuaro.gif) Sheet Set Manager.
2. In the Sheet Set Manager, open a sheet set.
3. In the Details panel, expand Sheet Set, and next to Callout Blocks, click Manage.
4. In the Callout Blocks dialog box, click the (+) Add button and specify a new block to be added to the list.
