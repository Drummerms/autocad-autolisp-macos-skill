# About Annotative Objects Workflow

This workflow provides an overview of the process in creating annotation objects that automatically resize based on the current
drawing or viewport scale.

1. Create or modify an annotation style.

   Use one of the annotation style managers to create or edit an annotation style.
2. Set an annotation style current.

   Set the annotation style current that you want to use for the new objects.
3. Set an annotation scale current.

   Set the annotation scale current which the annotative object should be displayed at.
4. Create an annotation object.

   Use one of the annotation commands from the user interface or Command prompt.
5. Optional, assign additional annotation scales to annotative object.

   If one or more annotation objects were created with an annotative style, assign additional scales as needed.
6. Reposition the new scale representations of the annotative object.

   After assigning a new scale to each annotative object, reposition the new scale representations as needed.
7. Create or set a layout current.

   Create or set one of the layouts current in the drawing.
8. Create a viewport and set its scale.

   Create a new viewport or select an existing viewport to set its scale to display the objects in model space at the appropriate
   size.
