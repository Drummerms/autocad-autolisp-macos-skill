# To Work With Updating Text Fields

## Update an Individual Field

1. Double-click text.
2. Select the field to update and right-click. Click Update Field.

## Update Multiple Fields

1. Click Tools menu![](../images/ac.menuaro.gif) Update Fields .At the Command prompt, enter UPDATEFIELD.
2. At the Select Objects prompt, select the objects that contain the fields you want to update and press Enter.

   All of the fields in the selected objects are updated.
