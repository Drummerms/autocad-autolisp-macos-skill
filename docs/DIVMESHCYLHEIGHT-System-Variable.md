# DIVMESHCYLHEIGHT (System Variable)

Sets the number of subdivisions between the base and the top of the mesh cylinder.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 3 |

![](../images/GUID-CCBEE90C-8A8A-4EB5-9280-49C226DBDA89-low.png)

This setting affects new mesh cylinders that are created with the [MESH](MESH-Command.md) command.
