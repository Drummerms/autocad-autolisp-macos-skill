# PAGESETUPEDIT (Command)

Edits the page setup settings for the current layout, Model or named.

## Access Methods

*Toolbar*:
![](../images/GUID-13B1323C-F949-429F-8C84-3999E8ACC9EA-low.png)

*File tab*:
![](../images/GUID-C2BA3961-A036-4F07-A055-D7A43119520E-low.png)Quick View. In the Quick View dialog box, right-click a layout and click Edit Page Setup.

*Shortcut menu*: Right-click the drawing area when a named layout is current and click Edit Page Setup.

## Summary

The Page Setup dialog box is displayed.
