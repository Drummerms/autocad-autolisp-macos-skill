# PAGESETUP (Command)

Controls the page layout, plotting device, paper size, and other settings for each new layout.

*Toolbar*:

File

*File tab*:
![](../images/GUID-C2BA3961-A036-4F07-A055-D7A43119520E-low.png)Quick View. In the Quick View dialog box, right-click a layout and click Page Setup Manager.

## Summary

The Page Setup Manager is displayed.

A page setup is a collection of plot device and other settings that determine the appearance and format of your final output.
These settings are stored in the drawing file and can be modified and applied to other layouts.
