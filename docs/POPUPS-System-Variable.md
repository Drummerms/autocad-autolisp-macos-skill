# POPUPS (System Variable)

Displays the status of the currently configured display driver.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 1 |

| 0 | Does not support dialog boxes and the menu bar |
| 1 | Supports these features |
