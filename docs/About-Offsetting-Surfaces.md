# About Offsetting Surfaces

Create a parallel surface or solid by setting an offset distance from a surface.

With SURFOFFSET, you specify the offset distance and whether or not the offset surface maintains associativity with the original
surface. You can also specify the offset distance with a mathematical expression.

When you offset a surface, you can do the following:

* Change the offset direction with the Flip option

  ![](../images/GUID-94C0C094-0A79-4769-929C-2DAA69F82726-low.png)
* Offset in both directions to create two new surfaces

  ![](../images/GUID-AAF9781D-13BD-4DC0-8192-214546B90373-low.png)
* Create a solid between the offset surfaces

  ![](../images/GUID-C3BFCA3B-D0A0-44DA-A0C6-C3A922119AE5-low.png)
* If you are offsetting more than one surface, you can specify whether the offset surfaces remain connected.

  ![](../images/GUID-7CBDCDF3-2F79-419F-BD28-CDA40AAB5AB1-low.png)
* Enter an expression that will constrain the distance of the offset surface to the original surface. This option is only available
  if associativity is on.
