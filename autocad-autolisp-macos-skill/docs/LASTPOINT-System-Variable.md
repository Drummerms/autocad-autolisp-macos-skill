# LASTPOINT (System Variable)

Stores the last point specified, expressed as UCS coordinates for the current space.

|  |  |
| --- | --- |
| Type: | 3D-point |
| Saved in: | Not-saved |
| Initial value: | 0.0000,0.0000,0.0000 |

You can reference the last point specified by entering the @ symbol at a point prompt. This is equivalent to entering @0,0,0.
