# USERI1-5 (System Variable)

Provides storage and retrieval of integer values.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

There are five system variables: USERI1, USERI2, USERI3, USERI4, and USERI5.
