# OBJECTISOLATIONMODE (System Variable)

Controls whether hidden objects remain hidden between drawing sessions.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 0 |

OJBECTISOLATIONMODE controls whether objects that are hidden using the [ISOLATEOBJECTS](ISOLATEOBJECTS-Command.md) or [HIDEOBJECTS](HIDEOBJECTS-Command.md) command remain hidden after saving and reopening the drawing.

| 0 | Isolated/hidden objects are temporary for current drawing session |
| 1 | Isolated/hidden objects persist between drawing sessions |
