# spacer Tile (DCL)

Tile used to add space between other tiles in a dialog definition.

*Supported Platforms:* Windows, Mac OS, and Web

```
: spacer {
   alignment fixed_height fixed_width
   height width
}
```

A spacer is a blank tile. It is used only for layout purposes to affect the size and layout of adjacent tiles. To ensure
consistency with other dialog boxes, use spacer tiles only in special cases, because the PDB feature handles spacing automatically.
See About Adjusting the Layout of Dialog Boxes (AutoLISP).

The
spacer tile has no additional attributes beyond the standard layout attributes.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
