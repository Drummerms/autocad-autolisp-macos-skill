# Dialog Box Opening and Closing Functions Reference (AutoLISP/DCL)

The following table provides summary descriptions of the DCL opening and closing functions.

| Dialog box opening and closing functions | | Platforms | | | | |
| Windows | | Mac OS | | Web |
| Function name | Description | AutoCAD | AutoCAD LT | AutoCAD | AutoCAD LT | AutoCAD |
| [(done\_dialog [status])](done_dialog-AutoLISP-DCL.md) | Terminates a dialog box | ✓ | ✓ | ✓ | -- | ✓ |
| [(load\_dialog dclfile)](load_dialog-AutoLISP-DCL.md) | Loads a DCL file | ✓ | ✓ | ✓ | -- | ✓ |
| [(new\_dialog dlgname dcl\_id [action [screen-pt]])](new_dialog-AutoLISP-DCL.md) | Begins a new dialog box and displays it, and can also specify a default action | ✓ | ✓ | ✓ | -- | ✓ |
| [(start\_dialog)](start_dialog-AutoLISP-DCL.md) | Displays a dialog box and begins accepting user input | ✓ | ✓ | ✓ | -- | ✓ |
| [(term\_dialog)](term_dialog-AutoLISP-DCL.md) | Terminates all current dialog boxes as if the user cancels each of them | ✓ | ✓ | ✓ | -- | ✓ |
| [(unload\_dialog dcl\_id)](unload_dialog-AutoLISP-DCL.md) | Unloads a DCL file | ✓ | ✓ | ✓ | -- | ✓ |
