# MIRROR (Command)

Creates a mirrored copy of selected objects.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Mirror.
![](../images/GUID-D688E1AF-F7AB-4F1D-B29B-93F48BE3CC25-low.png)

*Menu*:
Modify ![](../images/ac.menuaro.gif) Mirror.

You can create objects that represent half of a drawing, select them, and mirror them across a specified line to create the
other half.

![](../images/GUID-02FD32A4-435A-4176-ACA5-86CA5F69B856-low.gif)

NOTE:By default, when you mirror a text object, the direction of the text is not changed. Set the MIRRTEXT system variable to 1
if you do want the text to be reversed.

![](../images/GUID-DFAD04C6-98FB-4C2B-800C-2B4B029F6FC0-low.png)

The following prompts are displayed.

Select objects
:   Use an object selection method to select the objects to be mirrored. Press Enter to finish.

Specify first point, second point of mirror line
:   The two specified points become the endpoints of a line about which the selected objects are mirrored. For mirroring in 3D,
    this line defines a mirroring plane perpendicular to the
    *XY* plane of the user coordinate system (UCS) containing the mirror line.

    ![](../images/GUID-35F5BB8C-A5E2-4CC0-BCA0-8D284E60FD21-low.png)

Erase source objects
:   Determines whether the original objects are erased or retained after mirroring them.

    ![](../images/GUID-6F4DCD60-DCCB-4BE1-88A9-67EA1AA3B2BC-low.png)
