# To Turn Live Sectioning On or Off

1. Select a section plane object. Right-click and choose Active Live Sectioning.

When turned on, the cross sections of 3D objects intersected by the section object are displayed. Live sectioning only works
with objects that were created with SECTIONPLANE.
