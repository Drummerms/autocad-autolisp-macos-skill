# Italian and Spanish

AutoCAD for Mac now supports Italian and Spanish.

You can change the language in the Application Preferences dialog. Restart
AutoCAD for Mac to see the language change.

![](../images/GUID-568A2176-7021-4305-BA07-FE709ED347AD-low.png)
