# TRACEFADECTL (System Variable)

Controls the amount of fading when TRACEMODE is active. The setting affects only the objects not being edited - the host
drawing geometry or Trace geometry.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 40 |

The valid range is from 0 to 90.
