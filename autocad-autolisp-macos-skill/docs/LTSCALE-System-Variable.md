# LTSCALE (System Variable)

Sets the global linetype scale factor.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 1.0000 |

The linetype scale factor cannot equal zero. This system variable has the same name as a command. Use the SETVAR command to
access this system variable.
