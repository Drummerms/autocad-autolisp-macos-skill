# Bind External Definitions Dialog Box

Adds xref-dependent named objects (such as blocks, dimension styles, layers, linetypes, and text styles) to your drawing.

XBIND (Command)

*Menu*:
Modify
![](../images/ac.menuaro.gif) Object
![](../images/ac.menuaro.gif) External Reference
![](../images/ac.menuaro.gif) Bind.

![](../images/GUID-2DBB3219-61D2-49D2-9D20-F781853887EF-low.png)

## List of Options

The following options are displayed.

Xrefs
:   Lists the xrefs currently attached to the drawing. Selecting an xref (double-clicking) displays the named object definitions
    in the attached xref.

Definitions to Bind
:   Lists the xref-dependent named object definitions to bind to the host drawing.

Add
:   Moves the named object definitions selected in the Xrefs list into the Definitions to Bind list.

Remove
:   Moves the xref-dependent named object definition selected in the Definitions to Bind list back to its xref-dependent definition
    table.
