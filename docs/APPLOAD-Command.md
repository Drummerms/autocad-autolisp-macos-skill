# APPLOAD (Command)

Loads and unloads applications and defines which applications to load at startup.

## Access Methods

*Menu*:
Tools
![](../images/ac.menuaro.gif) Load Application.

## Summary

The Startup Suite option loads the specified applications each time the product first starts. Application files can be dragged
from the files list, or from any application with dragging capabilities.

The
[Load/Unload Applications dialog box](Load-Unload-Applications-Dialog-Box.md) is displayed.
