# Get Information from Drawings

Commands used to get information about the current drawing or objects within the current drawing.

Information About the Drawing

| To see... | Use... |
| File properties of the current drawing | DWGPROPS NOTE:Available on Windows only. |
| General drawing settings, including total number of objects | STATUS NOTE:Not available in AutoCAD LT. |
| Amount of time spent in the current drawing file | TIME |
| Whether the drawing was saved with or references data from an Educational License product | STUDENTDRAWING |

Information About Objects in the Drawing

| To see... | Use... |
| Database information for each object in the drawing | DBLIST NOTE:Not available in AutoCAD LT. |
| Property data for selected objects | LIST |
