# CURSORBADGE (System Variable)

Determines whether certain cursor badges are displayed in the drawing area.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 2 |

| 1 | Turns off cursor badges used in selection, inspection, and deletion. It also turns off the badges used in the following commands: COPY, MOVE, ROTATE, SCALE, and ZOOM. |
| 2 | Turns on all cursor badges |
