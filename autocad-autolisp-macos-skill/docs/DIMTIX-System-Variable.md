# DIMTIX (System Variable)

Draws text between extension lines.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Drawing |
| Initial value: | Off |

| Off | For linear and angular dimensions, dimension text is placed inside the extension lines if there is sufficient room. |
| On | Draws dimension text between the extension lines even if it would ordinarily be placed outside those lines. For radius and diameter dimensions, DIMTIX on always forces the dimension text outside the circle or arc. |
