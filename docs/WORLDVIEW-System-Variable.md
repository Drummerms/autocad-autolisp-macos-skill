# WORLDVIEW (System Variable)

Determines whether input to the DVIEW and VPOINT commands is relative to the WCS (default) or the current UCS.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| 0 | UCS remains unchanged |
| 1 | UCS changes to the WCS for the duration of the command; the command input is relative to the current UCS |
