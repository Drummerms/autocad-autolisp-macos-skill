# COLORSCHEME (System Variable)

Stores the current color scheme used by the program.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

NOTE:Obsolete. The COLORSCHEME system variable is functionally identical to the
[COLORTHEME](COLORTHEME-System-Variable.md) system variable.

| 0 | Dark |
| 1 | Light |
