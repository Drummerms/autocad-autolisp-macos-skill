# About Managing Named Plot Styles

You can add, delete, rename, and copy plot styles in a named plot style table using the Plot Style Table Editor.

The Plot Style Table Editor is also used to change plot style settings for both named and color-dependent plot style tables.

NOTE: You cannot delete or edit the NORMAL plot style. Also, you cannot add, delete, copy, or rename plot styles in a named plot
style table if a color mapping table has been attached to the plot style table. A color mapping table associates every plot
style with an ACI color.
