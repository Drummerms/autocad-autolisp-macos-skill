# DIMDISASSOCIATE (Command)

Removes associativity from selected dimensions.

## Summary

DIMDISASSOCIATE filters the selection set to include only associative dimensions that are not on locked layers, and that are
not in a different space than the current space (for example, if model space is active, associative dimensions in paper space
are excluded). DIMDISASSOCIATE then disassociates these dimensions and reports the number of dimensions that are filtered
out and the number that are disassociated.
