# About Using the Clipboard

## Cut Objects

Cutting deletes selected objects from the drawing and stores them on the Clipboard. The objects are now available to be pasted
into other programs.

## Copy Objects

You can use the Clipboard to copy part or all of a drawing into a document created by another application. The objects are
copied in vector format, which retains the high resolution in other applications. The information stored in the Clipboard
can then be pasted in other programs.

NOTE: If the copied object is embedded, updating the original drawing does not update the copy embedded in the other program.

## Paste Objects

Applications use different internal formats to store Clipboard information. When you copy objects to the Clipboard, information
is stored in all available formats. When you paste the Clipboard contents into a drawing, the format that retains the most
information is used.
