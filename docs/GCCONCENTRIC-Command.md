# GCCONCENTRIC (Command)

Constrains two arcs, circles, or ellipses to the same center point.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Parametric panel ![](../images/ac.menuaro.gif) Geometric Constraint drop-down ![](../images/ac.menuaro.gif) Concentric.
![](../images/GUID-852BE7A1-E566-4979-A967-69373627B0AC-low.png)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Parametric ![](../images/ac.menuaro.gif) Geometric Constraints ![](../images/ac.menuaro.gif) Concentric.

## Summary

This command is equivalent to the Concentric option in
[GEOMCONSTRAINT](GEOMCONSTRAINT-Command.md).

Following are the valid constraint objects and points:

* Circle
* Arc
* Polyline arc
* Ellipse

## List of Prompts

The following prompts are displayed.

Select first
[object](GCCONCENTRIC-Command.md):
*Select a circle, arc, or ellipse*

Object

Selects an object to be constrained.

First object
:   Selects the first circle, arc, or ellipse to be constrained.

Second object
:   Selects the second circle, arc, or ellipse to be constrained.
