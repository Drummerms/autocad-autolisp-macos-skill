# To Work With Creating Solid Spheres

## Create a Solid Sphere

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Solid drop-down ![](../images/ac.menuaro.gif) Sphere.
   ![](../images/GUID-82213F83-9B90-4937-A7F6-DB305011BA05-low.png)
2. Specify the center of the sphere.
3. Specify the radius or diameter of the sphere.

## Create a Solid Sphere Defined by Three Points

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Solid drop-down ![](../images/ac.menuaro.gif) Sphere.
   ![](../images/GUID-82213F83-9B90-4937-A7F6-DB305011BA05-low.png)
2. At the Command prompt, enter
   *3p* (Three Points). Specify the first point.
3. Specify the second point.
4. Specify the third point.
