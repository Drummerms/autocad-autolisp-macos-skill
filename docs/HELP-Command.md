# HELP (Command)

Displays Help.

## Access Methods

*Menu*:
Help ![](../images/ac.menuaro.gif) AutoCAD Help

*Command Line*:
*Cmd-/* or
*Fn-F1* (or *'help* for transparent use)

Executing the HELP command while a command is active displays Help for that command. Clicking the Help button in a dialog
box displays Help for that dialog box.

NOTE:Product Help is available online and is not installed with the product by default. Download the offline Help installation
by selecting Download Offline Help from the Help menu or from the online Help home page. If you download and install offline
Help, then offline Help displays by default if access to the internet is not available.
