# About Assigning Fonts to Text Styles

Fonts define the shapes of the text characters that make up each character set. You can use TrueType fonts in addition to
compiled SHX fonts.

A single font can be used by more than one text style. If your company has a standard font, you can modify other text style
settings to create a set of text styles that use this standard font in different ways.
