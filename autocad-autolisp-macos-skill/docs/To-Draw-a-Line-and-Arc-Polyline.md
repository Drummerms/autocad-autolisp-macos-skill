# To Draw a Line and Arc Polyline

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Polyline.
   ![](../images/GUID-30386B3B-BD89-447D-A95A-EFCE56889866-low.png)
2. Specify the start point of the polyline segment.
3. Specify the endpoint of the polyline segment.
   * Switch to Arc mode by entering
     *a* (Arc) at the Command prompt.
   * Return to Line mode by entering
     *L* (Line).
4. Specify additional polyline segments as needed.
5. Press Enter to end, or enter
   *c* to close the polyline.
