# To Customize the Properties Inspector

Set the properties you want to see on the My Properties tab of the Properties Inspector.

1. Select an object with properties you want to include in the My Properties tab of the Properties Inspector.
2. Click My to display the My Properties tab of Properties Inspector.

   ![](../images/GUID-58B64862-CEF5-4612-91EA-EBB930909C6A-low.png)
3. Click Customize.
   ![](../images/GUID-0783C82A-31E8-4DF0-8D49-17682322B712-low.png)

All the properties of the selected object type display.

4. Check the properties that you want to make visible in the My Properties tab and click Customize again.

The selected properties are now visible in the My Properties tab.
