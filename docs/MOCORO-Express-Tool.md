# MOCORO (Express Tool)

Moves, copies, rotates, and scales selected objects with a single command.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Move/Copy/Rotate.
![](../images/GUID-8F38D104-DCF0-4924-86A5-24C1CB813C50-low.png)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Express Tools ![](../images/ac.menuaro.gif) Move/Copy/Rotate.

Use MOCORO as a convenient method that combines the most commonly used modification commands.

## Options

| Move | Moves objects |
| Copy | Makes multiple copies of objects |
| Rotate | Rotates objects |
| Scale | Dynamically scales objects |
| Base | Specifies a new base point for the next modify option |
| Undo | Reverses the previous operation |
