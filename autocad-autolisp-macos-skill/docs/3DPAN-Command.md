# 3DPAN (Command)

When a drawing is in a Perspective view, starts the interactive 3D view and enables you to drag the view horizontally and
vertically.

## Access Methods

*Menu*:
View ![](../images/ac.menuaro.gif) Pan ![](../images/ac.menuaro.gif) Realtime.

*Shortcut menu*: Start any 3D navigation command, right-click in the drawing area, and click Other Navigation Modes ![](../images/ac.menuaro.gif) Pan (9).

## Summary

Moves in the direction that you drag. You can drag the view vertically, horizontally, or diagonally. 3DPAN changes the cursor
to a hand cursor.

You can view your entire drawing or select objects before entering 3DPAN.
