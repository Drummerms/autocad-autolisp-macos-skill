# OPENSHEETSET (Command)

Opens a selected sheet set.

## Access Methods

![](../images/ac.mouse.gif) Menu: File ![](../images/ac.menuaro.gif) Open Sheet Set

![](../images/ac.mouse.gif) Toolbar: Sheet Set Manager ![](../images/ac.menuaro.gif) Sheet Set action button ![](../images/ac.menuaro.gif) Open Sheet Set

## Summary

A
[standard file selection dialog box](Standard-File-Selection-Dialog-Boxes.md)) is displayed.

In this dialog box, you can select a sheet set data (DST) file to load sheet set information into the Sheet Set Manager.
