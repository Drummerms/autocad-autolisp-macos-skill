# PDFFRAME (System Variable)

Determines whether the PDF underlay frame is visible.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

The [FRAME](FRAME-System-Variable.md) system variable overrides the PDFFRAME setting. Use the PDFFRAME system variable after the FRAME system variable to reset
the PDF frame settings.

| 0 | The PDF underlay frame is not displayed or plotted.  The frame temporarily reappears during selection preview or object selection. |
| 1 | Displays and plots the PDF underlay frame. |
| 2 | Displays but does not plot the PDF underlay frame. |
