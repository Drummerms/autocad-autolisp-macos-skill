# To Create a Solid by Offsetting a Surface

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Surface panel ![](../images/ac.menuaro.gif) Surface Creation drop-down ![](../images/ac.menuaro.gif) Surface Offset.
   ![](../images/GUID-8EF587F6-00BA-4CDA-88FE-B51624F160AF-low.png)
2. In the drawing area, select a surface and press Enter. Arrows show the offset direction.
3. Enter
   *s* (Solid).
4. Enter an offset distance.

   A parallel face is created at the distance you specified to form a 3D solid object.
