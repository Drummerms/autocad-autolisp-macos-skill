# To Create a Line Using a Direction and a Distance

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Line.
   ![](../images/GUID-12971FA0-B529-418C-80D0-B2FD66874C09-low.png)
2. Specify the first point and then, move the cursor the same angle as the line that you want to create.

To constrain the cursor, you can turn on Ortho or polar tracking. You can also enter an angle override in the form of <*angle value*.

3. Enter a distance at the prompt.
