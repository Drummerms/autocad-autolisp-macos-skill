# To Save a Block as a Drawing (Block Editor)

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block Editor.
   ![](../images/GUID-E3DDBE51-C92A-4D07-A51D-C7C685E4F578-low.png)
2. Select the block and click Edit Block.
3. Click
   Block Editor visor ![](../images/ac.menuaro.gif) Save drop-down ![](../images/ac.menuaro.gif) Save New.
4. In the Save Block Definition As dialog box, enter a name for the new block definition.
5. Select the Save Block as Drawing File check box. Save Block.
6. In the Browse For Drawing File dialog box, click Save.
