# To Change the Settings for Selecting Similar Objects

1. With no objects selected, enter SELECTSIMILAR.
2. Enter *se* (settings).
3. In the Select Similar Settings dialog box, select the properties that must match for objects to be selected.

   If no properties are selected, objects of the same type (for example, all lines) are selected.
