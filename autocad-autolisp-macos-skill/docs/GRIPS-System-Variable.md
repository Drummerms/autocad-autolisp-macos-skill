# GRIPS (System Variable)

Controls the display of grips on selected objects.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 2 |

| 0 | Hides grips |
| 1 | Displays grips |
| 2 | Displays additional midpoint grips on polyline segments |

To adjust the size of the grips and the effective selection area used by the cursor when you snap to a grip, use [GRIPSIZE](GRIPSIZE-System-Variable.md).
