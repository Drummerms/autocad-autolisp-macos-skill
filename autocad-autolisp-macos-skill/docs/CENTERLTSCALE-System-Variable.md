# CENTERLTSCALE (System Variable)

Sets the linetype scale used by center marks and centerlines.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 1.0000 |

The CENTERLTSCALE system variable applies only to center marks and centerlines created with CENTERMARK and CENTERLINE commands.
This system variable accepts any real number except zero.
