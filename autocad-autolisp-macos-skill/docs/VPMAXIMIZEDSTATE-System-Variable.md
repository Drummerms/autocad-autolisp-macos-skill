# VPMAXIMIZEDSTATE (System Variable)

Indicates whether the viewport is maximized or not.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 0 |

The maximized viewport state is canceled if you start the PLOT command.

| 0 | Not maximized |
| 1 | Maximized |
