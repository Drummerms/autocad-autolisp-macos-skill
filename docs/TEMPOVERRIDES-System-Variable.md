# TEMPOVERRIDES (System Variable)

Turns temporary override keys on and off.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

A temporary override key is a key that you can hold down to temporarily turn on or turn off one of the drawing aids that are
set in the Drafting Settings dialog box; for example, Ortho mode, object snaps, or Polar mode.

| 0 | Off |
| 1 | On |
