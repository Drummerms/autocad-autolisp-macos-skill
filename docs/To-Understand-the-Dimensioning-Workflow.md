# To Understand the Dimensioning Workflow

1. Create a layer designated for dimensions, and make it the *current layer*.
2. Near the bottom-left corner of the application window, click a layout tab.
3. Right-click the layout tab, and click Page Setup Manager. Import a *page setup*, or choose the settings that you want to apply to the layout.
4. If necessary, insert a *title block*.
5. Add *layout viewports*, and adjust their size and location on the layout.
6. Use the Properties Inspector to adjust the scale of the model space objects that display within each layout viewport.
7. If necessary, double-click within each layout viewport and pan the model to display correctly. Double-click the layout to
   exit model space.
8. Click the Dimension menu or enter the dimension command and follow the prompts.
