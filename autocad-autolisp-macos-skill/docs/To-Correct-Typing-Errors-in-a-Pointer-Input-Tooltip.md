# To Correct Typing Errors in a Pointer Input Tooltip

1. When a dynamic input tooltip displays the red error outline, type over the selected value to replace it. You can also use
   the RIGHT ARROW, LEFT ARROW, Backspace, and Delete keys to correct your entry.
2. After you make the correction, press Tab, comma (,), or a left angle bracket (<) to remove the red outline and complete the
   coordinates.
