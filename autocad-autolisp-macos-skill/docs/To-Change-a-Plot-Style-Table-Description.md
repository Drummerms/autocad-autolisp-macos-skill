# To Change a Plot Style Table Description

1. Click
   Format menu ![](../images/ac.menuaro.gif) Plot Style.
2. Select the plot style table you want to change. Click Edit.
3. In the Plot Style Table Editor, Properties tab, enter the new description for the plot style table.
4. Click Save & Close.
