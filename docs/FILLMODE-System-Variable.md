# FILLMODE (System Variable)

Specifies whether hatches and fills, 2D solids, and wide polylines are filled in.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| 0 | Objects are not filled |
| 1 | Objects are filled |
