# DWGTITLED (System Variable)

Indicates whether the current drawing has been named.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 0 |

| 0 | Drawing has not been named |
| 1 | Drawing has been named |
