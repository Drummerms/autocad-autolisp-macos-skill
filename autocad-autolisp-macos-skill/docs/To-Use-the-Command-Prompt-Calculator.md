# To Use the Command Prompt Calculator

Do *one* of the following:

* At the Command prompt, enter CAL. Then, enter a CAL expression.
* At a prompt for a command in progress, enter ‘CAL to start the CAL command transparently. Then, enter a CAL expression to
  calculate a value for that prompt.
