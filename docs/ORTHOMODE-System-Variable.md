# ORTHOMODE (System Variable)

Constrains cursor movement to the perpendicular.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

When ORTHOMODE is turned on, the cursor can move only horizontally or vertically relative to the UCS and the current grid
rotation angle

| 0 | Turns off Ortho mode |
| 1 | Turns on Ortho mode |
