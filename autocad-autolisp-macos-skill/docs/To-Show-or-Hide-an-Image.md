# To Show or Hide an Image

1. Select the image to modify.
2. Right-click in the drawing area. Click Properties.
3. In the Properties Inspector, to show or hide the image, select Yes or No in the Show Image list.
4. To switch the background of the selected image between transparent and opaque, select Yes or No in the Transparency list.
