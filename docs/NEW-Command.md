# NEW (Command)

Creates a new drawing.

## Access Methods

*Toolbar*:
![](../images/GUID-897067F0-FDCF-4E30-8624-7643248620D6-low.png)

*Menu*:
File ![](../images/ac.menuaro.gif) New Drawing.

*Shortcut Menu*: right-click on any File tab.

## Summary

Displays a standard file selection dialog box where you select a drawing template. If the FILEDIA system variable is set
to 0 instead of 1, a
[Command prompt is displayed](NEW-Command-Prompt.md).
