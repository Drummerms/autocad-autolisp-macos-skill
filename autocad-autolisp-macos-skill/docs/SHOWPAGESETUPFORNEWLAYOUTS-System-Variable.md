# SHOWPAGESETUPFORNEWLAYOUTS (System Variable)

Specifies whether the Page Setup Manager is displayed when a new layout is created.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Do not display the Page Setup Manager when a new layout is created |
| 1 | Display the Page setup Manager when a new layout is created |
