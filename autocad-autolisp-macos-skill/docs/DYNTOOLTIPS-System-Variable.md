# DYNTOOLTIPS (System Variable)

Controls which tooltips are affected by tooltip appearance settings.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Only Dynamic Input value fields |
| 1 | All drafting tooltips |
