# To Create a Leader Attached to Block Content at an Angle

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Leader panel ![](../images/ac.menuaro.gif) Multileader Style Manager.
   ![](../images/GUID-35FD07B0-ECA0-4396-A6F4-B3DC82B3998C-low.png)
2. In the Multileader Style Manager, click the plus sign (+).
3. In the Create New Multileader Style dialog, specify a name for the new multileader style. Click Continue.
4. In the Modify Multileader Style dialog, Leader Structure tab, under Landing Settings, uncheck Automatically Include Landing.
5. On the Content tab, next to Multileader Type, choose Block content.
6. Under Block Options, next to Attachment, choose one of the following:
   * Center Extents: Attaches the leader line to the center extent of the block content
   * Insertion Point: Attaches the leader line to the block content from any point you specify
7. Click OK.
8. In the Multileader Style Manager, click Close.
9. Do one of the following:
   * Create a multileader object
   * Apply the new multileader style to an existing multileader object.
