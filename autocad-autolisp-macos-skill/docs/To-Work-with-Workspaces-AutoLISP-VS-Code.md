# To Work with Workspaces (AutoLISP/VS Code)

NOTE:It is recommended to manage AutoLISP source (LSP) files with AutoLISP projects. For information on using AutoLISP projects,
see
[Managing AutoLISP Files](Managing-AutoLISP-Files-AutoLISP-VS-Code.md).

## Save a Workspace

1. In Visual Studio Code, click File menu > Save Workspace As.
2. In the Save Workspace dialog box, browse to and select the folder you want to save to the current workspace.
3. In the File Name text box (or Save As text box on Mac OS), enter a name for the workspace and click Save.

## Open a Workspace

1. In Visual Studio Code, click File menu > Open Workspace.
2. In the Open Workspace dialog box, browse to and select the Workspace (*.code-workspace*) file you want to open.
3. Click Open.

## Add a Folder to a Workspace

1. In Visual Studio Code, on the Activity Bar, click Explorer and then click Add Folder (or click File menu > Add Folder to
   Workspace).

   ![](../images/GUID-C9C6103C-790E-44A0-9FAC-D135722B5F39-low.png)
2. In the Add Folder to Workspace dialog box, browse to and select the folder you want to add to the current workspace.
3. Click Add.
