# CENTERLINE (Command)

Creates centerline geometry associated with selected lines and polylines.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Centerline drop-down ![](../images/ac.menuaro.gif) Centerline.
![](../images/GUID-E4521EDC-0353-4060-9278-EF7387188402-low.png)

*Menu*:
Draw ![](../images/ac.menuaro.gif) Center Line.

The following prompts are displayed.

Select First Line
:   Specifies the first of the two lines required.

Select Second Line
:   Specifies the second of the two lines required.

    ![](../images/GUID-F7DFC100-AF79-4C64-80FE-A704E3D56603-low.png)

Layer
:   Assigns new centerlines to the specified layer, overriding the current layer. Enter
    *Use Current* or "*.*" to use the current layer. ([CENTERLAYER](CENTERLAYER-System-Variable.md) system variable)

NOTE:Annotative scaling and dimension style support is not available for centerlines.
