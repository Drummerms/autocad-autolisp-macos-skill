# To Unload an Unused Linetype

1. Click
   Format menu ![](../images/ac.menuaro.gif) Linetype.
2. In the Linetype Manager dialog box, choose the unused linetype, and click -.

NOTE:

* Several linetypes cannot be unloaded: BYLAYER, BYBLOCK, CONTINUOUS, and any linetypes currently in use.
* Use the PURGE command to remove all unused linetypes from a drawing.
