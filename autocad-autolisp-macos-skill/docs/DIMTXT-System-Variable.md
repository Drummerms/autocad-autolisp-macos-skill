# DIMTXT (System Variable)

Specifies the height of dimension text, unless the current text style has a fixed height.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 0.1800 (imperial) or 2.5000 (metric) |
