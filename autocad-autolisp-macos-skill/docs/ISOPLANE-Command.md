# ISOPLANE (Command)

Specifies the current isometric plane.

## Access Methods

![](../images/ac.keyboard.gif) Command entry:   *'isoplane* for transparent use

## Summary

![](../images/GUID-6FAA2BD9-D74F-42FB-A400-050361124BDF-low.png)

The isometric plane affects the cursor movement keys only when Snap mode is on and the snap style is Isometric. If the snap
style is Isometric, Ortho mode uses the appropriate axis pair even if Snap mode is off. The current isometric plane also determines
the orientation of isometric circles drawn by
[ELLIPSE](ELLIPSE-Command.md).

You can cycle through the isometric planes by pressing Fn-F5 (or F5).

## List of Prompts

The following prompts are displayed.

Enter isometric plane setting [Left/Top/Right] <Top>:
*Enter an option or press* Enter

Left
:   ![](../images/GUID-B90C9313-36CD-404C-BB83-17869CF1EBB5-low.png)

    Selects the left-hand plane, defined by the 90-degree and 150-degree axis pair.

Top
:   ![](../images/GUID-2085E535-CD8E-4C63-BA05-FA81785119AC-low.png)

    Selects the top face of the cube, called the top plane, defined by the 30-degree and 150-degree axis pair.

Right
:   ![](../images/GUID-E3E9EF2E-90B5-4CAB-8BC6-888B4CA20852-low.png)

    Selects the right-hand plane, defined by the 90-degree and 30-degree axis pair.
