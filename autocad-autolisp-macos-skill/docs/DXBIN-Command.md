# DXBIN (Command)

Imports an AutoCAD DXB (drawing interchange binary) file.

## Access Methods

![](../images/ac.mouse.gif) Menu:   Insert
 ![](../images/ac.menuaro.gif)  Drawing Exchange Binary Not available in menus in the current workspace

## Summary

AutoCAD DXB files contain only 2D vectors in binary format with 16-bit integer precision. These vectors are imported as line
objects, and they assume the current layer and object properties.

The Select DXB File dialog box (a
[standard file selection dialog box](Standard-File-Selection-Dialog-Boxes.md)) is displayed. Enter the name of the file to import.
