# SHPNAME (System Variable)

Sets a default shape name that must conform to symbol- naming conventions.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Not-saved |
| Initial value: | "" |

If no default is set, it returns "". Enter a period (.) to set no default.
