# To Work With Curvature Analysis

## Analyze Surface Curvature

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Surface panel ![](../images/ac.menuaro.gif) Analysis drop-down ![](../images/ac.menuaro.gif) Analysis Curvature.
   ![](../images/GUID-58955CC0-1231-41BD-B582-F04CF678A350-low.png)
2. Select a surface and press Enter.

   The curvature coloring display.

## Turn Off the Curvature Color Display

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Surface panel ![](../images/ac.menuaro.gif) Analysis Options.
   ![](../images/GUID-E43E5959-FA91-43AA-B25D-2754135C2B20-low.png)
2. In the Analysis Options dialog box, Curvature tab, click Clear Curvature Analysis.

   The curvature coloring disappears.

## Change the Curvature Analysis Options

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Surface panel ![](../images/ac.menuaro.gif) Analysis Options.
   ![](../images/GUID-E43E5959-FA91-43AA-B25D-2754135C2B20-low.png)
2. In the Analysis Options dialog box, Curvature tab, select the desired options.
