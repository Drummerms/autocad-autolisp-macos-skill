# To Include Text in Custom Linetypes

1. Create a simple linetype, as described in
   [To Add a Simple Linetype to a LIN File](To-Add-a-Simple-Linetype-to-an-LIN-File.md).
2. Add the text character descriptor within the linetype pattern, using the following format:

   ```
   ["text",text_style_name,scale,rotation,xoffset,yoffset]
   ```

   For example,

   ```
   A,.5,-.2,["HW",STANDARD,S=.1,U=0.0,X=-0.1,Y=-.05],-.2
   ```
