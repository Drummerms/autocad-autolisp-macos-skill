# To Imprint a 3D Solid

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Solid Edit - Edge drop-down ![](../images/ac.menuaro.gif) Imprint.
   ![](../images/GUID-C2408DF0-4EB3-4A5C-845C-33F2A8CFFAC0-low.png)
2. Select the 3D solid object.
3. Select an object that you want to imprint. (It must be coplanar to a face on the 3D solid.)
4. Press Enter to retain the original objects, or enter
   *y* (Yes) to delete them.
5. Select additional objects to imprint if needed.
6. Press Enter to complete the command.
