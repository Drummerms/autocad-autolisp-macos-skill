# To Align Objects Between Layout Viewports With MVSETUP

1. At the Command prompt, enter MVSETUP.
2. Enter
   *a*(Align).
3. Select one of the following alignments:
   * *Horizontal**.* Aligns a point in one layout viewport horizontally with a base point in another layout viewport.
   * *Vertical**.* Aligns a point in one layout viewport vertically with a base point in another layout viewport.
   * *Angled**.* Aligns a point in one layout viewport at a specified distance and angle from a base point in another layout viewport.
4. Make sure the layout viewport with the view that is to remain stationary is current. Then specify a base point.
5. Select the layout viewport with the view you want to realign. Then specify an alignment point in that view.
6. For angled alignments only, specify a distance and displacement angle from the base point to the alignment point in the second
   layout viewport.
