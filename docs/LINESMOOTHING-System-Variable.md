# LINESMOOTHING (System Variable)

Controls whether anti-aliasing is applied to 2D objects in 2D wireframe visual styles.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Registry |
| Initial value: | ON |

| Value | Description |
| OFF | Anti-aliasing is turned off |
| ON | Anti-aliasing is turned on |
