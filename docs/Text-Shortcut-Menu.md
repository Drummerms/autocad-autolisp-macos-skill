# Text Shortcut Menu

Displays options available for creating and modifying single-line text.

## List of Options

The following options are displayed.

Opaque Background
:   When checked, makes the background of the editor opaque.

Insert Field
:   Displays the [Insert Field dialog box](GUID-1B6DD22B-10D1-44ED-BAA2-E6D79FE52327.htm#WSC30CD3D5FAA8F6D8B82368FFC2D60192-7FFF), where you can select a field to insert in the text.

Find and Replace
:   Displays the [Find and Replace dialog box](GUID-39B391F5-F74D-4D6F-988D-61B24DEFEE52.htm#WSC30CD3D5FAA8F6D813D93F4FFC2D60E2B-7FB5).

Select All
:   Selects all the text in the single-line text object.

Change Case
:   Changes the case of selected text.
