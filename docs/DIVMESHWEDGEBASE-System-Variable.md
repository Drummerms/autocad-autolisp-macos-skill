# DIVMESHWEDGEBASE (System Variable)

Sets the number of subdivisions between the midpoint of the perimeter of triangular dimension of the mesh wedge.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 3 |

![](../images/GUID-01F63BEA-D4EF-475F-A0AD-9B4F56CB51A7-low.png)

This setting affects new mesh wedges that are created with the [MESH](MESH-Command.md) command.
