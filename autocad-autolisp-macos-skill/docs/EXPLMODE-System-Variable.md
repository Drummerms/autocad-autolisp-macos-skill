# EXPLMODE (System Variable)

Controls whether the EXPLODE command supports nonuniformly scaled (NUS) blocks.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 1 |

| 0 | Does not explode NUS blocks |
| 1 | Explodes NUS blocks |
