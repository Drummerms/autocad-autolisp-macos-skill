# INSNAME (System Variable)

Sets a default block name for the INSERT command.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Not-saved |
| Initial value: | "" |

The name must conform to symbol naming conventions. Returns "" if no default is set. Enter a period (.) to set no default.
