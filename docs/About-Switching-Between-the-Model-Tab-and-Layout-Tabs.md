# About Switching Between the Model Tab and Layout Tabs

You can control how you change between the Model tab and one or more named layout tabs.

By default, a Model tab and several named layout tabs are displayed at the bottom-left corner of the drawing area.

![](../images/GUID-BA034E2E-9482-4F5F-AEB5-A0B132DD0900-low.png)

Click the plus (+) icon to add more Layout tabs.
