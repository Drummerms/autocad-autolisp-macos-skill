# IMPORTUSERSETTINGS (Command)

Imports custom user settings from an export file created previously with EXPORTUSERSETTINGS. Settings can include user profile,
toolbar and toolset customizations, command aliases, plot settings, templates, and the My Properties settings for the Property
palette.

## Access Methods

*Menu*:
Tools ![](../images/ac.menuaro.gif) Import Settings

## Summary

A file selection dialog box is displayed. Browse to and select the export file created with EXPORTUSERSETTINGS.

The Import Settings dialog box is displayed.

![](../images/GUID-6516254F-DDDE-41CF-9D21-87E7615F02D3-low.png)

| Setting | File Type |
| User Profile | .plist |
| CUI | .cui |
| Command Alias | .pgp |
| Plot Files | .stb,.ctb |
| Templates | .dwt, .dst, .dwg |
| My Properties | UserPropertyData.plist |
| Hatch Patterns | .pat |
| Linetypes | .lin |
| Shapes | .shx |
| SHX Fonts | .shx |
