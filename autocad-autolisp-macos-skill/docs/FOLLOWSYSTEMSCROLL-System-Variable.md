# FOLLOWSYSTEMSCROLL (System Variable)

Controls whether the AutoCAD trackpad follows the system scrolling direction during panning or orbiting in 3D.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| Value | Description |
| 0 (OFF) | The trackpad does not follow the system scrolling direction. |
| 1 (ON) | The trackpad follows the system scrolling direction. |
