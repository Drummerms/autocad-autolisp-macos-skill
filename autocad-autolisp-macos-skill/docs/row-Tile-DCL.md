# row Tile (DCL)

Grouping tile used to organize tiles into a row without a border around it.

*Supported Platforms:* Windows, Mac OS, and Web

```
: row {
   alignment children_alignment
   children_fixed_height children_fixed_width
   fixed_height fixed_width height label width
}
```

A row can contain any kind of tile.

A row without a box has no additional attributes beyond the standard layout attributes.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
