# DISPLAYVIEWCUBEIN3D (System Variable)

Controls the display of the ViewCube tool when a 3D visual style is applied to a viewport.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Off. Do not display the ViewCube tool when a 3D visual style is applied |
| 1 | On. Display the ViewCube tool when a 3D visual style is applied |
