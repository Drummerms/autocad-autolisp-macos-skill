# LAYCUR (Command)

Changes the layer property of selected objects to the current layer.

## Access Methods

*Menu*:
Format
![](../images/ac.menuaro.gif) Layer Tools ![](../images/ac.menuaro.gif) Change to Current Layer.

## Summary

If you find objects that were created on the wrong layer, you can quickly change them to be on the current layer.

## List of Prompts

The following prompts are displayed.

Select objects to be changed to the current layer:
*Use an object selection method* *and press*Enter
*when you are finished*

N object(s) changed to layer X (the current layer).
