# To Invert a Clipped Image

1. Set IMAGEFRAME to 1. This displays the clipping boundaries.
2. Select the clipped boundary. You should see the grip at the midpoint on the first edge of the clipped boundary.
3. Click the grip to invert the current x-clip region. The grip now points the other way.
