# To Work With Changing Mesh Object Smoothness

## Increase the Smoothness of a Mesh Object

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Mesh panel ![](../images/ac.menuaro.gif) Smooth More.
   ![](../images/GUID-7909F658-4F28-43C8-BF76-4D7E770F26B6-low.png)
2. Select the mesh objects that you want to modify.

   The smoothness of each object that you select is increased one level.

## Decrease the Smoothness of a Mesh Object

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Mesh panel ![](../images/ac.menuaro.gif) Smooth Less.
   ![](../images/GUID-A8A3647B-4F19-4569-AAF6-E2FC07D797D9-low.png)
2. Select the mesh objects that you want to modify.

   The smoothness of each object that you select is decreased one level.

## Change Mesh Smoothness in the Properties window

1. If the Properties window is not displayed, select any object. Right-click the object and select Properties.
2. Select the mesh objects that you want to modify.
3. In the Properties window, Geometry area, Smoothness box, select a new smoothness level.
