# SSMSTATE (System Variable)

Indicates whether the Sheet Set Manager is open or closed.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | Varies |

| 0 | Closed |
| 1 | Open |
