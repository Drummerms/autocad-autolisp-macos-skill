# To Create a Drawing Set for Publishing

1. Open a drawing. Click Publish.
   ![](../images/GUID-4C887873-7EE8-497C-B87D-0D8DFF393268-low.png)

   In the Publish dialog box, you can modify the list of sheets by doing any of the following as needed:

   * *Add sheets.*In the page setup column, click the Add Sheets in the shortcut menu. In the Select Drawings dialog box, select drawings. Click
     Select to add them to the list of sheets in the Batch Publish dialog box.
   * *Remove sheets.*In the page setup column, click the Remove Sheets in the shortcut menu. In the Select Drawings dialog box, select one or more
     sheets in the list. Click the Remove Sheets button.
   * *Reorder sheets.*To reorder the sheets up or down one position in the list, select a sheet. Click either the Move Sheet Up or Move Sheet Down
     button. Sheets in the drawing set are viewed or plotted in the order shown in the list.
   * *Change page setups.* To change the page setup for a layout, click the page setup name and select another page setup from the list.

     NOTE: Change the page setup for each layout depending on your desired output. Model space page setups can only be applied to model
     space sheets; paper space page setups can only be applied to paper space sheets.
2. When your list of drawing sheets is assembled and set the Output Options to configure the way you want for your drawing set,
3. Click the Publish button.
