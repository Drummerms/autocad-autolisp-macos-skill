# DIMFRAC (System Variable)

Sets the fraction format when DIMLUNIT is set to 4 (Architectural) or 5 (Fractional).

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | Horizontal stacking |
| 1 | Diagonal stacking |
| 2 | Not stacked (for example, 1/2) |
