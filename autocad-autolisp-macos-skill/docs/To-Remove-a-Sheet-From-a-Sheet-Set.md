# To Remove a Sheet From a Sheet Set

1. In the Sheet Set Manager, right-click a sheet in the list.
2. Choose Remove Sheet.

NOTE:Removing sheets disassociates the sheets from the sheet set; it doesn't delete the drawing files or the layouts associated
with the sheets.
