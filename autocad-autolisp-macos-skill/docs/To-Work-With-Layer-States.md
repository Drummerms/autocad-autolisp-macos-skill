# To Work With Layer States

Save, restore, and import layer settings using layer states. Add layers, remove layers, and change layer settings within
a layer state.

Use the Layer States Manager to work with layer states.

1. Click
   Format menu ![](../images/ac.menuaro.gif) Layer States Manager to open the Layer State Manager dialog.

   NOTE:The Layer State Manager is also accessible from the Layers palette.

## Save Layer Settings to a Layer State

To include the viewport overrides in the layer state, double-click in a layout viewport to make it current before opening
the Layer States Manager.

1. Click New.
   ![](../images/GUID-6B850EA9-1427-42F5-8E66-E967AD5445C4-low.png)
2. In the Add New Layer State dialog box, enter a name for the new layer state, or choose a name from the list. You can also
   add a description. Click Add.
3. Click Save & Close.

NOTE:The Save button in the Layer States Manager is used only when you want to overwrite the selected layer state with the current
layer settings.

## Restore Layer Settings from a Layer State

1. Click a layer state.
2. Choose the layer properties and settings to restore.
3. Click
   ![](../images/GUID-370B208A-7720-44F1-9172-CE646CB91807-low.png) and click Set as Current.
4. Click Save & Close.

NOTE:The layer that was current when the layer state was saved is made current. If that layer no longer exists, the current layer
does not change.

## Add Layers to a Layer State

If you create layers after creating the layer state, you may want to add those layers to the layer state.

1. Click the layer state to which you want to add layers.
2. Click
   ![](../images/GUID-370B208A-7720-44F1-9172-CE646CB91807-low.png) and click Edit.
3. In the Edit Layer State dialog box, check the layers you want to add and click OK.
4. Click Save & Close.

## Remove a Layer from a Layer State

1. Click the layer state from which you want to remove a layer.
2. Click
   ![](../images/GUID-370B208A-7720-44F1-9172-CE646CB91807-low.png) and click Edit.
3. In the Edit Layer State dialog box dialog box, clear the check box for the layers that you want to remove and click OK.
4. Click Save & Close.

## Change Layer Settings Within a Layer State

1. Click the layer state you want to change the layer settings for.
2. Click
   ![](../images/GUID-370B208A-7720-44F1-9172-CE646CB91807-low.png) and click Edit.
3. In the Edit Layer State dialog box, change the individual layer settings, and click OK.
4. Click Save & Close.

## Delete a Layer State

1. Click the layer state you want to delete.
2. Click
   ![](../images/GUID-896F25FF-DE8C-4782-A6DB-13775428B0A5-low.png) and confirm.
3. Click Save & Close.
