# RTDISPLAY (System Variable)

Controls the display of raster images during Realtime ZOOM or PAN.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Displays raster image content |
| 1 | Displays outline only |
