# fixed\_height Attribute (DCL)

Specifies if a tile's height is allowed to fill the available space.

*Supported Platforms:* Windows, Mac OS, and Web

```
fixed_height = true-false;
```

If this attribute is
true, the tile does not fill the extra space that becomes available in the layout/alignment process.

Possible values are
true or
false (default:
false).

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
