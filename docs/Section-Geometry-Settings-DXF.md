# Section Geometry Settings (DXF)

The following group codes apply to Section geometry settings.

| Section geometry settings group codes | |
| Group code | Description |
| 90 | Section type |
| 91 | Geometry count |
| 92 | Bitflags |
| 63 | Color data |
| 8 | Layer name |
| 6 | Linetype name |
| 40 | Linetype scale |
| 1 | Plotstyle name |
| 370 | Line weight |
| 70 | Face transparency |
| 71 | Edge transparency |
| 72 | Hatch pattern type |
| 2 | Hatch pattern name |
| 41 | Hatch angle |
| 42 | Hatch scale |
| 43 | Hatch spacing |
| 3 | “SectionGeometrySettingsEnd” data marker |
