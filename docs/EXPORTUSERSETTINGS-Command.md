# EXPORTUSERSETTINGS (Command)

Exports custom user settings for import on another machine. Settings can include user profile, toolbar and toolset customizations,
command aliases, plot settings, templates, and the My Properties settings for the Property palette.

## Access Methods

*Menu*:
Tools ![](../images/ac.menuaro.gif) Export Settings

## Summary

The Export Settings dialog box is displayed.

![](../images/GUID-EA828087-F0BF-4F20-B194-C17E5F1B725D-low.png)

| Setting | File Type |
| User Profile | .plist |
| CUI | .cui |
| Command Alias | .pgp |
| Plot Files | .stb,.ctb |
| Templates | .dwt, .dst, .dwg |
| My Properties | UserPropertyData.plist |
| Hatch Patterns | .pat |
| Linetypes | .lin |
| Shapes | .shx |
| SHX Fonts | .shx |

Enter a file name and a location for the export file.
