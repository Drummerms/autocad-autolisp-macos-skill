# About Adding Text and Blocks to Tables

Table cell data can include text and multiple blocks.

When a table is created, the first cell is highlighted, and you can begin entering text. The row height of the cell increases
to accommodate the number of lines of text. To move to the next cell, press Tab, or use the arrow keys. You can edit cell
text by double-clicking in a selected cell or start entering text to replace the current content of a cell.

Inside the cell, the arrow keys move the cursor. Use the Table and Table Cell visors, Properties Inspector, and shortcut menu
to format text, import text, or make other changes to the text in the cell. Different options are available depending on whether
you click a cell border, click inside the cell, or double-click inside the cell.

When you insert a block into a table cell, either the block can automatically fit to the size of the cell, or the cell can
adjust to accommodate the size of the block. Blocks can be inserted from the Table cell visor. Multiple blocks can be inserted
in a cell. If there is more than one block in a cell, use the Manage Cell Content dialog box to customize the way the cell
content is displayed. You can also delete a block from a cell in the Manage Cell Content dialog box. Click inside the table
cell, then click Manage Cell Content from the expanded Table Cell visor.
