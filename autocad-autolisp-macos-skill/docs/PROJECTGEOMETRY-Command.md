# PROJECTGEOMETRY (Command)

Projects points, lines, or curves onto a 3D solid or surface from different directions.

## Summary

By projecting geometry you can do on of the following:

* Project points or curves onto a 3D solid or surface.
* Project and trim a selected 3D solid or surface.

To trim the surface, set the
[SURFACEAUTOTRIM](SURFACEAUTOTRIM-System-Variable.md) system variable to 1.

You can project any curve (line, arc, circle, ellipse, 2D polyline, 2D spline-fit polyline, 2D curve-fit polyline, 3D polyline,
3D spline-fit polyline, spline, or helix) onto any type of surface or solid.

## List of Prompts

The following prompts are displayed.

Project to UCS
:   Projects the geometry along the positive or negative
    *Z*axis of the current UCS.

    ![](../images/GUID-CC8C325F-F2C3-426E-89AF-8B16C74F33F3-low.gif)

Project to View
:   Projects the geometry based on the current view.

    ![](../images/GUID-43356F98-34F0-4266-A2F5-9C5C7DF587AF-low.gif)

Project to Two Points
:   Projects the geometry along a path between two points.

    ![](../images/GUID-65B40BC3-6C61-421C-A1A8-DCCC857DFF11-low.gif)
