# To Justify Multiline Text

1. Click Windows ![](../images/ac.menuaro.gif) Properties Inspector
2. Select the multiline text object.
3. On the Properties Inspector, select one of the Justify options.
4. Click outside the Properties Inspector.
5. As a workaround, you can also select the Justify options from the Text Editor visor.
