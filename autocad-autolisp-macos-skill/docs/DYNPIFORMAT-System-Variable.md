# DYNPIFORMAT (System Variable)

Controls whether pointer input uses polar or Cartesian format for coordinates.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Registry |
| Initial value: | 0 |

This setting applies only to a second or next point.

| 0 | Polar |
| 1 | Cartesian |

The  [DYNMODE](DYNMODE-System-Variable.md) system variable turns Dynamic Input features on and off.
