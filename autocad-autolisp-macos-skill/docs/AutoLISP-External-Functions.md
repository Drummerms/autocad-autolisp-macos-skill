# AutoLISP External Functions

#### Topics in this section

* [Externally Defined Commands (AutoLISP)](Externally-Defined-Commands-AutoLISP.md)
* [3dsin (AutoLISP/External Function)](3dsin-AutoLISP-External-Function.md)

  Imports a 3D Studio (.3ds) file
* [align (AutoLISP/External Function)](align-AutoLISP-External-Function.md)

  Translates and rotates objects, allowing them to be aligned with other objects
* [cal (AutoLISP/External Function)](cal-AutoLISP-External-Function.md)

  Invokes the on-line geometry calculator and returns the value of the evaluated expression

* [rotate3d (AutoLISP/External Function)](rotate3d-AutoLISP-External-Function.md)

  Rotates an object about an arbitrary 3D axis

  Creates profile images of three-dimensional solids
