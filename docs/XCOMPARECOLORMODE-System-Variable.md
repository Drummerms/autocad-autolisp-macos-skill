# XCOMPARECOLORMODE (System Variable)

Switches the visual effect of objects in the host drawing during an xref comparison.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | Objects in the host drawing are faded during comparison. |
| 1 | Objects in the host drawing are highlighted in white during comparison. |
