# SOLIDCHECK (System Variable)

Turns 3D solid validation on and off for the current session.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 1 |

| 0 | Off |
| 1 | On |
