# Font Not Found Dialog Box

![](../images/GUID-538BFFE9-15BA-4E73-81D0-6491E542CA95-low.png)

When you open a drawing with a missing font file, this dialog box is displayed to allow you to choose a replacement font file
for the style.

## List of Options

The following options are displayed.

Family
:   Lists the font family name for all SHX fonts in the
    *Fonts* folder or all registered TrueType fonts.

Typeface
:   Lists all available Asian-language typeface files when a typeface is missing.

    NOTE:Available only when a style is using a SHX font file.

Browse
:   Displays the Open dialog box (a standard file selection dialog box), in which you can select a different font file to use.
