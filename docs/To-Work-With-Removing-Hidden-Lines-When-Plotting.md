# To Work With Removing Hidden Lines When Plotting

## Remove Hidden Lines When Plotting from a Layout Tab

1. Select a layout viewport.
2. Double-click the viewport border to display the Properties Inspector.
3. On the Properties Inspector, click Shade Plot.
4. In the Shade Plot drop-down list, select Hidden.
5. Click OK to plot the drawing.

   ![](../images/GUID-B6B132D0-8BB3-4B3E-8FFE-779D8A9AC7A7-low.png)

## Remove Hidden Lines When Plotting from the Model Tab

1. Click Plot.
   ![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png)
2. In the Plot dialog box, under 3D Viewports, Shading drop-down list, select Hidden.
3. Click OK to plot the drawing.

   ![](../images/GUID-B6B132D0-8BB3-4B3E-8FFE-779D8A9AC7A7-low.png)
