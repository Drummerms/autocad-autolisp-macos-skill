# aspect\_ratio Attribute (DCL)

Specifies the ratio of the width of the image to its height (width divided by height).

*Supported Platforms:* Windows, Mac OS, and Web

```
aspect_ratio = real;
```

If zero (0.0), the tile is fitted to the size of the image.

Possible values are floating-point values (default: none).

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
