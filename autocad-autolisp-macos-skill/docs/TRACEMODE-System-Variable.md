# TRACEMODE (System Variable)

Indicates whether Trace is active and which mode is current - editing or viewing.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 0 |

| Value | Description |
| 0 | Trace mode is not active |
| 1 | Viewing trace (Host drawing space current) |
| 2 | Editing trace (Trace space current) |

NOTE:You can view traces in the desktop, web, or mobile apps. You can only create or edit traces while using web and mobile.
