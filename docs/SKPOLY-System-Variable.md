# SKPOLY (System Variable)

Determines whether the SKETCH command generates lines, polylines, or splines.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | Generates lines |
| 1 | Generates polylines |
| 2 | Generates splines |
