# PLACEMENTMODE (System Variable)

Controls whether block auto-placement is active and what commands it supports.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | Disable auto-placement suggestions |
| 1 | Enable auto-placement suggestions |
