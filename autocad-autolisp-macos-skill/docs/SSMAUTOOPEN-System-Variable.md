# SSMAUTOOPEN (System Variable)

Controls the display behavior of the Sheet Set Manager when a drawing associated with a sheet is opened.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 1 |

| 0 | Does not open the Sheet Set Manager automatically |
| 1 | Opens the Sheet Set Manager automatically |

SSMAUTOOPEN and
[SSLOCATE](SSLOCATE-System-Variable.md) must both be set to 1 to open a sheet set automatically in the Sheet Set Manager.
