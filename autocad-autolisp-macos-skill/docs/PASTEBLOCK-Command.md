# PASTEBLOCK (Command)

Pastes objects from the Clipboard into the current drawing as a block.

## Access Methods

Button

![](../images/ac.mouse.gif) Menu:   Edit
 ![](../images/ac.menuaro.gif)  Paste as Block Not available in menus in the current workspace

Shortcut menu: With no command active, right-click in the drawing area and click Clipboard ![](../images/ac.menuaro.gif) Paste as Block

## Summary

The objects copied to the Clipboard are pasted into the drawing as a block at the specified insertion point. The block is
given a random name.
