# PFACEVMAX (System Variable)

Sets the maximum number of vertices per face.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 4 |

This system variable affects only legacy polyface meshes such as those created by [PFACE](PFACE-Command.md).
