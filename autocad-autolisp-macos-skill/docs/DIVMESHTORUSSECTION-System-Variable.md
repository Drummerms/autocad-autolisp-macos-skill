# DIVMESHTORUSSECTION (System Variable)

Sets the number of subdivisions in the profile that sweeps the path of a mesh torus.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 8 |

![](../images/GUID-1C9FA8A7-3CFE-4DBF-B925-FE9B8AA0A008-low.png)

This setting affects new mesh tori that are created with the [MESH](MESH-Command.md) command.
