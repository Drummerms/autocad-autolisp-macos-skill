# To Change the Visibility of Objects in the Current Visibility State

The current visibility state determines what objects are visible in the Block Editor. You can select objects that you want
to show or hide as you work in the Block Editor.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block Editor.
   ![](../images/GUID-E3DDBE51-C92A-4D07-A51D-C7C685E4F578-low.png)
2. In the Edit Block Definition dialog box, select the block you want to modify and click Edit Block.
3. In the Block Editor visor, do one of the following:
   * Click
     Make Visible.
     ![](../images/GUID-3FBDAB3D-ED36-40BC-83F1-E5A7FCEA1257-low.png)

     The objects that are currently hidden are displayed in a dimmed state.
   * Click
     Make Invisible.
     ![](../images/GUID-899A866A-E971-4C8D-8CD1-F1558BDCFCDE-low.png)
4. In the drawing area, select the objects whose visibility you want to change and press Enter.
