# REDRAWALL (Command)

Refreshes the display in all viewports.

## Access Methods

![](../images/ac.keyboard.gif) Command entry:  *'redrawall* for transparent use

## Summary

Removes temporary graphics left by some operations in all viewports. To remove stray pixels, use the [REGENALL](REGENALL-Command.md)  command.
