# LOFTNORMALS (System Variable)

Controls the normals of a lofted object where it passes through cross sections.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

This setting is ignored when specifying a path or guide curves.

| 0 | Ruled |
| 1 | Smooth Fit |
| 2 | Start cross section |
| 3 | End cross section |
| 4 | Start and End cross sections |
| 5 | All cross sections |
| 6 | Use draft angle and magnitude |
