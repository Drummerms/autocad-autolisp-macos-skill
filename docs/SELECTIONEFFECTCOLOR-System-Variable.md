# SELECTIONEFFECTCOLOR (System Variable)

Sets the color of the glowing highlighting effect on object selection.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

Valid values are from 1 to 255 (AutoCAD Color Index (ACI) colors).

NOTE:The SELECTIONEFFECTCOLOR system variable only applies when SELECTIONEFFECT is set to 1.
