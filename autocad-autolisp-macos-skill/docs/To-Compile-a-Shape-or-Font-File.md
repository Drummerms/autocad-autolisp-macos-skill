# To Compile a Shape or Font File

1. At the Command prompt, enter *compile*.
2. In the Select Shape or Font File dialog box, select a shape definition file (SHP) or PostScript font file (PFB). Click Open.

   After you select the file name, compiling begins. If the program finds an error in the shape descriptions, a message is displayed
   telling you the type of error and the line number the error occurs on. When compiling is complete, the following messages
   are displayed:

   Compilation successful.

   Output file *name.shx* contains *nnn* bytes.
