# HATCHTOBACK (Command)

Sets the draw order for all hatches in the drawing to be behind all other objects.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Draw Order drop-down ![](../images/ac.menuaro.gif) Hatch to Back.
![](../images/GUID-4CFA1CC5-306A-469A-AF98-119FB50D90E6-low.png)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Draw Order ![](../images/ac.menuaro.gif) Send Hatches to Back.

Selects all hatches in the drawing, including hatch patterns, solid fills, and gradient fills, and sets their draw order
to be behind all other objects. Hatch objects on locked layers are also modified.
