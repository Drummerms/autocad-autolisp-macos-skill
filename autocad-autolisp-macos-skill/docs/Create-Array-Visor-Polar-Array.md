# Create Array Visor - Polar Array

Modifies the properties of the polar array as it is being created.

![](../images/GUID-27F94150-0BA9-4824-9BD0-FC60FCC4F379-low.png)

## Summary

The Create Array visor is displayed after selecting the objects to array and the center point or axis of rotation in which
the objects are arrayed around while using the Polar option of the ARRAY command or the ARRAYPOLAR command.

## List of Options

The following options are displayed.

Base Point

Specifies a base point for the array.

Items

Specifies the number of items in the array.

Fill Angle

Specifies the angle between the first and last item in the array using a value or expression.

Save Changes

Saves the changes to the array and exits the command.
