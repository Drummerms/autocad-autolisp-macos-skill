# REBUILD2DDEGREE (System Variable)

Sets the global degree when rebuilding a spline.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 3 |

Valid values are 1 to 11.

![](../images/GUID-24D1D0FF-274D-4B7E-9AB4-70B8DAFC4E57-low.png)

A curve with a degree of 1 is a straight line and a curve with a degree of 2 has one bend. A Bezier curve has a degree of
3. Higher degrees create more complex curves, but require computations.
