# PLOTSTAMP (Command)

Places a plot stamp on a specified corner of each drawing and logs it to a file.

## Summary

The
[Plot Stamp Settings dialog box](Plot-Stamp-Settings-Dialog-Box.md) is displayed.

If you enter
*-plotstamp* at the Command prompt,
[options are displayed](PLOTSTAMP-Command-2.md).
