# HPQUICKPREVTIMEOUT (System Variable)

Sets the maximum time for a hatch preview to generate before the preview is automatically cancelled.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 2 |

If the specified duration is reached, and the preview is not complete, the preview is cancelled, but the command remains active.

The duration is measured in seconds. Valid values range from 1 to 300.

NOTE:The
[HPQUICKPREVIEW](HPQUICKPREVIEW-System-Variable.md) system variable controls whether a preview is generated when using the
[HATCH](HATCH-Command.md) command.
