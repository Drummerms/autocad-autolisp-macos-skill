# About User Interface Customization on Mac OS

The user interface includes commands, menus, tool sets, and a toolbar. Methods to customize each element varies, depending
on the element.

## Commands and Menus

From the CUI editor's Customize dialog box , you can:

* Create new custom commands or modify existing commands
* Assign commands to various menus
* Add or change menus that are displayed on the Mac OS menu bar

## Tool Sets

Tool sets are customized using the buttons provided on the tool sets. You can:

* Display or hide tools on a standard panel
* Create, edit, or remove custom panels
* Change the panel order
* Collapse or expand panels

## Toolbar

Use the toolbar's context menu to customize the toolbar, selecting from the sets provided.
