# To Extract Attribute Information

1. At the Command prompt, enter *-attext*.
2. Select the appropriate file format: CDF, SDF, DXF, Objects in the command prompt.
3. Selects objects whose attributes you want to extract.

   You can select one or more blocks in the drawing.
4. In the Select Template File dialog box, enter the name of an existing attribute extraction template file.
5. In the Create Extract File dialog box, enter the name for the output file. The extract file's file name extension is .txt
   for CDF or SDF format. The extract file's file name extension is .dxx for DXF format.
