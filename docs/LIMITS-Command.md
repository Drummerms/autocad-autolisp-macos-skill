# LIMITS (Command)

Sets and controls the limits of the grid display in the current Model or named layout.

## Access Methods

![](../images/ac.mouse.gif) Menu:   Format  ![](../images/ac.menuaro.gif)  Drawing Limits Not available in menus in the current workspace

![](../images/ac.keyboard.gif) Command entry:  *'limits* for transparent use

## List of Prompts

The following prompts are displayed.

Specify [lower left corner](LIMITS-Command.md) or [[ON](LIMITS-Command.md)/[OFF](LIMITS-Command.md)] <*current*>: *Specify a point, enter* *on* *or* *off**, or press* Enter

Lower-Left Corner
:   Specifies the lower-left corner for the grid limits.

    ![](../images/GUID-5F60293F-E01C-432F-B6CB-8DC1E4D3C9E8-low.png)

On
:   Turns on limits checking. When limits checking is on, you cannot enter points outside the grid limits. Because limits checking
    tests only points that you enter, portions of objects such as circles can extend outside the grid limits.

Off
:   Turns off limits checking but maintains the current values for the next time you turn on limits checking.
