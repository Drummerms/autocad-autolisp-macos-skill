# DIMTMOVE (System Variable)

Sets dimension text movement rules.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | Moves the dimension line with dimension text |
| 1 | Adds a leader when dimension text is moved |
| 2 | Allows text to be moved freely without a leader |
