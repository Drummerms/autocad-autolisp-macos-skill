# About Creating a Solid or Surface by Sweeping

Create a 3D solid or surface by sweeping a profile along a path.

The SWEEP command creates a solid or surface by extending a profile shape (the object that is swept) along a specified path.
When you sweep a profile along a path, the profile is moved and aligned normal (perpendicular) to the path. Open profiles
create surfaces and closed curves create solids or surfaces.

You can sweep more than one profile object along a path.

## Options for Sweeping

When you extrude objects, you can specify any of the following options:

* *Mode.* Sets whether the sweep creates a surface or a solid.
* *Alignment.* If the profile is not on the same plane as the sweep path, specify how the profile aligns with the sweep path.

  ![](../images/GUID-60F3695F-29FB-4F66-99DE-148519BDE0C9-low.png)
* *Base Point.* Specify a base point on the profile to sweep along the profile.
* *Scale.* Specify a value that will change the size of the object from the beginning of the sweep to the end. Enter a mathematical
  expression to constrain the object scaling.

  ![](../images/GUID-4624190E-40DD-475B-BE9B-57E54A0B0CD7-low.png)
* *Twist.* By entering a twist angle, the object rotates along the length of the profile. Enter a mathematical expression to constrain
  the object ‘s twist angle.

  ![](../images/GUID-E63C841B-C927-412C-8979-F4E72AA09E3A-low.png)
