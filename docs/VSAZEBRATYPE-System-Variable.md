# VSAZEBRATYPE (System Variable)

Sets the type of zebra display when using zebra analysis (ANALYSISZEBRA).

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| 0 | Chrome Ball |
| 1 | Cylinder |
