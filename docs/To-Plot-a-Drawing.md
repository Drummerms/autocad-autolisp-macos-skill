# To Plot a Drawing

1. Click Plot.
   ![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png)

   ![](../images/GUID-8F40613C-B96D-4B7F-A2AB-6548E583647C-low.png)
2. Select a plotter.
3. Select paper size, plot area, plot scale, orientation and other options.

   NOTE:If you frequently plot using the same settings, select Default to Previous Plot to save the current plot settings and automatically
   use them on your next plot.
4. Click Plot.
