# CENTERMARKEXE (System Variable)

Determines whether centerlines extend automatically from new center marks.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Drawing |
| Initial value: | 1 |

The CENTERMARKEXE system variable applies only to center marks created using the CENTERMARK command.

| Value | Description |
| 0 | Hides the centerlines extending from a center mark |
| 1 | Displays the centerlines extending from a center mark |
