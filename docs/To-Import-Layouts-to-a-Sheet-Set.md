# To Import Layouts to a Sheet Set

1. In the Sheet Set Manager, right-click a sheet set or a subset.
2. Choose Import Layout as Sheet.
3. Browse to the desired location and select one or more drawings that you want to use.

   To select several drawings, use Shift or Ctrl when you click on the drawing files.
4. Click Open.
