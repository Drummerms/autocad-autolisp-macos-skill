# BREAKATPOINT (Command)

Breaks the selected object into two objects at a specified point.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Break drop-down ![](../images/ac.menuaro.gif) Break at Point.
![](../images/GUID-6FD61209-7835-4DA3-9F54-EE2F3ADB87D5-low.png)

You can break an open, 2D object into two objects at a specified point on the object.

![](../images/GUID-5C31638D-5A81-4896-B77A-F62DE794D1CC-low.gif)

If the point is off of the object, it's automatically projected on to the object. Valid objects include lines, arcs, and
open polylines. Closed objects such as circles cannot be broken at a single point.

If you want to break an object to include a gap, use the BREAK command specifying the gap with two points.
