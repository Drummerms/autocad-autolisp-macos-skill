# About Section Object States

## Section Object States

![](../images/GUID-8E745192-525F-4A6D-9324-86D6B286AE67-low.png)

Section objects have the following display states:

* *Section Plane.* The section line and transparent section plane indicator are displayed. The cutting plane extends infinitely in all directions.
* *Section Boundary.*A 2D box shows the
  *XY*extents of the cutting plane. The cutting plane along the
  *Z* axis extends infinitely.
* *Section Volume.* A 3D box shows the extents of the cutting plane in all directions.
* *Section Slice.* A 2D box shows the depth along the direction of the cutting plane.

You can switch between object states by clicking the menu grip that is displayed when you select the section object, or by
selecting from the shortcut menu.
