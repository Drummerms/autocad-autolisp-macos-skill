# CMLJUST (System Variable)

Specifies multiline justification.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | Top |
| 1 | Zero (Middle) |
| 2 | Bottom |
