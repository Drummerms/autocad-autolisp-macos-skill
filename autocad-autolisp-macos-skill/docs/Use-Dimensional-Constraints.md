# Use Dimensional Constraints

#### Topics in this section

* [About Dimensional Constraints](About-Dimensional-Constraints.md)
* [About Applying Dimensional Constraints](About-Applying-Dimensional-Constraints.md)

  Dimensional constraints maintain specified distances and angles between geometric objects or points on objects.
* [About Controlling the Display of Dimensional Constraints](About-Controlling-the-Display-of-Dimensional-Constraints.md)

  You can display or hide dynamic and annotational constraints within a drawing.
* [About Modifying Objects With Dimensional Constraints Applied](About-Modifying-Objects-With-Dimensional-Constraints-Applied.md)

  You can control lengths, distances, and angles of objects by changing constraint values, by manipulating dimensional constraints
  using grips, or by changing user variables or expressions associated with dimensional constraints.
