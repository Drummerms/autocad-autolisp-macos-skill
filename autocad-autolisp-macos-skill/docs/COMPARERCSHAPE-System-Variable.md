# COMPARERCSHAPE (System Variable)

Controls whether nearby individual changes are merged as a single large rectangle or a series of smaller rectangles in the
compare result drawing.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| Value | Description |
| 0 | Mark up differences with a single rectangular revision cloud |
| 1 | Mark up differences with several smaller rectangular revision clouds |
