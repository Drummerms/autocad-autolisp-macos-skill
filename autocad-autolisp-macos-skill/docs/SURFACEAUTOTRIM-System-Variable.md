# SURFACEAUTOTRIM (System Variable)

Controls whether surfaces are automatically trimmed when you project geometry onto them.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Surfaces are not trimmed when you project geometry onto them. |
| 1 | Surfaces are automatically trimmed to geometry that is projected using [PROJECTGEOMETRY](PROJECTGEOMETRY-Command.md). |
