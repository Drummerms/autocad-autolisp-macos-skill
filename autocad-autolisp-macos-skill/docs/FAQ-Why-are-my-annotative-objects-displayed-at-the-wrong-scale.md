# FAQ: Why are my annotative objects displayed at the wrong scale?

The display of annotative objects is controlled by the current annotation scale of model space and the Annotation Scale property
of a viewport. If you are in model space, make sure the Annotation Scale drop-down list on the status bar ( ![](../images/GUID-14C55011-FC01-4F6C-AD03-C7BC1EEE76A1-low.png) Find)shows the scale you want annotative objects to be displayed at.

For layout viewports, select the viewport and, on the status bar, click Viewport Scale Sync ( ![](../images/GUID-CA72EE34-6FD2-4A0F-BBAF-677CA1576550-low.png) Find). The Viewport Scale property and the Annotation Scale property for the viewport take the same value.
