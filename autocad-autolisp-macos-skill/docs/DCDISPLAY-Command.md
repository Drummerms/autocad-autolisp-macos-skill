# DCDISPLAY (Command)

Displays or hides the dynamic constraints associated with a selection set of objects.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Parametric panel ![](../images/ac.menuaro.gif) Show Dynamic Constraints.
![](../images/GUID-C87AB270-7C32-4039-97F9-BD9097576886-low.png)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Parametric ![](../images/ac.menuaro.gif) Dynamic Dimensions ![](../images/ac.menuaro.gif) Select Objects.

## List of Prompts

The following prompts are displayed.

Select objects:
*Select objects to display or hide the dimensional constraints*

Enter an option [[Show](DCDISPLAY-Command.md)/[Hide](DCDISPLAY-Command.md)] <Show>:
*Enter the appropriate value to show or hide dynamic constraints for the selection set of objects*

Show

Displays the dynamic dimensional constraints for any selection set of objects.

Hide

Hides the dynamic dimensional constraints for any selection set of objects.
