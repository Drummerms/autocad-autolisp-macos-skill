# To Turn On or Off the Display of Text

When you turn on Quick Text mode in drawings that contain a lot of text using complex fonts, only a rectangular frame defining
the text is displayed or plotted.

1. Set Qtext to 0 to turn off the Quick text mode. This displays the characters.
2. Set Qtext to 1 to turn on the Quick text mode. This displays a text boundary frame only in place of text.
3. At the Command prompt, enter *regen* to regenerate the drawing, updating the display.
