# DIVMESHPYRLENGTH (System Variable)

Sets the number of subdivisions along each dimension of a mesh pyramid base.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 3 |

![](../images/GUID-A5805296-8BA3-47C1-99FB-FDFA5E328555-low.png)

This setting affects new mesh pyramids that are created with the [MESH](MESH-Command.md) command.
