# SUPPRESSALERTS (System Variable)

Controls alerts about potential data loss when opening and saving newer drawings in older versions of the product.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 0 |

By default, an alert displays when a drawing is opened from a more recent version of the product that contains data that might
be lost if the drawing is edited and saved.

| 0 | The alert is displayed |
| 1 | The alert is suppressed, which might be desirable when running scripts |
