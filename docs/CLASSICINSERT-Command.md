# CLASSICINSERT (Command)

Inserts a block or drawing into the current drawing using the classic version of the INSERT command.

The classic Insert Block dialog box is displayed.

If you enter -INSERT at the Command prompt, options are displayed.

A good practice is to insert a block from a block library. A block library can be a drawing file that stores related block
definitions or it can be a folder that contains related drawing files, each of which can be inserted as a block. With either
method, blocks are standardized and accessible to multiple users.

NOTE:Use the INSERT command to display the Blocks palette.
