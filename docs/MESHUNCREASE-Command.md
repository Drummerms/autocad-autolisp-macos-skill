# MESHUNCREASE (Command)

Removes the crease from selected mesh faces, edges, or vertices.

## Access Methods

*Tool Set*:
Modeling tab ![](../images/ac.menuaro.gif) Mesh panel ![](../images/ac.menuaro.gif) Mesh Crease drop-down ![](../images/ac.menuaro.gif) Remove Crease.
![](../images/GUID-8225CFF1-841A-4ECA-8A93-820278413F29-low.png)

*Menu*:
Modify
![](../images/ac.menuaro.gif) Mesh Editing
![](../images/ac.menuaro.gif) Uncrease.

## Summary

Restore smoothness to an edge that has been creased.

![](../images/GUID-992F9F34-F87D-4E2C-8CF3-5C1844EB9B66-low.gif)

You can also remove a selected crease in the Crease area of
[Properties Inspector](Properties-Inspector.md) by changing the Type value to None when a face on a mesh is selected.

## List of Prompts

The following prompt is displayed:

Select crease to remove
:   Specifies which creased edges to smooth. You do not need to press Ctrl to select a creased subobject.
