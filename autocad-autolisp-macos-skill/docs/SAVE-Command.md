# SAVE (Command)

Saves the drawing under the current file name or a specified name.

Obsolete. The SAVE command is functionally identical to the
[QSAVE](QSAVE-Command.md) command. The Save option on the File menu is QSAVE.

## Summary

The Save Drawing As dialog box (a
[standard file selection dialog box](Standard-File-Selection-Dialog-Boxes.md)) is displayed. Save the drawing under the current file name, or enter a different file name to save a copy of the drawing
under that name.

If the drawing is read-only, use the
[SAVEAS](SAVEAS-Command.md) command to save changes under a different file name.
