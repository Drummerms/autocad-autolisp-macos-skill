# SPLKNOTS (System Variable)

Stores the default knot option for the SPLINE command when specifying fit points.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 0 |

| 0 | Chord distance |
| 1 | Square root of chord distance |
| 2 | Uniform distance |

NOTE:SPLKNOTS always defaults to 0 when AutoCAD 2026 starts.
