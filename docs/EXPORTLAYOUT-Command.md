# EXPORTLAYOUT (Command)

Creates a visual representation of the current layout in the model space of a new drawing.

## Access Methods

*Menu*:
File ![](../images/ac.menuaro.gif) Export Layout to Model.

*Layout tab*: Right-click a layout tab and click Export Layout to Model.

*File tab*:
![](../images/GUID-C2BA3961-A036-4F07-A055-D7A43119520E-low.png) Quick View. In the Quick View dialog box, right-click a layout and click Export Layout to Model .

Exported objects are duplicated visually in the new drawing and, as a result, they might be trimmed, scaled, copied, or exploded.

A dialog box is displayed. Select the location and enter the file name for the new drawing. The default file name combines
the names of the current drawing and the current layout. All drawings are exported as Drawing (*\*.dwg*).

NOTE: The EXPORTLAYOUT command is only available in a layout. The command is not available from the Model tab, while in the block
editor, during reference editing, or while using a maximized viewport.
