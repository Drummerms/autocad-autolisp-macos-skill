# TEXTEDIT (Command)

Edits a selected multiline or single-line text object, or the text in a dimension object.

## Access Methods

*Menu*:
Modify ![](../images/ac.menuaro.gif) Object ![](../images/ac.menuaro.gif) Edit Text.

*Shortcut menu*: Select a text object, right-click in the drawing area, and click Edit.

*Pointing device*: Double-click a text object.

## Summary

Displays the in-place text editor, and accepts your changes to the selected multiline text, single-line text, or dimension
object.

## List of Prompts

The following prompt is displayed.

Select an annotation object: *Select a text, mtext, or dimension object*
