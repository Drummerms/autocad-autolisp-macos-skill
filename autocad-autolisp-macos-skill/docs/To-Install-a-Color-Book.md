# To Install a Color Book

1. At the Command prompt, enter PREFERENCES.
2. In the Application Preference dialog box, Application tab, click Color Book Locations.
3. Click the Browse button to confirm the folder paths where color book files are located. If you want to add new folder for
   color books, click Add and enter the new folder path in the blank path box.
4. Click OK to close the Application Preference dialog box.
5. Using your operating system, copy the new color book (.*acb*) file to a specified color book folder.
