# Create New Cell Style Dialog Box

Specifies a name for the new cell style and specifies the existing cell style on which the new cell style will be based.

## Access Methods

TABLESTYLE (Command)

*Tool Sets*:
Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/ac.menuaro.gif) Table Style.
![](../images/GUID-EA6A1B89-9B6F-4552-AD8E-694113F12EA9-low.png)

*Menu*:
Format ![](../images/ac.menuaro.gif) Table Style.

On the Table Style Manager dialog box, select New
![](../images/GUID-956AE331-FA56-435B-B092-CDB1D4FA6458-low.png) or Modify.
![](../images/GUID-4AC4C87F-0D68-4B11-AD8D-D11171DD08B0-low.png) , then click Create A New Cell Style.
![](../images/GUID-CFA7D967-6EE3-4245-9C5E-B948246380C9-low.png)

![](../images/GUID-C68C6C83-C433-42D6-B268-E0200299D3B5-low.png)

## List of Options

The following options are displayed.

New Style Name
:   Names the new cell style.

Start With
:   Specifies an existing cell style whose settings are the default for the new cell style.

Continue
:   Returns you to the New or Modify Table Style dialog box, in which you define the new cell style.
