# To Find Plotter Configuration Files

Windows
:   1. Click the Application menu ![](../images/ac.menuaro.gif) Options.
    2. In the Options dialog box, Files tab, click the plus sign (+) to the left of Printer Support File Path.
    3. Click the plus sign (+) to the left of Printer Configuration Search Path.

       The locations the program looks in for plotter configuration files are displayed.

Mac OS
:   1. From the Mac OS menu bar, click *<Product Name>* ![](../images/ac.menuaro.gif) Preferences.

       NOTE:A drawing must be open to see the Preferences menu item.
    2. In the Application Preferences dialog box, Application tab, click the arrow to the left of Printer Support File Path.
    3. Click the arrow to the left of Printer Configuration Search Path.

       The locations the program looks in for plotter configuration files are displayed.
