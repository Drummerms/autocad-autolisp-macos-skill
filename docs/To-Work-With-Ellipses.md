# To Work With Ellipses

The shape of an ellipse is determined by two axes that define its length and width.

## Draw an Isometric Circle

If you are drawing on isometric planes to simulate 3D, you can use ellipses to represent isometric circles viewed from an
oblique angle.

1. Click
   Tools menu ![](../images/ac.menuaro.gif)Drafting Settings.At the Command prompt, enter dsettings.
2. In the Drafting Settings dialog box, Snap and Grid tab, under Snap Type and Style, click Isometric Snap. Click OK.
3. Click
    Draw
   ![](../images/ac.menuaro.gif) Ellipse drop-down
   ![](../images/ac.menuaro.gif) Axis, End.
   ![](../images/GUID-36ACD897-212E-469F-96D1-0FF1356A0DA3-low.png)At the Command prompt, enter
   *ellipse*.
4. Enter
   *i* (Isocircle).
5. Specify the center of the circle.
6. Specify the radius or diameter of the circle.

## Draw a True Ellipse Using Endpoints and Distance

1. Click
   Draw
   ![](../images/ac.menuaro.gif) Ellipse drop-down
   ![](../images/ac.menuaro.gif) Axis, End.
   ![](../images/GUID-36ACD897-212E-469F-96D1-0FF1356A0DA3-low.png)At the Command prompt, enter
   *ellipse*.
2. Specify the first endpoint of the first axis (1).
3. Specify the second endpoint of the first axis (2).
4. Drag the pointing device away from the midpoint, and click to specify a distance (3) for half the length of the second axis.

   ![](../images/GUID-0572912F-F5E2-48FA-91FF-7112E24D8426-low.png)

## Draw an Elliptical Arc Using Start and End Angles

1. Click
    Draw
   ![](../images/ac.menuaro.gif) Ellipse drop-down ![](../images/ac.menuaro.gif) Elliptical Arc.
   ![](../images/GUID-EC200F6F-5C85-4482-A84B-16AAE01A3ED5-low.png)At the Command prompt, enter
   *ellipse*.
2. Specify endpoints for the first axis (1 and 2).
3. Specify a distance to define half the length of the second axis (3).
4. Specify the start angle (4).
5. Specify the end angle (5).

   The elliptical arc is drawn counterclockwise between the start point and endpoint.

   ![](../images/GUID-DFD6C729-4055-444B-97F6-328E42EF3EED-low.png)
