# To Move an Object Between Paper Space and Model Space

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Change Space.
   ![](../images/GUID-1F36D443-FC78-43E3-88C7-6CD372388B90-low.png)
2. Select one or more objects to move.
3. Press Enter to end the command.
