# DIVMESHWEDGEHEIGHT (System Variable)

Sets the number of subdivisions for the height of the mesh wedge along the *Z* axis.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 3 |

![](../images/GUID-B823D233-0C73-4D3F-89AC-EC8525461574-low.png)

This setting affects new mesh wedges that are created with the [MESH](MESH-Command.md) command.
