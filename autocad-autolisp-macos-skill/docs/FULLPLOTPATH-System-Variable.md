# FULLPLOTPATH (System Variable)

Controls whether the full path of the drawing file is sent to the plot spooler.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Sends the drawing file name only |
| 1 | Sends the full path of the drawing file |
