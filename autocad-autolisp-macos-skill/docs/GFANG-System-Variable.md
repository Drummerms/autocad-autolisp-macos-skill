# GFANG (System Variable)

Specifies the angle of a gradient fill.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Not-saved |
| Initial value: | 0 |

Valid values range from 0 to less than 360 (degrees). Negative values and values equal to or greater than 360 are accepted,
but they are converted to the equivalent value between 0 and 360.
