# To Specify Center Mark and Centerline Properties

Center marks and centerlines can be modified in the Properties palette.

1. Select a center mark or centerline.
2. If the Properties Inspector is not displayed, right-click and select Properties.
3. In the Properties Inspector, modify the properties you want to change.
