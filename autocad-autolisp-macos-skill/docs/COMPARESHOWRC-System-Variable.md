# COMPARESHOWRC (System Variable)

Controls the display of revision cloud around the differences (change set) in the comparison drawing.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| Value | Description |
| 0 | The display of revision cloud is off. |
| 1 | The display of revision cloud is on. |
