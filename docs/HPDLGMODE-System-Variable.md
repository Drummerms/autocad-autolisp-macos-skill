# HPDLGMODE (System Variable)

Controls the display of the Hatch and Gradient dialog box and the Hatch Edit dialog box.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 2 |

| 0 or 2 | Hatch and Gradient dialog box is not displayed for HATCH and GRADIENT unless Settings is entered at the Command prompt. |
| 1 | Hatch and Gradient dialog box is displayed for HATCH and GRADIENT. |
