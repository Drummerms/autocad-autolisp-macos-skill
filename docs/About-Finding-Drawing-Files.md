# About Finding Drawing Files

You can search for a drawing using name, location, and date filters, properties such as keywords that you added to the drawing,
or text strings containing a specific word or phrase.

* On the Start tab, the recent and open files are displayed in a text or image list. The files are organized according to the
  date of access.
* Use Spotlight in Finder to search for drawings using name, location, and date filters.
* Use the OPEN command, right-click on a drawing file, and select Quick Look.
