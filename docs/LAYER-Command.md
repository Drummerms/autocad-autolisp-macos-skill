# LAYER (Command)

Manages layers and layer properties.

## Access Methods

*Menu*:
Format
![](../images/ac.menuaro.gif) Layers.

*Command entry*:
*'layer* for transparent use

## Summary

The
[Layers Palette](Layers-Palette.md) is displayed.

If you enter *-layer* at the Command prompt,
[options are displayed](LAYER-Command-2.md).

![](../images/GUID-6C464045-976D-475F-9855-2AAF63DC3B92-low.png)

Use layers to control the visibility of objects and to assign properties such as color and linetype. Objects on a layer normally
assume the properties of that layer. However, you can override any layer property of an object. For example, if an object’s
color property is set to BYLAYER, the object displays the color of that layer. If the object’s color is set to Red, the object
displays as red, regardless of the color assigned to that layer.
