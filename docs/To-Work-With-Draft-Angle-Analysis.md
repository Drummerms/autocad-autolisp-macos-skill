# To Work With Draft Angle Analysis

## Analyze the Draft Angle of a Model

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Surface panel ![](../images/ac.menuaro.gif) Analysis drop-down ![](../images/ac.menuaro.gif) Analysis Draft.
   ![](../images/GUID-C14948F4-DF4F-4C01-BC26-F82EAF12296E-low.png)
2. Select a model and press Enter.

   The draft analysis colors reflect the maximum and minimum draft angles, as set in the Analysis Options dialog box.

## Turn Off the Draft Analysis Color Display

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Surface panel ![](../images/ac.menuaro.gif) Analysis Options.
   ![](../images/GUID-E43E5959-FA91-43AA-B25D-2754135C2B20-low.png)
2. Click the Clear Draft Analysis.

## Change the Draft Analysis Options

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Surface panel ![](../images/ac.menuaro.gif) Analysis Options.
   ![](../images/GUID-E43E5959-FA91-43AA-B25D-2754135C2B20-low.png)
2. In the Draft Angle tab of the Analysis Options dialog box, select the desired options.
