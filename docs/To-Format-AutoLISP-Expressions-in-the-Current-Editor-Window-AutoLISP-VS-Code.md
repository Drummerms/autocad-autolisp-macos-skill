# To Format AutoLISP Expressions in the Current Editor Window (AutoLISP/VS Code)

## Change Format Settings

1. In Visual Studio Code, click File menu > Preferences > Settings.
2. On the User tab, expand Extensions and click AutoCAD AutoLISP Configuration.![](../images/GUID-8910638A-3E86-420C-A6F3-F02FA92690FE-low.png)
3. Change the value of one or more format settings:
   * Close Parenthesis Style
   * Long List Format Style
   * Max Line Chars
   * Narrow Style Indent

## Format AutoLISP Expressions

1. Activate the editor window that contains the AutoLISP expressions to format.
2. Do one of the following:
   * *Format Selection* – Select the expressions you want to format, right-click (or Control+click on Mac OS) and then choose Format Selection.
   * *Format Document* – Right-click (or Control+click on Mac OS) in the editor window and choose Format Document.
