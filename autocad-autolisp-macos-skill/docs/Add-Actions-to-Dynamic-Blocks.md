# Add Actions to Dynamic Blocks

#### Topics in this section

* [About Move Actions](About-Move-Actions.md)

  Move actions move selected objects a specified distance and angle.
* [About Scale Actions](About-Scale-Actions.md)

  Scale actions resize selected objects relative to the base point specified by the action.
* [About Stretch Actions](About-Stretch-Actions.md)

  Stretch actions move and stretch objects a specified distance in a specified direction relative to the base point specified
  by the action.
* [About Polar Stretch Actions](About-Polar-Stretch-Actions.md)

  Polar stretch actions rotate, move, and stretch objects a specified angle and distance.
* [About Rotate Actions](About-Rotate-Actions.md)

  A rotate action rotates selected objects.
* [About Flip Actions](About-Flip-Actions.md)

  A flip action mirrors selected objects in a dynamic block reference about a specified reflection line.
* [About Grips on Dynamic Blocks](About-Grips-on-Dynamic-Blocks.md)

  Custom grips can be used to manipulate the geometry of a dynamic block reference.
* [About Controlling the Visibility of Objects in Dynamic Blocks](About-Controlling-the-Visibility-of-Objects-in-Dynamic-Blocks.md)

  Using visibility states, you can create a block with different graphical representations.
