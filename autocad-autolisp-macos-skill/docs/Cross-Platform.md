# Cross Platform

#### Topics in this section

* [About Cross Platform Features](About-Cross-Platform-Features.md)

  Cross platform features provide functionality in AutoCAD desktop, the AutoCAD web app, and the AutoCAD mobile app.
* [Marking Up Drawings](topichead-96.htm)
* [Open in Desktop](Open-in-Desktop.md)

  If you're working in the AutoCAD web app and your computer has AutoCAD 2018 or later installed, you can use the Open in Desktop
  button to open a drawing in desktop AutoCAD.
* [Share Drawing (Share URL)](Share-Drawing-Share-URL.md)

  Share a link to a copy of the current drawing to view or edit in AutoCAD on the web ([web.autocad.com](https://web.autocad.com/)) or AutoCAD on mobile. All related DWG xrefs and images are included.
* [About Sheet Set Manager for Web](About-Sheet-Set-Manager-for-Web.md)

  Manage sheet sets from Autodesk Docs with Sheet Set Manager for Web.
