# To Save Part of a Drawing as a New File

Create new drawings from objects within the current drawing.

Use the Write Block command to create a new drawing from a set of objects in the curent drawing, or create a reusable block
of the entire drawing.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Write Block.
   ![](../images/GUID-E84A38E9-381C-46EC-BE0C-F4E1728D05DA-low.png)
2. Select one of the following sources:
   * Block
   * Entire Drawing
   * Objects
3. If the selected source is Objects, click Select Objects, and in the drawing area select objects to save.
4. Specify or pick a base point, if desired.
5. Enter a file name and path and change the units, as needed, and click OK.
