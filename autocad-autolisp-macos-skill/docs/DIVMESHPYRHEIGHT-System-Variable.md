# DIVMESHPYRHEIGHT (System Variable)

Sets the number of subdivisions between the base and the top of the mesh pyramid.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 3 |

![](../images/GUID-1754D2C8-FC2A-4B6D-9A80-89095A456825-low.png)

This setting affects new mesh pyramids that are created with the [MESH](MESH-Command.md) command.
