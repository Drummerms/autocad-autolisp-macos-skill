# To Create a Data Link from an Excel File

1. Click
   Window menu ![](../images/ac.menuaro.gif) Reference Manager.
2. Click Attach Reference in the Reference Manager palette.
   ![](../images/GUID-8F1CD821-F5B3-4571-8E03-B82409C83BBC-low.png)
3. In the Select Reference File, select the required Excel file.

   The New Excel Data Link dialog box opens.

   ![](../images/GUID-1A0F33B3-2D9E-4AF6-97C5-B7E14271E181-low.png)
4. Choose the necessary settings and click OK.
5. Select a point in the drawing to insert. The table is inserted into the drawing using the selected options. The table data
   is locked since it is linked to the spreadsheet.

   NOTE:If the Excel spreadsheet changes, use the DATALINKUPDATE command or reload from Reference Manager to refresh the table.
