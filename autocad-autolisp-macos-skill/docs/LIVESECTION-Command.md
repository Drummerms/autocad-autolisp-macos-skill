# LIVESECTION (Command)

Turns on live sectioning for a selected section object.

## Access Methods

Shortcut menu: Select a section plane object. Right-click and choose Activate Live Sectioning.

## Summary

When turned on, the cross sections of 3D objects intersected by the section object are displayed. Live sectioning only works
with objects that were created with SECTIONPLANE.

![](../images/GUID-08101007-6927-4260-B168-E7C87D7926F6-low.gif)

## List of Prompts

The following prompts are displayed.

Select section object
:   Turns on live sectioning for the section object you select.
