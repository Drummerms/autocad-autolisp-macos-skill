# Modify Raster Images

#### Topics in this section

* [About Clipping Raster Images](About-Clipping-Raster-Images.md)

  You can clip and display specific portions of a raster image in a drawing with a clipping boundary.
* [About Showing and Hiding Raster Image Boundaries](About-Showing-and-Hiding-Raster-Image-Boundaries.md)

  You can control whether the clipping boundaries of a raster image are displayed or hidden in a drawing.
* [About Changing Raster Image Brightness, Contrast, and Fade](About-Changing-Raster-Image-Brightness,-Contrast,-and-Fade.md)

  You can change several display properties of raster images in a drawing for easier viewing or special effects.
* [About Modifying Color and Transparency for Bitonal Raster Images](About-Modifying-Color-and-Transparency-for-Bitonal-Raster-Images.md)

  Bitonal images are images that consist only of a foreground color and a background color. You can change the foreground color
  and turn the transparency of the background color on and off.
