# To Work With Creating Solid Boxes

## Create a Solid Box Based on Two Points and a Height

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Solid drop-down ![](../images/ac.menuaro.gif) Box.
   ![](../images/GUID-E4DD1427-E3FC-4146-B9AF-6C7B5D088B0D-low.png)
2. Specify the first corner of the base.
3. Specify the opposite corner of the base.
4. Specify the height.

## Create a Solid Cube

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Solid drop-down ![](../images/ac.menuaro.gif) Box.
   ![](../images/GUID-E4DD1427-E3FC-4146-B9AF-6C7B5D088B0D-low.png)
2. Specify the first corner, or enter
   *c* (Center) to specify the center point of the base.
3. At the Command prompt, enter
   *c* (Cube). Specify the length of the cube and a rotation angle.

   The length value sets both the width and height of the cube.
