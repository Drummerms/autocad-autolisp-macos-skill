# To Discard all Changes

* Click
  Reference Editor visor
  ![](../images/ac.menuaro.gif) Discard Changes.

  Any changes you make to objects in the current drawing (not in the xref or block) are not discarded. If you delete any object
  that is not in the working set, the object is not restored even if you choose to discard changes.
