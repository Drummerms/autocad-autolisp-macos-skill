# To Change the Page Setup of One or More Sheets in a Drawing Set for Publishing

1. Click Publish.
   ![](../images/GUID-4C887873-7EE8-497C-B87D-0D8DFF393268-low.png)
2. In the Publish dialog box, ensure the sheets for which you want to change the page setups are listed.
3. Select one or more sheets in the list.
4. Right-click the selection. Click Change Page Setup.
5. In the Page Setup list, select a page setup to apply to the drawing sheets.

   NOTE:Change the page setup for each layout depending on your desired output. Model space page setups can only be applied to model
   space sheets; paper space page setups can only be applied to paper space sheets.
