# SELECTIONPREVIEW (System Variable)

Controls the display of selection previewing.

|  |  |
| --- | --- |
| Type: | Bitcode |
| Saved in: | Registry |
| Initial value: | 3 |

Objects are highlighted when the pickbox cursor rolls over them. This selection previewing indicates that the object would
be selected if you clicked. The setting is stored as a bitcode using the sum of the following values:

| 0 | Off |
| 1 | On when no commands are active |
| 2 | On when a command prompts for object selection |
