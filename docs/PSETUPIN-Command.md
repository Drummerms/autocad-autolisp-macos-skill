# PSETUPIN (Command)

Imports a user-defined page setup into a new drawing layout.

## Summary

The Select Page Setup From File dialog box (a [standard file selection dialog box](Standard-File-Selection-Dialog-Boxes.md)) is displayed in which you can select the drawing *(.dwg)*, template (*.dwt*), or drawing interchange format (*.dxf*) file whose page setups you want to import.

If [FILEDIA](FILEDIA-System-Variable.md) is set to 0 (zero) and you enter *-psetupin* at the Command prompt, PSETUPIN displays [command prompts](PSETUPIN-Command-2.md).

When you select the drawing file that you want to use, the [Import Page Setups dialog box](Import-Page-Setups-Dialog-Box.md) is displayed.
