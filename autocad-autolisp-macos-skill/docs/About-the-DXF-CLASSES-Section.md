# About the DXF CLASSES Section

The CLASSES section in DXF files holds the information for application-defined classes whose instances appear in the BLOCKS,
ENTITIES, and OBJECTS sections of the database. It is assumed that a class definition is permanently fixed in the class hierarchy.
All fields are required.
