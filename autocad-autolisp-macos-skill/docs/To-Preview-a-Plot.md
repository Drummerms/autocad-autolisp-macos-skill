# To Preview a Plot

You can preview a drawing before printing it. The preview shows exactly how the drawing will look when plotted, including
lineweights, fill patterns, and plot style options.

When you preview your drawing, the active toolbars and tool palettes are hidden. A temporary Preview toolbar provides buttons
to plot, pan, and zoom the drawing.

1. Click Plot.
   ![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png)
2. In the Plot dialog box, click Preview.

   The plot is previewed as a PDF file in a PDF reader.
3. Switch back to your drawing, and if necessary make additional adjustments to the plot settings and preview the plotted drawing
   again.
4. Once the settings are correct, click OK to plot the drawing.
