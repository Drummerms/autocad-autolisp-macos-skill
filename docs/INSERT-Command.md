# INSERT (Command)

Displays the Blocks palette, which you can use to insert blocks and drawings into the current drawing.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Insert.
![](../images/GUID-6857E103-73A1-42F1-AA87-3265189DB33B-low.png)

*Menu*:
Insert ![](../images/ac.menuaro.gif) Block.

![](../images/GUID-C997FCC0-1B2F-4EC2-ACBC-B488CD780FE3-low.png)

Select a block to insert by selecting from blocks in the current drawing, recently used blocks, or a block library. You can
also browse to a drawing to insert as a block.

* If you enter
  *-insert* at the Command prompt, options are displayed.
* Inserting a drawing file as a block also imports all of its block definitions into the current drawing.

NOTE:To access the classic Insert Block dialog box, use the CLASSICINSERT command.

TIP:A good practice is to insert a block from a block library. A block library can be a drawing file that stores related block
definitions or it can be a folder that contains related drawing files, each of which can be inserted as a block. With either
method, blocks are standardized and accessible to multiple users.
