# LOGFILEPATH (System Variable)

Specifies the path for the command history log files for all drawings in a session.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Registry |
| Initial value: | Varies |

You can also specify the path by using the OPTIONS command. The initial value is based on the installation folder.
