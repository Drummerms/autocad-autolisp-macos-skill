# NAVVCUBEOPACITY (System Variable)

Controls the opacity of the ViewCube tool when inactive.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 50 |

Valid values are from 0 to 100 percent. When set to 100, the ViewCube tool appears fully opaque against the drawing window
and obscures all objects under it in the viewport. When set to less than 100, the ViewCube tool fades into the drawing window,
which results in the objects under it to appear less obscured. If set to 0, the ViewCube tool is not displayed in the viewport
unless the cursor is positioned over top of the ViewCube tool location.
