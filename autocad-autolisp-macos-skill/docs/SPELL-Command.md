# SPELL (Command)

Checks spelling in a drawing.

## Access Methods

*Tool Sets*:
Drafting tab ![](../images/ac.menuaro.gif) Text panel ![](../images/ac.menuaro.gif) Spell.
![](../images/GUID-48124FBB-3839-46CF-A0A0-EDC87BE0172E-low.png)

*Menu*:
Tools
![](../images/ac.menuaro.gif) Spelling.

## Summary

When you enter
*spell* at the Command prompt, the
[Check Spelling dialog box](Check-Spelling-Dialog-Box.md) is displayed. Select the Start button to begin the spelling check.

If Check Spelling is set to Entire Drawing, spelling is checked on the Model layout, then on named (paper space) layouts.
If a flagged word is identified, the drawing area highlights and zooms to that word.

NOTE:Invisible text such as text on hidden layers, and hidden block attributes is not checked. Non-uniformly scaled blocks and
objects not on the supported annotation scale are also not checked.
