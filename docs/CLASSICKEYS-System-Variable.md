# CLASSICKEYS (System Variable)

Sets the behavior of the AutoCAD 2026 shortcut keys.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | To copy, press Cmd-C |
| 1 | To cancel, press Cmd-C |
