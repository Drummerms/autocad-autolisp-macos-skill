# To Work With Changing Prompt Order in Attribute Definitions

How to change the display order of the prompts that are displayed when you insert a block with attributes.

The order of the attribute prompts is the same as the order in which you selected the attributes when you created the block.
However, if you used crossing or window selection to select the attributes, the order of the prompts is the reverse of the
order in which you created attributes. You can use the Block Attribute Manager to change the order in which you are prompted
for attribute information when you insert the block reference.

## To Change the Prompt Order for Attribute Values (Block Attribute Manager)

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block Attribute Manager.
   ![](../images/GUID-29956DF2-A40C-4FF8-943C-CA1DBA0F5B9D-low.png)
2. In the Block Attribute Manager, select a block from the Block list, or click Select Block and select a block in the drawing
   area.

   For the selected block, attributes are listed in their prompt order.
3. Select the attribute and drag rows to change its order in the prompt list.

## To Change the Prompt Order for Attribute Values (Block Editor)

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block Editor.
   ![](../images/GUID-E3DDBE51-C92A-4D07-A51D-C7C685E4F578-low.png)
2. At the Command prompt, enter BATTORDER.
3. In the Attribute Order Dialog Box, drag rows to specify the order in which attributes are to be listed.
