# ACADPREFIX (System Variable)

Stores the directory path, if any, specified by the ACAD environment variable, with path separators appended if necessary.

(Read-only)

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Not-saved |
| Initial value: | Varies |
