# To Import a DXF File

1. Click
   File menu ![](../images/ac.menuaro.gif) Open.
2. In the dialog box, in the File Format drop-down, select DXF (\*.dxf).
3. Find and select the DXF file you want to import.
4. Click Open.

## Insert a DXF File as a Block by Dragging

1. From Finder, drag the DXF file into the drawing canvas.
2. Move the mouse to the location you want to place the DXF file.

   When you release the button, the DXF file is inserted as a block.
3. Use the Properties palette to change block properties or EXPLODE as needed.
