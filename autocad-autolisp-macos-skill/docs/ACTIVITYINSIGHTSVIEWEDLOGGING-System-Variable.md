# ACTIVITYINSIGHTSVIEWEDLOGGING (System Variable)

Turns on or off logging Viewed activities, such as opening a drawing.

|  |  |
| --- | --- |
| Type: | Bitcode |
| Saved in: | Registry |
| Initial value: | 0 |

| Value | Description |
| 0 | Turns off logging Viewed (drawing file open) activities |
| 1 | Turns on logging Viewed (drawing file open) activities |

NOTE:Past Viewed activities that were logged while the system variable was on are displayed in the Activity Insights palette.
