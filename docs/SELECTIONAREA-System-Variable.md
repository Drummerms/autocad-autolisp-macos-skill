# SELECTIONAREA (System Variable)

Controls the display of effects for selection areas.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

Selection areas are created by the Window, Crossing, WPolygon, and CPolygon options of [SELECT](SELECT-Command.md).

| 0 | Off |
| 1 | On |
