# Reference Editor Visor

Manages the working set of objects that are edited in a reference.

![](../images/GUID-43077F04-323B-4047-86F9-95A0445B007B-low.png)

## List of Options

The following options are displayed.

Add to Working Set

Adds objects to the current working set while editing an xref or a block definition in-place. (REFSET command)

Remove From Working Set

Removes objects from the current working set while editing an xref or a block definition in-place. (REFSET command)

Discard Changes

Discards the changes made to an xref or a block definition. (REFCLOSE command)

Save

Saves back the changes made to an xref or a block definition. (REFCLOSE command)
