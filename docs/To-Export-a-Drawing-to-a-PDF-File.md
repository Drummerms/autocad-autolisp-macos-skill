# To Export a Drawing to a PDF File

1. Click Plot.
   ![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png)
2. In the Plot dialog box, do one of the following:
   * Select a PDF option from the Printer drop-down menu and click Plot.
   * Click Save as PDF, enter a file name, and click Save.

## Publish Multiple Drawings to PDF

1. Click
   File ![](../images/ac.menuaro.gif) Batch Publish.
2. In the Batch Publish dialog box, select PDF in Publish To list.
3. Define the drawings and layouts to publish:
   * *+*. opens a standard file selection dialog box where you can select drawings to publish.
   * *-*. removes selected layouts from the list.
   * *Add open drawings*. adds all layouts from all open drawings.
4. In Publish Output, select from the available presets.
5. Set the remaining options to configure the way you want the drawings to be output.
6. Click Publish.
