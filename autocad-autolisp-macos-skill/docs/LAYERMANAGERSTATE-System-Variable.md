# LAYERMANAGERSTATE (System Variable)

Indicates whether the Layers palette is open or closed.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | Varies |

| 0 | Closed |
| 1 | Open and collapsed |
| 2 | Open and expanded |
