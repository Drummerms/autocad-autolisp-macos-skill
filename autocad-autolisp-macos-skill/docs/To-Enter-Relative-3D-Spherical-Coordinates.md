# To Enter Relative 3D Spherical Coordinates

* At a prompt for a point, enter the coordinates using the following format:

  *@* *X* *<* *angle from the x axis* *<* *angle from the xy plane*

For example, @4<60<30 represents a location that is 4 units from the last point, at 60 degrees from the positive *X* axis in the *XY* plane, and at 30 degrees from the *XY* plane.
