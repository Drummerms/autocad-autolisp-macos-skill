# About Program and Support Files

You can change the default directory structure for the program and support files to suit your needs.

#### Topics in this section

* [About File Organization](About-File-Organization.md)

  The default folder structure for program and support files is designed to efficiently organize those files into logical groups.
* [About Multiple Configurations](About-Multiple-Configurations.md)

  If you use more than one pointing device, you can set up more than one configuration file to make it easy to switch between
  devices.
* [About Multiple Drawing Folders](About-Multiple-Drawing-Folders.md)

  Keeping your drawing and other associated files in separate directories makes it easier to perform basic file maintenance.
* [About Customized File Locations](About-Customized-File-Locations.md)
