# To Assign a Shape to a Light

1. Select the light in a drawing.
2. Scroll down the Lighting Properties tab in the Properties Inspector to view the Shadow Detail.
3. Change the *Type* property to *Soft* (sampled).

   NOTE:The Shape property is displayed depending on the Type property that is selected.
4. Change the *Shape* property to an available option.

   Available shapes depend upon the type of light:
   * If Spotlight is selected: Rectangular and Disk shapes are available.

   If Point light, soft (sampled), and Web are selected: Linear, Rectangular, Disk, Cylinder, and Sphere shapes are available.
5. Change the Visible Render property to Yes.
6. Render the image.

   The light is shown in the shape selected.
