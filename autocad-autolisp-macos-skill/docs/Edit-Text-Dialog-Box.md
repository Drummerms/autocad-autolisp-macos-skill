# Edit Text Dialog Box

Edits a single-line text object.

DDEDIT (Command)

*Menu*:
Modify ![](../images/ac.menuaro.gif) Object ![](../images/ac.menuaro.gif) Edit Text.

## Summary

When the TEXTED system variable is set to 1, modifies text created with the TEXT command. Enter the new text and click OK.

To add a field to the text, right-click the text where you want to insert the field and click Insert Field on the shortcut
menu to display the Field dialog box.
