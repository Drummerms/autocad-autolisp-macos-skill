# The ViewCube Tool

The ViewCube tool is a handy tool to control the orientation of 3D views.

This tool is available in most Autodesk products, and provides a common experience when you switch between products.

![](../images/GUID-5CDE9E6F-BD95-4FA0-957F-A5260571A7BD-low.png)

Alternatively, you can use the 3DORBIT command to drag 3D views, and right-click for additional 3D viewing options.
