# XREFLAYER (System Variable)

Specifies a default layer for a new xref.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | use current |

Controls the layer that DWG external references are placed on when attached using the XATTACH, ATTACH, and XREF commands.

Values other than the "." (use current) value override the current layer (CLAYER system variable).
