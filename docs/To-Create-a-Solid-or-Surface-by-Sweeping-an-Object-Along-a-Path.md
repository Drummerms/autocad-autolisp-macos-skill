# To Create a Solid or Surface by Sweeping an Object Along a Path

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Modeling drop-down ![](../images/ac.menuaro.gif) Sweep.
   ![](../images/GUID-C86F5BE9-D23A-4E26-BF0D-DB9C1A521E0B-low.png)
2. Select objects to sweep and press Enter.
3. Select an object or edge subobject as a sweep path.

   After the sweep, the original objects are deleted or retained, depending on the setting of the DELOBJ system variable.
