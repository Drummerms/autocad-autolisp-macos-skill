# CMDINPUTHISTORYMAX (System Variable)

Sets the maximum number of previous input values that are stored for a prompt in a command.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 20 |

Display of the history of user input is controlled by the [INPUTHISTORYMODE](INPUTHISTORYMODE-System-Variable.md) system variable.
