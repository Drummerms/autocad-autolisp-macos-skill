# To Create a Section Object by Specifying Two Points

Create a straight cutting plane.

![](../images/GUID-3049BDE0-9D7E-4868-BA0E-B0F9AAC59DB4-low.png)

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Section Plane drop-down ![](../images/ac.menuaro.gif) Section Plane.
   ![](../images/GUID-EFEADBD7-601A-4BFF-9459-6F20FC6A74E7-low.png)
2. Specify the first point of the section object.
3. Specify the endpoint.

The section object is created between the two points. Live sectioning is turned off.
