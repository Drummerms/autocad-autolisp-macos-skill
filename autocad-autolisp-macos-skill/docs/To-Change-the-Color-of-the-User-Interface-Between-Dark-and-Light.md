# To Change the Color of the User Interface Between Dark and Light

1. Right-click in the drawing area, and choose Preferences.
2. In the Application Preferences dialog box, left column, click Look & Feel.
3. Under Interface Theme, click in the Themes box, and click the theme of your choice.
4. Click OK.
