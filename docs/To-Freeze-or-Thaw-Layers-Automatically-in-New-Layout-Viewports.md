# To Freeze or Thaw Layers Automatically in New Layout Viewports

You can set visibility defaults for specific layers in all new layout viewports.

1. Click a layout tab.
2. Click
   Window menu ![](../images/ac.menuaro.gif) Layers.
3. In the Layers palette, select the layer or layers you want to be automatically frozen or thawed in new viewports you create.

   For example, you can restrict the display of dimensions by freezing the DIMENSIONS layer in all new viewports. If you create
   a viewport that requires dimensions, you can override the default setting by changing the setting in the current viewport.
   Changing the default for new viewports does not affect existing viewports.
4. Click in the New VP Freeze column to change the state of the selected layers.
