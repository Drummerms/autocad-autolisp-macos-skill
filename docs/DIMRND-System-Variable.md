# DIMRND (System Variable)

Rounds all dimensioning distances to the specified value.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 0.0000 |

For instance, if DIMRND is set to 0.25, all distances round to the nearest 0.25 unit. If you set DIMRND to 1.0, all distances
round to the nearest integer. Note that the number of digits edited after the decimal point depends on the precision set by
DIMDEC. DIMRND does not apply to angular dimensions.
