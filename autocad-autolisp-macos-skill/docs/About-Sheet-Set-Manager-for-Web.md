# About Sheet Set Manager for Web

Manage sheet sets from Autodesk Docs with Sheet Set Manager for Web.

NOTE:Access the online
[AutoCAD help](https://help.autodesk.com/view/ACD/2026/ENU/?guid=GUID-1690E514-A7B0-44EB-8EB0-E1C897D78780) to see the latest information for Sheet Set Manager for Web.

![](../images/GUID-571B5D05-1860-46E7-9444-77946D456ABA-low.png)

A lightweight version of the legacy Sheet Set Manager, the Sheet Set Manager for Web, displays and manages named collections
of drawing sheets stored in Autodesk Docs. Opening a DST file from Autodesk Docs displays the Sheet Set Manager for Web by
default.

The Sheet Set Manager for Web provides the following core capabilities:

* Create and open cloud sheet sets
* Manage support files
* Create, remove, or move sheets and subsets
* Define sheet set and sheet properties
* Automatically number new sheets
* Rename or renumber sheets
* Import sheets
* Publish sheet sets or selected sheets to PDF
* Transmit sheet sets and selected sheets

IMPORTANT:Use Desktop Connector to upload your DST, DWG, and support files to BIM 360 or Autodesk Docs.

## Use Sheet Sets Collaboratively

When you use cloud sheet sets in a team, each member should have access to the sheet set data (DST) file and the drawing template
(DWT) files associated with the sheet set. Each team member can open the cloud sheet set to load the sheet set information
from the DST file into the Sheet Set Manager for Web. Click
![](../images/GUID-0AD3D90B-1AAA-4970-90E1-A674434A542A-low.png) to identify the team members who are collaborating on the same cloud sheet set.

![](../images/GUID-2243E5DC-FC3D-46FE-BD44-F4E4ADEA2461-low.png)

If each member of the team has access to the sheet set DWT files, new drawing files and their sheets are created using the
same drawing template file; page setups for these drawings are also standardized.

Other members of the team can automatically see changes to the sheet set in the Sheet Set Manager tree view.
