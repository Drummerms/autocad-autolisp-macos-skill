# FACETRES (System Variable)

Adjusts the smoothness of shaded and rendered objects and objects with hidden lines removed.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 0.5000 |

Valid values are from 0.01 to 10.0.
