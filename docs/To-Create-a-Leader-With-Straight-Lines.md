# To Create a Leader With Straight Lines

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Leader panel ![](../images/ac.menuaro.gif) Multileader.
   ![](../images/GUID-31CC8661-13A6-4D2B-968D-39BE64E908F8-low.png)
2. At the Command prompt, enter
   *o* to select options.
3. Enter
   *l* to specify leaders.
4. Enter
   *s* to specify straight leaders.
5. In the drawing, click a start point for the leader head.
6. Click an end point for the leader.
7. Enter your MTEXT content.
8. On the Text Formatting toolbar, click OK.
