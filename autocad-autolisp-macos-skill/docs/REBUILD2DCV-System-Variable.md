# REBUILD2DCV (System Variable)

Sets the number of control vertices when rebuilding a spline.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 6 |

Valid values are from 2 to 32767.
