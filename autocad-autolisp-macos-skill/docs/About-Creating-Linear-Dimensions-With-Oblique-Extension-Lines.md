# About Creating Linear Dimensions With Oblique Extension Lines

You can create linear dimensions with extension lines that are not perpendicular to their dimension lines.

Extension lines are normally perpendicular to the dimension line. However, if the extension lines conflict with other objects
in a drawing, you can change their angle after a linear dimension has been created.
