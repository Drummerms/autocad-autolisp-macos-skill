# CPLOTSTYLE (System Variable)

Controls the current plot style for new objects.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | ByColor |

If the current drawing you are working in is in color-dependent mode, CPLOTSTYLE is read-only and has a value of BYCOLOR.
If the current drawing is in named plot styles mode, CPLOTSTYLE can be set to the following values (BYLAYER is the default):

"BYLAYER"

"BYBLOCK"

"NORMAL"

"USER DEFINED"
