# To Map XREF Server Path Between PC and Mac

1. If you have a drawing with a reference file in the Windows server, connect to the server and find the drawing.
2. Copy the drawing from the Windows server to Mac machine and open it.

The References - Unresolved References File dialog box appears.

![](../images/GUID-1285E457-323D-4537-8895-67ED04EF87C1-low.png)

3. Click the
   *Mac and Windows server path mapping* option.

The Server Path Mapping dialog box appears.

![](../images/GUID-C5D43967-81F7-4A54-9D5C-35B0828E4526-low.png)

4. Click the + button.

The New Server Path Mapping dialog appears.

![](../images/GUID-4A4C5027-11EA-4450-A063-080ABE53049F-low.png)

5. Click the dropdown menu to find the Windows path of the missing xref file and map it to shared address in Mac.
6. Click OK and the reference file is now loaded.
