# To Resolve External References (Xrefs) while Sharing Drawings Between Mac and Windows

When main drawings are on local machine and references are on server, you need to resolve references using the Server Path
Mapping Dialog Box.

To resolve external references:

1. Open a main drawing located in Mac and its xref saved in Windows.

The References dialog opens.

![](../images/GUID-38AA17B0-597D-4888-8E17-EDA54596B23E-low.png)

2. Click the Mac and Windows Server Path Mapping pane.

The Server Path Mapping dialog opens.

![](../images/GUID-AA5A9625-DE6B-4314-A4BD-8B476F20C2BE-low.png)

3. Click the
   *+* button to open the New Server Path Mapping dialog

![](../images/GUID-182981FD-1AF2-4C81-8EE8-7341C202436D-low.png)

4. Click the drop-down menu to select the Windows server address of the missing xref and click OK.
5. Click OK on the Server Path Mapping dialog box.

The missing reference now appears in the drawing and the path appears in the Saved Path field of the Reference Manager dialog
box.

In the main drawing, set the Projectname sysvar to Project1. This value is saved in the drawing file.
