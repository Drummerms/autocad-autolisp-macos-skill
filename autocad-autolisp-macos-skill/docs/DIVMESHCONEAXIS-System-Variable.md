# DIVMESHCONEAXIS (System Variable)

Sets the number of subdivisions around the perimeter of the mesh cone base.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 8 |

![](../images/GUID-E63FF017-F1FD-447D-86C7-4A774F4E9332-low.png)

This setting affects new mesh cones that are created with the [MESH](MESH-Command.md) command.
