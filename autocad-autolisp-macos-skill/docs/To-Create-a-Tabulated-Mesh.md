# To Create a Tabulated Mesh

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Mesh panel ![](../images/ac.menuaro.gif) Tabulated Surface.
   ![](../images/GUID-3F02A81B-8AA1-4EC0-91D6-7925BD1D2F9B-low.png)
2. Specify an object to define the overall shape of the tabulated surface (the path curve).

   The object can be a line, arc, circle, ellipse, or a 2D or 3D polyline.
3. Specify an open line or polyline that defines the direction vector.

   The mesh is extended from the start point to the endpoint of the direction vector.
4. Erase the original objects if necessary.
