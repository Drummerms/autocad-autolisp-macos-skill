# To Remove Objects From the Working set

1. Click Reference Editor visor![](../images/ac.menuaro.gif)Remove from Working Set. At the Command prompt, enter REFSET, and click the Remove Objects from the Working Set button on the Refedit toolbar.
2. Select the objects you want to remove. You can also set PICKFIRST to 1 and create a selection set before using the Remove
   option.

   REFSET can be used only with objects in the space (paper space or model space) in which REFEDIT is initiated.
