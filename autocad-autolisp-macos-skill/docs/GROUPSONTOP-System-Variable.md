# GROUPSONTOP (System Variable)

Controls how layer groups are sorted in the Layers palette.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| Value | Description |
| 0 | Layers and layer groups are sorted together and alphanumerically by name |
| 1 | Layers and layer groups are sorted separately, layer groups appear at the top of the Layers list |
| 2 | Layers and layer groups are sorted separately, layer groups appear at the bottom of the Layers list |
