# HIDEOBJECTS (Command)

Hides selected objects.

## Access Methods

*Menu*:
Tools
![](../images/ac.menuaro.gif) Isolate
![](../images/ac.menuaro.gif) Hide Objects.

Shortcut menu: Right-click in the drawing area and click Isolate ![](../images/ac.menuaro.gif) Hide Objects.

## Summary

Temporarily hides selected objects in the current view. All other objects are visible.

## List of Prompts

The following prompt is displayed.

Select objects:
*Use an object selection method*
