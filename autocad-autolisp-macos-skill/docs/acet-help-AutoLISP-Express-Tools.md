# acet-help (AutoLISP/Express Tools)

Displays a help page.

*Supported Platforms:* AutoCAD for Windows only; not available in AutoCAD LT for Windows, or on Mac OS and Web

## Signature

```
(acet-help [topic])
```

*topic*
:   *Type:* String

    A keyword that identifies the help page to display. If not supplied, the Home page will be displayed.

## Return Values

*Type:* nil

Always returns
nil.
