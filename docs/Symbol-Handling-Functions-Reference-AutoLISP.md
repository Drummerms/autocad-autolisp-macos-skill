# Symbol-Handling Functions Reference (AutoLISP)

The following table provides summary descriptions of the AutoLISP symbol-handling functions.

| Symbol-handling functions | | Platforms | | | | |
| Windows | | Mac OS | | Web |
| Function | Description | AutoCAD | AutoCAD LT | AutoCAD | AutoCAD LT | AutoCAD |
| [(atom item)](atom-AutoLISP.md) | Verifies that an item is an atom | ✓ | ✓ | ✓ | -- | ✓ |
| [(atoms-family format [symlist])](atoms-family-AutoLISP.md) | Returns a list of the currently defined symbols | ✓ | ✓ | ✓ | -- | ✓ |
| [(boundp sym)](boundp-AutoLISP.md) | Verifies whether a value is bound to a symbol | ✓ | ✓ | ✓ | -- | ✓ |
| [(not item)](not-AutoLISP.md) | Verifies that an item evaluates to nil | ✓ | ✓ | ✓ | -- | ✓ |
| [(null item)](null-AutoLISP.md) | Verifies that an item is bound to nil | ✓ | ✓ | ✓ | -- | ✓ |
| [(numberp item)](numberp-AutoLISP.md) | Verifies that an item is a real or an integer | ✓ | ✓ | ✓ | -- | ✓ |
| [(quote expr)](quote-AutoLISP.md) | Returns an expression without evaluating it | ✓ | ✓ | ✓ | -- | ✓ |
| [(set sym expr)](set-AutoLISP.md) | Sets the value of a quoted symbol name to an expression | ✓ | ✓ | ✓ | -- | ✓ |
| [(setq sym1 expr1 [sym2 expr2 ...])](setq-AutoLISP.md) | Sets the value of a symbol or symbols to associated expressions | ✓ | ✓ | ✓ | -- | ✓ |
| [(type item)](type-AutoLISP.md) | Returns the type of a specified item | ✓ | ✓ | ✓ | -- | ✓ |
| [(vl-symbol-name symbol)](vl-symbol-name-AutoLISP.md) | Returns a string containing the name of a symbol | ✓ | ✓ | ✓ | -- | ✓ |
| [(vl-symbol-value symbol)](vl-symbol-value-AutoLISP.md) | Returns the current value bound to a symbol | ✓ | ✓ | ✓ | -- | ✓ |
| [(vl-symbolp object)](vl-symbolp-AutoLISP.md) | Identifies whether or not a specified object is a symbol | ✓ | ✓ | ✓ | -- | ✓ |
