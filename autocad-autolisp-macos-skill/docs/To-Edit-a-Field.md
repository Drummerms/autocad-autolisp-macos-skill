# To Edit a Field

1. Double-click a text object to display the appropriate text editing dialog box.
2. Double-click the field that you want to edit.

   The Insert Field dialog box is displayed.
3. Make any needed changes.
4. Click OK to exit the Insert Field dialog box.
5. Exit the text editor.
