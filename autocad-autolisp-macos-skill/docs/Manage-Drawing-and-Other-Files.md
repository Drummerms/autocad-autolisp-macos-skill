# Manage Drawing and Other Files

#### Topics in this section

* [About Drawings and Templates](About-Drawings-and-Templates.md)

  New drawings start from either a default drawing template file or a custom drawing template file that you create. Drawing
  template files store default settings, styles, and additional data.
* [About Units of Measurement](About-Units-of-Measurement.md)

  Objects are measured in drawing units. Before you start to draw, decide what one drawing unit represents in your drawing.
* [About Unit Format Conventions](About-Unit-Format-Conventions.md)

  Specify the format and number of decimal places for linear and angular units.
* [About Opening Drawings](About-Opening-Drawings.md)

  You open drawings to work on them just as you do with other applications.
* [About Counting Objects](About-Counting-Objects.md)

  Quickly and accurately count the instances of objects in a drawing. You can insert a table with the block count data into
  the current drawing.
* [About Saving Drawings](About-Saving-Drawings.md)

  Save drawing files for later use just as you do with other applications. You can also set up automatic saving and backup
  files and save only selected objects.
* [Repair, Restore, and Recover Drawings](topichead-7.htm)
* [Import and Export Drawing Data](topichead-8.htm)
