# Package Drawing Dialog Box

Packages a set of drawings and dependency files into one unit for sharing.

ETRANSMIT (Command)

*Toolbar*:
![](../images/GUID-D5ECD674-84D1-4B3A-AB6A-F2264AA36BC2-low.png)

![](../images/GUID-CBBBC520-CFB5-4B72-A06D-1434CD4AB9BE-low.png)

## Summary

By default, all files associated with the current drawing (such as related xrefs, plot styles, and fonts) are listed. You
can add files to the transmittal package or remove existing files.

If a sheet in the list is unavailable, the sheet is referenced (as an xref) by another sheet in the transmittal package,
and the unavailable sheet is automatically included in the transmittal package.

The drawings to be transmitted are listed under the following categories:

* *Current Drawing.* Lists the files associated with the current drawing.
* *User Added Files.* Lists the files that have been added manually with the Add File option.

## List of Options

The following options are displayed.

Files

Displays the list of files to be included in the transmittal package. You can add and remove files by clicking the plus (+)
and minus (-) signs.

Package Settings

Displays the Package Settings dialog box to specify additional transmittal options.

Package Notes

Provides a space to enter notes related to a transmittal package. The notes are included in the transmittal report.

View Report

Displays report information that is included with the transmittal package. Includes any transmittal notes that you entered
and distribution notes automatically generated that detail what steps must be taken for the transmittal package to work properly.
For example, if SHX fonts are detected in one of the transmittal drawings, you are instructed where to copy these files so
that they can be detected on the system where the transmittal package is being installed. If you have created a text file
of default notes, the notes are also included in the report.
