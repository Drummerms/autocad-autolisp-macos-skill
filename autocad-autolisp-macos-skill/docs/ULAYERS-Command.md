# ULAYERS (Command)

Controls the display of layers in a PDF underlay.

## Access Methods

![](../images/ac.mouse.gif) Toolbar: PDF Underlay visor ![](../images/ac.menuaro.gif) Underlay Layers.
![](../images/GUID-64C287A2-38AC-4691-A9AF-83DC92ADA84E-low.png)

Shortcut menu: Select an underlay. Right-click in the drawing area and choose View PDF Layers.

## Summary

The Underlay Layers dialog box is displayed.
