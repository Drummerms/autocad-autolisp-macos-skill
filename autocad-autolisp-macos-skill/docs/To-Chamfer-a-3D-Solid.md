# To Chamfer a 3D Solid

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Fillet Edge drop-down ![](../images/ac.menuaro.gif) Chamfer Edge.
   ![](../images/GUID-55ADD811-CA11-4FD5-8923-F2433BC22951-low.png)
2. Select the edge of the base surface to chamfer.

   One of two surfaces adjacent to the selected edge is highlighted.
3. Do one of the following:
   * To select a different surface, enter
     *n* (Next).
   * To use the current surface, press Enter.
4. Specify the base surface distance.

   The base surface distance is measured from the selected edge to a point on the base surface. The other surface distance is
   measured from the selected edge to a point on the adjacent surface.
5. Specify the location of the chamfer using one of the following options.
   * To specify an individual edge, select the edge.
   * To select all edges around the base surface, enter
     *l* (Loop). Specify an edge.
6. To complete the chamfer, press Enter.
