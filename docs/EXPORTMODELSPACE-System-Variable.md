# EXPORTMODELSPACE (System Variable)

Specifies what part of the drawing to export to a DWF, DWFx, or PDF file from Model space

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | Display |
| 1 | Extents |
| 2 | Window |
