# LENGTHEN (Command)

Changes the length of objects and the included angle of arcs.

## Access Methods

*Tool Sets*:
Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Lengthen.
![](../images/GUID-E835CDF6-D2B4-4965-87FD-E6841924D37D-low.png)

NOTE:Hidden by default. Click
![](../images/GUID-78DED619-51B3-49C1-AD71-F928B5F69C0A-low.png) to display the icon on the tool set panel.

*Menu*:
Modify
![](../images/ac.menuaro.gif) Lengthen.

You can specify changes as a percentage, an increment, or as a final length or angle. LENGTHEN is an alternative to using
TRIM or EXTEND.

![](../images/GUID-032BCDA0-D320-4AD9-90F6-0F702E8013E2-low.gif)

The following prompts are displayed.

## Object Selection

Displays the length and, where applicable, the included angle of the object.

LENGTHEN does not affect closed objects. The extrusion direction of the selected object need not be parallel to the
*Z* axis of the current user coordinate system (UCS).

## Delta

Changes the length of an object by a specified increment, measured from the endpoint that is closest to the selection point.
Delta also changes the angle of an arc by a specified increment, measured from the endpoint that is closest to the selection
point. A positive value extends the object; a negative value trims it.

Delta Length
:   Changes the length of the object by the specified increment.

    ![](../images/GUID-D4C9507E-F53B-41AC-BC83-C0A121F22AFC-low.png)

Angle
:   Changes the included angle of the selected arc by the specified angle.

    ![](../images/GUID-BBBFCF0C-6796-424E-B23A-F54EA3868157-low.png)

## Percent

Sets the length of an object by a specified percentage of its total length.

## Total

Sets the length of a selected object by specifying the total absolute length from the fixed endpoint. Total also sets the
included angle of a selected arc by a specified total angle.

![](../images/GUID-46F13A18-D15E-46C7-B335-06CDFF6A658A-low.png)

Total Length
:   Lengthens the object to the specified value from the endpoint that is closest to the selection point.

Angle
:   Sets the included angle of the selected arc.

    ![](../images/GUID-969A7B3C-E5B0-4432-A841-1ABD03F817D9-low.png)

## Dynamic

Turns on Dynamic Dragging mode. You change the length of a selected object by dragging one of its endpoints. The other end
remains fixed.
