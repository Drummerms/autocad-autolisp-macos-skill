# To Find the Cumulative Distance Between a Series of Points

1. Click
    Tools menu ![](../images/ac.menuaro.gif) Inquiry ![](../images/ac.menuaro.gif) Distance.
2. Specify the first point.
3. Enter
   *m* (Multiple).
4. Specify additional points.
