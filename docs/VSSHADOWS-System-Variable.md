# VSSHADOWS (System Variable)

Controls whether a visual style displays shadows.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| 0 | No shadows are displayed |
| 1 | Ground shadows only are displayed |
| 2 | Full shadows are displayed |

NOTE:Existing visual styles are not changed when you enter a new value for this system variable. Any new value entered for this
system variable temporarily creates an unsaved new visual style.
