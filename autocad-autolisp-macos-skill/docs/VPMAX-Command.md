# VPMAX (Command)

Expands the current layout viewport for editing.

## Access Methods

![](../images/ac.mouse.gif) Toolbar: Status bar ![](../images/ac.menuaro.gif) Maximize Viewport
![](../images/GUID-9E0D1D0A-2D7E-4D01-BE39-D581F3DE91B8-low.png)

Pointing device: Viewport Control menu (+) ![](../images/ac.menuaro.gif) Maximize Viewport

Shortcut menu: Select a named layout viewport to maximize. Right-click and click Maximize Viewport.

Pointing device: Double-click a named layout viewport.

## Summary

The viewport is expanded to fill the screen and switched to model space for editing.
