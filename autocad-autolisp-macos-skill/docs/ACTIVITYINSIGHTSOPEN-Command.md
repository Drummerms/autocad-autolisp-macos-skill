# ACTIVITYINSIGHTSOPEN (Command)

Opens the Activity Insights palette to view activities for the current drawing.

## Access Methods

*Toolbar*:
![](../images/GUID-9B46E37A-20F4-4690-85EE-031FE503F1A9-low.png) Find

*Menu:
Window ![](../images/ac.menuaro.gif) Activity Insights*

![](../images/GUID-5A3AC27F-DABD-43E5-A9CA-8D369711C7E4-low.png)

Once your
[Activity Insights palette](About-Activity-Insights.md) is open, you can
[filter activities within](To-Work-with-Activity-Insights.md) it by date, user, activity, or file name.
