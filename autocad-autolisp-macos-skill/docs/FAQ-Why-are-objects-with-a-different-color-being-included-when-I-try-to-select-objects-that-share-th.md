# FAQ: Why are objects with a different color being included when I try to select objects that share the same color?

Objects are considered similar if they are set to BYLAYER for properties like color, linetype, lineweight, plot style, material,
and transparency. For example, a blue polyline and a red polyline will both be selected if their color property is set to
ByLayer even though they are not the same color.
