# Work with Blocks

#### Topics in this section

* [Overview of Blocks](Overview-of-Blocks.md)

  A block can be composed of objects drawn on several layers with various properties. You can use several methods to create
  blocks.
* [About Block Units and Insertion Scale](About-Block-Units-and-Insertion-Scale.md)

  When a block is defined, it can be assigned a unit of measurement which controls the automatic scaling of the block during
  insertion into a drawing.
* [About Inserting Blocks](About-Inserting-Blocks.md)

  Save time and control the drawing size by inserting references to a set of objects that have been combined to form a block.
  When you insert a block, you create a block reference and specify its location, scale, and rotation.
* [About Modifying Block References](About-Modifying-Block-References.md)

  You can modify a single insertion of a block without affecting its definition.
* [About Modifying Block Definitions](About-Modifying-Block-Definitions.md)

  You can modify a block definition using several methods.
* [About Defining Blocks](About-Defining-Blocks.md)

  You can create blocks by associating objects and giving them a name or by creating a drawing to be used as a block.
* [About Working With the Block Editor](About-Working-With-the-Block-Editor.md)

  The Block Editor provides an easy way to define and edit blocks and to add dynamic behaviors to block definitions.
* [About Organizing Blocks](About-Organizing-Blocks.md)

  You can use block libraries to organize and manage blocks.
