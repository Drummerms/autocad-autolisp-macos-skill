# To Open Drawings Stored Online

## Open a Drawing from an FTP Site

1. First, connect to the FTP site outside of AutoCAD.
   1. Click
      Finder menu ![](../images/ac.menuaro.gif) Go ![](../images/ac.menuaro.gif) Connect to Server.
   2. Enter the server address in the box provided, ftp://{server}.
   3. Click Connect.
2. In AutoCAD, click
   File menu ![](../images/ac.menuaro.gif) Open.
3. In the dialog box, select the server from the available locations.
4. Browse to and double-click the file name.
