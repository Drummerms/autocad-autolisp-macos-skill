# LWEIGHT (Command)

Sets the current lineweight, lineweight display options, and lineweight units.

## Access Methods

*Menu*:
Format
![](../images/ac.menuaro.gif) Lineweight.

Shortcut menu: Right-click Show/Hide Lineweight on the status bar and choose Settings.

![](../images/ac.keyboard.gif) Command entry:  *'lweight* for transparent use

## Summary

The
[Lineweight Settings dialog box](Lineweight-Settings-Dialog-Box.md) is displayed.

If you enter
*-lweight* at the Command prompt,
[options are displayed](LWEIGHT-Command-2.md).
