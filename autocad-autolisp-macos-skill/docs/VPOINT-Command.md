# VPOINT (Command)

Sets the viewing direction for a 3D visualization of the drawing.

## Access Methods

![](../images/ac.mouse.gif) Menu:   View  ![](../images/ac.menuaro.gif)  3D Views  ![](../images/ac.menuaro.gif)  Viewpoint Not available in menus in the current workspace

## List of Prompts

The following prompts are displayed.

Specify a [view point](VPOINT-Command.md) or [[Rotate](VPOINT-Command.md)] <[display compass and tripod](VPOINT-Command.md)>: *Specify a point, enter* *r**, or press* Enter *to display a compass and axis tripod*

View Point
:   Creates a vector that defines a direction from which the drawing can be viewed. The view defined is as if the viewer is looking
    from the point back at the origin (0,0,0).

    ![](../images/GUID-BA0D6EED-C84B-4673-B1FF-7D72D532A0DB-low.png)

Rotate
:   Specifies a new viewing direction using two angles.

    * *Enter angle in XY plane from X axis.*

    The first angle is specified with respect to the *X* axis, in the *XY* plane.

    * *Enter angle from XY plane.*

    The second angle is specified up or down from the *XY* plane.

    ![](../images/GUID-49067B7A-DF74-4866-82DC-077C43CBDC73-low.png)

Compass and Axis Tripod
:   Displays a compass and axis tripod, which you use to define a viewing direction in the viewport.

    ![](../images/GUID-0E339EC2-0E75-4F97-8EDC-67ABA64B5FA8-low.png)

    The compass is a two-dimensional representation of a globe. The center point is the north pole (0,0,*n*), the inner ring is the equator (*n*,*n*,0), and the entire outer ring is the south pole (0,0,-*n*).

    As you move the crosshairs, the axis tripod rotates to conform to the viewing direction indicated on the compass. To select
    a viewing direction, move your pointing device to a location on the globe and click.
