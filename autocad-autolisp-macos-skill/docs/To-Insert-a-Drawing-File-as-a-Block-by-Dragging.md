# To Insert a Drawing File as a Block by Dragging

## From the Blocks Palette

1. Click
   Window menu ![](../images/ac.menuaro.gif) Blocks.
2. Select which pane to display: Current
   ![](../images/GUID-0AAC56FF-83EA-4D4F-84E3-260C5F78A5C2-low.png), Recent
   ![](../images/GUID-9238CBF0-4279-429A-89B3-A1819BD3C918-low.png), or Block Libraries
   ![](../images/GUID-07D74F18-E70E-406E-B4F3-E196C25F310B-low.png).
3. From the palette, drag the drawing file into the drawing canvas.
4. Move the mouse to the location you want to place the drawing and release the mouse button.

## From Finder

1. From Finder, drag the drawing file into the drawing canvas.
2. Move the mouse to the location you want to place the drawing.

   When you release the button, you are prompted whether you want to insert or attach the drawing.
3. Enter
   *I* to insert the drawing as a block or
   *A* to attach the drawing as an xref.
