# XEDIT (System Variable)

Controls whether the current drawing can be edited in-place when being referenced by another drawing.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| 0 | Cannot use in-place reference editing |
| 1 | Can use in-place reference editing |
