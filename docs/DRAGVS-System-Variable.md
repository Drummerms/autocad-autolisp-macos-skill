# DRAGVS (System Variable)

Sets the visual style that is displayed while creating 3D solid and mesh primitives and extruded solids, surfaces, and meshes.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | "" |

You can enter a period (.) to specify the current visual style. DRAGVS can only be set to a visual style that is saved in
the drawing.

DRAGVS has no effect when the current viewport's visual style is set to 2D Wireframe.

The visual style specified for DRAGVS cannot be removed with the
[PURGE](PURGE-Command.md) command.
