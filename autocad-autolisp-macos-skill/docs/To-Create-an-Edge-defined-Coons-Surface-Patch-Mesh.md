# To Create an Edge-defined Coons Surface Patch Mesh

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Mesh panel ![](../images/ac.menuaro.gif) Modeling, Meshes, Edge Surface.
   ![](../images/GUID-20D37DA1-E8A0-44AD-850C-90BED133FA5A-low.png)
2. Select four objects to define the four adjoining edges of the mesh patch.

   The objects can be arcs, lines, polylines, splines, or elliptical arcs that form a closed loop and share endpoints.

   The first edge you select determines the
   *M* direction of the mesh.
