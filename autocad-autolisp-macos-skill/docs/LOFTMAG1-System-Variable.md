# LOFTMAG1 (System Variable)

Sets the magnitude of the draft angle through the first cross section in a loft operation.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 0.0000 |

Controls how soon the surface starts bending back toward the next cross section.
