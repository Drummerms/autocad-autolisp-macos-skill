# DIMTOL (System Variable)

Appends tolerances to dimension text.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Drawing |
| Initial value: | 0 |

Setting DIMTOL to on (1) turns DIMLIM off (0).
