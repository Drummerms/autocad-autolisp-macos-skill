# Add Scales to Object Dialog Box

Adds a new scale to the selected annotative object.

## List of Options

The following option is displayed.

Scale List
:   Displays the list of scales that can be added to the selected annotative object.

    Multiple scales can be selected by holding down the Shift or Command key while selecting the scales.

    Use the
    [SCALELISTEDIT](SCALELISTEDIT-Command.md) command to add custom scales to this list.
