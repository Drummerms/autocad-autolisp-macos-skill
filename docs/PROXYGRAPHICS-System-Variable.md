# PROXYGRAPHICS (System Variable)

Specifies whether images of proxy objects are saved in the drawing.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| 0 | Does not save image with the drawing; a bounding box is displayed instead |
| 1 | Saves image with the drawing |
