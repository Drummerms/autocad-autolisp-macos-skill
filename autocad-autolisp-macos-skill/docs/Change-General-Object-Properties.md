# Change General Object Properties

#### Topics in this section

* [Overview of Object Properties](Overview-of-Object-Properties.md)
* [Display and Change the Properties of Objects](Display-and-Change-the-Properties-of-Objects.md)

  You can display and change the current properties for any object in your drawing.
* [About Layers](About-Layers.md)
* [About Setting the Color of Objects](About-Setting-the-Color-of-Objects.md)

  You can set the color of an object either by its layer or by specifying its color explicitly, independent of its layer.
* [About Linetypes](About-Linetypes.md)

  A linetype is a repeating pattern of dashes, dots, and blank spaces displayed in a line or a curve. You assign linetypes
  to objects either by layer or by specifying the linetype explicitly, independent of layers.
* [About Lineweights](About-Lineweights.md)

  Lineweight is a property assigned to graphical objects, hatches, leader lines, and dimension geometry that results in thicker,
  darker lines.
* [About the Draw Order of Overlapping Objects](About-the-Draw-Order-of-Overlapping-Objects.md)

  You can change the draw order, which is the display and plotting order, of specified objects.
* [About Making Objects Transparent](About-Making-Objects-Transparent.md)

  You can control the transparency for selected objects, or for all objects on a layer. You can also specify a default transparency
  value for new hatch and fill objects.
