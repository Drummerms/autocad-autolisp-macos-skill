# TRACEPALETTESTATE (System Variable)

Reports whether the Traces palette is open or closed.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| Value | Description |
| 0 | Closed |
| 1 | Open |
