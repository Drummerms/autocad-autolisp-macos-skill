# COMPARECOLOR2 (System Variable)

Sets the color of the objects that only exist in the second drawing in the comparison result drawing.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

Valid values include the following:

* AutoCAD Color Index (ACI): integer values from 1 to 255, or a color name from the first seven colors
* True Colors: RGB or HSL values from 000 to 255 in the form "RGB:255,0,0"
* Color Books: Text from standard PANTONE or custom color books, the DIC color guide, or RAL color sets
