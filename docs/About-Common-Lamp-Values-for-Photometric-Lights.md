# About Common Lamp Values for Photometric Lights

The color of light emitted by a photometric light can be adjusted using the light's Filter Color and Lamp Color properties.
When you adjust a light's Lamp Color property, you can specify a standard color or a Kelvin temperature value. The Kelvin
temperature value of a light can be in the range of 1,000 to 20,000.

A light can be set to one of the following standard colors:

* D65 White
* Fluorescent
* Cool White
* White Fluorescent
* Daylight Fluorescent
* Incandescent
* Xenon
* Halogen
* Quartz
* Metal Halide
* Mercury
* Phosphor Mercury
* High Pressure Sodium
* Low Pressure Sodium

## Light Tools on Tool Palettes

The Tool Palettes window contains a number of tool palettes with light tools that represent commonly used lamp values. Clicking
a light tool on a tool palette allows you to place a new light with a specified lamp value into a drawing. The following is
a list of the tool palettes that contain commonly used lamps:

* Fluorescent
* High Intensity Discharge
* Low Pressure Sodium
* Incandescent
