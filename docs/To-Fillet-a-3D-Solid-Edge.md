# To Fillet a 3D Solid Edge

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Fillet Edge drop-down ![](../images/ac.menuaro.gif) Fillet Edge.
   ![](../images/GUID-D10626BF-02B8-45ED-B696-FF47C3910AAB-low.png)
2. Select the edge of the solid to fillet.
3. Specify the fillet radius.
4. Select additional edges or press Enter.
