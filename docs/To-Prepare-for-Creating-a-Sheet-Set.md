# To Prepare for Creating a Sheet Set

1. *Consolidate drawing files.* Move the drawing files to be used in the sheet set into a small number of folders. This will simplify sheet set administration.
2. *Eliminate multiple layout tabs.* For each drawing used in a sheet set, it's recommended that the drawing contains a single named layout and that the name
   of the layout does not include a # character. Using only one named layout or sheet per drawing allows for better collaboration
   among multiple users. When opening a sheet from a drawing that contains multiple sheets, only one user can edit that drawing
   at a time.
3. *Create a sheet creation template.* Create or identify a drawing template (DWT) file to be used by the sheet set for creating new sheets. This drawing template
   file is called the sheet creation template. You specify this template file in the Sheet Set Properties dialog box or the Subset
   Properties dialog box.
4. *Create a page setup overrides file.* Create or identify a DWT file to store page setups for plotting and publishing. This file, called the
   *page setup overrides file*, can be used to apply a single page setup to all sheets in a sheet set, overriding the individual page setups stored in each
   drawing.
