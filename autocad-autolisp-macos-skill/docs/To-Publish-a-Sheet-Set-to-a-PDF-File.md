# To Publish a Sheet Set to a PDF File

1. Click
   Window menu ![](../images/ac.menuaro.gif) Sheet Set Manager.
2. In the Sheet Set Manager, right-click to add a sheet set, subset, or sheet.
3. Right-click the sheet set and click Publish.
4. In the Batch Publish dialog box, select PDF in Publish To list.
5. In Publish Output, select from the available presets.
6. Set the remaining options to configure the way you want the drawings to be output.
7. Click Publish.
