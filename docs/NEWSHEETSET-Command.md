# NEWSHEETSET (Command)

Creates a new sheet set data (DST) file that manages drawing layouts, file paths, and project information.

## Access Methods

*Menu*:
File ![](../images/ac.menuaro.gif) New Sheet Set

*Menu*:
Window ![](../images/ac.menuaro.gif) Sheet Set Manager ![](../images/ac.menuaro.gif) ![](../images/GUID-40517BFA-4A82-4F1B-9580-59197EDD94CD-low.png) ![](../images/ac.menuaro.gif) New Sheet Set

## Summary

The
[New Sheet Set dialog box](New-Sheet-Set-Dialog-Box.md) allows you to create a new blank sheet set data (DST) file, use a project template, or copy an existing sheet set structure
to create a new sheet set. After a sheet set data file is created, you can create new sheets or add layouts from existing
drawings to the sheet set.
