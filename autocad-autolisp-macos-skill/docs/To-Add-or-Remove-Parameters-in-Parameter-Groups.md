# To Add or Remove Parameters in Parameter Groups

1. Click Parametric tab![](../images/ac.menuaro.gif)Manage panel![](../images/ac.menuaro.gif)Parameters Manager.![](../images/GUID-6C8237C9-7325-4FDE-9739-185DA7AE46E7-low.png)
2. In the Parameters Manager, you can do the following:
   * *Add parameters to any user-defined parameter group.* Select the parameters that you want to add to a parameter group. Drag and drop the selected parameters into a parameter group
     in the filter tree.

     A parameter can belong to multiple groups.
   * *Remove parameters from a group.* Select the parameters you want to remove. Right-click the selected parameters and choose Remove from Group Filter.
