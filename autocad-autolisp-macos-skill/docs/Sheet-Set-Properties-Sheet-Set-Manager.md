# Sheet Set Properties (Sheet Set Manager)

Displays information that is specific to the sheet set.

SHEETSET (Command)

*Menu:* Window ![](../images/ac.menuaro.gif) Sheet Set Manager: Select a sheet set in Sheet Set Manager

![](../images/GUID-B5033B45-DEAA-4700-8781-104D3C3A1165-low.png)

## List of Options

The following options are displayed.

### Project

Displays fields commonly used to identify a project.

Project Number
:   Specifies the number assigned to the project.

Project Name
:   Specifies the name assigned to the project.

Project Phase
:   Specifies the current phase of the project.

Project Milestone
:   Specifies the current milestone of the project.

Description
:   Specifies a description of the project.

### Sheet Set

Displays fields commonly used to identify a sheet set.

Sheet Set Name
:   Specifies the name of the sheet set.

Sheet Set Data File
:   Displays the path and file name of the sheet set data (DST) file.

Label Block for Views
:   Displays the path and file name of the drawing template (DWT) or drawing (DWG) file that contains the label block for the
    sheet set.

    Click the Ellipsis button to display the
    [Label Block for Views dialog box](Label-Block-for-Views-Dialog-Box.md) and specify a label block for the sheet set.

Callout Blocks
:   Click Manage to display the
    [Callout Blocks dialog box](Callout-Blocks-Dialog-Box.md) and manage the callout blocks for the sheet set.

Page Setup Overrides File
:   Displays the path and file name for the drawing template (DWT) file containing the page setup overrides for the sheet set.

    Click the Ellipsis button to specify the drawing template (DWT) file that contains the page setup overrides to use for the
    sheet set.

Sheet Storage Location
:   Displays the path and name of the folder where new drawing (DWG) files are saved when new sheets are created.

    NOTE:Clearing Sheet Title and Drawing Title under the Name Synchronization item of the Sheet Set Action menu, located in the lower-right
    corner of the Sheet Set Manager palette, enables the Ellipsis button to specify a different folder for new drawing (DWG) files.

### Template Location

Displays settings used when creating a new layout in the current sheet set.

Sheet Creation Template
:   Displays the sheet to use when creating a new sheet.

Sheet Set Creation Template Location
:   Specifies the drawing template (DWT) or drawing (DWG) file to use when creating new drawing files for the subset.

Prompt for Template
:   Controls whether the
    [New Sheet dialog box](New-Sheet-Dialog-Box.md) is displayed in the expanded or collapsed state each time a new sheet is added.

### Sheet Set Custom Properties

Displays the user-defined custom properties associated with the sheet set.

You can manage user-defined custom properties by:

* Clicking Add Property to display the
  [Add Custom Property dialog box](Add-Custom-Property-Dialog-Box.md) and create a new property.
* Right-clicking over a property and choosing Remove Custom Property to remove the property.

  You can also choose New Custom Property to display the Add Custom Property dialog box and create a new property.

### Sheet Custom Properties

Displays the user-defined custom properties associated with each sheet in the sheet set.

You can manage user-defined custom properties by:

* Clicking Add Property to display the Add Custom Property dialog box and create a new property.
* Right-clicking over a property and choosing Remove Custom Property to remove the property.

  You can also choose New Custom Property to display the Add Custom Property dialog box and create a new property.
