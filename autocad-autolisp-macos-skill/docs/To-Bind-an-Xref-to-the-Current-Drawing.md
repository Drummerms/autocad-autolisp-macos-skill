# To Bind an Xref to the Current Drawing

1. Click
   Window menu ![](../images/ac.menuaro.gif) Reference Manager.
2. In the Reference Manager, select the DWG reference.
3. Right-click the selected DWG reference and select Bind from the shortcut menu.

   The objects in the xref are converted into a block reference. Named object definitions are added to the current drawing with
   a prefix of blockname $n$, where n is a number starting at 0.

   Insert also converts the objects in the xref into a block reference. Named object definitions are merged into the current
   drawing without adding prefixes.
