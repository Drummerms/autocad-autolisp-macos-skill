# COMPARECLOSE (Command)

Closes the DWG Compare visor and ends the comparison.

## Access Methods

*DWG Compare Visor*:
![](../images/GUID-9002B3B8-BF54-447F-B816-6EA0CDF91EFC-low.png)

NOTE:This command is available only during a drawing comparison.
