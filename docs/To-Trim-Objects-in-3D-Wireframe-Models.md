# To Trim Objects in 3D Wireframe Models

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Trim.
   ![](../images/GUID-53B8A7B8-0050-44CA-A923-870AF64E125A-low.png)
2. Select the cutting edges to use for trimming (1).
3. Enter
   *p* (Project).
4. Enter
   *n* (None).
5. Select the object to trim (2 and 3).

![](../images/GUID-B869F498-2E83-42CB-8852-86E15061C372-low.png)
