# password\_char Attribute (DCL)

Specifies the character to be used to mask user input.

*Supported Platforms:* Windows, Mac OS, and Web

```
password_char = "char";
```

If
password\_char is specified and is not null, that character is displayed in the edit box instead of the characters entered by the user.
The use of this attribute has no effect on your application's retrieval of the value entered by the user; it alters only the
display of the characters in the edit box.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
