# About the ViewCube Home View

You can define a Home view for a model so you can restore a familiar view when you use the navigation tools.

The Home view is a special view stored with a model that makes it easy to return to a known or familiar view. You can define
any view of the model as the Home view. The saved Home view can be applied to the current view by clicking the Home button.

When you open a drawing that was created in an AutoCAD-based product before 2008, the extents of a model are used as the
default Home view. Drawings created with AutoCAD 2014 have a Home view defined with a Top/Left/Front orientation. While you
can use the Home view to navigate back to a familiar view, you can also use it to generate the thumbnail preview when you
save a model instead of using the last saved view.
