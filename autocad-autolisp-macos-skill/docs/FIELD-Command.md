# FIELD (Command)

Creates a multiline text object with a field that can be updated automatically as the field value changes.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/ac.menuaro.gif) Insert Field.
![](../images/GUID-CC21923E-59F3-4EBE-B889-5DE61360D96C-low.png)

*Menu*:
Insert
![](../images/ac.menuaro.gif) Field.

*Shortcut Menu*: Right-click while editing text in an in-place text editor, and click Insert Field.

## Summary

Fields can be inserted in any kind of text except tolerances. The FIELDEVAL system variable and the UPDATEFIELD command determine
how fields are updated.

The
[Insert Field dialog box](Insert-Field-Dialog-Box.md) is displayed.
