# OPEN (Command)

Opens an existing drawing file.

## Access Methods

*Toolbar*:
![](../images/GUID-B50A0504-05B5-44A6-A31D-2E72778B8CB3-low.png)

*Menu*:
File ![](../images/ac.menuaro.gif) Open.

*Shortcut Menu*: right-click on any File tab.

## Summary

The Select File dialog box (a standard file selection dialog box) is displayed.

When FILEDIA is set to 0 (zero), OPEN displays a Command prompt.
