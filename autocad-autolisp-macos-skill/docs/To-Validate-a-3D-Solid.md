# To Validate a 3D Solid

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Solid Edit - Body drop-down ![](../images/ac.menuaro.gif) Check.
   ![](../images/GUID-98EE866E-464D-49DF-84FE-5A069C57C7E5-low.png)
2. Select the 3D solid object.
3. Press Enter to complete the command.

   If the object is a valid 3D object, a message is displayed at the Command prompt. If it is not valid, you continue to be prompted
   to select a 3D solid.
