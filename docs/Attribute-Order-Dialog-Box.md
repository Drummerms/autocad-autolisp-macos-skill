# Attribute Order Dialog Box

Specifies the order in which attributes are listed and prompted for when you insert or edit a block reference.

BATTORDER (Command)

NOTE:This command is available only in the Block Editor (BEDIT).

![](../images/GUID-036CB86C-028B-4844-9183-23BEAA6EE81B-low.png)

## List of Options

The following options are displayed.

Attribute Definitions
:   Lists the attribute definitions in the current block.

    Drag an attribute definition up or down in the list to reorder it.
