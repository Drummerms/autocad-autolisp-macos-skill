# COLOR (Command)

Sets the color for new objects.

## Access Methods

*Menu*:
Format
![](../images/ac.menuaro.gif) Color.

![](../images/ac.keyboard.gif) Command entry:  *'color* for transparent use

## Summary

The
[Color Palette dialog box](Color-Palette-Dialog-Box.md) is displayed.

If you enter
*-color* at the Command prompt,
[options are displayed](COLOR-Command-2.md).
