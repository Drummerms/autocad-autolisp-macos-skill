# PROPOBJLIMIT (System Variable)

Limits the number of objects that can be changed at one time with the Properties Inspector.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 25000 |

The valid range is 0 - 32767. If the limit is exceeded, changes to the property fields do not affect the selected objects.
