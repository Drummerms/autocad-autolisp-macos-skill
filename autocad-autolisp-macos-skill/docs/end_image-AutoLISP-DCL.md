# end\_image (AutoLISP/DCL)

Ends creation of the currently active dialog box image

*Supported Platforms:* Windows, Mac OS, and Web

## Signature

```
(end_image)
```

No arguments.

## Return Values

*Type:* nil

Always returns
nil.

## Remarks

This function is the complement of
start\_image.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
* AutoCAD 2021 and later on Mac OS
