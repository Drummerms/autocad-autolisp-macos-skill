# GEOMARKERVISIBILITY (System Variable)

Controls the visibility of geographic markers.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| 0 | Geographic marker is not visible |
| 1 | Geographic marker is visible |
