# TSTACKALIGN (System Variable)

Controls the vertical alignment of stacked text.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| 0 | Bottom aligned |
| 1 | Center aligned |
| 2 | Top aligned |
