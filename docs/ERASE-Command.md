# ERASE (Command)

Removes objects from a drawing.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Erase.
![](../images/GUID-342B2C46-0E0A-431A-829D-77901015ED51-low.png)

*Menu*:
Modify ![](../images/ac.menuaro.gif) Erase.

You can erase selected objects from the drawing. This method does not move objects to the Clipboard, where they can then
be pasted to another location.

If you are working with 3D objects, you can also erase subobjects such as faces, meshes, and vertices.

Instead of selecting objects to erase, you can enter an option, such as
*L* to erase the last object drawn,
*p* to erase the previous selection set, or
*ALL* to erase all objects. You can also enter
*?* to get a list of all options.

![](../images/GUID-57AABC27-7269-486E-BE37-55FA98247C6E-low.gif)
