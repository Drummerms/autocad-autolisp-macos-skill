# About Assigning Plot Style Tables to Layouts

By assigning different plot style tables to each layout in your drawing, you can control how objects in the layout are plotted.

The plot style table affects both model space and paper space objects. To plot the drawing without applying plot style properties,
select None from the list of plot style tables.

If you use named plot style tables, each object in the drawing either is assigned a plot style directly or inherits a plot
style from its layer.

To display the effects of a plot style table in a layout, select Print with Plot Styles under Plot Style Table in the Page
Setup dialog box.

NOTE:

If you insert an xref into your current drawing, all defined plot style tables are also inserted. You can modify the appearance
of your objects by editing the attached plot style tables with the Plot Style Table Editor.
