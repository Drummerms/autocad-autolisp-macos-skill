# -RENAME (Command)

Changes the names assigned to named objects from the Command prompt.

## List of Prompts

The following prompts are displayed.

Enter object type to rename [Block/Dimstyle/LAyer/LType/Material/multileadeRstyle/Plotstyle/textStyle/Tablestyle/Ucs/VIew/VPort]: *Enter a named object type to rename an object*

Enter old object name: *Enter the old name*

Enter new object name: *Enter the new name*
