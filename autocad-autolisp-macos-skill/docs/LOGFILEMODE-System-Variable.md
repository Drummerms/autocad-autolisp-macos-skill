# LOGFILEMODE (System Variable)

Specifies whether the contents of the command history are written to a log file.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Log file is not maintained |
| 1 | Log file is maintained |
