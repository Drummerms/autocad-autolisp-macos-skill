# About Trace

Trace provides a safe space to provide feedback to a drawing without altering the existing drawing. The analogy is of a virtual
collaborative tracing paper that is laid over the drawing that allows collaborators to add markups and comments on the drawing.

Create traces in the AutoCAD web app, the AutoCAD mobile app, or AutoCAD desktop (Windows or Mac), then send or share the
drawing to collaborators so they can view the trace and its contents.

NOTE:Access the online
[AutoCAD for Mac help](https://help.autodesk.com/view/ACDMAC/2026/ENU/?guid=GUID-5D5E0BA1-10E7-4662-8118-E046EC579B08) to see the latest information for Trace.

## Using Trace

1. Create a trace or open an existing trace. You can create traces in AutoCAD desktop (Windows or Mac), AutoCAD on the web,
   and AutoCAD on mobile.

   ![](../images/GUID-B08B9C64-8A22-4A00-A527-D1147D54562A-low.png)
2. Add markup or additional design details to the trace.

   ![](../images/GUID-9A41D7B1-5BC1-43EE-B921-9225C68E6FD8-low.png)
3. Save the drawing, then share it with a colleague or collaborator.

   ![](../images/GUID-9B40F6BD-170A-4631-9D78-5162FC654D92-low.png)

When colleagues open the drawing in AutoCAD desktop (Windows or Mac), they’re notified of any new traces. Recipients can open
the trace to see what was added, then make decisions on the feedback that’s included.

NOTE:Traces are only supported in drawing file formats 2010 or later and they are not supported in the DXF format.

## Copy Trace Objects into the Drawing

The COPYFROMTRACE command allows you to copy objects from a trace into the drawing.

## Desktop, Mobile, and Web Functionality

Functionality differs slightly depending whether you're using the desktop (Windows or Mac), web, or mobile version of AutoCAD.
While you can view traces in the desktop, web, or mobile apps, you can only create or edit traces while using web and mobile.

| Functionality | Web and Mobile | Desktop (Windows or Mac) |
| Create a trace | X | X |
| Edit a trace you created | X | X |
| View traces from others | X | X |
| Edit drawing while viewing trace | X | X |
| Add and remove contributions to a trace created by others | X | X |
| Delete all traces in a drawing |  | X |
| Use Markup Assist to automatically identify markups |  | X |
| Edit xrefs while in Edit Drawing mode |  | X |
