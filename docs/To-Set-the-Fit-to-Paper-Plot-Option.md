# To Set the Fit to Paper Plot Option

1. Click the layout tab for which you want to set the plot scale to Fit to Paper.
2. Click
   File ![](../images/ac.menuaro.gif) Page Setup Manager.
3. In the Page Setup Manager, Page Setups area, select the page setup that you want to modify.
4. Click Modify.
5. In the Page Setup dialog box, Under Scale, select Fit to Paper.

   NOTE:If the Plot Area is set to Layout, you cannot select the Fit to Paper option.
6. Click OK.
7. In the Page Setup Manager, click Close.
