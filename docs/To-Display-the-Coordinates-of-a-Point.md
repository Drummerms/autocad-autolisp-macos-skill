# To Display the Coordinates of a Point

1. Click Tools ![](../images/ac.menuaro.gif) Inquiry ![](../images/ac.menuaro.gif) ID Point.At the Command prompt, enter id.
2. Click the location that you want to identify.

   The *X,Y,Z* coordinate values are displayed at the Command prompt. With object snaps turned on, you can select an object and see the
   coordinates for a feature such as an endpoint, midpoint, or center.
3. You can reference the last point by entering @ at the next prompt that requests a point.
