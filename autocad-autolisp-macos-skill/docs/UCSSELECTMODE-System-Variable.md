# UCSSELECTMODE (System Variable)

Controls whether the UCS icon can be selected and manipulated with grips.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | The UCS icon is not selectable. |
| 1 | The UCS icon is selectable. |
