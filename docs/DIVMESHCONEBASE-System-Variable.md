# DIVMESHCONEBASE (System Variable)

Sets the number of subdivisions between the perimeter and the center point of the mesh cone base.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 3 |

![](../images/GUID-268E2A28-7FE4-4FF7-B13A-17EF1F97B1A5-low.png)

This setting affects new mesh cones that are created with the [MESH](MESH-Command.md) command.
