# acet-reg-machine-prodkey (AutoLISP/Express Tools)

Returns the AutoCAD Windows registry path in the HKLM (HKEY\_LOCAL\_MACHINE).

*Supported Platforms:* AutoCAD for Windows only; not available in AutoCAD LT for Windows, or on Mac OS and Web

## Signature

```
(acet-reg-machine-prodkey)
```

No arguments.

## Return Values

*Type:* String

A value containing the AutoCAD registry path.

## Remarks

The AutoCAD registry path can be used to register an application for demand loading.

NOTE:acet-reg-machine-prodkey replaces
acet-reg-prodkey.
