# To List the Linetypes in a Linetype Definition File

1. Click
   Format menu ![](../images/ac.menuaro.gif) Linetype.
2. In the Linetype Manager dialog box, click +, and in the Load or Reload Linetypes dialog box, click
   ![](../images/GUID-F7FBE0E7-8A17-43FA-8CCF-7F9A75BFAF27-low.png).
3. In the Select Linetype File dialog box, choose a LIN (linetype definition) file, and click Open.

   The dialog box displays the linetype definitions stored in the LIN file.
