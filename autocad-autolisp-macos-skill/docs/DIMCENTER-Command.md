# DIMCENTER (Command)

Creates the center mark or the centerlines of circles and arcs.

## Summary

The default sizes of the center mark components can be set from the Dimension Style Manager, Symbols and Arrows tab, Center
Marks (DIMCEN system variable).

![](../images/GUID-EC206B8B-D16F-4CAC-9B5F-8DCE4E6A811B-low.gif)

You can choose between center marks and centerlines and specify their size when you set up the dimension style. See
[DIMSTYLE](DIMSTYLE-Command-2.md). You can also change center mark settings using the
 [DIMCEN](DIMCEN-System-Variable.md) system variable.

![](../images/GUID-E88FE88F-E9BE-445C-A8CB-35F38F8156D8-low.png)
