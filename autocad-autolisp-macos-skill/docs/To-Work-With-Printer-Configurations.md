# To Work With Printer Configurations

To print a layout, you select a printer or plotter and configure the device as needed.

## Select a Printer for a Layout

1. Click the layout tab for which you want to specify a printer or plotter.
2. Click
   File ![](../images/ac.menuaro.gif) Page Setup Manager.
3. In the Page Setup Manager, Page Setups area, select the page setup that you want to modify.
4. Click Modify.
5. In the Page Setup dialog box, under Printer/Plotter, select a printer or plotter from the list. Click OK.
6. In the Page Setup Manager, click Close.
