# To Work With Attaching and Detaching Referenced Drawings

## Attach an Xref (Reference)

1. Click Insert menu![](../images/ac.menuaro.gif)DWG Reference![](../images/ac.menuaro.gif).At the Command prompt, enter XATTACH.
2. In the Select Reference File dialog box, select one or more files you want to attach and then click Open.
3. In the Attach External Reference dialog box, under Reference Type, select Attachment.
4. Specify the insertion point, scale, and rotation angle. Click Specify On-Screen to use the pointing device.

   Attachment includes all nested xrefs (references).
5. Click OK.

## Attach or Overlay an Xref Using DesignCenter

1. Click View tab ![](../images/ac.menuaro.gif) Palettes panel ![](../images/ac.menuaro.gif) DesignCenter. ![](../images/GUID-1CD3ADC1-E360-4EA9-9CF3-544673FEE5F3-low.png)
2. In the content area or in the Search dialog box, locate the DWG reference you want to attach or overlay.
3. Right-click. Drag the DWG reference into the open drawing.
4. Release the right-pointing device button. Click Attach as Xref (reference).
5. In the External Reference dialog box, under Reference Type, select Attachment or Overlay.
6. Enter values for Insertion Point, Scale, and Rotation, or select Specify On-Screen to use the pointing device.
7. Click OK.

   You can also attach an xref by dragging or by clicking Attach as Xref on the shortcut menu.

## Attach an Xref Using Autodesk Vault

You must be logged into the vault to perform vault tasks in the External References palette.

1. Access the External References palette using one of the following methods:
   * Click Tools menu![](../images/ac.menuaro.gif)Palettes![](../images/ac.menuaro.gif)Reference Manager.At the Command prompt, enter REFERENCEMANAGER.  Right-click the background of the dialog box, and then select Attach DWG.

     NOTE:To attach a DWF file, right-click the background of the dialog box, and then select Attach DWF.
2. In the Select Reference File dialog box, select the file you want to attach, and then click Open.
3. In the External Reference dialog box under Reference Type, select Attachment.
4. Specify the insertion point, scale, and rotation angle. Click Specify On-Screen to use the pointing device. Attachment includes
   all nested xrefs.
5. Click OK.

   NOTE:You can attach a new instance of an existing xref. Right-click one of the xrefs in the External References palette window,
   and then select Attach.

## Manage Autodesk Vault Check In, Refresh, and Logins

You must be logged into the vault to perform vault tasks in the External References palette.

* To Check In, Check Out, Undo Checkout, or Refresh File, right-click a file in the External References palette.

  NOTE:In the List View, multiple files can be selected by holding down the Ctrl key to select individual files or by holding down
  the Shift key to select a group of files.
* To Attach from Vault, Log into Vault, or Log out of Vault, right-click the background of the External References palette.

## Detaching Xrefs

1. Click Tools menu![](../images/ac.menuaro.gif)Palettes![](../images/ac.menuaro.gif) Reference Manager.
2. In the External Reference palette, select a DWG reference.
3. Right-click the selected DWG reference and select Detach from the shortcut menu.

## Adjust Xref Fading (Transparency)

1. Click Insert tab ![](../images/ac.menuaro.gif) Reference panel ![](../images/ac.menuaro.gif) Xref Fading.![](../images/GUID-C00D5B05-79BA-4168-9ECE-DACC584F8814-low.png)
2. Drag the slider to adjust the fade display of the DWG xref.

   The DWG xref fades as you drag the slider.

## View Referenced Files by Type

1. Click Insert tab ![](../images/ac.menuaro.gif) Reference panel ![](../images/ac.menuaro.gif) Dialog box launcher.![](../images/GUID-6EC866D6-17B4-4BF1-81D1-9AB5FB49A6CB-low.png)
2. In the External References palette, in the File References pane, click the List View button.
3. Click the Icon column.

   The attached referenced files are sorted by type in the list view.
