# HPISLANDDETECTIONMODE (System Variable)

Controls whether internal closed boundaries, called islands, are detected.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| 0 | Off |
| 1 | On (recommended). Hatches or ignores islands according to HPISLANDDETECTION. |
