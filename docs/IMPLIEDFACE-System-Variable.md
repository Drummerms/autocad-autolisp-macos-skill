# IMPLIEDFACE (System Variable)

Controls the detection of implied faces.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Implied faces cannot be detected. |
| 1 | Implied faces can be detected. |

An *implied face* is a closed area that creates a boundary by intersecting or overlapping a face on a 3D solid object.

This variable affects the ability to use implied faces when you press Ctrl-Shift-E and drag the area inside the boundary to
create an extrusion dynamically.

This variable must be set to 1 if you want to select and modify implied faces.
