# CENTERRESET (Command)

Resets the centerlines to the current value specified in the CENTEREXE system variable.

The CENTERRESET command does not restore the center mark or centerline back to the originally created size. This system variable
resets only the length of the centerline extension as specified in the CENTEREXE system variable.

The following prompt is displayed.

Select Center Mark or Centerline
:   Selects the object with the centerlines you want to reset.
