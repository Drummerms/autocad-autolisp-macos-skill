# TOLERANCE (Command)

Creates geometric tolerances contained in a feature control frame.

## Access Methods

*Tool Sets*:
Drafting tab ![](../images/ac.menuaro.gif) Dimension panel ![](../images/ac.menuaro.gif) Tolerance.
![](../images/GUID-A9AB53BA-3EAF-4417-A1B5-CF3FF3F53BB0-low.png)

*Menu*:
Dimension ![](../images/ac.menuaro.gif) Tolerance.

## Summary

The
[Geometric Tolerance dialog box](Geometric-Tolerance-Dialog-Box.md) is displayed.

Geometric tolerances show acceptable deviations of form, profile, orientation, location, and runout. Feature control frames
can be created with leader lines using TOLERANCE, LEADER, or QLEADER.

![](../images/GUID-D8CE5BEC-5E95-4972-835D-B556EDF84D6A-low.gif)
