# Calculate the Length of a Vector (CAL Command)

The
*abs*function calculates the length of a vector.

abs(v)
:   Calculates the length of vector
    *v*, a nonnegative real number.

    In spherical coordinates (*dist<ang<ang*), the
    *dist* is the length of the vector.

    The following example calculates the length of the vector [1,2,3]:

    *abs([1,2,3])*
