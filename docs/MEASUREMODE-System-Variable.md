# MEASUREMODE (System Variable)

Controls whether the MEASUREGEOM command always defaults to the Quick option.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 0 |

| Value | Description |
| 0 | The last option used is the default |
| 1 | Defaults to the Quick option |
