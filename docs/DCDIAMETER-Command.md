# DCDIAMETER (Command)

Constrains the diameter of a circle or an arc.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Parametric panel ![](../images/ac.menuaro.gif) Dimensional Constraint drop-down ![](../images/ac.menuaro.gif) Diameter.
![](../images/GUID-116B2A88-E8A3-47B7-9696-4E33857BFCB1-low.png)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Parametric ![](../images/ac.menuaro.gif) Dimensional Constraints ![](../images/ac.menuaro.gif) Diameter.

## Summary

This command is equivalent to the Diameter option in
[DIMCONSTRAINT](DIMCONSTRAINT-Command.md).

The following table outlines the valid contraint objects and points:

| Valid Objects or Points | Characteristics |
| Circle  Arc | * Constrains the diameter of the circle or arc. |

## List of Prompts

The following prompts are displayed.

Select arc or circle:
*Select an arc or a circle to be constrained*

Dimension Line Location

Determines where the dimension line is located on the constrained object.
