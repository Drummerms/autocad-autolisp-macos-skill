# CENTERREASSOCIATE (Command)

Associates or reassociates a center mark or centerline object to selected objects.

The following prompts are displayed.

Select a Center Mark or Centerline
:   Specifies a center mark or centerline to reassociate to a selected object.

Select Circle or Arc
:   Specifies a circle or an arc to associate the selected center mark.

Select First Line
:   Specifies the first of the two lines required to associate the selected centerline.

Select Second Line
:   Specifies the second of the two lines required to associate the selected centerline.
