# To Create a Procedural Surface by Extruding Along a Path

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Modeling drop-down ![](../images/ac.menuaro.gif) Extrude.
   ![](../images/GUID-CA5C3149-50BE-4B87-9D21-1B4BD3E58ED4-low.png)
2. Select the objects or edge subobjects to extrude.
3. At the Command prompt, enter
   *p* (Path).
4. Select the object or edge subobject to use as the path.

   After the extrusion, the original objects are deleted or retained, depending on the setting of the DELOBJ system variable.
