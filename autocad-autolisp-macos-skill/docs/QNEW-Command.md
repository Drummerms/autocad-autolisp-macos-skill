# QNEW (Command)

Starts a new drawing from a selected drawing template file.

## Access Methods

*Toolbar*:
![](../images/GUID-328B7211-ADE2-46C6-BE30-98CF021E3440-low.png)

## Summary

QNEW creates a new drawing from the default drawing template file and folder path specified in Default Template File Name
for QNEW in the Application section of the Application Preferences dialog box.

When a default drawing template file is set to
*None* or is not specified, QNEW displays the Select Template File dialog box (a
[standard file selection dialog box](Standard-File-Selection-Dialog-Boxes.md)).

If the system variable, FILEDIA, is set to 0 instead of 1, a
[Command prompt](NEW-Command-Prompt.md) is displayed.
