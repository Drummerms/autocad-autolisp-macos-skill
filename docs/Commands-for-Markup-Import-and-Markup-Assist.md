# Commands for Markup Import and Markup Assist

#### Commands

* [-MARKUPASSIST](GUID-26A9073B-A2E2-4E2C-AB90-53614C9F9795.htm "At the Command prompt, analyzes an imported markup and can help place text callouts and revision clouds faster and with less manual effort.
  		")
* [-MARKUPIMPORT](GUID-5B1F0D89-D858-45DB-89C5-4093037DF959.htm " At the command prompt, imports a marked up drawing (image/pdf) in-place into your DWG as a new trace.
  		")
* [FADEMARKUP](GUID-D8793553-96C2-4177-A082-5359A4A85F45.htm "Fades individual markups so they are less visible on a trace.
  		")
* [MARKUPASSIST](GUID-D505C129-7D84-415E-97D8-2B20F59730BC.htm " Analyzes an imported markup and can help place text callouts and revision clouds faster and with less manual effort.
  		")
* [MARKUPIMPORT](GUID-2C39EFFC-FA21-419A-9B7A-4E06B4FFF554.htm " Imports a marked up drawing (image/pdf) in-place into your DWG as a new trace.
  		")

#### System Variables

* [COMMENTHIGHLIGHT](GUID-1409E1EB-D265-4FD4-94FA-452E9B75CBF2.htm "Controls the display of the indicator badge on PDF text comments.
  		")
* [MARKUPASSISTMODE](GUID-7C483FAC-793E-422F-B500-4D6F7AE96A89.htm " Controls whether identified markups are highlighted.
  		")
* [MARKUPPAPERDISPLAY](GUID-9F391B6F-5065-40C3-AF02-5E252F9FD807.htm "Indicates whether or not a digital markup is currently active.
  		")
* [MARKUPPAPERTRANSPARENCY](GUID-49945E9C-9B38-42DD-9197-B3561CB1C945.htm "Controls the level of transparency when a digital markup is active.
  		")
* [MARKUPSELECTIONMODE](GUID-49945E9C-9B38-42DD-9197-B3561CB1C945.htm "Enables selection using boundary markup assist boxes as criteria.
  		")
* [TRACEMARKUPFADECTL](GUID-4A3C5739-32D4-4C46-B718-4E5F0FE0CA1E.htm "Controls the transparency of faded markups. The lower the number, the more visible the markup is.
  		")
* [TRACEVPSUPPORT](GUID-5AEC6D3B-597A-41CD-A654-3A078F729FDD.htm "Controls whether markup boxes are actionable within a currently active model space viewport.
  		")
