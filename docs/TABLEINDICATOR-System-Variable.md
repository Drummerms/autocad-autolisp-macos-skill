# TABLEINDICATOR (System Variable)

Controls the display of row numbers and column letters when the In-Place Text Editor is open for editing a table cell.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Off |
| 1 | On |
