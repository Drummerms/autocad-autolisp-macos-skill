# To Set Dimension Scale for Creating Dimensions in a Layout

1. Click a layout tab to switch to paper space.
2. To create dimensions in paper space with the correct model space dimension values, use object snap modes to snap to points
   in model space from paper space or select the objects directly.

   The DIMLFAC system variable can be changed if you need to convert the linear dimension values between the imperial and metric
   measurement systems.
