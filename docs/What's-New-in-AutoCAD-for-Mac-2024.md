# What's New in AutoCAD for Mac 2024

Learn about new features in this release of AutoCAD for Mac 2024 release.

## Updates

## 2024.1 Highlights

* [Markup Import and Assist Improvements](Markup-Import-and-Markup-Assist-3.md) - Markup detection can now identify markups in flattened PDFs.
* [Trace Enhancements](Trace.md) - Traces palette has been updated to make it easier to create a new trace or import markup.

## Additional enhancements

*File Open*: Improved file open performance when opening a drawing that contains large numbers of annotation scales. Large numbers of
annotation scales can occur when there are lots of drawing references that might pull in additional annotation scales.

*Pan and Zoom*: General performance improvements when using pan and zoom commands.

## 2024 Highlights

* [Apple Silicon Native Support](Apple-Silicon-Support.md):
  AutoCAD for Mac now runs natively on Apple Silicon Mac machines. Overall performance improved significantly.
* [Italian and Spanish](Italian-and-Spanish.md):
  AutoCAD for Mac now supports Italian and Spanish.
* [Markup Import and Markup Assist](Markup-Import-and-Markup-Assist.md): Markups can be imported as a PDF, PNG, or JPG and are overlaid on top of your drawing in the Trace workspace. Markups in
  the imported file are automatically identified as mtext, mleaders, and revision clouds. Markup Assist lets you insert those
  identified markups into the drawing as geometry.
* [Trace Updates](Trace-Updates-2.md) - The Trace environment now includes new settings controls on the toolbar.
* [Smart Blocks](Smart-Blocks-Placement.md): As you insert a block from the Blocks palette, the Block Placement engine displays placement suggestions based on where
  you've placed that block before in the drawing.
* [3D Color Background Application Preference Updates](3D-Color-Background-Application-Preference-Updates.md): In the AutoCAD Preferences dialog, the Look and Feel tab now has settings to customize the background color for 3D visual
  styles including both parallel and perspective mode.
* [3Dconnexion Device Support](3Dconnexion-Device-Support.md): You can use the 3Dconnexion device to reorient and navigate a model's view.

## Additional enhancements

*PDFSHX System Variable*: Set PDFSHX to 2 to store text objects that use SHX fonts as hidden text when the drawing is exported to a PDF. This makes
the text searchable and selectable in the PDF.

*3D Graphics*: Below is a list of improvements that have been made to the Shaded (Fast) and Shaded with Edges (Fast) visual styles.

* Selection (Fence, Lasso, Polygon)
* Paper space support
* TrueType font support
* Line weights
* Locked layer dimming
* Xref dimming
* General stability, visual fidelity, and performance improvements

*2D Graphics*: Below is a list of improvements that have been made to the 2D graphics system.

* TrueType font display quality is improved
* Pan performance is improved for drawings with raster images or wipeouts
* Display performance is improved for trace background and ViewCube
* Layout switching performance improvement
