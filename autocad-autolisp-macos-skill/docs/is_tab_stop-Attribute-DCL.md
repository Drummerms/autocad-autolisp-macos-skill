# is\_tab\_stop Attribute (DCL)

Specifies whether the tile receives keyboard focus when the user moves between tiles by pressing the Tab key.

*Supported Platforms:* Windows only

```
is_tab_stop = true-false;
```

Possible values are
true or
false (default:
true). If the tile is disabled, it is not a tab stop even if this attribute is
true. If
false, the tile is not a tab stop.

## Release Information

*Releases:*

* AutoCAD R12 and later on Windows
