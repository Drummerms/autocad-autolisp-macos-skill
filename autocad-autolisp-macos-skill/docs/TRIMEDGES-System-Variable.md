# TRIMEDGES (System Variable)

Controls whether trimming and extending to hatches with Quick mode is limited to the edges of the hatches or includes the
objects within hatch patterns.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | Uses all hatch objects as boundaries |
| 1 | Uses hatch edges only |

NOTE:When set to 1, to extend objects to hatch boundaries only, use Shift-select to switch the TRIM command to extend mode.
