# ACISOUT (Command)

Exports a body object, solid, or region to an ACIS file.

Exported files are saved in SAT (ASCII) format.

The following prompt is displayed.

Select objects
:   Select 3D solids, regions, or legacy body objects to export. (Other types of objects are ignored.)

    Specify a name and location for the file.
