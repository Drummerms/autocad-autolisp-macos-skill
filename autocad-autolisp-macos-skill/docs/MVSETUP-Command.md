# MVSETUP (Command)

Sets up the specifications of a drawing.

## Summary

When you enter *mvsetup,* the prompts displayed depend on whether you are on the [Model layout (model space)](MVSETUP-on-the-Model-Layout.md) or on a [named layout (paper space)](MVSETUP-on-a-Named-Layout.md).
