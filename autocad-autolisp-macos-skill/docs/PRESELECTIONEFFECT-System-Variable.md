# PRESELECTIONEFFECT (System Variable)

Specifies the visual effect used for preselection of objects.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| Value | Description |
| 0 | Dashed and thickened lines (legacy behavior) |
| 1 | Glowing line highlighting effect |
