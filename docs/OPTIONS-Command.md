# OPTIONS (Command)

Customizes the program settings.

## Access Methods

*Menu*:
AutoCAD 2026 ![](../images/ac.menuaro.gif) Preferences

*Shortcut menu*: With no commands active and no objects selected, right-click in the drawing area. Click Preferences.

## Summary

The
[Application Preferences dialog box](Application-Preferences-Dialog-Box.md) is displayed.
