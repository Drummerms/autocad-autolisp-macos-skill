# To Clip a Layout Viewport Boundary

1. Click Modify ![](../images/ac.menuaro.gif) Clip ![](../images/ac.menuaro.gif) Viewports.At the Command prompt, enter VPCLIP.
2. Select the viewport to clip.
3. (Optional) Enter *d* (Delete) to delete an existing clipping boundary.
4. Do one of the following:
   * Enter *p* (Polygonal) to specify a series of points to define a polygonal boundary.
   * Select a paper space object that will define the new viewport boundary.
