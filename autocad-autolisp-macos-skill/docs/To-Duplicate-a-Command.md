# To Duplicate a Command

1. From the Mac OS menu bar, click Tools menu ![](../images/ac.menuaro.gif) Customize ![](../images/ac.menuaro.gif) Interface (CUI) or at the Command prompt, enter *cui* and press Enter.
2. In the Customize dialog box, Commands tab, Commands list, select the command to duplicate.
3. Click the Options action menu below the Commands list, Gear icon, and click Duplicate.

   Make the desired changes to the new copy of the command.
