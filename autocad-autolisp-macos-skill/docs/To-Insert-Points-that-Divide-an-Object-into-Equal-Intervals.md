# To Insert Points that Divide an Object into Equal Intervals

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Point drop-down ![](../images/ac.menuaro.gif) Divide.
   ![](../images/GUID-6EC11511-8D3E-4D01-92C3-883E60D59ED7-low.png)
2. Select a line, circle, ellipse, polyline, arc, or spline.
3. Enter the number of intervals.

   A point is placed between each interval.
