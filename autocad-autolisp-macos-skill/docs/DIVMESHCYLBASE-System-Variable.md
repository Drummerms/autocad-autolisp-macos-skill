# DIVMESHCYLBASE (System Variable)

Sets the number of radial subdivisions from the center of the mesh cylinder base to its perimeter.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 3 |

![](../images/GUID-9FDD2BF4-16C6-4E42-AD0E-88004A71C057-low.png)

This setting affects new mesh cylinders that are created with the [MESH](MESH-Command.md) command.
