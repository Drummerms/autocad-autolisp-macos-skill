# To Change the Color of Selected Objects

1. Select the objects whose color you want to change.
2. Right-click in the drawing area, and choose Properties.
3. In the Properties Inspector, click Color, and then click the down arrow. From the drop-down list, choose the color that you
   want to assign to the objects.
4. Press Esc to remove the selection.

The color that you choose overrides the default layer color for the selected objects.
