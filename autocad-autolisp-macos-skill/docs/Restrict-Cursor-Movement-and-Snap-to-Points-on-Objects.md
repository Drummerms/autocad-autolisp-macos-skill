# Restrict Cursor Movement and Snap to Points on Objects

#### Topics in this section

* [About Adjusting the Grid and Grid Snap](About-Adjusting-the-Grid-and-Grid-Snap.md)

  To enhance drawing speed and efficiency, you can display and snap to a rectangular grid. You can also control its spacing,
  angle, and alignment.
* [About Orthogonal Locking (Ortho Mode)](About-Orthogonal-Locking-Ortho-Mode.md)

  You can restrict cursor movement to horizontal and vertical directions for convenience and precision when creating and modifying
  objects.
* [About Polar Tracking and PolarSnap](About-Polar-Tracking-and-PolarSnap.md)

  Polar tracking restricts cursor movement to specified angles. PolarSnap restricts cursor movement to specified increments
  along a polar angle.
* [About Specifying Distances, Lengths, and Angles](About-Specifying-Distances,-Lengths,-and-Angles.md)

  When creating or moving objects, you can specify a point by moving the cursor to indicate a direction and then entering the
  distance.
* [About Using Object Snaps](About-Using-Object-Snaps.md)

  Object snaps provide a way to specify precise locations on objects whenever you are prompted for a point within a command.
* [About Using Object Snaps in 3D](About-Using-Object-Snaps-in-3D.md)

  Object snaps function the same way in 3D as they do in 2D with the exception that they can optionally be projected.
* [About Suppressing Running Object Snaps](About-Suppressing-Running-Object-Snaps.md)

  You can suppress running object snaps temporarily by using an override key.
