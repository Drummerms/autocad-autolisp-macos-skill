# REPORTERROR (System Variable)

Controls whether an error report can be sent to Autodesk if the program closes unexpectedly.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

Error reports help Autodesk diagnose problems with the software.

| 0 | The Error Report message is not displayed, and no report can be sent to Autodesk. |
| 1 | The Error Report message is displayed, and an error report can be sent to Autodesk. |

An error report can be sent through the operating system if REPORTERROR is set to 0 and the operating system supports error
reporting.
