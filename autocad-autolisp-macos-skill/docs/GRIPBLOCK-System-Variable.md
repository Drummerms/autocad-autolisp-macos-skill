# GRIPBLOCK (System Variable)

Controls the display of grips in blocks.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Assigns a grip only to the insertion point of the block |
| 1 | Assigns grips to objects within the block |
