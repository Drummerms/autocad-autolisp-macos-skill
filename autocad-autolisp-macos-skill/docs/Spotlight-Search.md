# Spotlight Search

The Spotlight search in macOS now supports searching for drawings based on drawing content. You can search on mtext, mleader,
field, or block attribute values. You can also search on named objects such as blocks, layers, or layouts, and system attributes
such as date or author.

Search for drawings using Finder or when opening a drawing in the product based on the following drawing content:

* Block name
* Layer name
* Layout name
* Text value contained in block attributes, text, mtext, mleaders, or fields
* Author

NOTE:The drawing must be saved using AutoCAD for Mac 2023 or later before it is searchable based on its contents.

## Using Search Attributes

In Finder or in the file selection dialog box in the product when opening a drawing:

1. Enter a value in the search box.

   By default the search attribute is Name.
2. To modify the search attributes shown on the drop-down menu, click the plus sign below the search box.

   ![](../images/GUID-FD527756-1FDB-44AD-9450-26D04B73B6CF-low.png)
3. Select Other on the attribute drop-down.

   ![](../images/GUID-3A0DE015-EDE4-4B2C-B5A9-C920BE503BF1-low.png)

   The list of available search attributes displays.
4. Select each attribute you want to include in the drop-down menu for easy selection.

   ![](../images/GUID-9021EB7B-182F-4F88-8601-7371A69FE0F0-low.png)
5. Click OK to dismiss the dialog box.
6. Select the search attribute you want to search on, for example Blocks, Layers, or Layouts.

   ![](../images/GUID-02B86E14-D887-44A9-A889-639F677640A3-low.png)

   The search results update to include drawings that contain the selected attribute's value. Only drawings saved in AutoCAD
   for Mac 2023 or later appear in the search results.
