# To Remove All Layer Property Overrides From All Layout Viewports

1. Display the layout with the layout viewports from which you want to remove the overrides.
2. Click
   Window menu ![](../images/ac.menuaro.gif) Layers.
3. In the Layer Properties Manager, right-click anywhere in the list of layers.
4. Click Remove Viewport Overrides For![](../images/ac.menuaro.gif) All Layers ![](../images/ac.menuaro.gif) In All Viewports.
