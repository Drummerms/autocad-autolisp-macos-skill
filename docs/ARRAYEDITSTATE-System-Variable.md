# ARRAYEDITSTATE (System Variable)

Indicates whether the drawing is in the array editing state, which is activated while editing an associative array’s source
objects.

(Read-only)

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 0 |

| 0 | Off |
| 1 | On |
