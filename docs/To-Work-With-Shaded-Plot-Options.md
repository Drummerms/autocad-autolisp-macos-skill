# To Work With Shaded Plot Options

You can plot a drawing that contains shaded 3D solids as it is displayed, in wireframe, with hidden lines removed, or as rendered.

## Set Shaded Viewport Options

1. Click the layout tab for which you want to set shaded viewport options.
2. Click Page Setup Manager.
   ![](../images/GUID-C6B05F20-1FAB-48DF-8CA1-621D77A17680-low.png)
3. In the Page Setup Manager, Page Setups area, select the page setup that you want to modify.
4. Click
   ![](../images/GUID-3811A988-85ED-4568-A59C-A09C6074AEC6-low.png) ![](../images/ac.menuaro.gif) Edit.
5. In the Page Setup dialog box, under 3D Viewports, select the required settings.
6. Click OK.
7. In the Page Setup Manager, click Close.

## Specify a Resolution Level for Shaded Plotting

You can set the resolution of shaded plots for either greater speed or higher fidelity.

1. Click Plot.
   ![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png)
2. In the Plot dialog box, under 3D Viewports, Quality drop-down list, select an option.

   If you want to enter a custom dpi, select Custom and specify a dpi in the DPI box.

## Modify the Shade Plot Setting of a Viewport

1. Make sure you are on a layout tab.
2. Double-click the border of the viewport you want to modify.
3. In the Properties Inspector, select Shade Plot, and then select an option for plotting.

## Modify the Shade Plot Setting in Model Space

1. Make sure you are on the Model tab and no objects are selected.
2. Click Plot.
   ![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png)
3. In the Plot dialog box, under 3D Viewports, Shading drop-down list, select an option.
