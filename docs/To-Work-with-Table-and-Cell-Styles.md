# To Work with Table and Cell Styles

## Create a Table Style

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/ac.menuaro.gif) Table Style ![](../images/GUID-EA6A1B89-9B6F-4552-AD8E-694113F12EA9-low.png), or enter TABLESTYLE at the command prompt.

   Existing table styles for the current drawing are listed in the Table Style Manager dialog box.

   ![](../images/GUID-B2E1CD3D-2358-4C3E-B847-2FEA8A29869D-low.png)
2. Click
   ![](../images/GUID-956AE331-FA56-435B-B092-CDB1D4FA6458-low.png).

   The Create New Table Style dialog box displays.

   ![](../images/GUID-F3A78278-05C3-41F5-9021-B1DDD12B5208-low.png)
3. Enter the style name.

   The New Table Style dialog box displays.

   ![](../images/GUID-508AF8D8-F349-4998-A426-82F276C52FEC-low.png)

   There are 3 default cell styles that make up a table style: Title, Header, and Data.

   ![](../images/GUID-8E38C430-2D54-40B1-BBAD-645CFB77F2CA-low.png)
4. Select the cell style from the drop-down list.
5. Select the properties for that cell style. Click the tabs General, Text, and Borders, to see all available properties for
   the selected cell style.
6. Click OK to save all changes to the table style.
7. Click Close to exit the Table Style Manager.

## Modify a Table Style

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/ac.menuaro.gif) Table Style ![](../images/GUID-EA6A1B89-9B6F-4552-AD8E-694113F12EA9-low.png), or enter TABLESTYLE at the command prompt.

   Existing table styles for the current drawing are listed in the Table Style Manager dialog box.
2. Select the table style you want to modify, right-click and select Modify. The Modify Table Style dialog box displays which
   has the same controls as the New Table Style dialog box.
3. Define the table style and cell style properties similar to when creating a table style.
4. Click OK to save all changes to the table style.
5. Click Close to exit the Table Style Manager.

## Rename a Table Style

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/ac.menuaro.gif) Table Style ![](../images/GUID-EA6A1B89-9B6F-4552-AD8E-694113F12EA9-low.png), or enter TABLESTYLE at the command prompt.

   Existing table styles for the current drawing are listed in the Table Style Manager dialog box.
2. Select the table style you want to rename.
3. Click
   ![](../images/GUID-4AC4C87F-0D68-4B11-AD8D-D11171DD08B0-low.png) and select Rename.
4. Enter the new name for the table style.
5. Click Close.

## Delete a Table Style

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/ac.menuaro.gif) Table Style ![](../images/GUID-EA6A1B89-9B6F-4552-AD8E-694113F12EA9-low.png), or enter TABLESTYLE at the command prompt.

   Existing table styles for the current drawing are listed in the Table Style Manager dialog box.
2. Select the table style you want to delete.
3. Click
   ![](../images/GUID-36652FED-6611-41B0-95B6-14B33F2D8DB1-low.png).
4. Confirm that you want to delete the table style.
5. Click Close.

NOTE:You can't delete the current table style or the Standard table style. To change the current table style, right-click a table
style and select Set Current.

## Create a Cell Style

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/ac.menuaro.gif) Table Style ![](../images/GUID-EA6A1B89-9B6F-4552-AD8E-694113F12EA9-low.png), or enter TABLESTYLE at the command prompt.
2. On the Table Style Manager dialog box, select New
   ![](../images/GUID-956AE331-FA56-435B-B092-CDB1D4FA6458-low.png) or select an existing table style and click Modify.
   ![](../images/GUID-4AC4C87F-0D68-4B11-AD8D-D11171DD08B0-low.png)
3. Click Create A New Cell Style.
   ![](../images/GUID-CFA7D967-6EE3-4245-9C5E-B948246380C9-low.png)
4. Enter the cell style name.
5. Select the cell style to start with and click Continue.
6. Set the cell style properties on the General, Text, and Borders tabs.
7. Click OK.

## Modify a Cell Style

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/ac.menuaro.gif) Table Style ![](../images/GUID-EA6A1B89-9B6F-4552-AD8E-694113F12EA9-low.png), or enter TABLESTYLE at the command prompt.
2. On the Table Style Manager dialog box, select New
   ![](../images/GUID-956AE331-FA56-435B-B092-CDB1D4FA6458-low.png) or select an existing table style and click Modify.
   ![](../images/GUID-4AC4C87F-0D68-4B11-AD8D-D11171DD08B0-low.png)
3. Select the cell style you want to modify from the Cell style drop-down.
4. Set the cell style properties on the General, Text, and Borders tabs.
5. Click OK.

## Delete a Cell Style

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/ac.menuaro.gif) Table Style ![](../images/GUID-EA6A1B89-9B6F-4552-AD8E-694113F12EA9-low.png), or enter TABLESTYLE at the command prompt.
2. On the Table Style Manager dialog box, select an existing table style and click Modify.
   ![](../images/GUID-4AC4C87F-0D68-4B11-AD8D-D11171DD08B0-low.png)
3. Click Manage Cell Styles.
   ![](../images/GUID-8C54AC47-A01F-4F42-AF9D-D9F3BF219B00-low.png)
4. Select the cell style you want to remove and click
   ![](../images/GUID-36652FED-6611-41B0-95B6-14B33F2D8DB1-low.png).
5. Click Yes to confirm.
6. Click Close.
