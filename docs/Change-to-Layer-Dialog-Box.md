# Change to Layer Dialog Box

Displays a list of layers to choose as the destination layer. You can also create a new destination layer.

LAYMCH (Command)

*Menu*:
Window ![](../images/ac.menuaro.gif) Layers ![](../images/ac.menuaro.gif) Layer Match.
![](../images/GUID-BD3247A5-4A0D-4151-B000-B802DC09511D-low.png)

## List of Options

The following options are displayed.

Current Layer
:   Displays the currently selected layer. Create a new layer name by entering a name.

Destination Layer list
:   Displays a list of layers in the drawing.
