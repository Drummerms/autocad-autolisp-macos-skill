# To Customize Shortcut Keys

Create or modify keyboard shortcuts for commands.

## Display the Customize (CUI) Editor

1. Click
   Tools menu ![](../images/ac.menuaro.gif) Customize ![](../images/ac.menuaro.gif) Interface (CUI).
2. Select the Shortcuts tab.

   ![](../images/GUID-6A44A6A8-8622-4641-A15E-A473BF2A6799-low.png)

## Create a Shortcut key

1. Click
   ![](../images/GUID-9174311E-7DC8-49C8-B39F-991EC3DAD0E2-low.png) to add a shortcut key.

   The visor displays.
2. Start typing the command name you want to assign the shortcut key to. Select the command from the drop-down list.

   ![](../images/GUID-0820C90D-6691-4789-B4E3-1C584E0C817A-low.png)
3. Click Record Shortcut.
4. Press the keys you want to assign to the command.

   NOTE:If the shortcut keys are already assigned to another command, an alert displays indicating which command uses the shortcuts
   keys.

   ![](../images/GUID-5C153FCB-843E-4667-BF7A-8282F49A18B6-low.png)
5. Select one of the following:
   * *Add*. Add the shortcut key and exit the visor
   * *Add Another*. Add the shortcut key and leave the visor open
   * *Cancel*. Don't add the shortcut key and exit the visor
6. Click Apply to apply the changes and continue making changes or click OK to apply the changes and exit the dialog.

NOTE:Click Reset to Default to remove all custom shortcut keys.

## Remove a Shortcut Key

1. Select the shortcut key assignment you want to remove.
2. Click
   ![](../images/GUID-A140F0E8-0D4A-41B4-8078-48722E2E2CDB-low.png).

## Edit a Shortcut Key

1. Select the shortcut key assignment you want to edit.
2. Right-click and select Edit.

   The visor displays.
3. Click the "x" next to the current shortcut key assigned.
4. Follow the steps described previously to add a shortcut key.
