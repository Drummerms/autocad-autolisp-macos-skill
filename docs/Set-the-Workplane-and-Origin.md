# Set the Workplane and Origin

#### Topics in this section

* [About the User Coordinate System (UCS)](About-the-User-Coordinate-System-UCS.md)

  The User Coordinate System (UCS) is a movable coordinate system, which is a fundamental tool both for 2D drawing and 3D modeling.
* [About Aligning the UCS Dynamically With Faces](About-Aligning-the-UCS-Dynamically-With-Faces.md)

  With the dynamic UCS turned on, you can temporarily and automatically align the XY plane of the UCS with a planar face on
  a 3D solid, a planar mesh element, or a planar point cloud segment while creating an object.
