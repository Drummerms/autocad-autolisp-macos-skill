# AUDITCTL (System Variable)

Controls whether AUDIT creates an audit report (ADT) file.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Prevents writing of ADT files |
| 1 | Writes ADT files |
