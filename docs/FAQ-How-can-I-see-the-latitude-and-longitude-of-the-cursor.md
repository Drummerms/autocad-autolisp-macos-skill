# FAQ: How can I see the latitude and longitude of the cursor?

When the COORDS system variable is set to 3, the status bar displays latitude and longitude.

1. Right-click the coordinate display located on the left end of the status bar ( ![](../images/GUID-44FB590E-5C3C-4CB5-BDCF-E48927891449-low.png) Find).
2. Select Geographic.

   NOTE:This option is available only if the drawing file contains geographical location data.

NOTE:If Coordinate Display is not displayed on the status bar, on the status bar click the Customization button and select Coordinates.
