# NAVVCUBESIZE (System Variable)

Specifies the size of the ViewCube tool.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 4 |

| 0 | Small |
| 1 | Normal |
| 2 | Large |
| 3 | Tiny |
| 4 | Automatic; the size of the ViewCube tool is scaled up or down based on the size of the active viewport, zoom factor of the active layout, or drawing window |
