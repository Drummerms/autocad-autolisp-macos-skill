# To Set the Lineweight Scale for a Layout

1. Click the layout tab for which you want to scale lineweights.
2. Click
   File ![](../images/ac.menuaro.gif) Page Setup Manager.
3. In the Page Setup Manager, Page Setups area, select the page setup that you want to modify.
4. Click Modify.
5. In the Page Setup dialog box, under Plot Scale, select Scale Lineweights.

   The lineweights in the current layout are scaled in proportion to the designated plot scale. When you are working in the Model
   tab, this option is not available.
6. Click OK.
7. In the Page Setup Manager, click Close.
