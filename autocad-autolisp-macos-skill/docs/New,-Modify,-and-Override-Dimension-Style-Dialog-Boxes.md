Set properties for dimension styles.

## Summary

When you choose Continue in the Create New Dimension Style dialog box, the New Dimension Style dialog box is displayed. You
define the properties for the new style in this dialog box. The dialog box initially displays the properties of the dimension
style that you selected to start the new style in the Create New Dimension Style dialog box.

Choosing either Modify or Override in the Dimension Style Manager displays the Modify Dimension Style or the Override Dimension
Style dialog box. The content of these dialog boxes is identical to the New Dimension Style dialog box, although you are modifying
or overriding an existing dimension style rather than creating a new one.

The sample image on each tab displays the effects of each option.
