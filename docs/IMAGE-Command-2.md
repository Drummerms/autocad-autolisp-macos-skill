# IMAGE (Command)

Displays the Reference Manager palette.

## Access Methods

*Menu*:
File ![](../images/ac.menuaro.gif) Attach.

*Menu*:
Insert ![](../images/ac.menuaro.gif) Reference Manager.

## Summary

The
[Reference Manager palette](Reference-Manager-Palette.md) is displayed.

If you enter
*-image* at the Command prompt,
[options are displayed](IMAGE-Command.md).
