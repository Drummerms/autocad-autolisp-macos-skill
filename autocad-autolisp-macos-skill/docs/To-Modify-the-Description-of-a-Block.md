# To Modify the Description of a Block

Use this procedure to add or modify the description for a block.

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block Editor.
   ![](../images/GUID-E3DDBE51-C92A-4D07-A51D-C7C685E4F578-low.png)
2. In the Edit Block Definition dialog box, select the block. Click Edit Block.

   The Block Editor opens up. Edit the drawing.
3. If the Properties Inspector isn't open, click
   Window menu ![](../images/ac.menuaro.gif) Properties Inspector.
4. In the Description box, enter or modify the description of the block.

   The name can have up to 255 characters and can include letters, numbers, blank spaces, and any special character not used
   by the operating system or the program for other purposes.
5. Click Save on the Block Editor visor to save the changes to the block.
6. Click X to close the Block Editor.
