# DIMSAH (System Variable)

Controls the display of dimension line arrowhead blocks.

|  |  |
| --- | --- |
| Type: | Switch |
| Saved in: | Drawing |
| Initial value: | Off |

| Off | Use arrowhead blocks set by  [DIMBLK](DIMBLK-System-Variable.md) |
| On | Use arrowhead blocks set by  [DIMBLK1](DIMBLK1-System-Variable.md) and  [DIMBLK2](DIMBLK2-System-Variable.md) |
