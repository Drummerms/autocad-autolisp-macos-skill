# HPISLANDDETECTION (System Variable)

Controls how islands within the hatch boundary are treated.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Drawing |
| Initial value: | 1 |

| 0 | Normal. Hatches islands within islands. |
| 1 | Outer. Hatches only areas outside of islands. |
| 2 | Ignore. Hatches everything within the boundaries. |

An island is an enclosed area within the area to be hatched.
