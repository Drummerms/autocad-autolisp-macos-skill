# To Add Custom Properties for Sheet Sets and Sheets

1. In the Sheet Set Manager, select a sheet set.
2. In the Details panel, expand Sheet Set Custom Properties or Sheet Custom Properties.
3. Click Add Property to create a new sheet set or sheet property.
   * Sheet set properties are typically specific to a project, for example, the contract number.
   * Sheet properties are typically specific to each sheet, for example, the name of the designer.
4. In the Add Custom Property dialog box, choose whether you want to add a new property for the sheet set or the sheet.
5. Enter the name and the default value.
6. Click Add Property.
