# COUNTLIST (Command)

Opens the Count palette to display and manage the counted blocks or other selected objects.

## Access Methods

*Menu*:
Window ![](../images/ac.menuaro.gif) Count.

The Count palette allows you to view the blocks on the current drawing and insert a table with the specified blocks and the
corresponding count into the drawing. You can also view the count of other selected objects.
