# CMDDIA (System Variable)

Controls the display of the In-Place Text Editor for the LEADER and QLEADER commands.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Off |
| 1 | On |
