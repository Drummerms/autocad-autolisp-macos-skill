# CMATERIAL (System Variable)

Sets the material of new objects.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | BYLAYER |

Valid values are BYLAYER, BYBLOCK, and the name of a material in the drawing.
