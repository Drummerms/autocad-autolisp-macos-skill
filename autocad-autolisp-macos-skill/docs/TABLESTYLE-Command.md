# TABLESTYLE (Command)

Creates, modifies, or specifies the current table style.

## Access Methods

*Tool Sets*:
Drafting tab ![](../images/ac.menuaro.gif) Table panel ![](../images/ac.menuaro.gif) Table Style.
![](../images/GUID-EA6A1B89-9B6F-4552-AD8E-694113F12EA9-low.png)

*Menu*:
Format ![](../images/ac.menuaro.gif) Table Style.

The Table Style Manager dialog box is displayed.

You can specify the current table style to determine the appearance of all new tables. You can also create and modify table
styles. A table style includes settings for background colors, margins, borders, text, and other table characteristics.
