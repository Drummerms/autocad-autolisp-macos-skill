# AREA (System Variable)

Stores the last area computed by the AREA command.

(Read-only)

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Not-saved |
| Initial value: | 0.0000 |

This system variable has the same name as a command.

Use the SETVAR command to access this system variable.
