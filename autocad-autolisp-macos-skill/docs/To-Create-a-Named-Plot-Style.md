# To Create a Named Plot Style

1. Click Format menu ![](../images/ac.menuaro.gif) Plot Style.
2. Select New from the Plot Style Table drop down list.
3. In the Plot Style Table editor, enter the required values.
4. Click Save as to save the new plot style table and to specify the location.
5. Click Save and close in Plot Style Table Editor to exit.
