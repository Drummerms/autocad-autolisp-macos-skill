# To Freeze or Thaw Layers in the Current Layout Viewport

1. Double-click within a layout viewport to make it current.
2. Click
   Window menu ![](../images/ac.menuaro.gif) Layers.
3. In the Layers palette, select the layers to freeze.
4. Click in the VP Freeze column to change the state of the selected layers.

NOTE:The VP Freeze column is not shown unless you are on a layout.
