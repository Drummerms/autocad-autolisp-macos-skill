# To Set the Current Annotation Scale for Annotative Objects

## Model Space

1. Enter QVDRAWING on the command line.
2. The Quick view model and layout thumbnails are displayed.
3. Click the Model thumbnail.
4. On the status bar, click the Annotation Scale drop-down list and select the scale to set current.

## Layout Viewport

1. Enter QVDRAWING on the command line.
2. The Quick view model and layout thumbnails are displayed.
3. Click the thumbnail of the layout to set current.
4. Select a viewport.
5. Right-click in the drawing area and choose Properties.
6. On the Properties window, select the Annotative Scale property and select the scale to set current.
