# PASTEORIG (Command)

Pastes objects from the Clipboard into the current drawing using the original coordinates.

## Access Methods

Button

![](../images/ac.mouse.gif) Menu:   Edit
 ![](../images/ac.menuaro.gif)  Paste to Original Coordinates Not available in menus in the current workspace

Shortcut menu: With no command active, right-click in the drawing area, and click Clipboard ![](../images/ac.menuaro.gif) Paste to Original Coordinates

## Summary

The objects copied to the Clipboard are pasted into the current drawing at the same coordinates used in the original drawing.
This command functions only when the Clipboard contains PRODNAME data from a drawing other than the current drawing.
