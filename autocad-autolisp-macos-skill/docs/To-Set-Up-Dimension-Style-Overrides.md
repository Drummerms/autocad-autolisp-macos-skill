# To Set Up Dimension Style Overrides

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Dimension panel ![](../images/ac.menuaro.gif) Dimension Style.
   ![](../images/GUID-89D5BC05-30EE-4627-B80B-8680F08A01A9-low.png)
2. In the Dimension Style Manager, under Styles, select the dimension style for which you want to create an override. Click Options
   ![](../images/GUID-0888F513-2A7D-45B1-AD99-683EBA803A86-low.png) ![](../images/ac.menuaro.gif) Override.
3. In the Override Current Style dialog box, make changes to the dimension style by clicking the appropriate tab.
4. Click OK to return to the Dimension Style Manager.

   The dimension style overrides are listed below the style they modify in the dimension style name list.
5. Click Close.
