# To Import an ACIS File

1. Click File ![](../images/ac.menuaro.gif) Import. At the Command prompt, enter IMPORT.
2. In the Import File dialog box, in File Format pane, select ACIS (\*.sat).
3. Find and select the ACIS file you want to import, or enter the name of the ACIS file at File Name.
4. Click Open.
