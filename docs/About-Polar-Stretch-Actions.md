# About Polar Stretch Actions

Polar stretch actions rotate, move, and stretch objects a specified angle and distance.

## Apply a Polar Stretch Action to a Polar Parameter

A polar stretch action can only be applied to a polar parameter. Like the stretch action, selected objects stretch when this
action is applied. However, a polar stretch action also allows you to stretch the objects at any angle.
