# XEDGES (Command)

Creates wireframe geometry from the edges of a 3D solid, surface, mesh, region, or subobject.

## Access Methods

*Tool Sets*:
Modeling tab ![](../images/ac.menuaro.gif) Solid panel ![](../images/ac.menuaro.gif) Solid Edit - Edge drop-down ![](../images/ac.menuaro.gif) Extract Edges.
![](../images/GUID-D38EE01F-6646-42AF-BE45-7ACE70090076-low.png)

*Menu*:
Modify ![](../images/ac.menuaro.gif) 3D Operations ![](../images/ac.menuaro.gif) Extract Edges.

## Summary

With the XEDGES command, you can create wireframe geometry by extracting all the edges from the following objects:

* 3D solids
* 3D solid history subobjects
* Meshes
* Regions
* Surfaces
* Subobjects (edges and faces)

![](../images/GUID-6C0F623F-926D-4C4F-8B01-C9708D3FCA5B-low.gif)

Press and hold Ctrl to select faces, edges and component objects, repeating if necessary. Objects such as lines, arcs, splines,
or 3D polylines are created along the edges of the selected objects or subobjects.
