# To Unload or Reload Images

1. Click
   Insert ![](../images/ac.menuaro.gif) Reference Manager.
2. In the Reference Manager palette, in the File References pane, right-click the image name, and then click Unload or Reload.

   The status of the selected image changes.

   All instances of the selected attached images are unloaded or reloaded.
