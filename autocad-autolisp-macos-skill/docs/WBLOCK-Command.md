# WBLOCK (Command)

Saves selected objects or converts a block to a specified drawing file.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Write Block.
![](../images/GUID-E84A38E9-381C-46EC-BE0C-F4E1728D05DA-low.png)

The Write Block dialog box is displayed.

Entering
*-wblock* at the Command prompt displays a standard file selection dialog box in which to specify a name for the new drawing file,
followed by command prompts. If FILEDIA is set to 0, the standard file selection dialog box is suppressed.
