# To Edit an Xref in a Separate Window

1. Click
   Window menu ![](../images/ac.menuaro.gif) Reference Manager
   .
2. In the External References palette, select the xref that you want to edit.
3. Right-click and choose Open.
4. The source file for the selected xrefs opens in a new window, where you can edit, save, and close the drawing.
