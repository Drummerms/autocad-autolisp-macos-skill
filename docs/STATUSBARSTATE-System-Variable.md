# STATUSBARSTATE (System Variable)

Indicates whether the layout tabs and status bar are visible or not.

(Read-only)

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Registry |
| Initial value: | ON |

| Value | Description |
| ON | Layout tabs and Status bar are visible. |
| OFF | Layout tabs and Status bar are not visible. |
