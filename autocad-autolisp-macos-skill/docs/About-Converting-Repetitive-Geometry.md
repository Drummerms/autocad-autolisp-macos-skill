# About Converting Repetitive Geometry

The BSEARCH command allows you to easily find and convert multiple instances of the specified geometry into blocks.

When you select geometry for conversion, AutoCAD finds and highlights all instances of the same geometry. You can then choose
to convert the selected object and the found instances into a block.

![](../images/GUID-39D544BD-446D-493B-A178-2AFD7D772941-low.png)

To exclude certain instances from the conversion, click them individually to remove them from the process.

![](../images/GUID-671DE19B-C591-411F-BBDF-C31C333A1E0A-low.png)

## Convert to an Existing Block

When converting the geometry or its instances into an existing block, you can pick a block from the current drawing by selecting
it directly. Alternatively, in the Convert dialog box, choose a block from list of blocks in the current drawing.

After selecting an existing block definition, you can adjust the scale and rotation to determine how the selected block definition
replaces the found instances.

![](../images/GUID-4133D1AA-B18B-437D-8CD5-83BC47A45850-low.png)

## Convert to a New Block

You can also convert the source object or the selected instances to a new block. In the Convert dialog box, define a new block
by specifying a block name and insertion point. By default, the insertion point is set to the center of the selected geometry.

![](../images/GUID-24800F6D-3387-4E32-B9AF-3BA1F673ED51-low.png)

Converting repetitive geometry into blocks streamlines the design process, minimizes redundancy, and offers flexibility in
organizing drawings.
