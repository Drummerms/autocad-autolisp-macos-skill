# UNISOLATEOBJECTS (Command)

Displays previously hidden objects.

## Access Methods

![](../images/ac.mouse.gif) Menu:   Tools  ![](../images/ac.menuaro.gif)  Isolate  ![](../images/ac.menuaro.gif)  End Object Isolation Not available in menus in the current workspace.

Shortcut menu: Right-click in the drawing area and click Isolate ![](../images/ac.menuaro.gif) End Object Isolation.

## Summary

Displays objects previously hidden with the [ISOLATEOBJECTS](ISOLATEOBJECTS-Command.md) or [HIDEOBJECTS](HIDEOBJECTS-Command.md) command.
