# FAQ: How do I get the best results from an online Help search?

![](../images/GUID-85FD13B1-4854-488E-B4FA-EABEEE2831AD-low.png)

* When you enter a search string, use several specific terms. For example, entering
  *add layer state*, will give you better results than entering
  *layer*.
* By default, search results include only content that includes all of the words you enter. If you want the search to find content
  that includes any of the search words, enter OR between each word. For example, enter "erase OR delete" to find any content
  that contains either of the words.
* As you type in the word, notice that the type-ahead feature provides several choices you can select.

  ![](../images/GUID-3A97DEB0-D3F6-4287-8796-C38E70C3B156-low.png)
* To narrow your results, choose from the available filters in the left pane. The filters can change based on the search results.

  ![](../images/GUID-C8CB005C-CC8D-4D5B-BFDC-AD8095A4A044-low.png)
