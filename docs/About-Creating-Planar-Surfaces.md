# About Creating Planar Surfaces

Create planar surfaces in the space between edge subobjects, splines and other 2D and 3D curves.

With PLANESURF, planar surfaces can be created from multiple closed objects and the edges of surface or solid objects. During
creation, you can specify the tangency and bulge magnitude.
