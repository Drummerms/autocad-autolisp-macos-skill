# LAYMCUR (Command)

Sets the current layer to that of a selected object.

## Access Methods

*Menu*:
Window ![](../images/ac.menuaro.gif) Properties Inspector ![](../images/ac.menuaro.gif) Make Current.
![](../images/GUID-F4E69652-F4B8-47E4-ACDC-203F2C9C9910-low.png)

*Menu*:
Format
![](../images/ac.menuaro.gif) Layer Tools
![](../images/ac.menuaro.gif) Make Current.

## Summary

You can change the current layer by selecting an object on that layer. This is a convenient alternative to specifying the
layer name in the Layers palette.
