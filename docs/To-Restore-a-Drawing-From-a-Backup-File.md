# To Restore a Drawing From a Backup File

Change the extension of a BAK file to DWG in File Explorer.

1. In File Explorer, locate the backup file identified by the .*bak* file extension.
2. Right-click a file and choose Rename.
3. Enter a new name using the .*dwg* file extension.
4. Open the file as you would any other drawing file.

## Display File Extensions in Finder

If file extensions are not displayed in Finder, you need to display them so you can edit the .*bak* file extension.

1. Switch to Finder.
2. Click
   Finder menu ![](../images/ac.menuaro.gif) Preferences.
3. In the Finder Preferences dialog box, Advanced tab, click Show All Filename Extensions.
4. Close the dialog box.
