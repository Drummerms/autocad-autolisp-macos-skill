# GFSHIFT (System Variable)

Specifies whether the pattern in a gradient fill is centered or is shifted up and to the left.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Not-saved |
| Initial value: | 0 |

| 0 | Centered |
| 1 | Shifted up and to the left |
