# BATTMAN (Command)

Manages the attributes for a selected block definition.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Block panel ![](../images/ac.menuaro.gif) Block attribute Manager.
![](../images/GUID-29956DF2-A40C-4FF8-943C-CA1DBA0F5B9D-low.png)

*Menu*:
Modify ![](../images/ac.menuaro.gif) Object ![](../images/ac.menuaro.gif) Attribute ![](../images/ac.menuaro.gif) Block Attribute Manager.

The Block Attribute Manager is displayed.

If the current drawing does not contain any blocks with attributes, a message is displayed.

This command controls all attribute properties and settings of a selected block definition. Any changes to the attributes
in a block definition are reflected in the block references.
