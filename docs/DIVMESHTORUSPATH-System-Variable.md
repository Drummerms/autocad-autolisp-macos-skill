# DIVMESHTORUSPATH (System Variable)

Sets the number of subdivisions in the path that is swept by the profile of a mesh torus.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | User-settings |
| Initial value: | 8 |

![](../images/GUID-0EDA2870-2A0A-4201-AD8F-7E6EA5DAFC25-low.png)

This setting affects new mesh tori that are created with the [MESH](MESH-Command.md) command.
