# To Align Objects Between Viewports Using a Construction Line

1. Click
   Drafting tab ![](../images/ac.menuaro.gif) Draw panel ![](../images/ac.menuaro.gif) Xline drop-down ![](../images/ac.menuaro.gif) Construction Line.
   ![](../images/GUID-85D09769-91B2-46C6-9756-261AC6D52C84-low.png)
2. Specify a point in the first viewport. Specify a second point to determine a line for the alignment.

   Choose a point that can be aligned with objects in the second viewport. Use object snaps for precision.
3. Click
   Drafting tab ![](../images/ac.menuaro.gif) Modify panel ![](../images/ac.menuaro.gif) Move.![](../images/GUID-DE17DBB8-D05E-46A6-A54D-BF9B6E37D8BC-low.png)
4. Select the viewport to align to the first viewport. Press Enter.
5. When prompted for a base point, specify a point in the second viewport. Choose a point that corresponds with the point selected
   in the first viewport.
6. When prompted for the second point, hold down Shift and right-click. Click Object Snap![](../images/ac.menuaro.gif)Perpendicular.
7. Click the construction line you created.

   The first and second viewports, and the objects in the viewports, are aligned.

   NOTE:When aligning objects in viewports, the scale of the viewports should be the same.
