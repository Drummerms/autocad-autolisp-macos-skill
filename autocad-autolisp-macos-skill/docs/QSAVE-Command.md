# QSAVE (Command)

Saves the current drawing using the specified default file format.

## Access Methods

*Toolbar*:
![](../images/GUID-53306AAB-BD76-49AA-B456-94F9D7027476-low.png)

*Menu*:
File ![](../images/ac.menuaro.gif) Save.

*Shortcut Menu*: right-click on the File tab for the drawing you want to save.

## Summary

If the drawing is named, the program saves the drawing and does not request a new file name. Use the
[SAVEAS](SAVEAS-Command.md) command if you need to save a drawing with a different name. If the drawing is read-only, use the SAVEAS command to save
the changed file under a different name.

If the drawing is unnamed, the Save Drawing As dialog box (see
[SAVEAS](SAVEAS-Command.md)) is displayed and the drawing is saved with the file name and format you specify.
