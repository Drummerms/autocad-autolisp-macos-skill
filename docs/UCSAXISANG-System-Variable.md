# UCSAXISANG (System Variable)

Stores the default angle when rotating the UCS around one of its axes using the X, Y, or Z option of the UCS command.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 90 |

Its value must be entered as an angle in degrees (valid values are: 5, 10, 15, 18, 22.5, 30, 45, 90, 180).
