# To Change the Linetype of Selected Objects

1. Select the objects.
2. Right-click in the drawing area, and choose Properties from the shortcut menu.
3. In the Properties Inspector, click Linetype, and then the down arrow. From the drop-down list, choose the linetype that you
   want to assign to the objects.
4. Press Esc to remove the selection.

NOTE:

* The linetype that you specify overrides the default layer linetype for the selected objects.
* If the linetype you want to use is not on the drop-down list, follow the steps in the task, To Load a Linetype.
