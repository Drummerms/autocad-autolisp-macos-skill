# VIEWPLOTDETAILS (Command)

Displays information about completed print and publish jobs.

## Summary

The
[Print Details dialog box](Print-Details-Dialog-Box.md) is displayed.

You can view detailed information about all completed print jobs, or just the errors that have occurred. You can also copy
the information displayed in the dialog box to the Clipboard.
