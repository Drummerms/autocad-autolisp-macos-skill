# DTEXTED (System Variable)

Specifies the user interface displayed for editing single-line text.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 2 |

Obsolete command. Replaced by the [TEXTED](TEXTED-System-Variable.md) system variable.
