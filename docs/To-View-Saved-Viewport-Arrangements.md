# To View Saved Viewport Arrangements

1. Click
   View menu ![](../images/ac.menuaro.gif) Viewports ![](../images/ac.menuaro.gif) Named Viewports.
2. In the Viewports dialog box, click the Named Viewports tab.

The list of saved viewports is displayed in the Viewports dialog box, Named Viewports tab.
