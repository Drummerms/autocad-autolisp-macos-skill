# About Big Font Descriptions

Some languages, such as Japanese, use text fonts with thousands of non-ASCII characters. In order for drawings to contain
such text, the product supports a special form of shape definition file called a Big Font file.

#### Topics in this section

* [About Defining a Big Font](About-Defining-a-Big-Font.md)

  Special codes in the first line of a Big Font file specify how to read two-byte hexadecimal codes.
* [About Defining an Extended Big Font File](About-Defining-an-Extended-Big-Font-File.md)

  To reduce the size of composite Kanji characters, you can define an extended Big Font file. Extended big fonts use the subshape
  code, followed immediately by a 0.
* [About Using Big Font Text in a Drawing](About-Using-Big-Font-Text-in-a-Drawing.md)

  A Big Font file can be used for drawing text by creating a text style with the STYLE command that specifies the name of the
  Big Font file.
* [About Using a Big Font to Extend a Font](About-Using-a-Big-Font-to-Extend-a-Font.md)

  To include special symbols in text strings, you can use a Big Font instead of extending a standard text font.
