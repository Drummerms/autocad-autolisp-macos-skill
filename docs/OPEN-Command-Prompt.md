# OPEN Command Prompt

Opens an existing drawing file.

When
[FILEDIA](FILEDIA-System-Variable.md) is set to 0 (zero), OPEN displays the following command prompt.

Enter name of drawing to open:

Enter
*~* (tilde) at the prompt to ignore FILEDIA and display the
[Select File dialog box](Standard-File-Selection-Dialog-Boxes.md), a standard file selection dialog box.
