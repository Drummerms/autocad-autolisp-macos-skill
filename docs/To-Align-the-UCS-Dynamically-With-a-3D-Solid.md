# To Align the UCS Dynamically With a 3D Solid

1. Start a command supported by dynamic UCS.
2. If necessary, click the Dynamic UCS button on the status bar (![](../images/GUID-3BDFADB2-6B00-4576-835A-9927035523D3-low.png)) or press F6 to turn it on.

   NOTE:Dynamic UCS is not displayed by default on the status bar. Click
   ![](../images/GUID-3811A988-85ED-4568-A59C-A09C6074AEC6-low.png) on the status bar and select Dynamic UCS to display it.
3. Move your cursor over a planar face on a 3D solid.
4. Complete the command.
