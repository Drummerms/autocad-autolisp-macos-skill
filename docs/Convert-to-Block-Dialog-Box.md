# Convert to Block Dialog Box

Converts selected entities and identical instances into blocks.

Stub topic for Convert dialog box in Mac.

BSEARCH (Command):
 *Select object to find matching instances to convert to blocks*

![](../images/GUID-5647943F-9CD1-447A-A43F-36FE5432A722-low.png)

## List of Options

The following options are displayed.

### Select

Provides option to select and review instances selected for block conversion. It also displays the number of total instances.

Include existing blocks
:   Specifies whether to convert blocks in the current drawing and geometry containing blocks or only the source object.

Include text variations
:   Specifies whether text variations will be searched for conversion into block attributes.

Define attribute tags
:   Creates attribute tags for text variations.

### Convert instances to

Provides option to choose whether to convert selected instances into an existing block or create a new block.

Existing block
:   Converts the selected instances into an existing block.

    NOTE:When converting text to attributes, the Existing Block tab is not available.

    * *Select a block from current drawing.* Displays a list of available blocks in the drawing.
    * *Select from drawing* (![](../images/GUID-8FE3247D-F38B-430C-897E-CC866E46BC92-low.png))*.* Temporarily closes the dialog box so that you can specify a block from the current drawing that you want to convert the selected
      instances into.

New block
:   Converts the selected instances into a new block entity.

    * *Name the new block*. Specifies a name for the newly created block.
    * *Basepoint.*Specifies the insertion point of the new block which determines its position within the drawing. By default, the insertion
      point is set to the center of the specified geometry.

## Assign block to layer

Provides option to select a layer to assign the block to.
