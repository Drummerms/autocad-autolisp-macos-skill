# To Find Non-Purgeable Blocks

Preview and locate the block references that you can't purge in the current drawing.

1. Do one of the following:
   * On the toolbar, click
     ![](../images/GUID-089301D9-6EA7-4E69-B931-839242981CFB-low.png).
   * At the Command prompt, enter PURGE.
2. In the Purge dialog box, click Show Details if necessary.
3. Click Non-Purgeable Items.
4. In the left pane, expand the tree view for Blocks.
5. Select a block definition in the tree view that you want to preview and locate in the current drawing.
6. Under Details in the right pane, click Select Objects.

   ![](../images/GUID-6CD45C16-1E0A-487F-9299-3F4EF8F44F8B-low.png)

   The dialog box closes, the specified block references are all selected, and the drawing automatically zooms to include them
   all.

NOTE:If a block reference is on a layer that is frozen or turned off, when you click on Select Objects, a task dialog box displays
that provides the option to thaw and turn on the layer of the non-purgeable block references.
