# To Extract Part of an Existing List

1. Double-click a multiline text object to open the In-Place Text Editor.
2. Select a sequence of list items or place the cursor at the beginning of the item where you want to start the new list.
3. Right-click in the editor. Click Bullets and Lists ![](../images/ac.menuaro.gif) Restart.

   The selected items are renumbered as a separate sequence. If you select items in the middle of a list, unselected items below
   the selected items also become part of the new list.
4. To continue the original list below the new list, select the first item below the new list.
5. Right-click in the editor. Click Bullets and Lists ![](../images/ac.menuaro.gif) Continue.

   The selected item and the items following it are renumbered to continue the previous list.
6. To save your changes, Click in the drawing outside the editor. Or, Press CMD+ENTER
