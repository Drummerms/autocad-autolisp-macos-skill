# ARRAYTYPE (System Variable)

Specifies the default array type.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 0 |

| 0 | Rectangular array |
| 1 | Path array |
| 2 | Polar array |
