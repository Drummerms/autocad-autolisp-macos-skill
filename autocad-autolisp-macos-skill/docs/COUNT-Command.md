# COUNT (Command)

At the command prompt, counts and highlights the instances of the selected object in the drawing.

## Access Methods

*Toolbar*:
![](../images/GUID-33A7C371-502D-4F78-B130-3C911A5598F8-low.png)

*Menu*:
Tools ![](../images/ac.menuaro.gif) Count.

When you are in an active count, the Count toolbar is displayed at the top of the drawing area, and all the instances of the
selected object or block are highlighted.

The following prompts are displayed.

## First Corner

Specifies the first corner of the rectangular count area.

Opposite Corner
:   Specifies the diagonally opposite corner of the rectangular count area.

## Current Area

Specifies the last area defined in the previous count session.

## Entire Model Space

Specifies the entire model space as the count area boundary.

## Object

Specifies an existing closed polyline as the count area boundary.

## Polygonal

Specifies a polygonal count area boundary.

Start Point
:   Specifies the starting point of the polygonal count area.

Next Point
:   Specifies the next point of the polygonal count area. Press Enter to close the polygonal area.
