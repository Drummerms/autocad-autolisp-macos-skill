# CELTSCALE (System Variable)

Sets the current object linetype scaling factor.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 1.0000 |

Sets the linetype scaling for new objects relative to the [LTSCALE](LTSCALE-Command.md) command setting. A line created with CELTSCALE = 2 in a drawing with LTSCALE set to 0.5 would appear the same as a line created
with CELTSCALE = 1 in a drawing with LTSCALE = 1.
