# MTEXTCOLUMN (System Variable)

Sets the default column setting for an mtext object.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 2 |

| Value | Description |
| 0 | No column |
| 1 | Dynamic columns with auto height |
| 2 | Dynamic column with manual height |
