# To Create an Ordinate Dimension

1. Click
   Modeling tab ![](../images/ac.menuaro.gif) UCS panel.
2. At the Specify New Origin Point prompt, specify an origin point.

   The origin point specified is used to define the value assigned to the ordinate dimension. Typically, the origin point is
   defined on the model.
3. Click
   Drafting tab ![](../images/ac.menuaro.gif) Dimension panel ![](../images/ac.menuaro.gif) Dimension drop-down ![](../images/ac.menuaro.gif) Ordinate.
   ![](../images/GUID-E5FF7802-872E-4FBE-91F9-26E0442108EB-low.png)
4. If straight ordinate leaders are required, turn Ortho mode on.
5. At the Select Feature Location prompt, specify a point location.
6. Enter
   *x* (X Datum) or
   *y* (Y Datum).

   You can skip this step by making sure that the ordinate leader endpoint is close to vertical for an
   *X* datum or close to horizontal for a
   *Y* datum.
7. Specify the ordinate leader endpoint.
