# Manage Raster Images

#### Topics in this section

* [About Assigning Descriptive Names to Raster Images](About-Assigning-Descriptive-Names-to-Raster-Images.md)

  When the name of a raster image file is not sufficient to identify an image, you can add a descriptive name using the External
  Reference palette.
* [About Viewing Raster Image Information](About-Viewing-Raster-Image-Information.md)

  You can view file-specific information about the raster images that are attached to a drawing. You can also load and unload
  the images and perform other operations using the Reference Manager palette.
* [About Changing the Path to Raster Images](About-Changing-the-Path-to-Raster-Images.md)

  With the Reference Manager palette, you can change the file path to a referenced raster image file or search for a referenced
  image when it is reported as not found.
* [About Loading and Unloading Raster Images](About-Loading-and-Unloading-Raster-Images.md)
* [About Improving the Display Speed of Raster Images](About-Improving-the-Display-Speed-of-Raster-Images.md)

  To increase the display speed of images, you can change image display quality, hide images not currently needed, use image
  tiling, or suppress image selection highlighting.
