# PLOT (Command)

Outputs a drawing to a printer or file.

## Access Methods

*Toolbar*:
![](../images/GUID-E98FB764-041C-4F90-8F08-3FD0683C3611-low.png)

*Menu*:
File ![](../images/ac.menuaro.gif) Print.

*File tab*:
![](../images/GUID-C2BA3961-A036-4F07-A055-D7A43119520E-low.png) Quick View. In the Quick View dialog box, right-click a layout and click Print.

## Summary

The Plot dialog box is displayed. Click OK to begin printing with the current settings.

TIP: To enable the Plot dialog box to remember the last used printer, do the following:

1. Click Mac menu ![](../images/ac.menuaro.gif) System Preferences menu.
2. In the System Preferences dialog box, click the Printers and Scanners icon.
3. In the Printers and Scanners dialog box, Choose Last Printer Used from the Default Printer drop down.

If you enter
*-plot* at the Command prompt, options are displayed.
