# MTEXT (Command)

Creates a multiline text object.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Text panel ![](../images/ac.menuaro.gif) Multiline Text.
![](../images/GUID-3CA3F6E0-15CB-47EC-990F-61AF97C78859-low.png)

*Menu*:
Draw ![](../images/ac.menuaro.gif) Text ![](../images/ac.menuaro.gif) Multiline Text.

## Summary

You can create several paragraphs of text as a single multiline text (mtext) object. With the built-in editor, you can format
the text appearance, columns, and boundaries.

After you specify the point for the opposite corner the
[Text Editor visor](Text-Editor-Visor.md) is displayed which contains text formatting options.

If you specify one of the other options, or if you enter *-mtext* at the Command prompt, MTEXT bypasses the In-Place Text editor and displays additional
[Command prompts](MTEXT-Command-2.md).
