# BOUNDARY (Command)

Creates a region or a polyline from an enclosed area.

## Access Methods

*Tool Set*:
Drafting tab ![](../images/ac.menuaro.gif) Hatch panel ![](../images/ac.menuaro.gif) Boundary.
![](../images/GUID-D54FC086-4D7A-4EAA-8243-41FF6509F5EC-low.png)

*Menu*:
Draw ![](../images/ac.menuaro.gif) Boundary.

The Boundary Creation dialog box is displayed.

If you enter *-boundary* at the Command prompt, options are displayed.

Each point that you specify identifies the surrounding objects and creates a separate region or polyline.

![](../images/GUID-980F7CDE-30B1-4B3D-BB8E-69C6D0C35A4A-low.gif)
