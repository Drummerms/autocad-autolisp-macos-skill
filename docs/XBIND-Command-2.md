# XBIND (Command)

Binds one or more definitions of named objects in an xref to the current drawing.

## Access Methods

*Menu*:
Modify
![](../images/ac.menuaro.gif) Object
![](../images/ac.menuaro.gif) External Reference
![](../images/ac.menuaro.gif) Bind.

## Summary

The
[Bind External Definitions dialog box](Bind-External-Definitions-Dialog-Box.md) is displayed.

If you enter
*-xbind* at the Command prompt,
[options are displayed](XBIND-Command.md).

NOTE:The Bind option of XREF binds the xref file. Use XBIND for individual dependent definitions.
