# REVCLOUDPROPERTIES (Command)

Controls the approximate chord length for the arcs in a selected revision cloud.

![](../images/GUID-FB370E92-2A90-44E9-A76C-72AF38A57123-low.png)

The following option is displayed.

Arc Length
:   Specifies an approximate
    *chord* length for each arc in the revision cloud, which is the distance between the endpoints of the arc. The default value is automatically
    determined by the size of the first revision cloud created in the drawing.

NOTE:You can also change the arc chord length of a selected revcloud object in the Properties Inspector palette. You can control
whether the chord lengths of the arcs vary or are uniform with the REVCLOUDARCVARIANCE system variable.
