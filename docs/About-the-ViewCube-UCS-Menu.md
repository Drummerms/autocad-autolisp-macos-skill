# About the ViewCube UCS Menu

Change the current UCS or define a new one.

The UCS menu, located below the ViewCube, displays the name of the current UCS in the model. From the menu, you can restore
one of the named UCSs saved with the model, switch to WCS, or define a new UCS. With the WCS item on the menu, you can switch
the coordinate system from the current UCS to WCS. With the new UCS, you can rotate the current UCS based on one, two, or
three points to define a new UCS. When you click New UCS, a new UCS is defined with the default name of Unnamed. To save the
new defined UCS with a name, use the Named option in the UCS command.

You can orient the ViewCube with the current UCS or WCS. By orienting the ViewCube to the current UCS, you know in which direction
you are modeling. Orienting the ViewCube with the WCS, you can navigate the model based on the North and Up directions of
the model. The settings for controlling the orientation of the ViewCube are in the ViewCube Settings.
