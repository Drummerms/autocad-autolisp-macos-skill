# ARRAYASSOCIATIVITY (System Variable)

Sets the default behavior of new arrays to be associative or non-associative.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

Associative arrays are treated as a single array object, which is helpful when you want to modify spacing, angles, or replace
or modify the source object.

If an array is non-associative, each copied item is treated as a separate object. Editing one item does not affect the other
items.

| 0 | Non-associative |
| 1 | Associative |
