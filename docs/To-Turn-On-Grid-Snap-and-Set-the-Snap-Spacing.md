# To Turn On Grid Snap and Set the Snap Spacing

1. Click
   Tools ![](../images/ac.menuaro.gif) Drafting Settings menu.
2. In the Drafting Settings dialog box, Snap and Grid tab, click Snap On.
3. Under Snap Type, make sure Grid Snap and Rectangular Snap are selected.
4. In the Snap
   *X* Spacing box, enter the horizontal snap spacing value in units.
5. To use the same vertical snap spacing, press Enter. Otherwise, enter a new distance in the Snap
   *Y* Spacing box.

NOTE:

* Grid snap can be turned on and off with several other controls, including the F9 key and the Snap Mode button on the status
  bar. ( ![](../images/GUID-EA84C3AD-9C15-4BD6-A0B7-A349A9B2BCCE-low.png) Find)
* You can suppress snap
  *temporarily* by holding down F9 while you create or modify an object.
* The grid can be turned on and off with several other controls, including the F7 key and the Grid Display button on the status
  bar ( ![](../images/GUID-3F4A035C-77C7-4203-8596-35D902ACA21E-low.png) Find).
