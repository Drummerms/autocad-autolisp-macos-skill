# HPASSOC (System Variable)

Controls whether hatches and fills are associative.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 1 |

| 0 | Hatches and fills are not associated with their defining boundary objects |
| 1 | Hatches and fills are associated with their defining boundary objects and are updated when the boundary objects change |
