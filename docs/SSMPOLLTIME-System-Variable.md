# SSMPOLLTIME (System Variable)

Controls the time interval between automatic refreshes of the status data in a sheet set.

|  |  |
| --- | --- |
| Type: | Integer |
| Saved in: | Registry |
| Initial value: | 60 |

The SSMPOLLTIME timer sets the time in seconds between automatic refreshes of the status data of sheets in a sheet set. Valid
values are 20-600. The
[SSMSHEETSTATUS](SSMSHEETSTATUS-System-Variable.md) system variable must be set to 2 for the timer to operate.
