# LOFTANG1 (System Variable)

Sets the draft angle through the first cross section in a loft operation.

|  |  |
| --- | --- |
| Type: | Real |
| Saved in: | Drawing |
| Initial value: | 90 |

The 0 direction is measured outward from the curve on the plane of the curve. The positive direction is measured toward the
next cross section. Valid values include 0 to less than 360.
