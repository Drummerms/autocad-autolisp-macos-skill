# To Copy Properties From One Hatch Object to Another

Applies the properties of a selected object to other objects.

1. Click Match Properties.
   ![](../images/GUID-F83BA1A0-36C4-43E4-B80B-8EF3E0976E54-low.png)
2. Select source object.
3. Click one or more destination hatch objects.
4. To match the hatch origin, use Inherit Properties in HATCH or HATCHEDIT. This is available only for hatch objects.

NOTE:By default, all applicable properties are automatically copied from the first object you selected to the other objects. If
you don't want a specific property or properties to be copied, use the Settings option to suppress the copying of that property.
You can choose the Settings option at any time after selecting the source object.
