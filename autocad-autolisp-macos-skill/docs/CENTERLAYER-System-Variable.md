# CENTERLAYER (System Variable)

Specifies a default layer for new center marks or centerlines.

|  |  |
| --- | --- |
| Type: | String |
| Saved in: | Drawing |
| Initial value: | "use current" |

The CENTERLAYER system variable applies only to center marks and centerlines created with the CENTERMARK and CENTERLINE commands.

Values other than the "." (use current) value override the current layer (CLAYER system variable).
